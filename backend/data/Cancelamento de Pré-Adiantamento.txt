Para cancelar o pedido de pré-adiantamento a fornecedor, acessar o menu 
Movimentação / Financeiro / Contas a Pagar / 
Manutenção de Pré-Adiantamento a Fornecedores
 - botão Cancela.




Clicar no botão Ok para cancelar o pré-adiantamento. 


O cancelamento do pedido de pré-adiantamento será permitido se: 




o pedido tiver com status 
pendente
 ou 
aprovado
.


o usuário deve ter permissão para cancelar. Para habilitar a permissão, deve acessar o menu {{Ferramentas / Usuário / painel
