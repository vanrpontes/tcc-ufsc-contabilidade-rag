Para poder fazer o Ajuste de Estoque Inicial é necessário dar permissão no perfil do usuário. Acesse a partir do Menu

Para ajuste de estoque vá em: Usuário → aba Perfil → aba Estoque
Regras:
Nos depósitos sem fechamento de estoque podem ser informadas as quantidades finais no dia e o custo médio. Será providenciado um lançamento de ajuste (entrada e saída) para chegar nesta quantidade e automaticamente um fechamento de estoque para a data. 
Opção útil para início do controle de estoque, ao invés de se fazer o primeiro fechamento de estoque, se utilize o ajuste de estoque inicial, para a partir dos próximos dias se começar a fazer fechamento.
Nos depósitos com fechamento p/ data só pode ser informado o custo médio, que será atualizado neste fechamento. 
Opção útil para início de controle do CMV, a empresa já faz o fechamento de estoque e a partir de então vai atualizar o custo médio inicial para contabilização do CMV.
Nos depósitos com fechamento posterior não pode ser alterada nenhuma informação, pois o fechamento posterior garante a integridade das quantidades e custo médio. Se necessário for, repita o processo informando como data final uma data posterior.
Logicamente a data final informada deve ser no máximo a data atual, pois fica incoerente fechar o estoque com data posterior.
Quando for fazer o primeiro ajuste desse depósito é preciso desmarcar a opção 
Trava quando existir fechamento de estoque posterior ao movimento.
