TITULO: Painel Compartimentos (Frotas)
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2918974011
--------------------------------------------------
Nesta aba será informada a compartimentação do frota homologada pelo Imetro indicado no documento MOP. Só disponível para frotas que NÃO sejam do tipo Cavalo, Dolly ou Passeio.
Para Incluir um novo compartimento clique no botão "Incluir"
Será apresentado a tela de indicação dos dados da compartimentação. Preencha os seguintes campo:
Compartimento: 
Informe o número do compartimento
Seta Máxima: 
capacidade máxima do compartimento
Altura: 
altura para frotas do tipo vagão
Para alterar um registro de compartimento já existe clique no botão "Alterar"
Será apresentado a tela para alteração, permitindo alterar a 
Seta Máxima, 
caso o frota esteja com o campo 
Multissetas
 marcado na aba Detalhes, os campos Inclui e  Exclui ficarão habilitados e permitirá edição
.
Para excluir um registro de compartimento já existe, clique no botão "Excluir"
Para cadastrar Multisstas, o frota precisa estar com o campo 
Multissetas
 marcado na aba Detalhes.
Compartimento: 
informe o número do compartimento.
Seta Máxima: 
capacidade máxima do compartimento.
Para incluir uma seta adicional clique no botão.
O campo ficará disponível para informar a capacidade no quadro Demais Setas.
Para excluir uma seta adicional, basta clicar no botão.
Exemplo: 
Um Frota com dois compartimentos, cada compartimento tem a capaciadade máxima de 20.000 litros, porém esse mesmo compartimento tem mais duas setas adiconais de 10.000 e  15.000 litros, possibilitando o carregamento nas setas desejadas.
