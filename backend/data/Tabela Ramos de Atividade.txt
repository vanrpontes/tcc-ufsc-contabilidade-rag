TITULO: Tabela Ramos de Atividade
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878898479
--------------------------------------------------
Nesta tabela são armazenados os Ramos de Atividades de clientes, utilizados no preenchimento dos relatórios relacionados ao DCP. 


A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Estoque / Ramo de Atividade
. 


 


Para acrescentar um novo Ramo de Atividade à tabela, acione o botão de inclusão de registro: 




O Sistema exibe a tela para preenchimento dos novos dados. 


 


Preencha os campos Descrição e Código DCP para o novo Ramo de Atividade. 

Acione o botão OK para atualizar os dados no Sistema. 


O Sistema retorna a tela de navegação. 

Para retornar ao Menu principal acione o botão Sair:
