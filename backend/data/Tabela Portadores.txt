TITULO: Tabela Portadores
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210603
--------------------------------------------------
Esta tabela armazena os locais onde estão depositadas as cobranças da Empresa, por exemplo: cartório, bancos, advogados, etc.
Para mudar o local da cobrança, basta ir na tela de Manutenção de Contas a Receber (
Movimentação / Financeiro / Contas a Receber / Manutenção de Lançamentos
), selecionar os lançamentos que deseja mudar o portador e apertar a tecla Portador, selecionando o portador desejado.
A tela que permite a manutenção da tabela de Portadores é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Financeiro / Portadores
:
Para adicionar um portador à tabela, acione o botão de inclusão de registro 
O Sistema exibe a tela abaixo.
Digite o código do novo portador e sua descrição.
Gera ocorrência de restrições financeiras
 para o cliente caso deseje que os títulos a receber com este portador apareçam na tela de restrições financeiras do cadastro de clientes. Esta opção pode ser útil, por exemplo, para saber rapidamente se determinado cliente já teve títulos em cartório, ou já foi para processo jurídico, etc.
Não permite baixar os títulos deste portador
 é útil para bloquear baixas de títulos no Contas a Receber, ou seja, fazer com que nos processos de baixa, não seja permitido baixar um título que já foi mandado para protesto. Portanto, com essa opção marcada indica que esse portador não irá permitir a baixa. No momento que o título for para protesto, deve-se mudar o portador para este portador que bloqueia baixa. Nos processos de baixa serão verificados o portador do título, permitindo ou não a baixa. Para conhecer os processos de baixas, vide 
.
Acione o botão 
OK
 para atualizar os dados no Sistema.
O Sistema retorna à tela de navegação, acione então o botão 
Sair
