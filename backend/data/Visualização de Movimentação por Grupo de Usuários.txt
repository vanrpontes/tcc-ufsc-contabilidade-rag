TITULO: Visualização de Movimentação por Grupo de Usuários

--------------------------------------------------
Na versão 8.11, foi desenvolvida uma forma de restringir os usuários a só visualizarem movimentação que foi incluída por determinados Grupos de Usuários.
Para habilitar esse perfil é necessário acessar o menu 
Ferramentas > Sistema > aba Dados Gerais
, e marcar a opção 
Restringe visualização de movimentação conforme Grupo de Usuários.

No menu 
Ferramentas > Grupos de Usuários > Cadastro de Grupos
 é possível cadastrar os grupos de cada usuários. 
Para incluir um novo grupo, basta clicar no botão Inclusão 
 
O Sispetro não aceita cadastro de grupos com a mesma Descrição.
Não é possível excluir um grupo que tenha usuários vinculados a ele.
 
Para vincular os usuário aos grupos, acesse o menu 
Ferramentas > Usuário > aba Grupos de Usuários
.

Marcando a opção Restringe visualização de movimentação conforme Grupo de Usuários, o usuário selecionado terá acesso somente às informações dos grupos em que ele está cadastrado.
No exemplo, o usuário terá acesso apenas às movimentações do GRUPO 1.
 
Para cadastrar os Perfis de cada Usuário, veja mais informações em 
 
Um usuário pode participar de mais de um grupo.
Caso a opção Restringe visualização de movimentação conforme Grupo de Usuários não esteja selecionada, o usuário não participará do controle de grupos, tendo acesso às informações de todos os Grupos mesmo que esteja cadastrado algum grupo no perfil do usuário. Da mesma forma, caso a opção Restringe visualização de movimentação conforme Grupo de Usuários esteja marcada e não exista nenhum grupo cadastrado, o usuário não conseguirá visualizar nenhuma movimentação.
