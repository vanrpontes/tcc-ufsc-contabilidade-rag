TITULO: Manutenção de PMPF / MVA e UF de Origem
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926206063
--------------------------------------------------
A divulgação do PMPF,MVA e UF Origem é única, porém é necessário definir os valores para cada empresa.
Desta forma, como em algumas situações, mesmo que as empresas sejam da mesma UF e não haveria variação inclusive na digitação dos dados de MVA (que varia para operações internas e interestaduais), implementamos a tela de Manutenção de PMPF/MVA e UF Origem, a fim de facilitar a digitação. As alterações são válidas para produtos marcados como Ativo.
A apresentação dos dados será o maior valor conforme as empresas indicadas, caso a base para apresentação for outra empresa, o usuário não precisa sair para se conectar nela.
Movimentação / Faturamento / Manutenção de PMPF/MVA e UF de Origem:
Alteração em lote
É possível selecionar mais de um produto para alteração dos valores de MVA, PMPF e UF de Origem
Filtros
Empresas:
 selecione uma ou mais empresas para ver os valores definidos
Produto:
 selecione o produto ou mais produtos para ver os valores definidos
Vigência atual:
 traz a data de vigência atual cadastrada
É possível limpar a seleção dos dados através do botão.
 
Para pesquisar os dados, de acordo com os filtros definidos, clique no botão.
Parametros para salvamento de dados 
Nova Data Vigência:
 permite que seja cadastrada nova data de vigência para os dados
UF a serem alteradas
Todas as UF indicada na tela: 
alterações serão aplicadas em todas as UF exibidas na tela, caso esteja filtrando mais de uma empresa e uma delas não possua informação para UF alterada.
Somente as UF já existentes para a empresa: 
 alterações serão aplicadas apenas para as UF já existentes para a empresa, caso esteja filtrando mais de uma empresa e uma delas não possua informação para UF alterada.
Abas
 PMPF
: mostra as informações de UF e base de cálculo
 MVA
: mostra as informações de UF, % MVA interno e % de MVA Interestadual
 UF de Origem: 
mostra as informações de UF, % Nac, %Imp e Total
Visualiza somente UF com valor:
 ao selecionar esta opção, o SisPetro apresentará somente as UFs que possuem valores definidos. Não é possível informar valores que sejam maiores que 100%. 
     
Botões
Inclui UF:
 inclui UFs para que os valores sejam cadastrados
Salvar
: salva alterações feitas nos dados
Cancela:
 cancela alterações feitas nos dados
As informações salvas também podem ser consultadas através da tela de 
Margem Subst. Tributária
, no cadastro de produtos.
