TITULO: Atualização dos dados do Patrimônio no Cadastro de Produtos
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2897281573
--------------------------------------------------
Tenha numa planilha os tipos de produtos que a empresa terá, contendo as seguintes informações:
Código do Produto: 
informe o código do produto que está cadastrado no SisPetro
Nome Usual:
 informe o nome usual do produto
Descrição Completa:
 informe a descrição completa do produto
Descrição para NF:
 informe a descrição do produto usada na nota fiscal
Ativo ?: 
informe 0 para Não e 1 para Sim
CC Custo Alienação
: informe o código reduzido da conta na contabilidade real. Conta de Despesa.
CC Receita Alienação
: informe o código reduzido da conta na contabilidade real. Conta de Receita.
CC Despesa Depreciação
: informe o código reduzido da conta na contabilidade real. Conta de Despesa.
CC Depreciação Acumulada
: informe o código reduzido da conta na contabilidade real. Geralmente é uma conta redutora do Ativo.
CC Imobilizado
: informe o código reduzido da conta na contabilidade real. Conta de Ativo.
Taxa Depreciação:
 informe a taxa anual de depreciação
Vida Útil:
 informe o número de meses para o bem depreciar. Se for 5 anos, coloque 60
CC ICMS a Recolher
: informe o código reduzido da conta na contabilidade real. Conta de Passivo.
CC ICMS a Recuperar
: informe o código reduzido da conta na contabilidade real. Conta de Ativo.
Número Meses CIAP
: informe  48 (a não ser que exista alguma regra que diferencie esse bem), caso tenha.
Sai no SPED PIS/COFINS:
 informe 0 para Não e 1 para Sim. Informar os campos abaixo somente se igual a 1.
CST COFINS Depreciação:
 informe o código CST para a COFINS conforme tabela utilizada pelo SPED Contribuições. Essa tabela está cadastrada no SisPetro.
CST_PIS_Deprec:
 informe o código CST para o PIS conforme tabela utilizada pelo SPED Contribuições. Essa tabela está cadastrada no SisPetro.
CST PIS Baixa:
 informe o código CST para o PIS Baixa conforme tabela utilizada pelo SPED Contribuições. Essa tabela está cadastrada no SisPetro.
CST COFINS Baixa:
 informe o código CST para a COFINS Baixa conforme tabela utilizada pelo SPED Contribuições. Essa tabela está cadastrada no SisPetro.
Identificação Bem:
 informe o código que identifica o bem conforme tabela utilizada pelo SPED Contribuições. Essa tabela está cadastrada no SisPetro.
Classificação:
 informe o código de classificação do bem conforme tabela de Classificação de Produtos do SisPetro.
Envie para o email 
suporte@futuranet.com.br
 para que possamos avaliar a possibilidade de fazermos essa atualização via script.
