TITULO: Inclusão de Observações no PMM
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926194801
--------------------------------------------------
 
A partir da versão 9.13 é possível gravar diversas observações no PMM. 
 
Configuração de Usuário
No perfil de usuário no 
Painel Compras (Usuários)
 tem que ser marcado se o usuário pode incluir e excluir Observações.
 
Incluindo Observações no PMM
O usuário pode realizar a inclusão de observações acessando a aba de Observações nas telas de 
Manutenção de Pedidos de Movimentação de Material:
Ou
 Consulta de Pedidos de Movimentação de Material Pendentes:
 
Ao acionar o Botão Inclui, será exibida a tela de Observações do PMM. Após inserir a observação, basta confirmar acionando o botão OK. 
O botão 
Inclui 
só ficará habilitado para usuários que estejam com o
 perfil de Usuário
 'Pode Incluir Observações'
 habilitado.
 
As observações que são adicionadas ao PMM serão apresentadas no Grid de 
Observações Gerais do PMM.
Excluindo uma Observação
Para excluir uma observação, basta ir na aba 
Observações
 na tela de 
Manutenção de Pedidos de Movimentação de Material , 
selecionar a observação do PMM e acionar o botão Exclui. 
O botão 
Exclui 
só ficará habilitado para usuários que estejam com o
 perfil de Usuário
 'Pode Excluir Observações'
 habilitado.
