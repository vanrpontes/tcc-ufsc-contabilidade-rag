A partir da versão 8.34 é possível criar mais de uma 
Tabela de Distância
, podendo criar fretes para o mesmo trecho com valores diferentes e vincular estas tabelas as transportadoras. 
Este recurso pode ser utilizado quando, para um mesmo trecho, o valor do frete for diferente para as transportadoras cadastradas. Veja abaixo o que é necessário para configurar
Tabela de Distância Padrão
As distâncias antes cadastradas no SisPetro serão todas migradas para esta tabela padrão, para que não haja nenhum impacto caso a empresa não trabalhe com mais de uma tabela. Esta tabela padrão será utilizada para o cálculo do frete nas seguintes telas:
Pedido de Movimentação de Material
Simulação do Cálculo do Frete
