Histórico Padrão para Ordens de Carregamento
Ao configurar este perfil, na tela de Inclusão de Ordem de Carregamento o histórico será sugerido automaticamente. Para cada Tipo de Ordem de Carregamento deve-se definir o histórico a ser utilizado.
Visualização de Históricos na Inclusão OC Sispetro App 
Com base nas opções informadas abaixo, a partir da versão 10.30 é possível definir se os históricos serão exibidos na tela de inclusão de Ordem de Carregamento via Sispetro App

Histórico Padrão para Contrato de Mútuo
Ao configurar este perfil, na tela de Contrato de Mútuo o histórico será sugerido automaticamente de acordo com tipo de contrato.
Histórico Padrão para Ordem de Fabricação
Ao configurar este perfil, na tela de Ordem de Fabricação o histórico será sugerido automaticamente, tanto para Entrada como para Saída.
Histórico Padrão para Adiantamentos
Ao configurar este perfil, na tela de Inclusão de Adiantamento de Clientes ou de Fornecedores o histórico será sugerido automaticamente.
