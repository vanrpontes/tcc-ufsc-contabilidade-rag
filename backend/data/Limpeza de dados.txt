Objetivo
A Limpeza de Dados reduz o espaço ocupado no banco, preservando apenas informações necessárias e mantendo integridade e auditoria. O processo é ativado pelo menu Ferramentas, onde o usuário define dia, hora e quantidade de dias a preservar. Cada tipo de dado possui um período padrão, que pode ser alterado conforme necessidade. Após configurado, o SisPetro executa a limpeza automaticamente, removendo dados antigos segundo os critérios definidos.

Permissão
Para ativar a limpeza é preciso: 
Dar permissão para o usuário ter acesso à tela, através do Menu 
Ferramentas/ Usuário
, 
habilitar a tela 
Limpeza de Dados
, conforme imagem abaixo.
Marcar a opção 
Ativar Limpeza de Dados
Informar o 
Dia do Mês
 que o SisPetro executará a limpeza de dados.
Informar a quantidade de 
Dias para Preservar,
 que são os dias anteriores a data da limpeza que 
não serão apagados
. Quando o valor informado for 
igual a zero, 
a limpeza será ignorada.
