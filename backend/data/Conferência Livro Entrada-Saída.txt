Filtra CFOP ? - se marcada:
Exatamente igual a:
 filtra o CFOP informado.
Exemplo:
 se informar no campo CFOP 11210, no relatório será exibido somente os registros cujo CFOP é 11210.
Inicia com:
 filtra os CFOPs que começam com o código informado.
Exemplo:
 se informar no campo CFOP 12, no relatório será exibido os registros cujos CFOPs comecem com 12: 12001, 12102, 12303, ...
Termina com:
 filtra os CFOPs que terminam com o código informado.
Exemplo:
 se informar no campo CFOP 03, no relatório será exibido os registros cujos CFOPs terminem com 03: 11003, 12203, 22303, ...
Contém o:
 filtra os CFOPs que contenham o código informado.
Exemplo:
 se informar no campo CFOP 22, no relatório será exibido os registros cujos CFOPs contenham 22: 11223, 12220, 20223, ...
Importante:
 
Os dados que serão impressos dependerão da
 
Data do Livro Fiscal a ser considerada na Apuração
. Ou seja, se a apuração foi feita por data de emissão serão exibidos os dados cuja data de emissão atenda o período informado em tela.
