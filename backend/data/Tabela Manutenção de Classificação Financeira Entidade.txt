TITULO: Tabela Manutenção de Classificação Financeira Entidade
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2936701760
--------------------------------------------------
Nesta tabela são cadastrados as classificações financeira de entidade, utilizados para diferenciação dos clientes no cadastro.
A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções -
 
Tabelas Auxiliares / Classificação Financeira Entidade
:
Para acrescentar uma nova classificação financeira de entidade, acione o botão de inclusão de registro.
O Sistema exibe a tela para preenchimento dos novos dados.
Será necessário preencher os seguintes campos:
Descrição:
 
texto que denomina a classificação financeira.
Ativo:
  Opção que define se a classificação ficara ativa ou inativa, só é possível utilizar classificações ativas em clientes.
Acione o botão
 
OK
 
para atualizar os dados no Sistema.
O Sistema retorna a tela de navegação.
Só é permitida a exclusão de  uma classificação financeira que não estão sendo utilizada em nenhuma entidade.
A classificação financeira será incluída nos clientes através da aba de 
padrão para vendas
.
