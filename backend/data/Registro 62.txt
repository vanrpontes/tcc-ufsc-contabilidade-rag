TITULO: Registro 62
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878899954
--------------------------------------------------
É o registro que informa o Estoque por Fornecedor para cada tipo de produto.


Os dados a serem preenchidos nesse registro são:


01 - Tipo de registro - 62

02 - Produto - código fixo conforme manual do SCANC

03 - Data do Estoque

04 - Quantidade do produto

05 - Tipo ICMS (1 Com retenção ou 0 Sem retenção)

06 - CNPJ - Fornecedor

07 - UF - Fornecedor

08 - IE - Fornecedor


O sistema busca as informações para o preenchimento desse registro a partir painel 
Apuração Média e Proporção de Fornecedores - Quadro 2
. Vide tópico 
 a parte que explica sobre o painel para maiores detalhes.


Para maiores informações de configurações que devem ser feitas antes da geração do arquivo, vide tópico
