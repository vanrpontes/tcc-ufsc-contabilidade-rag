TITULO: Tabela Padrão Contábil
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210538
--------------------------------------------------
Nesta tabela são cadastrados padrões contábeis que ao serem vinculados a uma entidade (no cadastro de 
Clientes
 ou 
Fornecedores
), já sugere as contas que serão contabilizadas na Inclusão do 
Contas a Receber
 ou do 
Contas a Pagar
.
A tela que permite manipular esta tabela deve ser acessada pelo Menu Tabelas Auxiliares / Controladoria / Padrão Contábil: 
Para acrescentar um novo Padrão Contábil acione o botão de inclusão de registro: 
 
O Sistema exibe a tela para preenchimento dos novos dados. 
O usuário deve informar a 
descrição,
  o 
tipo 
e o 
escopo:
 
Tipo:
Débito: 
se for utilizado para Fornecedores
Crédito:
 se for utilizado para Clientes.
Escopo:
Conta e Centro de Custo:
 quando selecionado, serão informadas as contas contábeis e os centros de custo.
Centro de Custo:
 quando selecionado, serão informados somente os centros de custos. No painel 
Contas
, os campos ficam indisponiveis. Somente os padrões contábeis com este escopo ficam disponíveis na 
Manutenção de Setores
.
Através do painel 
Contas
, deve ser informado as contas que serão utilizadas neste padrão e o percentual de cada uma delas, cujo escopo seja Conta e Centro de Custo. Deve-se atentar para que a soma dos percentuais seja sempre 100%. 
O usuário pode utilizar a tela de pesquisa para facilitar a localização da conta ou das contas que serão cadastradas. Para abrir a tela de pesquisa, deve-se posicionar o cursor no campo "Código Conta" e teclar <ENTER> ou clicar no botão (...). 
Dessa forma, o sistema abrirá a tela de pesquisa. 
Para informar um ou mais centros de custos, o usuário deve entrar no painel 
Centro de Custos
. 
Vale lembrar, que a soma dos percentuais deve ser sempre 100% e que o usuário pode utilizar a tela de pesquisa para localizar o centro de custo desejado.
Para gravar as informações deve ser utilizado o botão 
OK
. Após finalizar o processo de gravação o sistema retornará para a tela de navegação.
Para retornar ao Menu principal, o usuário deve clicar botão "Sair"
