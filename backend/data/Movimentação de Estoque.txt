A Consulta Movimentação de Estoque apresenta as movimentações de entrada e saída de produtos do estoque.
Esta consulta se encontra no Menu 
Consulta / Estoque / 
Movimentação de Estoque
:
Filial 
Unidade de negócios:
 informe todas as filiais ou uma filial ou um grupo de filiais
Filial:
 caso selecione uma filial ou um grupo de filiais em Unidade de Negócios
Dados Específicos
Depósito específico:
 para analisar os pedidos somente de um depósito. Se marcado, informe o código do depósito
Período
Data Inicial:
 informe a data inicial desejada
Data Final:
 informa a data final desejada
Filtros
Por Tipo de Produto
Somente Combustíveis:
 mostra os produtos cadastrados com tipo de produto igual a Combustível
Não Combustíveis:
 mostra os produtos cadastrados com tipo de produto diferente a Combustível
Todos
: mostra todos os tipos de produtos
Produto
Informe o código do produto
Layout de usuário
Permite salvar ou excluir layouts específicos por usuário:
 informe um nome para o layout e clique em Salvar. Caso queira excluir, selecione o layout e clique em Excluir
Sempre que alterar um layout específico (adicionando ou retirando campos). deve clicar em Salvar. 
Campos Disponíveis para uso
O layout é dividido em Colunas/Valores (Dados), Linhas e Colunas. Os campos podem ser adicionados/retirados conforme segue:
Colunas/Valores (Dados):
 campos que podem ser adicionados/retirados 
Qtde:
 indica a quantidade ambiente
Qtde 20c:
 indica a quantidade a 20 graus
Linhas e Colunas
Data: 
data de movimentação
Depósito: 
depósito da movimentação
Empresa:
 filial da movimentação
Histórico: 
histórico utilizado na movimentação 
Produto: 
produto movimentado
Razão Social/Histórico: 
mostra o cliente ou fornecedor referentes às notas que movimentaram o estoque ou Histórico caso sejam movimentações de ajustes
Tipo: 
entrada ou saída
