TITULO: Venda Município
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878211002
--------------------------------------------------
A partir do painel 
Detalhes
 é possível acionar o painel 
Bloqueio Movimentação
 para informar quais os municípios que pode ser vendido o produto, por exemplo, para os casos de produtos como o Diesel Metropolitano e o Óleo Diesel, que podem ser vendidos somente para determinados municípios.
Cabe lembrar que o padrão é não bloquear. Portanto, para estar disponível esse recurso é preciso marcar uma opção no menu 
Ferramentas / Usuário / painel Produtos
:
Será apresentada a tela a seguir:
Bloqueio de Venda p/ Município
a. Não bloqueia venda por município:
b. Permite venda somente p/ os municípios cadastrados abaixo:
c. Bloqueia venda p/ os municípios cadastrados abaixo:
Informe o código DNC do município desejado ou tecle <ENTER> para escolher o município pela descrição. Clique no botão 
"Inclui"
 para armazenar o município escolhido. Caso deseje retirar um município, basta selecionar o município e clique no botão 
"Exclui"
.
E deve-se, também, configurar o perfil de sistema para definição de qual município será considerado no bloqueio, se será o município do endereço padrão do cliente (que irá considerar a região e o município), do endereço de cobrança, do endereço de entrega ou do endereço de faturamento.
Para configurar esse perfil de sistema, é necessário acessar o menu 
Ferramentas / Sistema / Clientes
:
Cuja opções são as seguintes:
Cadastro do Cliente
: conforme o endereço cadastrado para o cliente na aba de 
Dados Gerais
;
Endereço de Entrega
: conforme endereço de entrega definido no cadastro de cliente, aba 
Padrão para Vendas (Clientes)
;
Endereço de Cobrança
: conforme endereço de entrega definido no cadastro de cliente, aba 
Padrão para Vendas (Clientes)
.
Endereço de Faturamento
: conforme endereço de entrega definido no cadastro de cliente, aba 
Padrão para Vendas (Clientes)
;
O bloqueio
Para maiores informações sobre o bloqueio de venda por município, veja o tópico 
Bloqueando ou permitindo um produto para venda por município
.
