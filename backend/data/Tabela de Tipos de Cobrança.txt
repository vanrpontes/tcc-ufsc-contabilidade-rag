TITULO: Tabela de Tipos de Cobrança
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878211198
--------------------------------------------------
Nesta tela, o usuário irá configurar todos os tipos de cobrança que a empresa usa para receber do cliente. Os tipos mais comuns são: Cheque, Ordem de Pagamento, DOC, TED, Adiantamentos, Depósitos, Ficha de Compensação, Duplicata, etc. 
Para mais informações ver, 
Cobrança Eletrônica
.
Cada tipo de cobrança deve ter seu layout de documento definido, pois no momento do faturamento, o sistema imprime a cobrança de acordo com o layout vinculado. Para se alterar o layout do tipo de cobrança ou amarrar outro tipo de cobrança a um determinado cliente, acesse o tópico 
Padrão para Vendas (Clientes)
.
Para manipular a Tabela com os Tipos de Cobrança permitidos na Empresa, o usuário deve entrar na tela de Manutenção de Tipos de Cobrança, através de Tabelas Auxiliares / Vendas / Tipos Cobrança:
Painel Navegação:
Todas as cobranças cadastradas no sistema são armazenadas e apresentadas por esse painel, inclusive as cobranças que estão inativas.
Para inserir um novo Tipo de Cobrança, deve-se clicar no botão de inclusão de registro.
O Sistema entrará em modo de edição, entrando no painel 
Detalhes
. 
Para mais informações ver,
 
Painel Detalhes - Tipo de Cobrança
As informações sobre Cobrança Eletrônica poderão ser encontradas através do tópico 
Painel Por Empresa - Tipo de Cobrança
.
Para sair da tela de Tipo de Cobrança, utilize o botão 
Sair
.
