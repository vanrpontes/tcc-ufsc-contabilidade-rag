TITULO: Roteiro Descarga na Base
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878898491
--------------------------------------------------
Os processos para fazer uma descarga na base podem ser feitos manualmente ou através de integrações com sistemas de terceiros.
Processos de Descarga:
Agendamento de descargas
: permite agendar horário para que um caminhão seja descarregado.
Descarga via Transferência entre Depósitos
: para distribuidoras com base própria
Descarga via Ordem Descarga Avulsa (descarga de terceiros)
Descarga via Transferência entre Empresas (descarga própria)
Chamada de Motoristas
: realiza a chamada de motoristas para entrada na base dentro de uma determinada ordem
Controle de Inspeção de Auto-Tanque (bola preta)
: permite realizar sorteio aleatório para inspeção dos caminhões
Processos com integrações com sistemas de terceiros:
Inclusão automática de OC através de API - SispetroApp
: faz a integração com os sistemas das empresas que armazenam na base, permitindo incluir automaticamente as OCs emitidas por essas empresas
Biometria dos motoristas (controle de acesso)
: permite controlar o acesso à base através de biometria da empresa Henry (utiliza o modelo Primme SF Acesso)
Balanças
: permite obter o peso
Balança Jundiaí
Balança Toledo
Balança Toledo Guardian 6
SmartCard
 (em desuso)
View
 (em desuso)
Integração com Sistemas de Arquivo:
 somente Zylix. Permite realizar as leituras, de forma automática, dos arquivos gerados por esse sistema
Leitura Automática de Tanque:
 desenvolvida para atender a leitura de tanque da empresa DPM. Como utiliza um layout de arquivo pré-definido, então qualquer sistema que gerar nesse layout, o SisPetro poderá realizar a leitura automática
