Abaixo detalhamos os perfis que podem ser definidos para o SispetroWeb/SispetroApp.
Para acessar esta tela, acesse o 
SispetroMon / SispetroWeb / Definição Perfil Cliente:
Tela inicial:
 a tela inicial que o cliente terá acesso no logon do SispetroWeb/SispetroApp.
Tela Cliente
: ao logar no SispetroWeb/SispetroApp, será apresentada a tela de dados de clientes
Tela Estoque:
 ao logar no SispetroWeb/SispetroApp, será apresentada a tela de consulta resumida de estoque
Perfil Cliente SispetroWeb
: é possível definir se o cliente a acessar o SispetroWeb/SispetroApp tem perfil de Congênere ou Perfil de Posto Revenda ou se Ambos.
Congênere
: apresentará os menus de Dados Clientes (notas fiscais), Tabelas (transportadoras e frotas), Ordem de Carregamento, Estoque e Base.
Posto Revenda
: apresentará os menus de Dados Clientes (dados clientes, notas fiscais e títulos) e Pedidos
Ambos
: visualizará todos os menus.

OC Visualização Colunas Temperatura/Densidade:
 permite que sejam exibidas ou não as colunas de temperatura e densidade dos produtos na OC.
OC Visualização Informação Bocas
: permite que seja exibido ou não as informações de bocas na OC.
OC Visualização Rodapé Assinaturas
: permite que sejam exibidos campos específicos para assinatura do usuário do drive, do operador e do motorista.
OC Visualização Produtos Primários
: permite que sejam exibidos o detalhamento da composição dos produtos (produtos primários com suas respectivas quantidades).
Todos os perfis de OC referem-se tanto a visualização quanto a impressão da OC.

Tipo Distribuição Cpts Auto Atendimento: 
permite configurar os passos para utilização do Auto Atendimento, onde temos:
         - 
Sem distribuir compartimentos antes lacres:
 selecionando essa opção, os lacres e envelopes são informados antes da distribuição dos compartimentos e o AutoAtendimento leva em consideração o número total de compartimentos dos frotas informados.
         - 
Distribui compartimentos antes de mostrar lacres:
 nessa opção, o AutoAtendimento distribui os pedidos escolhidos nos compartimentos e o usuário informa os lacres após esta distribuição. Serão mostrados os compartimentos e o usuário inclui os lacres para cada compartimento. 
A tela de envelopes segue a mesma regra. No final, aparece a tela de resumo onde ele pode alterar o que quiser. Quando o usuário não vai utilizar todos os compartimentos, este tipo de processamento é vantajoso pois ele terá de colocar somente os lacres e envelopes dos compartimentos realmente utilizados.
         - 
Distribui compartimentos e os mostra antes de mostrar lacres:
 na última opção, o AutoAtendimento mostra a tela de compartimentos (com a distribuição sugerida) antes de mostrar os lacres e envelopes. O usuário tem condição de, antes de informar os lacres e envelopes, alterar os compartimentos para, então, informar os lacres e envelopes. Ao final, como sempre, aparece a mesma tela de resumo onde ele pode alterar tudo. A vantagem nesta configuração é que o usuário pode alterar a compartimentação sugerida antes de informar os lacres e envelopes.
