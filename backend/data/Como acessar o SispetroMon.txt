Como iniciar o SispetroMon
O 
SispetroMon
 depende de duas configurações para ser executado:
Porta ServidorWeb:
 O padrão é a porta 
11000.
A partir daí, o usuário deve utilizar a interface Web digitando <endereço do SispetroMon>:11000
Nesta tela, podemos ver o log no próprio 
SispetroMon
. Caso ocorra algum erro inesperado, pode-se visualizá-lo no log. Como é possível que a própria ação do usuário (em outra tela) gere algum erro, para visualizá-lo, seria necessário abrir uma outra aba no Browser apontando para o 
SispetroMon
 para poder ler o log. O log é sempre gerado independente da sua visualização via web.
Para mais informações sobre a administração do SispetroMon, acesse 
SispetroMon
.
Se deseja monitorar o SispetroWeb, acesse 
Monitorando o SispetroWeb
.
Se deseja configurar o monitoramento do 
, clique em 
Monitorando o Servidor Sispetro
.
