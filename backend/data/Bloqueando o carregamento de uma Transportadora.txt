TITULO: Bloqueando o carregamento de uma Transportadora
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878440152
--------------------------------------------------
O Sispetro permite a desativação de uma Transportadora cadastrada, ou seja, após desabilitada, uma Transportadora essa não mais poderá ser selecionada para efetuar fretes. Para isso, acesse a tela de transportadoras, 
Tabelas Auxiliares / Transportadoras
. 




Na tela será exibida a lista com as Transportadoras cadastradas. Selecione a transportadora que deve ser desabilitada. Clique no painel 
Detalhes
, posicionado na parte superior da tela. 




Localize e marque o campo 
"Bloqueado"
. Se for o caso, para desbloquear, basta desmarcar o campo. 


Acione o botão 
Observações
 para digitar o motivo da desativação. 




Digite o motivo para constar como histórico da Transportadora. 

Clique no botão OK para salvar as alterações.
