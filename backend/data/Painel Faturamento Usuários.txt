o
 usuário só poderá excluir todas as notas geradas e/ou todos os itens das notas. Caso um XML importado gere 3 notas ou 1 nota com 3 itens, o usuário deverá selecionar as três notas/os 3 itens de uma nota e clicar para excluir.

o 
usuário, mesmo que o XML gerar duas notas/itens das notas, poderá excluir uma nota/um item e deixar a outra nota/outro item caso queira.
Um XML pode ter vários itens com CFOP diferentes. Se o De Para estiver apontado para Naturezas diferentes, no SisPetro, os itens desse XML irá gerar várias notas.
