Através do menu
 
Relatórios / Auxiliares / Contábil / Contabilização de Adiantamentos
 
é possível emitir a relação de todos adiantamentos num período com a indicação se os mesmos foram contabilizados e no caso positivo quais as contas movimentadas.


Informe a filial, o período e o tipo de Adiantamento (Cliente, Fornecedor ou todos) para que sejam filtrados os dados.


O layout do relatório será gerado conforme o arquivo
 
ContabilizacaoAdto.RTM
 
que deve estar localizado no mesmo diretório que o executável. Sua existência é
 
imprescindível.
 
Para alterar o layout utilize o botão 




Clique no botão OK para confirmar e o relatório será apresentado em tela. Para imprimí-lo clique no botão
