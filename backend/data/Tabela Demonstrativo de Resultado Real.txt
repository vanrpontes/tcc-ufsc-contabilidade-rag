A tabela de Demonstrativo de Resultado é utilizada para criar uma estrutura para o Demonstrativo de Resultado e será utilizada no 
Plano de Contas
 para as contas do tipo Receita e Despesa. A manutenção desta tabela é acionada a partir do Menu principal com as opções -
 
Arquivos / Demonstrativo de Resultado:
Campos:
Sequencial:
 automático, definido pelo SisPetro.
Código Ordenação:
 informado e definido pelo usuário. Define a ordenação da estrutura na emissão do DRE.
Tipo:
 define o tipo de estrutura do demonstrativo. São 4 tipos:
Analítico:
 conta contábil que contém os lançamentos.
Totalizador Analítico:
 
totaliza
 as contas contábeis analíticas 
abaixo
 
desta estrutura
 até o próximo Totalizador Analítico ou até o próximo Totalizador Geral ou até o próximo Acumulador.
Totalizador Geral:
 
totaliza
 as contas contábeis analíticas 
abaixo desta estrutura
 até o próximo Totalizador Geral ou até o próximo Acumulador.
Acumulador:
 
totaliza
 as contas contábeis analíticas 
acima desta estrutura
.
Ativo:
 ativa/inativa uma estrutura.
Grupo Empresa:
 caso a estrutura seja usada especificamente para um Grupo Empresa, informe o código. Caso a 
estrutura seja de uso comum
 para todos os Grupos Empresa, não informe o código. Caso tenho informado e não devia, clique no botão 
.

Inclusão de Contas em uma estrutura analítica:
 só é permitido incluir contas em uma estrutura analítica.
Selecione uma estrutura analítica e clique na aba 
Contas

Clique no botão 
Seleciona Contas:
 na tela que abre serão mostradas as contas contábeis de receitas e despesas analíticas do 
Plano de Contas Real
. Selecione as contas que deseja vincular à estrutura.
Para que o Demonstrativo seja emitido corretamente - valores e ordenação - é essencial que o SeqDemo esteja RIGOROSAMENTE na ordem correta.
Com as alterações ocorridas na estrutura do DRE e na emissão do relatório, pode ser necessário que as empresas tenham que ajustar suas estruturas para o padrão usado pela Futura. Sendo necessário, solicitar exemplo para o suporte da Futura. 
O padrão usado pela Futura é o padrão determinado pelo Conselho Federal de Contabilidade.
Selecionadas as contas, clique no botão 
OK
 para confirmar/atualizar os dados.
