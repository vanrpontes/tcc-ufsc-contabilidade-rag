Para utilizar a 
aprovação por alçada
 é necessário fazer as seguintes configurações:


Criar 
perfis de aprovação


Neste item é definido que cargos/perfis podem aprovar de forma geral, sem associar especificamente um usuário.

Aqui deve ser indicado se este cargo terá um limite de aprovação. Quanto será esse limite, depois informaremos no usuário.


Configurar cada 
usuário que fará a aprovação


Aponte para o usuário que tipo de perfil de aprovação que ele terá e quanto será seu limite. Atente-se que isso pode ser feito para PMM e para o Contas a Pagar, configure a área do seu interesse.


Configurar no 
sistema
 como será a aprovação da área desejada


Aqui serão informados quais os perfis (criados num tópico anterior) serão utilizados para aprovaçao e o tipo de aprovação.


A partir daí o usuário responsável pela aprovação deverá acessar regularmente a tela de aprovação para realizar este procedimento. Nenhuma baixa do Contas a Pagar poderá ser feita se o título não estiver aprovado.
