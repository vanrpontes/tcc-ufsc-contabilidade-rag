Caros Clientes,


A Futura Tecnologia em todos esses anos vem buscando se atualizar em relação à Tecnologia, Obrigações Fiscais e Novas Funcionalidades no Sispetro para facilitar o dia-a-dia dos usuários. 


Agora estamos dando mais um grande passo: a atualização do banco de dados Sybase SQLAnywhere versão 7 para a versão 12. 


A versão 12 do SQLAnywhere permitirá:




Utilizar mais memória para cache já que roda em 64bits nativamente.


Utilização de recursos mais avançados no que se refere a banco de dados, igualando-se aos demais bancos de mercado como Oracle e SQL Server


Otimização na gravação e consulta aos dados, garantindo uma melhor performance


Monitoramento das conexões, agilizando a identificação de eventuais travamentos e sua resolução


Administração efetiva do banco de dados através de ferramentas para log, tunning, profiling, etc




Para utilizarmos todos os recursos citados é necessário migrar seu banco da versão 7 para a versão 12, pois iremos utilizar cada vez mais estes novos recursos e alguns deles são incompatíveis com a versão atual do banco. 


Sendo assim, estamos abrindo uma janela de atualização de quatro meses a partir de hoje (dia 01/12/2011) dentro da qual 
cada empresa deverá providenciar a migração para a versão 12 do banco de dados
. 

Durante este período, serão lançadas versões do Sispetro compatíveis com ambas as versões do banco de dados (7 e 12).


A primeira versão a partir de 01/04/2012 será 
exclusiva para utilização na versão 12
 do SQLAnywhere sem garantia de compatibilidade ou funcionamento na versão 7
. 


O processo de migração não é complicado, mas exige um conhecimento mínimo em banco de dados. Estamos disponibilizando um roteiro de migração caso a equipe de TI da sua empresa deseje fazer o processo sozinha. 


Se preferir que esse trabalho seja feito por um Analista da Futura agende com nosso Suporte Técnico uma visita. Mas lembre-se: temos uma agenda concorrida e quanto mais próximo da data final menor a possibilidade de encontrar uma data disponível.
