TITULO: Inclusão a partir de NF
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210354
--------------------------------------------------
Para incluir um bem a partir de uma determinada Nota Fiscal, utilize o botão 
Inclui pela NF
 da tela de Manutenção de Bens. Será apresentada a tela a seguir:
Informe o 
número da Nota Fiscal
 desejada (deve ser uma nota de entrada que representa a compra do(s) bem(ns)).
Será apresentada a 
entidade
 (fornecedor), 
data
 e 
valor
 da Nota Fiscal. Na parte inferior serão apresentados os produtos desta nota, quantidade e preço.
Confirme a inclusão dos bens clicando no botão 
OK
, o sistema incluirá um número de bens de acordo com a quantidade adquirida de cada produto, usando como dados para o bem os dados do produto. Ex.: Se for 3 cadeiras o sistema incluirá três bens com a descrição Cadeira.
Caso o valor do bem seja zerado na entrada da Nota Fiscal, o Bem não será gerado e será apresentada mensagem ao usuário.
Para habilitar o botão 
EDITAR
, é necessário que a natureza de operação utilizada para incluir a nota fiscal deve estar marcada a opção 
Permite alterar valor do bem na inclusão via NF
. Para mais informações veja 
Dados Gerais (Painel Geral)
.
Permite alterar somente o valor do bem.
Os dados do bem criado poderão ser alterados pelo botão 
Altera
 na tela de Manutenção de bens, quando selecionado o bem desejado.
Uma nota fiscal só poderá ser incluída uma única vez, a não ser que todos os bens incluídos por esta Nota Fiscal sejam excluídos, aí será possível providenciar uma nova inclusão.
