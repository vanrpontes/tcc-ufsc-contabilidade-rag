As inclusões e exclusões de cadastro de gerenciadoras serão feitas pela Futura. 
Esta tabela armazena o cadastro da gerenciadoras de averbação  
O acesso a essa tabela é através do Menu
 
Tabelas Auxiliares / Transportes / Gerenciadora de Averbação
Aba Detalhes
Quadro Dados Gerenciadora
ID: 
número identificador da Gerenciadora, gerado automaticamente pelo SisPetro
Descrição: 
Descrição da Gerenciadora de Averbação (já preenchido pela Futura)
Ativo: 
quando marcado, a gerenciadora está ativa para uso.
Ambiente: 
Homologação: 
ambiente de testes
Produção: 
ambiente de trabalho real
Quadro URL
Homologação: 
aqui serão informadas as URL de teste
URL CT-e
URL NF-e
URL MDF-e
URL Token
Produção: 
aqui serão informadas as URL de produção
URL CT-e
URL NF-e
URL MDF-e
URL Token
Aba Identificação
Aba Dados: 
nessa aba constam os dados de acesso do usuário para API
Nome Usuário: 
o nome do usuário fornecido pela gerenciadora
Senha: 
senha do usuário fornecido pela gerenciadora
Código ID: 
o código ID fornecido pela gerenciadora
Ativo: 
se o usuário está ativo ou não. Quando está marcado o usuário está ativo.
Aba Empresas
Gerenciadora de Averbação - Identificação: 
o SisPetro mostrará os usuários cadastrados anteriormente para serem selecionados
Empresa: 
o SisPetro mostrará as empresas cadastradas para ser selecionada. 
Aba Comunicação  
A aba 
Comunicação
 contém o histórico dos documentos enviados e recebidos da Gerenciadora.
Por padrão o SisPetro manterá os últimos 90 dias de consulta.
