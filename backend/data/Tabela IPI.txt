TITULO: Tabela IPI
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210563
--------------------------------------------------
Tabela controladora das alíquotas de IPI aplicadas. É constituída do código do IPI mais a alíquota aplicada.
 Esta associada à tabela de Produtos via código de IPI. Cada produto tem associado o código de IPI.
Para se cadastrar uma nova alíquota de IPI, a partir do menu principal acione as opções - 
Tabelas Auxiliares / Faturamento / Tabela IPI
.
 
Acione o botão de inserção de novo registro.
 
O Sistema exibe a tela abaixo.
 
Preencha os campos:
Código
 da nova alíquota
% de IPI
 - neste campo deverá ser informada a alíquota, que será buscada para os produtos que usarem esse IPI, permitindo valores com casas decimais.
Acione o botão OK para atualizar no Sistema os novos dados.
 Para retornar ao Menu principal acione o botão Sair
