TITULO: Painel Carga (Usuários)
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926189817
--------------------------------------------------
Nesta tela é possível definir quais empresas o usuário poderá realizar o fechamento de carga e a manutenção dos pedágios. Além disso há outros perfis relacionados a carga e pedágio.
O acesso é feito através do Menu 
Ferramentas / Usuário / Aba Perfil / Aba Inferior Carga
Quadro Carga
Pode fazer composição de carga: 
define se o usuário vai poder realizar a composição da carga.
Pode fechar carga: 
define se o usuário vai poder fechar carga.
Pode Cancelar carga: 
permite que o usuário possa realizar o cancelamento da carga.
Pode Encerrar/Cancelar Encerramento de Cargas?: 
permite que o usuário possa realizar o encerramento manual de Cargas sem Integração com o Monitoramento ou cancelamento do agendamento da carga.
Pode excluir itens da carga incluídos por outros usuários: 
define se o usuário pode excluir cargas inclusas por outros usuários.
Pode alterar carga se Requer Monitoramento: 
define se o usuário vai poder Requer Monitoramento neste caso permite alterar a carga.
Pode incluir/Cancelar Monitoramento: 
permite que o usuário possa incluir e cancelar o Monitoramento da carga.
Autoriza/Desautoriza carga (valor do frete): 
permite autorizar desautorizar o valor do frete da carga.
Visualização das Cargas:
 qual será o filtro para visualização de Pedidos e OC permitidos para inclusão nas cargas
Somente da filial conectada: 
usuário só vai ver as cargas na filial ativa
Das Empresas para Carga (padrão): 
usuário irá visualizar as cargas das empresas definidas no quadro 
Filiais para Carga
. 
Das empresas que tem acesso: 
usuário irá visualizar as cargas das empresas definidas na aba 
Empresas Acesso
Todas: 
irá visualizar a carga de todas as empresas.

Pode Renovar Token Pedágio:
 define se o usuário vai pode renovar o Token.

Pode Alterar na Carga se Exige Confirmação:
 caso o usuário esteja com esse perfil marcado, é possível alterar na emissão da carga se exige ou não a confirmação. Esse perfil age em conjunto com o perfil de Sistema 
Utilização de Vale Pedágio
. Para mais informações, ver

Pode Alterar na Carga ID do Gerenciador de Pedágio:
 este perfil define se o usuário pode ou não alterar na carga o gerenciador de pedágio. Esse perfil age em conjunto com o perfil de Sistema 
Utilização de Vale Pedágio
. Para mais informações, ver  
.
Processamento de Vale Pedágio: 
esse perfil define se o usuário pode ou não realizar o processamento de um vale pedágio, e em que situação o mesmo pode processar. Esse perfil age em conjunto com o perfil de Sistema 
Utilização de Vale Pedágio
. Para mais informações, ver  
.
Não pode processar compra pedágio: 
utilizando essa opção o usuário não conseguira realizar o processamento do vale pedágio.
Pode processar somente com confirmação posterior: 
ao utilizar essa opção o usuário só terá permissão para processar o vale pedágio, deixando o mesmo pendente de confirmação. Esse perfil é exclusivo para quem trabalha com o gerenciador de pedágio da Target.
Pode processar Compra sem confirmação e Confirmação: 
ao utilizar esse perfil o usuário pode processar pedidos de vale pedágio com e sem confirmação. A utilização desse perfil pode ser tanto para o gerenciador de pedágios da Target quanto para a DBTrans.
Quadro Cancelamento de Vale Pedágio
Pode cancelar Pedidos Não Processados ou Sem Valor: 
define se o usuário poderá realizar cancelamento de Pedidos de Pedágio não processados e com valor de pedágio zerado.  
Pode cancelar Pedidos Processados com Valor: 
define se o usuário poderá realizar cancelamento de Pedidos de Pedágio processados e com Pedágio que possua valor.
