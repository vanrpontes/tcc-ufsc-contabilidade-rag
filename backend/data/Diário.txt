Para emitir o Diário, o usuário deve acessar o menu
 
Relatórios / Diário.
O layout do diário será gerado conforme o arquivo "diario.rtm" que estiver no mesmo local do executável, sendo que a sua existência é imprescindível. Para alterar o layout utilize o botão 
Para incluir informações sobre o Responsável Legal pela empresa, acesse 
Dados do Responsável Legal pela empresa nos Relatórios Contábeis
.
 
O usuário deve ter uma atenção especial para as atualizações que são disponibilizadas para o sistema. Essas atualizações podem conter implementações para o arquivo "diario.rtm". Para que as alterações feitas sejam consideradas, é necessário sobrepor o .rtm existente pelo mais atual.
 Algumas informações, como informações adicionais, estão disponíveis no layout do diário, porém não estão sendo utilizadas. Caso a empresa deseje que sejam impressas essas informações, basta alterar o seu layout para contê-las.
Para informar uma determinada filial, o usuário deve-se marcar a opção
 
"Filial específica"
 
e informar a filial, preenchendo o campo "Filial"
.
Caso queira imprimir o diário de todas as filiais, basta o usuário desmarcar o campo
 
"Filial específica".
 
Imprimindo o diário de todas as filiais, é possível a cada dia apresentado, apresentar um cabeçalho que agrupa os dados de cada filial, para isso deve ser marcada a opção
 
"Quebra por filial"
. Não sendo marcada esta opção, o código da filial será apresentado em uma coluna do relatório.
Deve-se informar um intervalo de data, no campo
 
"Período"
, para o qual se deseja emitir o diário.
É importante lembrar que, se a empresa trabalha com códigos estendidos, a partir do processamento dos novos códigos estendidos, o período da emissão dos relatórios deve ser em função da data inicial deste período. Não é recomendado emitir um relatório com um período que contemple dois ou mais códigos estendidos diferentes, pois os resultados podem ser afetados.

O usuário pode optar por agrupar os dados tanto por data quanto por código do histórico.
Após preencher os critérios, o usuário deve clicar no botão
 
"OK"
 
para que o relatório seja apresentado em tela e para imprimí-lo deve-se clicar no botão: 
 
O diário será impresso na seguinte ordem: Data, Filial, Número do Lote e primeiro as contas de débito e depois as contas de crédito de cada lote.
Para imprimir os totais de débitos e créditos, deve-se marcar a opção: 
Neste caso, serão apresentados totais agrupados por data e no final do relatório um total geral de débitos e créditos.
