O SisPetro utiliza o arquivo 
SCG.Ini
 para fazer sua instalação (na primeira execução), atualização e configuração para a sua execução.
Um arquivo INI, é do tipo texto que contém informações separadas por seções, onde seu inicio é definido por [
Nome da Seção
] e é finalizada ao encontrar o inicio de uma nova seção ou o final do arquivo.
O SisPetro utiliza no arquivo SCG.Ini com as seguintes seções:
Geral
: Possui diversos parâmetros para iniciar o SisPetro;
Splash
: Possui o parâmetro de tempo para configuração da tela de apresentação. Esta é uma seção opcional;
Automação
: Possui informações para Integração com o sistema de Automação da View.
Servidor Sefaz
: Nesta seção definimos cada servidor de controle associado ao servidor de processamento. 
O nome do servidor serve mais como orientação para quem está configurando.
Acesse 
 para o modelo do arquivo.
Outras configurações:
DLL's
: Além das configurações dos arquivos INI, são necessárias as dll's de execução do Sispetro e de validação da Inscrição Estadual (caso utilize este perfil).
O arquivo 
sptCont.ini
 (Controladoria) segue a mesma estrutura do arquivo 
SCG.ini
.
