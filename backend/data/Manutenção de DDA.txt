Objetivo
DDA é a sigla para Débito Direto Autorizado, serviço bancário que permite receber os boletos de cobrança de forma eletrônica. Tem como principal característica a praticidade de ser notificado quando um boleto é emitido para o CNPJ da empresa, sendo reunidos em um único arquivo enviado pelo banco onde nele consta todos os boletos emitidos para o CNPJ. Assim sendo, o intuito desta ferramenta no SisPetro consiste em ler os arquivos enviados pelo agente bancário, nele constando todos os boletos emitidos para o CNPJ da empresa. 
O acesso é feito através do menu 
Movimentação / Financeiro / Contas a Pagar

Como Utilizar:
O SisPetro irá procurar na pasta de 
Arquivos a Processar,
 indicado no serviço, os arquivos do mais antigo para o mais recente. Após isso, faz a 
Leitura do Arquivo
 e caso não encontre nenhuma informação válida, move o arquivo para a pasta de 
Erros
. Se não encontrar empresa válida com o CNPJ da empresa no (header do arquivo) ficará como
 Não Localizada.
Após o 
Processamento do Arquivo
, caso tenha ocorrido o processo normalmente, sem erro, o arquivo será salvo na pasta 
Processados
. 
Para cada item do arquivo o Sispetro verifica se foi encontrado um único título na empresa para o Fornecedor ativo (pelo CNPJ), com o mesmo Valor e mesmo Vencimento. 
Será salvo a
 Linha Digitável
 no título do Contas a Pagar.
