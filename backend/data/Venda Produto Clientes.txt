TITULO: Venda Produto (Clientes)

--------------------------------------------------
Visando atender a algumas portarias da ANP, onde alguns produtos não são permitidos para venda em determinadas cidades para Postos, Consumidores Finais e Congêneres, o Sispetro permite que sejam realizados bloqueios de Venda de determinados produtos para determinadas Atividades Econômicas.
O acesso é feito através do menu 
Arquivos / Clientes / Aba Detalhe / Aba inferior Venda Produtos
Permissão de Venda de Produto: 
nessa opção é definida como será feito o bloqueio de venda. A verificação pode ser feita de duas maneiras: 
Verifica Produtos permitidos para 
empresa
: padrão de bloqueio geral que será aplicado a todos os clientes que estiverem com esse perfil
Verifica Produtos permitidos para este 
cliente: 
aplicado uma regra específica para esse cliente, sobrepondo a configuração geral por empresa.
Para alterá-la o usuário deve ter os seguintes perfis de usuários habilitados, 
Alteração de Cliente
 e  
Pode Alterar Permissão e Dados de Vendas Produtos.
 Para mais informações, acesse 
Painel Segurança (Usuários)
.
Quadro Restrição de Depósitos
Restringe depósitos para venda?: 
marcando essa opção, na finalização do pedido de venda, o SisPetro irá verificar se o depósito utilizado está no 
Conjunto de Depósitos Permitidos
. Caso não esteja o SisPetro mostrará a mensagem: 
O depósito xx não está incluído no conjunto...
 conforme imagem abaixo:
Conjunto de Depósitos Permitidos: 
funciona em conjunto com a opção 
Restringe depósitos para Venda. 
 Aqui são listados os conjuntos de depósito cadastrados em 
Conjunto de Depósitos Permitidos
.
Para mais informações sobre o bloqueio de venda de alguns produtos para o cliente por atividade econômica, acesse 
Bloqueio de Venda por Atividade Econômica
