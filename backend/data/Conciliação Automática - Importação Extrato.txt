TITULO: Conciliação Automática - Importação Extrato
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878898199
--------------------------------------------------
Após realizar todo o processo de inclusão de um período para a conta a ser conciliada, conforme explicação do tópico 
Conciliação Automática - Informações Gerais
, o usuário deverá realizar a importação do arquivo.
Nesse processo, o usuário terá a sua disposição um arquivo de extrato disponibilizado pelo banco onde se terá todos os movimentos de um determinado período. A idéia é importar esse arquivo para dentro do sistema, evitando assim o processo de digitação dos lançamentos.
Lembrando que nem todos os bancos estão disponíveis para esse tipo de procedimento dentro do Sistema, hoje trabalhamos com os Banco do Brasil e Safra, caso não esteja disponível o Banco desejável o usuário deverá se informar com o suporte técnico a respeito.
Além de não permitir a digitação dos dados, o processo de importação, não permitirá que um extrato importado seja aberto após estar pronto para conciliação. Neste caso, o arquivo deverá ser excluído, porém, será necessário o ter o perfil de usuário, localizado no menu 
Ferramentas / Usuário / Financeiro
: 
Para iniciar o procedimento de importação do arquivo de extrato enviado pelo banco, o usuário deverá acessar o menu 
Movimentação / Financeiro / Conciliação automática / Manutenção de conciliação automáticas
:
O usuário deve clicar na aba Arquivos de Extrato e no botão 
Arq. Banco
, o sistema abrirá uma tela onde o usuário deverá informar o caminho onde se encontra o arquivo, utilizando o botão localizado ao lado direito da tela. Não será possível o usuário digitar diretamente o caminho.
O sistema abrirá a estrutura do diretório da rede, onde o usuário poderá selecionar o caminho do arquivo.
Após informar o nome e o local do arquivo, o usuário deve clicar no botão 
OK
.
O sistema irá validar o arquivo e se não houver nenhum erro, o arquivo será importado para dentro do sistema, sendo apresentado no painel 
Arquivos de Extrato
.
Havendo algum erro no arquivo, o sistema irá emitir uma mensagem de alerta para o usuário.
O próximo passo a ser tomado pelo usuário é conciliar os lançamentos do arquivo. Esse procedimento pode ser consultado no tópico 
.
