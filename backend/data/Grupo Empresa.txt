TITULO: Grupo Empresa
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210618
--------------------------------------------------
Essa tabela permite ao usuário cadastrar Grupos de Empresas que serão controlados pelo sistema. É preciso cadastrar o grupo de empresas, para que sejam utilizados para análise de dados em relatórios e tela de consultas.


Para acessar a Tabela de Natureza de Operação acesse o menu 
Tabelas Auxiliares / Grupo Empresa
:




Clique no botão 
Inserir um novo registro
.




O sistema exibirá a tela que permite o cadastramento de um novo Grupo Empresa, através da aba 
Detalhes
.




O Sequencial é sugerido pelo sistema, mas caso seja necessário é possível alterar para um sequencial desejado. Informe a descrição desejada.


Clique no botão 
OK
 para salvar os dados.




Para cancelar o processo, clique no botão 
cancelar




Em seguida, é preciso informar no cadastro da empresa que ela faz parte de um grupo de empresa e qual grupo de empresa. Para isso, veja maiores informações no tópico 
Painel Dados Gerais
. Caso seja necessário configurar a empresa para trabalhar com Contabilidade Real, veja maiores informações 
Painel Contab. Real
.
