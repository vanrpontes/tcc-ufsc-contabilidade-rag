O acesso a essa tela se dá através de 
Ferramentas / Usuário / painel Perfil / painel inferior Produtos
.
Conhecendo os perfis existentes:
Cadastro de Produtos
Geral
Pode incluir produto no cadastro?: 
permite que o usuário inclua produtos no cadastro.
Pode alterar produto no cadastro?: 
permite que o usuário altere os produtos no cadastro.
Pode excluir produto no cadastro?: 
permite que o usuário exclua produtos do cadastro.
Visualização de Dados
Visualiza painel movimentação:
 permite que o usuário veja a aba de movimentações no cadastro de produtos.
Visualiza painel pendências: 
permite que o usuário veja os pedidos pendentes no cadastro de produtos.
Visualiza painel Cotações: 
permite que o usuário veja as cotações realizadas para o produto no cadastro de produtos.
Visualiza painel quantidade/gráfico: 
permite que o usuário veja a aba quantidade / informações no cadastro de produtos.
Pode analisar Margem Pedidos?: 
Pode visualizar % lucro bruto?: 
permite que o usuário veja e altere o % de lucro bruto do produto.
Margem Substituição Tributária
Acessa a tela de substituição tributária produto: 
permite que o usuário acesse a tela de margem de substituição tributária no cadastro de produtos.
Pode incluir data vigência - Margem Subst.? 
permite que o usuário inclua data de vigência na tela de margem de substituição tributária no cadastro de produtos.
Pode excluir data vigência - Margem Subst.? 
permite que o usuário exclua data de vigência na tela de margem de substituição tributária no cadastro de produtos.
Permite copiar valores alíquotas substituição tributária entre empresas: 
permite que o usuário copie as alíquotas para outra empresa na tela de margem de substituição tributária no cadastro de produtos.
Pode alterar custos? 
permite que o usuário altere os custos do produto.
Pode alterar dados Automação Base: 
permite que o usuário consiga realizar alterações no quadro de automação Basecom do 
Cadastro de Produtos
.
Pode incluir, excluir e alterar crédito provisório: 
permite inclusão, alteração e exclusão de crédito provisório no cadastro de produtos.
Pode dar manutenção em Produtos correlatos:
 permite que o usuário inclua 
produtos correlatos
 no cadastro de produtos.
Pode Alterar Temperatura e Densidade: 
permite alterar dados da tela de temperatura e densidade no cadastro de produtos.
Vendas
Pode bloquear venda produto: 
permite que o usuário bloqueie vendas de um determinado produto.
Pode vender produto bloqueado: 
permite que o usuário venda produtos que estejam bloqueados.
Endereço utilizado no bloqueio de movimentação do Produto (Município ou UF): 
permite que o usuário defina a permissão ou o bloqueio de venda produtos por Municípios ou UF.
Pode incluir data vigência - Tabela Preço? : 
permite ao usuário incluir uma data de vigência para uma determinada tabela de preço. Para mais informações, ver 
Manutenção de Tabela de Preços
.
Pode excluir data vigência - Tabela Preço?: 
permite ao usuário excluir uma data de vigência para uma determinada tabela de preço. Para mais informações, ver 
Manutenção de Tabela de Preços
.
Pode configurar se para o produto o preço de tabela é o mesmo do pedido: 
permite que o usuário configure se o preço do produto no pedido é o mesmo do preço de tabela.
Pode controlar a cor do produto por entidade?:
 permite que o usuário controle a cor do produto vendido por cliente, 
 
Em conjunto com os perfis, Pode Alterar e Pode Incluir Clientes. -
