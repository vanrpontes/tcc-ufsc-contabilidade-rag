TITULO: Encerramento de MDF-e pela Chave de acesso
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2954362902
--------------------------------------------------
No módulo de Faturamento é normal que os usuários só realizem operações referente a dados da empresa conectada. Porém, no caso do Encerramento de MDF-e não é bem assim, já que, normalmente o usuário da filial que recebeu o produto pode estar finalizando um MDF-e emitido em outra filial. 
Para evitar que o usuário possa realizar outras operações na Manutenção de MDF-e, é possível 
o  Encerramento de MDF-e pela Chave de Acesso por uma tela especifica.
Configurações
Perfil de usuário: O primeiro passo é liberar o acesso a tela, para isso basta ir em Ferramentas / Usuário / Acesso Menus. Selecione o menu e marque as opções Visível e Habilitado. Para mais informações, ver 
.
No menu Movimentação / Faturamento / Manifesto Eletrônico / Encerramento de MDF-e.
Informe a chave de acesso do MDF-e que deseja encerrar.
Se a chave for válida e estiver pendente de encerramento, abrirá a tela de
 
Encerramento
 
para que o usuário preencha/confira as informações.
Encerramento Automático para NF-e de entrada via serviço
Apenas para MDF-e referentes a nota de entrada e para responsável pela emissão do MDF-e, ao dar entrada na NF-e vinculada ao MDF-e o SisPetro fará o encerramento automatícamente via serviço. 
Para mais informações, ver
