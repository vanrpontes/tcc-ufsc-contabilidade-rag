TITULO: Remessa para Armazenagem Automática
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2902164202
--------------------------------------------------
No processo de Venda para uma congênere, na Ordem de Carregamento, é indicada a necessidade de fazer a remessa de armazenagem. Ao ser feito o processo de Portaria (de Entrada e Saída) dessa Ordem de Carregamento de Venda, ao finalizar, o Sispetro gera uma Ordem de Carregamento de Descarga de Terceiros para essa congênere e faz automaticamente o processo de Portaria, para devida movimentação do Estoque Físico.
Para isto, é necessário indicar na tela de clientes, aba Cadastro Detalhado, que tipo de Armazenagem Automática o cliente deve ser configurado. Mais detalhes em 
Cadastro Detalhado
.
Para maiores detalhes em como fazer Remessa e Retorno de Armazenagem: 
Armazenadora - Remessa e Retorno de Armazenagem
Ao fazer uma Ordem de Carregamento de Vendas, há três situações:
Para entidade configurada para "Nunca" fazer Armazenagem Automática, o campo "Armazenagem Automática" aparecerá desabilitado;
Para entidade configurada para "Sempre" fazer Armazenagem Automática, o campo "Armazenagem Automática" aparecerá habilitado, sem opção para desabilitar o mesmo;
Para entidade configurada para "Conforme configuração do usuário" fazer Armazenagem Automática, o campo "Armazenagem Automática" aparecerá desabilitado com a opção de habilitar o mesmo;
