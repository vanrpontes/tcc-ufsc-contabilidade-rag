TITULO: Troca Código Extendido de Conta
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210442
--------------------------------------------------
A partir da versão 9.30 esta tela foi desabilitada, já que, 
após o advento de múltiplos códigos extendidos, essa troca de código implica em alterar toda a movimentação dessa conta.
Abaixo seguem algumas informações importantes:
Se a conta não foi utilizada, ela pode ser excluída e incluída outra com o código desejado.
Se a conta foi utilizada, mas ainda é o primeiro ano de utilização (ainda não fechado), contactar Suporte Futura para providenciar alteração.
A partir do segundo ano de utilização, criar um novo conjunto de Código Estendido (
Manutenção de Códigos Estendidos
) para que o primeiro ano de utilização fique íntegro.
 
O código extendido de uma conta é o que a define na hierarquia do Plano de Contas, por isso sua troca está numa tela específica, acionada pelo menu 
Ferramentas / Contabilidade / Troca Código Extendido de Conta
.
 
Informe a conta desejada que deverá ser analítica. Será apresentado o código extendido atual.
Informe o novo código extendido. Este novo código deverá ser da mesma natureza que o anterior (Ativo, Passivo, Receitas ou Despesas), não pode já estar cadastrado no Sistema e deve ter uma conta sintética acima dele.
Acione o botão OK para atualizar os dados no Sistema.
