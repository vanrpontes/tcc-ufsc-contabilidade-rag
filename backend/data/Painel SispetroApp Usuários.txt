Esta página foi criada para explicar onde se configura, no SisPetro Normal, os parâmetros para acesso do Sispetro App.
Para qualquer usuário acessar o SisPetro App, é preciso que tenha permissão para acessa-lo e um e-mail válido.
Na tela de Manutenção de Usuários, aba SispetroApp:
SisPetro App
Tipo de Acesso: 
nesta fase o Administrador definirá se o usuário terá acesso apenas ao SisPetro APP, SisPetro Normal, remoto e controladoria ou ambos.
Ambos: 
terá acesso ao SisPetro Normal e APP.
SisPetro Normal (Windows): 
terá acesso apenas ao SisPetro Normal.
SisPetro APP: 
terá acesso apenas ao SisPetro APP.
E-mail: 
deve ser informado o e-mail do usuário, lembrando que é com ele que o usuário fará login.
Administrador: 
caso seja marcada esta opção, o usuário poderá atualizar o SisPetro APP e alterar as configurações do envio de e-mail. 
Resetar senha e enviar: 
caso o usuário tenha esquecido sua senha, esta é a opção para lhe reenviar.
Depois de feito todos os passos, o novo usuário receberá um e-mail com o link para cadastrar. 
Caso seja necessário a restrição da movimentação (Pedido e Faturamento) ou mesmo da carteira de clientes, existem configurações especificas para isso.
Venda Spot
Permite aprovação de solicitação de abastecimento: 
com este perfil marcado, o usuário poderá aprovar uma solicitação de abastecimento.
Permite cancelamento de solicitação de abastecimento: 
com este perfil marcado, o usuário poderá cancelar uma solicitação de abastecimento.
Permite visualização de solicitação de abastecimentos: 
neste campo, podemos configurar para o usuário as opções abaixo:
Somente as próprias: 
o usuário poderá visualizar somente suas próprias solicitações de abastecimento.
Todas as solicitações: 
o usuário poderá visualizar todas solicitações de abastecimento.
Roteamento após login: 
neste campo, podemos configurar para o usuário as opções abaixo:
SisPetroApp: 
após o login, irá para a tela do SispetroApp.
Bloqueado: 
após o login, não irá acessar nada.
Venda Spot: 
após o login, não irá acessar a tela do Spot.
Permite inclusão de clientes: 
neste campo, podemos configurar para o usuário as opções abaixo:
Ver Sistema: 
selecionando essa opção, o SisPetro irá respeitar a configuração do 
Perfil de Sistema > Painel SispetroApp
. 
Não: 
não permitirá a inclusão de Clientes
Sim
: permitirá a inclusão de Clientes
Tipo usuário Venda Spot
: neste campo, podemos configurar para o usuário as opções abaixo:
Sem acesso
: o usuário não terá acesso ao Venda Spot
Cliente (usuário externo)
: o usuário terá acesso como Cliente no Venda Spot
Abastecedor (usuário interno)
: o usuário terá acesso como Administrador no Venda Spot
Vendas
Permite visualizar cadastro de clientes:
 com este perfil marcado, será possível no SisPetro APP visualizar o cadastro de clientes.
Permite visualizar manutenção de Pedidos de Venda: 
com este perfil marcado, será possível no SisPetro APP visualizar manutenção de Pedidos de Venda.
Permite visualizar manutenção de Notas Fiscais: 
com este perfil marcado, será possível no SisPetro APP visualizar manutenção de Notas Fiscais.
