TITULO: Inclusão de Cotações

--------------------------------------------------
A inclusão de uma cotação pode ser acionada a partir do Menu 
Movimentação > Movimentação de Material > Cotação > Inclusão de Cotação. 
Ela pode ser incluída também por:
Manutenção de Cotações
, através do botão Incluir
Busca de Produtos para Cotação
A tela abaixo é apresentada:
Para incluir uma cotação é necessário informar:
Status: 
Informar qual o Status da cotação:
Cotação para envio a fornecedor: 
a cotação ficará com 
Status 7
 na 
Manutenção de Cotações
Aguardando recebto informações fornecedor: 
cotação ficará com 
Status 8
 na 
Manutenção de Cotações
Para escolha da melhor cotação: 
cotação ficará com 
Status 9
 na 
Manutenção de Cotações
Informa Fornecedor Cadastrado? ou Fornecedor: 
o usuário pode escolher se deseja informar um fornecedor já cadastrado no SisPetro ou digitar o nome do Fornecedor. Caso o usuário informe um fornecedor já cadastrado no SisPetro, os campos Fornecedor, Contato, E-mail e Telefone serão preenchidos automaticamente com os dados contidos no Cadastro de
 Fornecedores
.
Informa Condições de Pagamento Cadastrado? ou Condição de Pagamento: 
é o mesmo caso acima, caso a condição de pagamento negociada para a cotação já esteja cadastrada no SisPetro, marcar a opção 
Informa Condições de Pagamento Cadastrado?.
 Se não for cadastrada, digitar a informação no campo
 
Condição de Pagamento.
Contato: 
informar o nome da pessoa com quem esta negociando, esta cotação.
E-mail: 
caso seja informado um Fornecedor cadastrado no SisPetro, e este estiver com o e-mail cadastrado, o este campo será preenchido automaticamente. Caso contrário informar o e-mail do fornecedor.
Telefone: 
o mesmo caso acima, se o Fornecedor estiver cadastrado no SisPetro o campo Telefone será preenchido automaticamente, caso contrário é preciso informar.
Validade da Cotação:
 o usuário pode ou não informar a validade desta cotação. É um dado interno para a empresa, podendo ser desnecessário em alguns casos.
Setor de Compras:
 informar o setor que é responsável pela compra da mercadoria. 
Observações e Observações Internas: 
é um espaço para caso o usuário deseje registrar alguma informações importante inerente a cotação. Podem ser visualizadas em 
Manutenção de Cotações
 aba 
Cotações
Essas informações, caso sejam preenchidas, serão transferidas para o PMM caso o usuário emita um PMM a partir dessa cotação.
Preenchidas as informações acima será necessário incluir o 
Item da Cotação. 
Para isso pressione o botão 
Incluir
. O SisPetro irá mostrar a tela abaixo:
 
Para concluir a inclusão do produto é necessário informar:
Produto: 
informar o código do produto ou apertar a tecla 
ENTER
 para pesquisar. 
Qtde cotação
: quantidade solicitada pela empresa.
Qtde Fornecedor
: quantidade que o fornecedor tem para a compra.
Preço inicial
: preço negociado com o fornecedor, este campo é apenas para dados estatísticos, não se altera durante a negociação. É o preço Inicial requerido pela empresa.
Preço unitário
: preço passado pelo fornecedor conforme negociação. 
Esse será o preço utilizado no PMM.
Prazo de Entrega (Dias)
: informe o prazo de entrega do item.
Unidade
: unidade de medida do item, que deve ser previamente cadastrada na 
. Este campo só será habilitado caso o usuário tenha acesso. Para mais informações, acesse 
Painel Compras (Usuários)
.
Característica: 
informe a característica do produto .
O campo 
Características
 só irá aparecer caso o perfil 
Trabalha com Característica de Produtos
 esteja habilitado para trabalhar com o Característica. Para mais informações, ver 
Painel Compras (Usuários)
.
Será questionado se já deseja enviar a cotação por e-mail ao Fornecedor. Caso seja optado pelo envio do e-mail o status será mudado para 8 (Aguardando recebto informações do fornecedor).
Para que o e-mail seja enviado corretamente, o serviço de e-mail deve estar em pleno funcionamento no SisPetro. Para mais informações, ver 
Serviço de E-mail
.

Compras,
 do perfil de Sistema uma maneira de vincular esta conta de e-mail. Para mais informações, ver 
 e
