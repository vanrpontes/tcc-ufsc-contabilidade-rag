Configurando o Plano de Contas de Clientes / Fornecedores
As contas  contábeis de Clientes e Fornecedores serão geradas automaticamente pelo sistema  a partir da inclusão dos mesmos. Porém, antes é necessário configurar alguns  parâmetros e incluir as contas para os fornecedores e clientes (entidades) já  cadastrados até o momento. Acione no Menu Principal as opções -
 
Ferramentas  / Sistema.
 
Vá para o painel  "Contab. Real".




Informe a conta  contábil sintética de Clientes e de Fornecedores, pois a partir deles serão  geradas as contas contábeis das entidades. Tais contas precisam ter o último  nível livre para que nas contas analíticas das entidades seja prenchido com o  código da entidade. Caso deseje que o último nível seja preenchido de forma  sequencial, marque a opção:


O código extendido será sequencial?




Pressione o botão OK para confirmar a operação.


Em seguida,  pressione o botão
 
Inclui Conta Contábil das Entidades
. Serão criadas  contas contábeis para todas as entidades que ainda não as tenha, portanto este  processo pode ser repetido, pois não duplicará informações.


O botão
 
Converte códigos extendidos para sequenciais
 
só precisa ser utilizado  quando se fez o processo de inclusão de contas sem ter marcada a opção de código  extendido sequencial e depois, em outro momento, decidiu-se trocar para que este  estilo, assim será preciso converter as contas para que tenham o código  extendido sequencial.
