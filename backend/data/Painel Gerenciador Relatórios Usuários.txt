Painel inferior Gerenciador Relatórios

Conhecendo os perfis existentes:
Permitir criar relatórios
 - permite que o usuário crie relatórios no Gerenciador de Relatórios
Permite alterar relatórios
 - permite que o usuário altere os relatórios do Gerenciador de Relatórios
Permite excluir relatórios e pastas
 - permite que o usuário exclua os relatórios ou pastas do Gerenciador de Relatórios
Permite importa/salvar relatórios formato RTM
 - permite que o usuário importe e salve relatórios como RTM.
Permite criar pasta
 - permite que o usuário crie pastas no Gerenciador de Relatórios
Permite renomear pasta
 - permite que o usuário remomeie pastas no Gerenciador de Relatórios
Permite mover pasta
 - permite que o usuário mova pastas no Gerenciador de Relatórios
Permite modificar segurança das pastas próprias
 - permite que o usuário modifique as pastas criadas pelo próprio usuário no Gerenciador de Relatórios
Permite modificar segurança das pastas terceiros
 - permite que o usuário modifique as pastas criadas por outros usuários no Gerenciador de Relatórios
Permite capturar propriedade
 - 
Permite alterar segurança próprios relatórios
 - permite que o usuário altere a segurança das pastas criadas pelo próprio usuário no Gerenciador de Relatórios
Permite alterar segurança relatórios de terceiros
 - permite que o usuário altere a segurança das pastas criadas por outros usuários no Gerenciador de Relatórios
Sugere data atual para parâmetros do tipo data
 - permite que o sistema sugira a data atual nos parâmetros de data.
