Vendedor é a Entidade no Sistema, responsável pela venda dos produtos, e que portanto, deve receber uma comissão.
O vendedor está amarrado ao cliente comprador ou faturado no cadastro de clientes, ou seja, no cadastramento de um cliente, deve ser associado um vendedor.
Existe uma tela que permite manipular o cadastro de Vendedores. Para ativá-la, acione a partir do Menu 
Arquivos / Vendedores
:
 
No painel 
Navegação
 serão mostrados todos os vendedores cadastrados e também suas informações.
Para cadastrar um novo Vendedor no Sistema, acione o botão de inserção de registro.
O Sistema exibe a tela que permite a entrada dos dados do novo registro.
