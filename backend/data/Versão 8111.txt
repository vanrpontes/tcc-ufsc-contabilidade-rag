Usuário Vinculado a Usuário de Perfil não Permite Acesso ao SisPetro
Na versão passada foi criada uma restrição para não permitir acesso ao SisPetro de usuário criados como perfil. Com essa alteração, não foi possível mais conectar ao SisPetro com usuários de acesso vinculados a usuários de perfil. Esse problema foi corrigido.

Ambiente da NF-e para Manifestação
As notas fiscais de terceiros, importadas, que deveriam ser manifestadas, ao serem enviadas a Sefaz estão gerando uma mensagem de erro avisando que o ambiente em que elas foram geradas está errado. Esse problema foi corrigido.
