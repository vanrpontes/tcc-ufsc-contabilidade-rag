TITULO: Grupo Econômico - Fornecedores
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926201916
--------------------------------------------------
É possível configurar Grupo Econômico para Fornecedores, funcionalidade que permite ao usuário realizar operações como recebimentos de nota fiscal e inclusão de adiantamentos, alterando o Fornecedor desde que o mesmo pertença ao mesmo Grupo Econômico.
Para definir um Responsável do Grupo econômico basta ir em Arquivos / 
Fornecedores
 / 
Aba Observações
 marcar o perfil 
Pertencente ao Grupo Econômico
, e selecionar o Fornecedor Responsável do Grupo Econômico.
O usuário só conseguirá marcar essa opção caso o mesmo tenha o perfil de usuário 
Permite Indicar Grupo econômico de Fornecedores
 marcado.
Após definir nos fornecedores, o Responsável pelo Grupo Econômico, é possível visualizar o grupo econômico através da aba 
Grupo Econômico
 na Tela de 
Manutenção de Fornecedor
.
Para a empresa trabalhar com grupo econômico é necessário utilizar a opção 
Permite Fornecedor Responsável do Grupo Econômico
 do perfil de empresa 
Fornecedores X Entradas (Adiantamento e NF).
