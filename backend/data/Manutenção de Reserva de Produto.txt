none
Objetivo
Apresentar o funcionamento da manutenção de reserva de produto. O acesso é feito através do Menu 
Movimentação / Estoque / Reserva de Produto
Filtros
É possível filtrar a manutenção para exibir dados por :
Período de exibição de dados (Data) 
Ativos / Inativos:
 marcando a opção 
Exibe Inativos
 mostra os registros inativos. 
Obs.: essa opção é desmarcada por padrão
Depósito 
Empresa
Os filtros de depósito e / ou empresa, caso estiverem em branco, serão exibidos todos os dados
Botões
Incluir / Editar: 
Inclui ou edita um registro incluído
Copiar: 
é possível efetuar uma cópia de Reserva por um determinado período
Excluir: 
exclui o registro inserido
Layout Padrão: 
volta ao layout padrão, quando uma vez alterado a ordem
Atualizar: 
atualiza o status da tela
Sair: 
sai da tela
Abas
Reservas: 
mostra as reservas de produtos
Movimentações: 
Tela onde mostra os produtos que tiveram movimentação
