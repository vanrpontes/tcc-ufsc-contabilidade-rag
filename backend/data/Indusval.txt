Dados Gerais

O Banco Indusval trabalha em conjunto com o Banco Bradesco (237). Assim, a empresa tem conta no Banco Indusval, onde adquire o produto Cobrança Eletrônica, porém, ao imprimir o boleto, o imprime com as características do Banco Bradesco. Dessa forma, o cliente que recebe o boleto não precisa localizar uma agência do Banco Indusval para efetuar o pagamento, podendo fazer isso perfeitamente no Banco Bradesco, que tem muito mais agências disponíveis em todo o país. Para atender essa necessidade usaremos a configuração de Banco Correspondente do SisPetro. Cabe lembrar que o arquivo de remessa é conforme o layout do Banco Indusval.

Dados Técnicos




Nosso Número:
 o nosso número pode ter até 11 dígitos. É salvo no SisPetro com o número de dígitos que estiver no título, porém é impresso com 12 dígitos no seguinte formato:NNNNNNNNNNN-D


 O dígito verificador é calculado no módulo 11, utilizando até a base 7. Para este cálculo é usada a carteira do Banco Correspondente.
