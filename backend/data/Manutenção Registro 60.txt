TITULO: Manutenção Registro 60
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878899492
--------------------------------------------------
O registro 60 é o registro do Sintegra que possui informações de Cupom Fiscal. A manutenção destes dados é feita pelo menu 
Movimentação / Faturamento / Cupom Fiscal / Manutenção do Registro 60
, a partir da seguinte tela:




Há dois tipos de Registro 60: M (Mestre) e A (Analítico). O Registro 60 M tem informações do equipamento de Cupom Fiscal numa determinada data, enquanto o Registro 60 A detalha, a cada data os tipos de tributação realizados.


Os registros 60 podem "chegar" ao SisPetro de 3 formas distintas:




Via 
Inclusão Manual
, onde o usuário obtém as informações direto de um Cupom Fiscal (não atrelado ao SisPetro) e através do botão 
Inclui
 desta tela de Manutenção, providencia a digitação dos dados.


Via 
Importação
, onde o usuário tem um arquivo texto no layout exigido pelo SisPetro e importa os dados, através do botão Importa desta tela de Manutenção.


Via 
ECF
 (Emissor de Cupom Fiscal), que é quando o equipamento está devidamente configurado no SisPetro (vide tópico ECF - Emissor de Cupom Fiscal) e transmite as informações para esta tela através do botão 
Gera
.




Cada uma das funções citadas acima, como as demais a serem explicadas neste tópico, estão disponíveis para o Registro 60A e para o Registro 60M, dependendo do painel que estiver selecionado (no exemplo abaixo seria o Registro 60M):




No subPainel 
"Válidos"
 são apresentados todos os registros que estarão disponíveis para uso no Sintegra. No subPainel 
"Não Importados"
 são apresentados os registros que foram solicitados a Importação, mas que por algum motivo não conseguiu ser importado (geralmente por problemas de duplicidade).


Tanto para registros 
"Válidos"
 como para os registros 
"Não Importados"
 é possível alterar os dados, pelo botão 
Altera
 ou excluí-los pelo botão Exclui.


Utilize o filtro da Empresa e o Período para apresentação dos dados. Em seguida, sempre clique no botão Atualiza.


Através do botão 
Legenda
 é possível verificar fontes e cores utilizadas para apresentação dos dados e seus significados.


Mude a ordem e posição das colunas como desejar, através do botão 
Layout Padrão
 é possível apresentar os dados da forma original, desfazendo as alterações realizadas na apresentação das colunas.
