TITULO: Efetivação de Vendor
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878898432
--------------------------------------------------
A efetivação de vendor consiste em atribuir um contrato para os títulos que estão sem contrato de vendor.


A tela de Efetivação de Vendor é acessada pelo menu 
Movimentação / Financeiro / Manutenção de Vendor / botão Efetiva
:




Para atribuir um contrato de vendor é necessário preencher os seguintes campos:




Data:
 informar a data do contrato.






Conta Contábil de Banco:




 






Banco:




 



O banco será preenchido automaticamente se existir um banco correspondente à conta contábil de banco.






Conta contábil de juros:
 Informação sugerida através da configuração do perfil de sistema, veja informações mais abaixo.



 






Conta Contábil de Despesa:
 Informação sugerida através da configuração do perfil de sistema, veja informações mais abaixo.



 






Conta Contábil de Vendor:
 Informação sugerida através da configuração do perfil de sistema, veja informações mais abaixo.



 




Através de uma nova configuração de sistema, os campos de 
CC Juros
, 
CC Despesa
 e 
CC Vendor
 podem ter seus respectivos conteúdos sugeridos.


Para configurar estas contas acesse o menu 
Ferramentas/ Sistema/ aba Contas Contábeis
 e localize o quadro de 
Vendor
:

 

Em seguida, configure as contas desejadas para cada caso.


Código do Contrato:
 deve ser preenchido o código do contrato que será associado ao vendor.






Código do Histórico:
 informar o código do histórico.



 






Selecionando título:




 



Nesta grid serão apresentados os títulos que estão marcados como vendor (ver 
Tabela Tipos de Cobrança
) e devem ser selecionados os títulos aos quais serão atribuídos o contrato de vendor.


As colunas 
% dos Juros
 e 
Nova Data Vencto.
 podem ser preenchidas.






Botão de marca e desmarca:




 



Esse botão serve para marcar ou desmarcar todos os títulos para serem atribuídos ou não contrato de vendor.






Valores do Vendor:




 



A figura acima mostra o valor total dos títulos selecionados, bem como o valor total dos juros desses títulos e a somatória do valor dos títulos + valor juros.


O valor transferido pelo banco pode ser preenchido. Como sugestão este campo é preenchido com o valor total (valor dos títulos + valor dos juros)




O resultado do valor das despesas bancárias é a diferença entre o valor total e o valor transferido.






Botão Calcula:




 



O botão Calcula irá recalcular os valores do(s) título(s) selecionado(s).






Botão Imprime:




 



Esse botão serve para imprimir os dados exibidos na grid da tela.






Botão OK:




 

Esse botão serve para fazer a contabilização do vendor gerando lotes e alterando o status do vendor para vendor com contrato.






Botão Cancela:




 

O botão cancela sai da tela sem efetuar as alterações nos títulos.
