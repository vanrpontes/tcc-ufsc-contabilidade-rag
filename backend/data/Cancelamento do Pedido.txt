TITULO: Cancelamento do Pedido
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878440084
--------------------------------------------------
Um pedido de vendas pode ser cancelado pelas seguintes formas: 


Tela de manutenção de Pedidos:
 


Para acessar a tela de manunteção de Pedidos, basta acessar o menu 
Movimentação/ Estoque / Manutenção Pedidos
: 




Nesta tela, o sistema concentra todos os pedidos que foram feitos, independente do pedido estar aprovado ou não. Para continuar o processo de cancelamento, deve-se localizar e marcar o pedido para depois, clicar no botão "Cancela". 




O botão 
"Cancela"
 estará habilitado desde que o usuário tenha permissão para cancelar o pedido através da tela de manutenção de pedidos. Essa permissão pode ser configurada através do menu 
Ferramentas / Usuário / Venda
, marcando a opção: 




Após clicar no botão "Cancela", o sistema emitirá uma mensagem solicitando a confirmação do cancelamento para continuar ou não o processo. 




Antes de cancelar efetivamente o pedido, será aberta uma tela para informar o motivo pelo qual o pedido está sendo cancelado. 




O sistema não cancelará o pedido, se nesta tela o usuário clicar no botão cancelar. Vale lembrar, que apenas os motivos de cancelamento ativos serão mostrados na tela de pesquisa. Maiores informações no tópico 
Tabela Motivos Cancelamento
. 


Após confirmar o motivo do cancelamento, o sistema cancelará o pedido e o mesmo aparecerá na tela de manutenção de pedidos na cor vermelha, isso se a opção "Visualiza Pedidos Cancelados" estiver marcada. 




Tela de Pedidos Pendentes:
 


Para acessar a Tela "Pedidos Pendentes", o usuário deve utilizar o menu 
Consulta / Pedidos Pendentes
: 




Basta o usuário selecionar o pedido desejado e clicar no botão "Cancela Pedido". Os procedimentos são os mesmos da tela de Manutenção de Pedido de Vendas. 


Tela de pedidos aprovados e não faturados:
 


Para acessar a Tela de pedidos aprovados e não faturados, basta acessar o menu 
Consulta/Pedido Aprovados
 p/ Faturamento. 




Através do botão 
"Canc.Pedido"
, o usuário pode cancelar o pedido de vendas. Os procedimentos são os mesmo que são realizados pela tela de Manutenção de Pedidos. O usuário precisa marcar o pedido a ser cancelado e informar um motivo para o cancelamento. Nesse caso, após cancelado, o pedido desaparece dessa tela, só podendo ser consultado pela tela de Manutenção de Pedidos, onde deverá aparecer em vermelho. 


Para que o pedido possa ser cancelado, ele não pode ter nenhuma Ordem de Carregamento ou Nota Fiscal vinculada a ele. Caso tenha, o usuário deverá cancelar primeira a ordem de carregamento ou a nota fiscal e depois a OC.
 


Caso o pedido tenha utilizado um 
Adiantamento Quantificado
, a quantidade informada para o produto será acrescida ao saldo do mesmo no Adiantamento para que possa ser utilizado em outro pedido.
