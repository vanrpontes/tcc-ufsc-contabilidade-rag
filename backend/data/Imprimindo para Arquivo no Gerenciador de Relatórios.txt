TITULO: Imprimindo para Arquivo no Gerenciador de Relatórios
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2879653459
--------------------------------------------------
Existem 
relatórios mais antigos
 no Gerenciador de Relatórios que a opção para 
imprimir para arquivo não está habilita
.
O que fazer?!
Vamos habilitar a função!
Acesse o Gerenciador de Relatório:
Localize e abra o relatório a ser alterado.
Localize Propriedade de Report e vá para 
Output - File
:
Marque o campo 
AllowPrintFile
Salve normalmente (Arquivo/ Salvar).
Com isso, ao imprimir o relatório, a opção para imprimir para arquivo estará habilitada.
