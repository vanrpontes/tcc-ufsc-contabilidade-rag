TITULO: Tabela Motivos Cancelamento
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210589
--------------------------------------------------
Nesta tela são cadastrados os motivos de cancelamento a serem utilizados no Sistema.


A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - Tabelas Auxiliares / Faturamento / Motivos Cancelamento.




O sistema já pré-cadastra 2 motivos: 
Erro preenchimento dados
 e 
Erro impressão
. No entanto, é possível alterá-los. Para acrescentar um novo motivo de cancelamento à tabela, acione o botão de inclusão de registro:




O Sistema exibe a tela para preenchimento dos novos dados.




O usuário deve informar a descrição do motivo e marcar o campo "Ativo".


Os motivos cadastrados serão exibidos para escolha no momento do cancelamento de uma Nota Fiscal, cancelamento de um pedido ou desmembramento de pedido.


Lembrando, que os Motivos de Cancelamentos que não estiverem ativos, ou seja, possuirem o campo "Ativo" desmarcado, não aparecerão nas telas de pesquisa, caso o usuário deseje, por exemplo, cancelar um pedido de vendas ou uma nota fiscal.


Clicando no botão OK, o usuário atualiza os dados no Sistema.


O Sistema retorna a tela de navegação.


Para melhor identificação, os registros que não estiverem ativos, serão apresentados na cor vermelha, na tela de navegação.
