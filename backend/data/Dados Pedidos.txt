Com as atualizações realizadas para emissão do DPMP de refinarias, uma das implementações realizadas foram que pedidos de outras entras e outras saídas pudessem ser apresentados na manutenção do DPMP.


Isto é possível se o pedido estiver utilizando um código de histórico que participe do Dpmp.


Mas é importante lembrar que estes pedidos não podem substituir operações realizadas por notas fiscais, pois nem todos os dados gerados pelas notas são gerados pelos pedidos.


A tabela abaixo lista os campos que são gerados pelos pedidos.

Uma aplicação pratica para os pedidos no DPMP é a utilização dos históricos de perda de ganhos pela variação de estoque, se caso estes históricos estiverem com o perfil de participação no DPMP estes podem ser apresentados na manutenção do mesmo.
