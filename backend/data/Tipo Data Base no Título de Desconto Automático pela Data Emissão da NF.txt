TITULO: Tipo Data Base no Título de Desconto Automático pela Data Emissão da NF
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2902164188
--------------------------------------------------
Para gerar esses títulos de Desconto Automáticos (Retenção) há diversas configurações para identificarmos cada campo, uma delas é referente à data base para emissão e vencimento. Até a versão 7.18 essa data base é pela data de emissão ou vencimento do título original, mas temos uma situação onde isso não atende a necessidade.


É normal configurar a data de emissão de um título a Pagar pela Data de Movimento da NF, mas em alguns casos, poderá ser necessário gerar o título automático pela Data de Emissão da NF.


À partir da versão 7.19 do Sispetro será possível definir a data base para cálculo do vencimento à partir da Data de Emissão da NF de Entrada.


Para realizar essa configuração acesse a tela 
Geração de Titulos de Desconto Automático
 no menu 
Tabelas Auxiliares / Financeiro




O Campo 
Tipo Data Base
 Defina para 
Data Emissão NF (se não tiver, do Título)


A data base é utilizada para identificar a faixa de vencimento do título de retenção. A data de emissão do título de retenção é igual à data de emissão do título original.
