Objetivo
Essa é uma das tabelas que serão utilizadas para o cálculo do IS. Através dessa tabela serão feitas as  configurações necessárias para fazer os cálculos e também enviar os 
dados no XML.
O acesso é feito através do Menu Tabelas Auxiliares / Faturamento / Alíquota de IS
