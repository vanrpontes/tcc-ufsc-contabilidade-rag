TITULO: Especificação Máquina Virtual SispetroWeb
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926193840
--------------------------------------------------
Objetivo
Listar os requisitos mínimos para execução da máquina virtual (VM) do SispetroWeb.
Requisitos Mínimos
Ter um servidor que execute a máquina virtual VMWare versão de hardware 11 (VMWare Player, Workstation v11, ESXi 6.0, Fusion 7.x ou Server).
A máquina virtual alocada deve ter os seguintes requisitos mínimos:
Processador com 1 Core ou mais.
2Gb de memória.
10Gb de hard-drive.
Funcionamento
Não será necessário nenhuma configuração adicional a não ser o que está detalhado no manual do administrador. Subir a máquina virtual, anotar o IP que está sendo mostrado na tela e, como um teste, apontar o browser  na porta 11000 para acessar o  Sispetro Monitor. A partir daí, deve-se configurar os dois ambientes a partir do processo descrito no manual do administrador.
