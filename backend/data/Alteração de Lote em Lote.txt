TITULO: Alteração de Lote em Lote
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2897281307
--------------------------------------------------
A tela de Alteração de Lote em Lote, possibilita ao usuário alterar vários lotes de uma só vez. Esta tela pode ser acessada pelo menu Movimentação / Alteração de Lote em Lote.


Para que esta opção esteja disponível, é necessário habilitar no perfil do usuário em Usuários > Perfil > aba Contabilidade > "Pode Alterar Lotes em lote" no painel Contabilidade.



Marcando esta opção, o campo fica habilitado para o usuário alterar os lotes (perfil configurável por usuário).



Tela Inicial




Grupo Contábil:
 
O usuário pode escolher um grupo contábil;








Código de Contabilização:
 
O usuário pode escolher o código de contabilização de acordo com o Grupo Contábil informado;








Filial:
 
Escolhe qual filial alterar;








Período:
 
Define o periodo que estão os lotes;






Clicando em 
, o sispetro traz todos os lotes informados para aquele período. Há uma opção de filtros adicionais na qual pode ser feito uma busca mais refinada:




Filtro Adicional:
 
O Filtro Adicional permite que filtre a busca pelos lotes com os campos desejados
 





É possível criar filtros para busca por meio dos campos do lado esquerdo da tela. Se for o caso, mais de uma pesquisa pode ser feita, basta selecionar um campo, colocar o intervalo inicial/final, clicar em outro campo, colocar o intervalo inicial/final novamente e assim por diante. Na aba 
Pesquisados
 mostra todos os campos usados no filtro. Em 
Ver Resumo
, mostra o sumário da busca.

Se quiser desfazer do filtro, clique em Nova pesquisa.


Dados de Alteração






Cód. Histórico Lote:
 
Altera o código do histórico;








Histórico:
 
Conforme o código do histórico informado, o campo histórico lote é alterado de acordo com seu código;








Cód. Histórico Itens Lote:
 
Altera o código do Item do lote;






Obs.: Se for selecionado somente a linha Débito (ou vice versa), apenas o código do Débito(ou vice-versa) será alterado, deixando o Crédito inalterado.




Histórico Itens Lote:
 
Altera o Histórico do Item de acordo com o código informado anteriormente;






Obs.: Se for selecionado somente a linha Débito (ou vice-versa), apenas o Histórico do Débito (ou vice-versa) será alterado, deixando o Crédito inalterado.




Conta Débito:
 
Altera a conta de
 
Débito
 
.
 
Ela só será alterada se a linha do Débito no item do lote estiver selecionada
;








Conta Crédito:
 
Altera a conta
 
Crédito
 
atual para a nova conta selecionada.
 
Ela só será alterada se na linha de Crédito o item do lote estiver selecionada
;






Data do Lote


É possível também alterar a data do lote, basta selecionar  campo "Altera Data do Lote?" e inserir a nova data.
