TITULO: Inclusão de Pedido de Bombeio Web
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926194290
--------------------------------------------------
Para incluir um Pedido de Bombeio via SispetroWeb clique no Menu localizado no lado esquerdo/superior da tela
Selecione a opção
 Base / Incluir Pedido de Bombeio:
Selecione a programação de bombeio e o sistema apresentará o 
Produto
 e a 
data do pedido
.
Defina a quantidade, se houver necessidade, inclua uma observação.
Para confirmar, selecione a opção 
Salva
, ou cancele para retornar para a 
Manutenção de Bombeios
.
Para mais informações, ver 
Consulta de bombeios abertos
.
