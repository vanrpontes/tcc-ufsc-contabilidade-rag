--------------------------------------------------
O acesso a essa tela se dá através de 
Ferramentas / Usuário / painel Perfil / painel inferior Fiscal
Abaixo detalharemos os perfis e suas funcionalidades:
Manutenção de Apuração Fiscal
Pode incluir:
 permite que o usuário inclua um período para a apuração fiscal.
Pode encerrar
: permite que o usuário possa encerrar a apuração fiscal de um período.
Pode cancelar
: permite que o usuário possa cancelar um período de apuração fiscal.
Livro Fiscal
Pode incluir
: permite que o usuário inclua registros nos livros fiscais
Pode alterar
: permite que o usuário altere registros nos livros fiscais.
Pode excluir/cancelar
: permite que o usuário exclua ou cancele registros nos livros fiscais.
Pode cancelar/excluir
 diretamente no Livro Fiscal: permite que o usuário cancele ou exclua registros diretamente no Livro Fiscal.
Anexo Distribuidora
Pode editar layout do Anexo
: permite que o usuário altere o layout do Anexo.
Pode imprimir remotamente os anexos
: permite que o usuário imprima os anexos.
Anexo Refinaria
Pode incluir/alterar anexo
: permite que o usuário inclua ou altere um anexo da Refinaria
Pode cancelar anexo
: permite que o usuário cancele um anexo da Refinaria.
REINF
Pode dar manutenção de dados
: permite que o usuário possa fazer a inclusão dos dados a serem enviados.
Geração/Transmissão de dados
: define o tipo de acesso do usuário com relação a geração do arquivo xml e da transmissão dos dados
Não Gera/Transmite Dados
: usuário não poderá gerar xml nem transmitir os dados.
Só Gera xml
: usuário só poderá gerar xml dos dados.
Gera e Transmite Dados
: usuário poderá gerar xml e transmitir os dados gerados.
Pode fazer inventário
Permite que o usuário faça inventário.
PIS/COFINS
Retenções / Faz manutenção nos dados
Não faz: 
não tem permissão de fazer retenção.
Somente sem Apuração: 
caso tenha apuração em andamento, o sistema barra a inclusão com está opção selecionada.
Com Apuração em Andamento:
 permite lançamento de retenção mesmo com apuração em andamento.
Apuração
Pode Incluir/Alterar/Imprimir:
 terá acesso para inclusão, alteração e impressão do relatório de apuração em aberto.
Pode Encerrar/Reabrir:
 encerrar uma Apuração em Andamento. Reabrir uma apuração que já está encerrada.
Importante: 
para reabrir uma apuração de um mês anterior, você precisará cancelar todas as apurações posteriores a esse mês.
Pode Cancelar: 
terá permissão para cancelar uma apuração.
Altera Crédito/Contribuições. (Resumo e Detalhe)
Não altera: 
não faz alterações.
Somente na Primeira Apuração: 
permite somente alteração na primeira Apuração
Sempre (Apuração não será recalculada): 
permite alterações.
