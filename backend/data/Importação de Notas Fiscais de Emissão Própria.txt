URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/4082597889
--------------------------------------------------
Objetivo
Permitir importar 
notas fiscais
 de emissão própria emitidas em outro sistema ou casos em que a 
empresa tem um banco de dados para operação como Distribuidora de Combustíveis, mas tem um segundo banco de dados onde ela é a síndica de uma base armazenadora, emitindo Notas de Retorno de Armazenagem com o mesmo CNPJ e IE.
Esse processo foi criado para ser utilizado na importação via Serviço. 
Para mais informações, ver 
Monitoramento da Importação de NF-e/CT-e

A partir da versão 10.57 
foi ajustada a validação do CNPJ da Chave de Acesso durante a emissão e importação de Nota Fiscal, passando a contemplar a validação para casos de Nota Fiscal própria.
Recomendamos criar uma Natureza de operação e um De-Para apenas para importação de notas próprias
Esse processo foi criado para notas fiscais. Caso o CT-e seja incluído nesse processo o SisPetro mostrará a mensagem 
Não é permitido informar dados de CT-e de Terceiros para NF que não seja do tipo Entrada.
