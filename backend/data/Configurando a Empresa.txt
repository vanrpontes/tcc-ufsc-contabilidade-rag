TITULO: Configurando a Empresa
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210505
--------------------------------------------------
Será  necessário cadastrar um ou mais Grupos de Empresas que agrupará as empresas já  cadastradas.


A tela que permite a manipulação da tabela Grupo Empresa é  acionada a partir do Menu principal com as opções Tabelas  Auxiliares / Grupo Empresa.




Para  acrescentar um novo Grupo de Empresas à tabela, acione o botão de inclusão de  registro:




O Sistema exibe a tela para preenchimento dos dados.




Digite o conteúdo para os campos: Código e Descrição do Grupo de  Empresas.

Acione o botão OK para atualizar os  dados no Sistema.


O Sistema retorna a tela de navegação.


Para  retornar ao Menu principal acione o botão Sair:




Após cadastrar o Grupo de  Empresas, será necessário configurar para cada empresa o Grupo de Empresas ao  qual ela pertence. Isto será feito a partir do Menu principal com as opções - Ferramentas  / Sistema.




1.  Informe no campo Grupo Empresa que se encontra logo abaixo do Telefone o Grupo  Empresa correspondente à empresa em que você está conectado.


Será  necessário também habilitar a opção que permite a utilização do módulo contábil  no Sispetro. Vá para o painel "Contab. Real".




2. Marque a opção  "Habilita Contabilização Real" e pressione o botão OK para confirmar.


Conecte-se em todas as empresas cadastradas e repita os processos 1 e 2.
