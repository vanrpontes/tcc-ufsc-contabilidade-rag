TITULO: Painel Carregamento (Usuários)

O acesso a essa tela se dá através do Menu 
Ferramentas / Usuário / aba Perfil / aba inferior Carregamento
Conhecendo os perfis existentes:
Quadro OC
Pode incluir carregto na fila manualmente: 
permite que o usuário inclua uma OC de carregamento na fila manualmente.
Pode cancelar inclusão na fila carregto: 
pode cancelar a fila da OC de carregamento.
Pode alterar Informações Adicionais após emissão da OC
: permite ao usuário alterar as informações adicionais da ordem de carregamento após ela ter sido emitida.
Pode informar o frete na inclusão da OC (qualquer tipo): 
permite que o usuário informe o frete na inclusão de uma OC.
Pode pegar fila manualmente p/ OC Transf.Depósitos: 
em casos de OC de transferência entre depósitos, permite que o usuário pegue fila manualmente para a OC.
Pode alterar produtos na compartimentação da OC: 
permite que o usuário altere os produtos na compartimentação da OC.
Pode alterar o vínculo NF / OC: 
permite que o usuário altere o vínculo de uma OC com uma Nota Fiscal. Para mais informações, ver 
Verificando o vínculo entre Ordens de Carregamento e Notas Fiscais
.
Pode alterar OC (Transf. Depósitos ou Própria): 
permite que o usuário altere uma OC, desde que ela seja de Transferência entre depósitos ou que seja de Carregamento ou Descarga Próprios.
Pode imprimir Boletim de Conformidade pela OC: 
permite que o usuário imprima o boletim de conformidade do tanque.
Pode alterar Qtde da OC de Venda
: Permite que usuário altere a quantidade da OC de venda, desde que não esteja faturada.
Pode alterar Qtde da OC de Carregamento/Descarga Própria
: Permite ao usuário alterar a quantidade da OC de Carregamento e Descarga Própria, sem que esteja já tenha sido faturada. Para mais informações Ver 
Alterar quantidade de Ordens de Carregamentos.
 
Informa unidade de medida do produto na inclusão da OC: 
Permitirá ao usuário informar o Fator de Conversão Unidade na inclusão da OC, habilitando os campos no quadro de Itens da OC.
Pode indicar posição da OC: 
permite que o usuário defina a posição de uma OC, seja ela dentro ou fora da base (se está em trânsito, na Portaria, no Faturamento...). Para mais informações, ver 
Indicação da Posição da OC
.
Pode incluir OC: 
permite ao usuário incluir OC.
Pode cancelar OC: 
permite que o usuário cancele uma Ordem de Carregamento.
Pode Confirmar OC: 
permite realizar a confirmação de ordens de carregamento. Para mais informações, ver 
Confirmação de Ordem de Carregamento
.
Pode Cancelar Confirmação da OC: 
permite realizar o cancelamento da confirmação de ordens de carregamento. Para mais informações, ver 
Confirmação de Ordem de Carregamento
.
Pode alterar Série e Controle de Lacres/Envelopes: 
permite ao usuário incluir, excluir e alterar série e controle de lacres e envelopes através da tela de Manutenção de Série de Lacres.
Pode alterar Série e Controle de Lacres/ Envelopes na OC: 
permite ao usuário alterar série e controle de lacres e envelopes em OC já emitidas através da tela de Manutenção de Ordem de Carregamento, botão Compartimentação.
Permite editar a capacidade na OC: 
permite ao usuário alterar a capacidade do frota na inclusão da OC.
Pode emitir Ordem Carregamento/Descarga própria
: permite ao usuário emitir ordens de carregamento e descarga própria
Pode alterar Transportadora Auxiliar na OC: 
permite que o usuário altere a transportadora auxiliar na OC
Pode mudar data OC: 
permite que o usuário mude a data de uma Ordem de Carregamento.
Pode alterar histórico na inclusão da OC: 
permite que o usuário altere o histórico na Inclusão da Ordem de Carregamento.
Altera Transportad. OC após emissão: 
permite que o usuário altere a Transportadora na Ordem de Carregamento após a emissão.
Direciona impressão OC?: 
questiona se a Ordem de Carregamento vai ser impressa após a emissão.
Pode dar manutenção nas Entregas da OC?: 
permite ao usuário incluir ou alterar os registros da Aba Entregas da Manutenção de Ordem de Carregamento. Para mais informações, ver 
Controle de Entrega e Viagens
Quadro Tanques
Permite incluir leituras de tanque:
 marcando essa opção permite ao usuário realizar a leitura de tanque de forma manual. Para mais informações, ver 
Como efetuar leitura do tanque
.
Permite incluir leituras de Tanque via arquivo:
 marcando essa opção permite ao usuário incluir leituras de tanque via arquivo. Para mais informações, ver 
Leitura Automática DPM
.
Permite alterar estoque tanque via leitura:
 permite que seja alterado o estoque quando for realizada a leitura do tanque.
Permite alterar dados das réguas tanque: 
quando marcada, permite que o usuário altere os dados da régua de leitura do tanque.
Permite incluir tanque:
 quando marcada, permite que o usuário. realize a inclusão de 
Tanques
.
Permite alterar tanque:
 permite que o usuário altere os dados dos 
Tanques
.
Permite excluir tanque: 
permite que o usuário realize exclusão de 
Tanques
.
Permite ativar tanque: 
permite que o usuário ative ou inative um tanque.
Permite incluir fator conversão para combustível:  
permite que seja incluído o fator de conversão.
Permite alterar código do produto do tanque: 
permite que o usuário troque o produto do tanque.
Pode dar manutenção em Boletim de Conformidade: 
permite que o usuário possa incluir e excluir os boletins de conformidade do tanque.
Quadro Portaria
Pode incluir carregamento manual: 
permite que o usuário faça a portaria de entrada e de saída, informando manualmente os dados de carregamento ou descarga.
Pode dar entrada/saída portaria c/ data diferente: 
permite que o usuário faça a portaria de entrada e de saída com data diferente da data da OC.
Pode fazer portaria de Entrada sem biometria do motorista: 
permite que o usuário realize a portaria de entrada sem a biometria do motorista.
Quadro Pesagem
Pode Alterar Peso de Descargas: 
permite que o usuário altere o peso das OCs de descarga.
Pode Alterar Peso de Carregamentos: 
permite que o usuário altere o peso das OCs de carregamento.
Pode alterar a balança na Portaria de Entrada: 
permite que o usuário altere a balança de pesagem definida como padrão para o deposito . Para mais informações, ver 
Gerenciamento de Balanças
.
Quadro Sorteio Inspeção
Pode visualizar dados de sorteio: 
permite usuário a visualizar dados de sorteio, assim como as abas e colunas relacionadas a inspeção.
Pode editar dados de inspeção:
 permite ao usuário incluir dados de inspeção de uma OC (frota) selecionada ou não para sorteio. Para mais informações, ver 
Inspeção de Auto Tanque - Sorteio de Bola Preta
Pode cancelar dados de inspeção:
 permite ao usuário cancelar uma inspeção de uma OC (frota) selecionada para sorteio. Para mais informações, ver 
Inspeção de Auto Tanque - Sorteio de Bola Preta
Permite Edição dos Lacres/ Envelopes?: 
Não Permite:
 não permite editar lacres/ envelopes em OC emitidas
Permite com OC emitida sem NF:
 permite editar lacres/ envelopes somente em OC sem notas fiscais emitidas
Permite com OC emitida mesmo com NF (não afeta as NF's emitidas):
 permite editar lacres/ envelopes em OC mesmo com notas fiscais emitidas. As notas fiscais não são afetadas, ou seja, os números dos lacres nas notas continuarão originais.
Para mais informações, ver 
Controle de Lacres
.
Pode marcar/desmarcar OC para Serviço de Exportação
Não pode: 
não pode marcar ou desmarcar.
Somente marcar: 
permite marcar a OC.
Somente desmarcar: 
permite somente desmarca a OC.
Marcar e desmarcar: 
permite marcar e desmarca a OC.
Quadro Impressão OC x Faturamento
Permite ao usuário controlar a impressão de OC de 
Vendas, Transferência entre Depósito, Carregamento Próprio e Descarga Própria
. Podendo definir se a impressão da OC será 
conforme o perfil de Depósito
 (para mais informações, ver 
Depósitos - Aba Layout OC
), se será impressa 
Somente se estiver faturada
 ou se poderá imprimir inclusive as que 
Não estão faturadas
.
Quadro Manutenção Carregamentos/Descargas
Pode cancelar carregamento manual: 
permite que o usuário cancele um carregamento manualmente.
Pode mudar data entrada/saída base na Man. Carregto.
: permite ao usuário alterar a data de entrada ou saída da base através da tela de Manutenção de Carregamento/Descarga.
Pode mudar data Carregamento Real: 
permite que o usuário altere a data do carregamento.
Pode alterar dados descarga
: permite ao usuário alterar, dando duplo clique, na tela de fechamento de estoques, aba Detalhes, os dados de temperatura e densidade do lançamento selecionado.
Quadro Limitação no Carregamento/Descarga
Verificar descarga real fora do limite permitido: 
quando marcado, checa se o volume da descarga a ser realizada está dentro do limite permitido.
% de tolerância de descarga real em relação ao previsto: 
percentual de diferença tolerável, considerando o volume da OC (previsto) com o volume real.
Verificar carregamento real fora do limite permitido: 
quando marcado, checa se o volume do carregamento a ser realizada está dentro do limite permitido.
% de tolerância de carregamento real em relação ao previsto: 
percentual de diferença tolerável, considerando o volume da OC (previsto) com o volume real.
Libera quantidade/peso fora do limite permitido: 
permite que seja feito o carregamento ou a descarga com o volume fora do limite permitido.
Informação a ser verificada na descarga e no carregamento
: é possível indicar perfil diferente para Carregamento e Descarga. Isso pois a validação por Peso, por exemplo, não é útil no Carregamento pelo fato do caminhão ter vários produtos, mas pode ser interessante na Descarga. 
Somente Volume:
 verifica somente o volume
Somente Peso:
 verifica somente o peso
Volume e Peso:
 verificar o volume e o peso
Quadro Sistema de Automação
Faz Portaria Entrada/Saída:
 define se o usuário vai ter acesso à integração do Basecom com a Portaria de entrada/saída.
Quadro Smart Card
Pode gravar cartão:
 permissão para usuário gravar dados no cartão
Pode Cancelar gravação do cartão:
 permissão para o usuário cancelar a gravação do cartão.
Pode ler cartão gravado:
 permissão para usuário ler dados do cartão.
Mostra tela configuração smart card:
 permissão para o usuário visualizar
Volta para Portaria de entrada s/ Cartão:
 com este perfil marcado é possível voltar a portaria de entrada sem que seja necessária a leitura do cartão, este perfil foi criado afim de aumentar a segurança durante o processo de carga e descarga na base.
Quadro Transferência de Produtos entre Tanques
Transfere produtos de empresas diferentes: 
permite que o usuário transfira produtos entre empresas diferentes.
Transfere produtos com quantidades diferentes: 
permite que o usuário transfira os produtos com quantidades diferentes.
Transfere entre tanques de produtos diferentes: 
permite que o usuário transfira produtos diferentes entre tanques.
Quadro Rateio de Estoque
Pode incluir rateio: 
permite que o usuário inclua um rateio de estoque. 
Pode cancelar rateio: 
permite que o usuário cancele o rateio de estoque feito anteriormente.
Pode imprimir rateio: 
permite que o usuário imprima o rateio de estoque realizado.
Pode alterar custo médio: 
permite ao usuário, na inclusão de rateio, alterar o custo médio do produto para aquela operação. O custo apresentado é utilizado, por exemplo, no pedido de OE quando há ganho no rateio do estoque. 
Para mais informações, ver 
Rateio de Estoque
 e 
Custo de Produtos (CMV)
Quadro SisPetro App
Pode incluir Ordem Carregamento: 
define se o usuário poderá incluir ou reagendar ordem de carregamento ou descarga de terceiros via SisPetro App
Pode emitir relatórios de estoque: 
define se o usuário poderá emitir os relatórios de movimentação de estoque (resumido e detalhado) no SisPetro App.
Pode alterar informação sobre Coleta Amostra-Testemunha:
 com este perfil marcado, o usuário poderá definir se a OC precisará ou não de coleta de Amostra-Testemunha.
Pode alterar tipo de compartimentação na descarga:
 define se o usuário pode ou não desmarcar a opção de compartimentação automática na descarga de terceiros.
Pode alterar tipo de compartimentação no carregamento:
 define se o usuário pode ou não desmarcar a opção de compartimentação automática no carregamento de terceiros.
Pode visualizar fila de entrada na base
: define se o usuário pode ou não visualizar a fila de carregamento e descarga na portaria de entrada da base
Permite acessar procedimentos de base
: define se o usuário pode ou não visualizar o menu Base no Sispetro App
Pode Reagendar OC?
: possibilita o reagendamento de OC, via SispetoApp.
Quadro API Via App
Access Key: 
chave de acesso para autenticação do usuário na nuvem, em caso de utilização de API para inclusão automática de OC de carregamento e descarga de terceiros via SisPetro App.
Gera Nova Access Key: 
gera nova chave de acesso para autenticação do usuário.
Secret Key: 
chave de secreta de acesso para autenticação do usuário na nuvem, em caso de utilização de API para inclusão automática de OC de carregamento e descarga de terceiros via SisPetro App.
Gera Nova Secret Key: 
gera nova chave secreta de acesso para autenticação do usuário.
Habilitar API para inclusão OC neste usuário: 
habilita API de inclusão automática de OC de carregamento e descarga de terceiros via SisPetro App.

Quadro Alterações na Portaria de Saída
Pode alterar a quantidade da descarga: 
permite que o usuário altere o volume na portaria de saída, para OC de descarga.
Pode alterar a densidade e temperatura da descarga: 
permite que o usuário altere a temperatura e a densidade na portaria de saída, para OC de descarga.
Pode alterar a quantidade do carregamento: 
permite que o usuário altere o volume na portaria de saída, para OC de carregamento.
Pode alterar a densidade e temperatura do carregamento: 
permite que o usuário altere a temperatura e a densidade na portaria de saída, para OC de carregamento.
Pode voltar da Portaria de Saída para Portaria de Entrada?
: permite que o usuário que estiver com o perfil habilitado volte uma OC da portaria de saída para a portaria de entrada.
