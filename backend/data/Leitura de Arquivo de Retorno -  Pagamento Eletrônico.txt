TITULO: Leitura de Arquivo de Retorno -  Pagamento Eletrônico
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878899311
--------------------------------------------------
A leitura do arquivo de retorno, enviado pelo banco deverá ser feito acessando a tela de Leitura de Arquivo de Retorno - Pagamento Eletrônico, localizada no Menu 
Movimentação / Financeiro / Leitura Arquivos Pagto. Eletrônico / Leitura Arquivo Retorno
.
O usuário deve informar a Conta Contábil do Banco.
O próximo passo será informar ao sistema o local onde se encontra o arquivo que será processado, ou seja, feito a leitura, através do campo 
Nome do arquivo
.
O usuário pode digitar diretamente o local onde se encontra o arquivo:
Caso não saiba o caminho, o usuário poderá acessar a tela de pesquisa de arquivos, informando assim o local do arquivo para o sistema. Para acessar essa tela, o usuário deve clicar no botão 
...
, com o campo 
Nome do arquivo
 não preenchido.
Após preencher corretamente o campo 
Nome Arquivo
, o usuário deverá clicar no botão 
Lê Arquivo
, para iniciar o processo de leitura e processamento do arquivo.
No caso de encontrar erros na leitura não será permitido fazer o processamento do arquivo, os erros estarão disponíveis no painel 
Erros na Leitura
 e o sistema dará uma mensagem de aviso.
Se o processo de leitura ocorreu com sucesso o botão 
Processa
 ficará habilitado para o caso de se desejar fazer o 
processamento
 (baixa) na seqüência. Se este não for o caso é possível fazer o processamento posteriormente pela tela de Manutenção de Arquivos do Pagamento Eletrônico.
Títulos gerados por PMMs de 
 não poderão ser 
Processados
 via Pagamento de Fornecedores (PagFor).
