TITULO: Tabela Irregularidade em NF
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210587
--------------------------------------------------
Nesta tabela são cadastradas as irregularidades possíveis de ocorrerem para a emissão da Carta de Correção de uma Nota Fiscal.


A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - Tabelas Auxiliares / Faturamento / Irregularidade em NF.




O sistema já pré-cadastra 42 tipos de irregularidades. No entanto, é possível alterá-las.


Para acrescentar uma nova irregularidade à tabela, acione o botão de inclusão de registro:




O Sistema exibe a tela para preenchimento dos novos dados.




Informe a descrição desejada.


Acione o botão OK para atualizar os dados no Sistema.


O Sistema retorna a tela de navegação.

Para retornar ao Menu principal acione o botão Sair:
