Cenário 5: 
Para empresas que trabalham com representantes, onde cada representante fecha contrato com a Usina, passa as informações para a Distribuidora, que providencia a inclusão do PMM.
Na sequência, é feita a venda dos produtos por esses Representantes, utilizando seus respectivos estoques. Para identificação das vendas deve ser utilizada a figura do Comprador, ou seja, cada Representante se torna um comprador e as vendas são vinculadas, através do Comprador, aos seus Representantes. No caso do PMM o representante será informado na tela de inclusão.
Para utilizar esse recurso é necessário seguir os passos abaixo:
Configurações de empresa
 - Perfil de empresa, denominado "Informação de Representante no PMM" com 3 opções:
Não informa Representante (padrão)
 - caso está opção seja selecionada o usuário inclui o PMM normalmente sem informar o representante;
Permite informar mas não obriga
 - caso está opção seja selecionada o usuário tem a opção de informar ou não o Representante;
Obriga informar Representante
 - caso está opção seja selecionada o usuário é obrigado a informar um representante, se não for informado o PMM não será incluído.
