Objetivo
Permite visualizar e incluir contabilizações gerenciais no SisPetro. O acesso é feito através do Menu 
Movimentação / Controladoria / Manutenção de Lotes
Nesta tela podem ser visualizados todos os lotes contábeis. Um lote representa dois ou mais lançamentos de débitos, ou créditos em determinadas contas.
Os lotes são gerados de forma automática por diversos procedimentos do SisPetro, entre eles, Nota Fiscal, inclusão de Contas Pagar/Receber, Baixas do Contas a Pagar/Receber, lançamentos feitos pela tela de Manutenção de Bancos, etc. Em cada tela de manutenção específica tem o número do lote gerado para o processo, este número de lote poderá ser consultado nesta tela. A numeração de lote é por empresa e na tela só são apresentados lotes da empresa na qual o usuário está conectado.
Além disso, é possível incluir lotes manuais, para movimentação de contas fora de procedimentos do SisPetro.

Detalhes
Informe os dados do cabeçalho do lote, estes dados serão os mesmos para todas as contas movimentadas neste lote:
Data:

Travamento Contábil

Cód. Histórico:

#Centro de Custo: 
centro de custo, informar em apenas um dos itens.
Histórico:
 caso tenha sido informado ou selecionado um código de histórico, o histórico já é preenchido automaticamente, permitindo incluir mais informações ou mesmo ser alterado totalmente.
Informações Adicionais:

Moeda:
 geralmente, informe a moeda REAL.
#Docto:

Sugere valor da diferença entre itens?:
 deixe esta opção marcada para que a cada conta informada o SisPetro sugere a diferença necessária para compor os valores, já que os valores de crédito e débito devem ser iguais.
Restrito:
 obsoleto, não marcar.
Copia Histórico para os Itens:
 esta opção faz com que, no momento de uma inclusão de lote manual, o SisPetro copie o histórico lançado no cabeçalho do lote para todos os itens, ou seja, para as contas de débito e crédito que serão movimentadas no lote.
Além de copiar, o usuário pode alterar ou cadastrar um novo histórico e uma nova informação adicional para cada item do lote, porém, isso só será possível se o usuário tiver o perfil localizado nas configurações do usuário marcado. Este perfil pode ser acessado pelo
