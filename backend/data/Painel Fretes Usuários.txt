Quadro Manutenção de Transportadora
Permissão no Cadastro de Transportadora/Motorista: 
esse perfil define se o usuário vai poder cadastrar somente transportadoras, somente motorista ou os dois.
Ambos:
 permite cadastrar e editar tanto motorista quanto transportadoras.
Só Transportadora:
 permite ao usuário cadastrar, editar e excluir somente transportadoras.
Só Motorista:
 permite ao usuário cadastrar, editar e excluir somente Motoristas.
Pode incluir e alterar Transportadora: 
este perfil é utilizado para controlar o usuário que pode incluir e alterar o cadastro das transportadoras, tanto para o SisPetro Normal quanto para o SisPetroApp.
Pode excluir Transportadora: 
este perfil permite a exlcusão de uam transportadora. Só será possível a exclusão caso a mesma não possua Notas Fiscais ou Ordens de Carregamento atreladas.

Quadro Frete
Alteração de Dados na Manutenção de Fretes: 
esse perfil dá acesso a 3 opções de alteração
Não altera: não pode ser nada alterado
Altera: Pode alterar os campos como observação, valor e data
Altera somente observação: Apenas a observação é alterada
Permite cancelar Fretes: 
esse perfil permite ou não ao usuário cancelar fretes na Manutenção de Fretes. 
Permite cancelar lançamento frete no financeiro: 
caso o frete tenha sido vinculado a algum título, no financeiro, este perfil permite ou não o usuário de cancelar este vínculo. 
Pode Alterar Status do Frete: 
esse perfil permite ao usuário a permissão de alterar o status das apurações de fretes. 
Pode apurar Frete: 
esse perfil dá ao usuário a permissão de apurar as diferenças de descargas na manutenção de frete. 
Permite incluir frete avulso: 
caso seja habilitado, permite ao usuário incluir frete avulso na Manutenção de Fretes. 
Permite lançar frete no módulo financeiro: 
dá ao usuário a permissão de lançar ou vincular títulos ao fretes já gerados. 
Permite cancelar o vínculo de frete e contas a pagar:
Pode Alterar Frete já apurado: 
depois de apurada a diferença de descargas, com este perfil marcado possibilita ao usuário alterar tal frete. Pode ser alterado valor do frete líquido, data e observação. 
Pode Inativar Frete: 
esse perfil permite ao usuário inativar fretes que estejam pendentes na tela de Manutenção de Frete.
Para mais informações, ver 
Manutenção de Frete
