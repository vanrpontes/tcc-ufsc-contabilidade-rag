TITULO: Tabela de Seguros
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2891088337
--------------------------------------------------
A tabela de seguros permitirá a empresa cadastrar todos os tipos de seguro existentes dentro da mesma.
Para utilizar a função de seguro é necessário habilitar o perfil de Sistema em
 Ferramentas / Sistema / Aba Vendas / Quadro Habilita Seguros ?  
Painel Vendas
E alterar o cliente em 
Arquivos / Cliente / Aba superior Detalhes / Aba Inferior Padrão p/ Vendas / Quadro Seguros. E selecionar dentre as três opções. 
Padrão para Vendas
Este painel permite a inclusão desta tabela é aciona a partir do Menu 
Tabelas Auxiliares / Vendas / Seguros:
Para acrescentar um novo Seguro à tabela, acione o botão de inclusão de registro 
.
O Sispetro exibe a tela para preenchimento dos dados.
Digite o conteúdo para os campos:
Descrição: 
informe uma descrição para o seguro.
Tipo Cálculo: 
informe o tipo de cálculo a ser utilizado:
Alíquota = 
irá considerar o percentual do valor base. 
Valor Fixo = 
irá considerar um valor fixo para o seguro.
Seguro Ativo: 
marcando o seguro estará ativo, quando desmarcado ficará inativo.
Acione o botão OK para atualizar os dados no Sispetro.
