TITULO: Assistente de Venda
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2892890285
--------------------------------------------------
No Cadastro de Clientes, existe um campos chamado 
Assistente Vendedor
. Este campo será lançado o vendedor que dentro da empresa atua no cadastro do pedido de venda. Este assistente é um vendedor que esta atrelado ao vendedor responsável pelo cliente.


Este campo só aparecerá quando no perfil 
Controla Assistente de Venda
 menu 
Ferramentas / Sistema / Venda
 estiver habilitado.




Este assistente poderá também ser utilizado como Representante de venda, que são os atravessadores que acabam participando da venda ou mesmo arrumando os compradores. E isso poderá acontecer de duas formas.




Ele trabalha para mais de um vendedor


E não substituirá o vendedor já que ambos serão comissionados.




Para isso teremos mais um  perfil de sistema em combinação com o do já existente 
"Controla Assistente de Venda"
 no menu 
Ferramentas / Sistema / Vendas
, que é o 
"Vinculo do Assistente ao Vendedor"
 com duas opções:




Vinculado a um único vendedor :
 Que é o padrão de vinculo único, trabalhando como o assistente vinculado a um vendedor.


Sem vendedor pré-definido:
 Poderá escolher qualquer vendedor para ser o representante ou assistente






Com o perfil habilitado, como "Sem vendedor pré-definido", permitirá a escolha de qualquer vendedor cadastrado. 




Mesmo com o assistente pré-definido no cadastro do cliente também poderá alterado-lo no momento da inclusão do pedido. 




No Contas a Receber

Para facilitar a identificação de qual assistente de venda que foi utilizado no contas a receber no grid é apresentado os campos “#Vendedor Assistente” e “Nome do Vendedor Assistente”.
