TITULO: Tipos de pagamento por Banco
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2956066880
--------------------------------------------------
A partir da versão 10.16 foi criada uma lista para auxiliar o usuário a cadastrar os tipos de pagamento habilitados para 
pagamento eletrônico
. 
Essa lista pode ser visualizada na inclusão ou na alteração da forma de pagamento. Após acessar 
 - Tabelas Auxiliares / Financeiro / Forma Pagamento 
guia 
Detalhes, 
informar qual banco que deseja configurar para realizar um tipo de pagamento eletrônico, no campo 
Classificação de Pagamento
 o sistema irá mostrar os tipos de pagamento que estão disponíveis juntamente com seu 
Código de Banco 
e 
Código Banco Adicional. 
No exemplo abaixo perceba que o banco 
033 - Santander
 possui diversos tipos diferentes de pagamento eletrônico.
Porém se o banco selecionado não possuir tipo de pagamento eletrônico a lista de 
Classificação de Pagamento 
permanecerá vazia.
Abaixo a lista dos bancos que temos disponíveis atualmente
