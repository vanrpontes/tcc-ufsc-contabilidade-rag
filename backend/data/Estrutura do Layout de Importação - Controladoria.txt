--------------------------------------------------
Este modelo foi projetado inteiramente pela Futura Tecnologia com a finalidade de importar os saldos de contas.
Layout do Modelo - Futura
Campo
Descrição
Conta
Conta Gerencial (código do campo Código Exportação da tabela de Plano de Contas)
Data
Data do saldo final informado dd/mm/yyyy
Valor
Valor do saldo final na data informada (utilizar vírgula para separar decimais, com duas casas decimais)
Para elaboração desta estrutura todos os campos devem ser separados com 
;
 e a primeira linha deve conter o cabeçalho.
Exemplo:
Cod.Exportação;Data;Valor
 123456;29/07/2011;1200,00
