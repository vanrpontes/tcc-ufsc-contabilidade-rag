TITULO: Acesso Menus
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2924288397
--------------------------------------------------
O SisPetro possibilita que o administrador do sistema possa restringir a visualização e acesso a todas as telas. Veja abaixo como fazer.
É preciso localizar a tela a partir do menu acima. Veja que são todas as telas do SisPetro, com todos os menus.
Depois de localizada a tela, é possível escolher entre:
Visível:
 Caso seja marcado a tela ficará visível para o usuário, porém não poderá acessa-la.
Habilitado:
 Caso seja marcada essa opção a tela poderá ser acessada.
Então note que para o usuário ter acesso a tela é preciso marcar as duas opções.
