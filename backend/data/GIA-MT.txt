Para gerar a mídia GIA-MT, o usuário deve acessar o menu
 
Ferramentas / Geração Mídia Gia/Gim
.
Através do painel
 
GIA-MT
, o usuário poderá gerar um arquivo contendo as informações necessárias para serem validadas pelo programa da Gia-MT.
Geração do Arquivo:
De: 
informe a data inicial e a data final 
Tipo:
 
Normal ou Substitutiva
Motivo:
 
Normal, Baixa, Paralisação Temporária, Reativação, Mudança de Domicilio Fiscal ou Revisão de Estimativa
Tipo Contabilista:
 
Para Contabilista, Escritório ou Individual
Periodicidade:
 
Mensal, Anual ou Semestral
Gera dados para o Anexo III (GIA-ICMS): 
caso marcado
, o SisPetro incluirá no arquivo um grupo para o anexo III. O ideal nesse caso, é optar por gerar dados do anexo III em dezembro, já que a periodicidade dele é anual. 
Movimentação Interestadual no Exercício: 
caso marcado, será
 gerada a movimentação interestadual do período selecionado para a geração do arquivo.
Dados do Arquivo:
Layout de Exportação:
 selecione o layout a ser utilizado para gerar o arquivo. Para mais informações, ver 
Manutenção Exportação de Dados
Nome do arquivo:
 informe a pasta onde o arquivo será gravado e o nome do arquivo
Clique no botão 
OK 
para iniciar a geração do arquivo ou no botão 
Cancela 
para cancelar.
Algumas considerações devem ser observadas para que o arquivo seja validado com sucesso:
Cadastrar o código de todos os municípios de Mato Grosso na 
 (Sispetro Normal), campo
 
Código do Município (uso da UF)
Nas configurações da empresa, 
, preencher os campos 
Data Cadastro Junta Comercial
 e 
Data Cadastro Secretaria Fazenda
Configurar devidamente todas as naturezas de operação, no SisPetro Normal em 
) que devem sair no Anexo 1 e 2, através do quadro GIA-MT, preenchendo os campos Grupo de Operações e COP
Conferir o código para GNRE na 
 em SisPetro Normal, campo 
Código Estadual
, pois será utilizado no Anexo III (GIA-ICMS).
O programa da Gia-MT é muito criterioso, portanto, é importante que os dados do livro fiscal (CFOP, Valor Contábil, UF, E/S) não contenham inconsistências, evitando assim problemas na validação do arquivo gerado.
