TITULO: Temperatura - Densidade
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210992
--------------------------------------------------
Objetivo
Através deste painel, o usuário poderá informar valores de temperatura e densidade que serão gravados para uso na 
Impressão da Nota Fiscal
.
O acesso é através do menu 
Arquivos / Produtos / Aba Detalhes / Aba inferior Temperatura/Densidade
Para uma melhor compreensão do assunto abordado nesse tópico, é aconselhado ao usuário ler o tópico
 
Temperaturas e densidade na Impressão da Nota Fiscal
.
Conforme configuração da empresa (detalhada no tópico 
Temperaturas e densidade na Impressão da Nota Fiscal
), os valores de temperatura, densidade e número do laudo poderão ser informados por 
Empresa
 ou 
Depósito
. O SisPetro verifica a o perfil de empresa e, conforme essa configuração, habilita apenas 
um dos painéis
 para que as informações sejam cadastradas.
O usuário precisa ter permissão para alterar os dados de temperatura e densidade. Para isso é necessário configurar através do 
Menu Ferramentas / Usuário / Produto, o perfil:
Também é possível acessar o painel para informar 
Mudança de Temperatura e Densidade
, por meio do Menu 
Movimentação / Faturamento / Muda Temperatura e Densidade.
Abas Laterais
Temperatura por Empresa:
 para informar os dados de temperatura e densidade 
por empresa
, o usuário deve utilizar o botão 
Edita
.
O SisPetro abrirá a tela abaixo. O usuário deverá informar:
Temp. Tanque
Temp. Amostra
Densidade
Núm. Laudo
O campo 
Número Laudo
 pode ser utilizado para informar o número do documento do boletim de conformidade.
Esta tela permite também ao usuário consultar a tela de Fator de Conversão, através do botão 
Fator de Conversão.
Após digitar os dados, o usuário deve utilizar o botão 
Gravar
 para salvar os dados, ou utilizar o botão 
Fechar
 para cancelar o processo. Lembrando que, no momento da inclusão do registro, o SisPetro pode ser configurado para validar ou não os dados cadastrados de temperatura e densidade.
Para mais informações, ver  
Temperaturas e densidade na Impressão da Nota Fiscal
.
Caso a empresa não esteja configurada para trabalhar com dados de temperatura e densidade por empresa, o botão 
Edita, não estará habilitado
. 
Temperatura por Depósito:
 neste painel, o usuário definirá as informações de temperatura e densidade por depósito.
Através do botão 
Edita
 ,o SisPetro abrirá a tela abaixo. O usuário deverá informar
Temp. Tanque
Temp. Amostra
Densidade
Núm. Laudo
Além dos dados acima, caso o usuário trabalhar com 
Venda Spot
, irá informar o campo 
Encerrante. 
Após digitar os dados, o usuário deve utilizar o botão 
Gravar
 para salvar os dados, ou utilizar o botão
 Fechar 
para cancelar o processo. 
Botões
Incluir Depósitos:
 o usuário adicionará os depósitos ativos configurados no SisPetro. Esse procedimento deve ser realizado se novos depósitos forem criados posteriormente.
Copiar Temperatura de Empresa:
 copiará as informações de temperatura e densidade de uma determinada empresa (painel Temperatura por Empresa) para todos os depósitos ou para um depósito selecionado.
O usuário deve informar no campo 
Filial
, o código da empresa que servirá de cópia para as informações. Caso deseje que todos os depósitos tenham essa informação, marcar a opção 
Copia temperaturas/densidade para todos os depósitos
. 
Caso contrário, antes de entrar nessa tela, o usuário deve selecionar o depósito desejado. 
Para que as informações sejam copiadas, o usuário deve clicar no botão 
Copiar
 ou no botão 
Fechar
 para cancelar o processo.
Caso o produto não esteja configurado como 
Produtos Claros (Alcoois)
 ou 
Produtos escuros (Gasolina / Diesel) 
Aba Ficha Produto / campo 
Tipo de fator de Conversão,
 o SisPetro não permitirá a inclusão de temperatura e densidade, apresentando ao usuário a mensagem de erro dizendo que o produto não esta configurado para conversão.
