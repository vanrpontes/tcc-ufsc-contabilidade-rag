TITULO: Status do Cliente X Situação do cadastro Ativo - Inativo
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2888204562
--------------------------------------------------
Muitos clientes confundem o Status do Cliente (Ativo / Livre / Potencial) com o Situação do Cadastro do Cliente (Ativo / Inativo).




Na figura acima podemos visualizar uma situação onde, de acordo com a legenda, o cadastro do cliente está 
Inativo
 (legenda vermelha) porém, na coluna de Status, está marcado como 
Ativo
. Esse é o típico caso que gera diversas dúvidas, pois alguns pensam que o cliente está com o cadastro ativo.


Para entender melhor:

A coluna que mostra ATIVO na verdade refere-se ao Status do cliente, utilizado para alocação/controle de vendedores.

Os status podem ser:


LIVRE
: esse cliente pode ser utilizado por qualquer vendedor.


POTENCIAL
: esse cliente está em fase de "testes" com o vendedor para tornar-se um cliente.


ATIVO
: cliente de um vendedor específico.

Para maiores detalhes veja o tópico 
.


Já a situação cadastral (ativo / inativo) definimos se esse é um cliente que está ou não apto para efetuar pedidos de compras no SisPetro.


ATIVO
: Apto a realizar pedido


INATIVO
: Não Apto a realizar pedido

Para maiores detalhes veja o tópico 
.


Caso a coluna de status ainda continue gerando confusão, é possível arrastá-la para o final do grid dessa forma ela não ficará visível na tela. 

Obs.: A posição da coluna será salva apenas ao fechar a tela.
