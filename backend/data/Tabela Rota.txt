TITULO: Tabela Rota
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878439625
--------------------------------------------------
Nesta tabela são cadastrados as rotas utilizadas para auxílio na 
Composição de Cargas
.
A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Vendas / Rotas
:
Para acrescentar uma nova Rota acione o botão de inclusão de registro 
.
O Sistema exibe a tela para preenchimento dos novos dados.
Informe os dados necessários:
Descrição
.
Cidade de Origem:
 de onde vai partir a rota.
Itens da Rota:
 cidades por onde vai passar esta rota. Utilize o botão 
Inclusão
 e 
Exclusão
.
Preferencial:
 indica se esta rota é a preferencial, ou seja, a de melhor acesso.
Utilize o botão 
Atualiza Entidades
 para que o sistema inclua esta rota para clientes cujo Código DNC Região façam parte desta rota. Porém isso só ocorrerá se o cliente não estiver com esta opção 
Trava Rota
 marcada. Maiores esclarecimentos em Tabela Clientes - 
.
Acione o botão 
OK
 para atualizar os dados no Sistema.
O Sistema retorna a tela de navegação.
Para retornar ao Menu principal acione o botão 
Sair
 
.
