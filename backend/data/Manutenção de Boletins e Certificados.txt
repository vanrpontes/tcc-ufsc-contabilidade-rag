Movimentação / Base / Manutenção de Boletins e Certificados
Itens inativos estarão destacados em
 vermelho
Essa tela possui duas funcionalidades: 
Quem 
Produz 
combustível: a cada batelada (quantidade produzida) deve ser feito um Certificado de Produção.
Quem 
Distribui 
combustível: na emissão da Ordem de Carregamento, deve ser emitido um Boletim de Conformidade
Objetivo
Realizar a inclusão e manutenção dos boletins (tabela) e certificados para posteriormente 
fazer a impressão
.
Botões
Inclui: 
permite incluir um boletim ou certificado. Clicando nesse botão o SisPetro mostra a tela abaixo
para fazer a inclusão basta preencher as seguintes informações:
Tipo:
 selecionar o quê será cadastrado. Pode ser:
Boletim
Certificado
Data:
 informar a data da análise. O SisPetro traz como padrão a data atual
Laudo:
 informe código do laudo
Químico Responsável:
 informe o nome do químico que foi responsável pela análise
Observações:
 caso tenha alguma observação, preencha aqui
Depósito:
 selecione qual o depósito que o usuário irá utilizar
Produto: 
selecione qual o produto que foi analisado. 
Atualiza: 
após preencher as informações acima, ao clicar no botão atualiza, o SisPetro irá carregar os itens de análise previamente cadastrados em 
Depósitos - aba Boletim / Certificado
 
No quadro, Itens
 de Análise, 
preencher as colunas
Valor: 
informar o valor da análise. 
OK: 
caso o item analisado esteja em conformidade, marque o 
ok
Altera: 
abre a mesma tela do incluí, permitindo o usuário alterar as informações, desde que 
não 
haja vínculo
Cancela: 
cancela o boletim / certificado
Copia: 
permite copiar um boletim ou certificado para outro produto e / ou depósito. O SisPetro irá abrir a tela abaixo:
Para fazer a cópia, basta o destino das informações preenchendo: 
Depósito
Produto
Data
Imprime: 
imprime o Boletim o Certificado que o usuário estiver selecionado. 
A impressão utiliza o RTM informado no Depósito. Para mais informações, ver 
Depósitos - aba Boletim / Certificado
 
Legenda: 
mostra a legenda das cores utilizadas
Atualiza: 
atualiza as informações na tela
Sair: 
saí da tela de manutenção. 
Abas
Boletins e Certificados 
Nessa aba é possível visualizar todos os boletins / certificados cadastrados, incluindo o status, usuário que fez o cadastro e se ele está cancelado ou não. As informações são apresentadas são de acordo com 
Período para exibição de dados: 
filtro de data
Depósito: 
caso preenchido, filtra um depósito específico
Produto: 
caso preenchido, filtra um produto específico
Itens de Análise
Nessa aba é possível ver quais itens foram analisados para o Boletim ou Certificado em questão e se, no momento da inclusão, o item ficou com o 
ok 
marcado. 
Vínculos
Nessa aba é possível visualizar onde o certificado ou boletim está vinculado, lembrando que: 
Certificados são vinculados a 
Ordem de Fabricação
Boletins são vinculados ao 
Tanque
