TITULO: Manutenção de Exportação de Dados
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878899339
--------------------------------------------------
Nesta tabela são cadastradas os layouts de Exportação de Dados. Tais exportações podem ser utilizadas para sistemas externos, bem como no 
 e na Geração do SCANC.
A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - Ferramentas / Manutenção Exportação Dados.
Para acrescentar um novo layout acione o botão de inclusão de registro: 
 
O Sistema exibe a tela para preenchimento dos novos dados.
Informe a 
descrição
 deste layout. Caso já tenha um layout pronto (arquivo .rtm gerado pelo Gerenciador de Relatórios) utilize o botão 
Carrega de Arquivo
 para já armazenar este layout.
Clique em OK para confirmar os dados.
No caso de ter que projetar o layout, após a confirmação da inclusão, selecione o layout desejado, vá ao painel Detalhes e clique em 
Projeta Layout
. Será apresentada a tela do Gerenciador de Relatórios (vide maiores explicações no manual específico). Ao terminar, salve o relatório quando for questionado.
Mude o tipo de arquivo a ser gerado conforme a necessidade e clique em 
Ok
 para confirmar.
Para imprimir um determinado layout é possível utilizar o botão 
Imprime Layout.
 Para facilitar a manipulação de usuários que só terão acesso a impressão de layouts, no menu Ferramentas há o item de menu Exportação de dados que apresenta a tela a seguir onde só pode ser selecionado um layout já existente e providenciada a sua impressão.
