TITULO: Como carregar arquivo na Manutenção Exportação de Dados
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2910486656
--------------------------------------------------
Vá no menu Ferramentas / Exportação de Dados / Manutenção Exportação de Dados
Na parte superior da tela clique no botão 
Ao clicar sobre o botão, automaticamente será direcionado para a aba de 
Detalhes
, observe os campos abaixo:
Descrição:
 Dê um nome correspondente ao layout a ser carregado
Tipo de Arquivo:
 Informe TextFile para gerar arquivo texto
Clique então no botão 
Carregar Arquivo 
e na tela apresentada, indique o arquivo 
*.rtm
 que deseja carregar, clique em 
OK
.
Após o 'OK', o novo layout irá aparecer na aba de 
Navegação
, conforma abaixo. Dê um duplo clique sobre o layout:
Como o layout já está criado a partir do arquivo carregado, clique sobre o botão 
Imprime Layout
 e insira o filtro desejado para que seja exportado o arquivo texto contendo os dados:
Clique 
OK
, na tela abaixo desconsidere o campo da 
Impressora
, os campos abaixo devem ser atentados:
Tipo:
 Arquivo Texto
Onde:
 Local para escolha em que o arquivo será exportado
Pronto, o arquivo contendo os dados foi exportado com sucesso.
Após realizar a configuração do layout, a Exportação de Dados também poderá ser feita através da tela propriamente dita, um exemplo é a geração de arquivo para o DPMP. 
Basta selecionar o layout que será possível gerar o arquivo:
Movimentação/Estoque/DPMP/Atualiza/Aba - Geração Mídia/Escolha o Layout de Exportação/Escolha o local em que irá salvar o arquivo 'Caminho para geração da mídia magnética/Gerar Mídia
