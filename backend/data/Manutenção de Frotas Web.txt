TITULO: Manutenção de Frotas Web
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2927269631
--------------------------------------------------
Para alterar um Frota no SispetroWeb:
Clique no Menu que fica no lado esquerdo/superior
Selecione a opção 
Tabelas / Frota
Selecione a Frota pela placa ou pelo código e clique no código:
O SispetroWeb apresentará a tela para alteração:
Clique em 
Salva
 ou 
Cancela
 para finalizar a alteração.
O menu de Frota só é habilitado caso o perfil de cliente definido no SispetroMon esteja como 
Congênere
 ou 
Ambos
 (
Menu SispetroWeb / Definição Perfil Cliente
).
