TITULO: Ajuste de Saldo

--------------------------------------------------
Para alterar o saldo final de uma conta para um valor determinado pelo usuário (contador), basta acessar o menu 
Movimentação / Ajuste de Saldo
.
Para filtrar o código das contas pelo código extendido ou resumido, vide help 
 e 
 (sistema)
O sistema solicitará o código da empresa, código da conta a ter o saldo final alterado, o código da conta de ajuste, o código do histórico, a data da movimentação e o saldo final que a conta receberá.
Através da tela do Plano de Contas, painel
 
 
também é possível fazer esse ajuste.
Para finalizar, clique em Ok e o sistema irá mostrar o número do lote gerado.
