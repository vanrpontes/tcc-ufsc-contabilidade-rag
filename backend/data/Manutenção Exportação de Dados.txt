TITULO: Manutenção Exportação de Dados

--------------------------------------------------
Nesta tela são cadastradas os layouts de Exportação de Dados. Tais exportações podem ser utilizadas para sistemas externos, bem como na Geração da Mídia Magnética do Sintegra ,GIA/GIM, DIME, DECLAN-RJ entre outros.


Para carregar os layouts acesse a partir do Menu principal / Ferramentas / Manutenção Exportação Dados:




Acione o botão de inserção de registro para incluir um layout novo.






O Sistema sugere o sequencial, portanto informe a
 
descrição
 
deste layout e seleciona o tipo do arquivo. Caso já tenha um layout pronto (arquivo .rtm gerado pelo Gerenciador de Relatórios) utilize o botão
 
Carrega de Arquivo
 
para já armazenar este layout.


Cabe lembrar que nele é possível exportar, projetar e gerar um layout de exportação desenvolvido para o SisPetro Controladoria. Portanto, nesta tela serão somente listados os layout's para este tipo de conexão com o banco de dados, não exibindo os layout's do SisPetro Normal.


Clique em
 
OK
 
para confirmar os dados.


No caso de ter que projetar o layout, após a confirmação da inclusão, selecione o layout desejado, vá ao painel Detalhes e clique em
 
Projeta Layout.
 
Será apresentada a tela do Gerenciador de Relatórios (vide maiores explicações no manual específico). Ao terminar, salve o relatório quando for questionado.


Mude o tipo de arquivo a ser gerado conforme a necessidade e clique em
 
Ok
 
para confirmar.


Para imprimir um determinado layout é possível utilizar o botão
 
Imprime Layout.
