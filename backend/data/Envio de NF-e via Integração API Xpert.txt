Para facilitar o controle de estoque desse sistema, o SisPetro encaminhará alguns dados da nota utilizando a API da Xpert.

Como utilizar
Após realizar a emissão ou cancelamento da 
Nota Fiscal de Venda,
 o SisPetro irá automaticamente enviar as informações para API da Xpert. É possível visualizar como está o processo de envio em 
Manutenção de Notas Fiscais
 / Aba Transportes / Sub Aba Averbação.
Em 
Tabelas Auxiliares / Transporte / Gerenciadora de Averbação 
é possível visualizar as comunicações e, assim como na tela acima, ver as notas que foram enviadas com sucesso. 
Reenvio ou Cancelamento de envio de uma Nota
Se houver algum erro de comunicação é possível fazer o reenvio da informação. Além disso é possível também cancelar a comunicação com a Xpert.  Em 
Manutenção de Notas Fiscais
  / Aba Transportes / Sub Aba Averbação há dois botões que permitem ou o reenvio ou o cancelamento de envio da informação. Os botões são habilitados no perfil de usuário, 
Aba Fretes
.
Inativar a comunicação
Caso seja necessário inativar a comunicação, o usuário deverá verificar primeiro se n
ão tem 
nenhum documento
 com Status de 
Envio Pendente.
 Além disso, a Xpert não pode estar v
inculada a nenhuma empresa. 
Com essas condições sanadas a inativação será feita. 
É bom ressaltar que os documentos gerados até o momento da inativação ficarão na fila para ser enviados, a não ser que o usuário cancele o envio.
Histórico de Comunicação
O SisPetro, por padrão, armazenará os últimos 90 dias do histórico das comunicações entre a Xpert e o SisPetro. Caso queira alterar isso, basta acessar a 
 e modificar a quantidade de dias a preservar.
