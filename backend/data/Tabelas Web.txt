No SispetroWeb é possível Incluir e Alterar as transportadoras e os Frotas
Os menus de transportadora e de frota são apresentados caso o perfil de cliente definido no SispetroMon esteja como 
Congênere
 ou 
Ambos
 (
Menu SispetroWeb / Definição Perfil Cliente
).
Na inclusão de uma nova transportadora ou de um novo frota, o SispetroWeb irá bloquear automaticamente, com o motivo cadastrado em 
Menu Ferramentas / Sistema - Painel Ordem Carregto
:
Para incluir uma transportadora, acesse 
Inclusão de Transportadoras Web
Para incluir um frota, acesse 
Inclusão de Frota Web
