TITULO: Edição de dados do Fator de Conversão
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2909897293
--------------------------------------------------
A Edição só poderá ser feita nas seguintes situações:
Produto = 
Derivados Petróleo
Forma de Cálculo = 
Fórmula
Para que este campo esteja como Fórmula, é preciso configurar o perfil de sistema 
Fator de Conversão para Derivados de Petróleo, 
Ferramentas / Sistema / aba Estoque, esteja configurado como 
Utiliza cálculo através de Fórmula.
Produto = 
Álcool
Forma de Cálculo = 
Tabela
O acesso a essa tela se dá através de Movimentação / Base / Fator Conversão.
Primeiramente deve-se definir o tipo de produto que será feito o cálculo, podendo ser:
Derivados de Petróleo
Álcool
Em seguida será necessário informar os parâmetros, sendo:
Densidade da amostra:
 densidade obtida na amostra do combustível.
Temperatura da amostra:
 temperatura obtida na amostra do combustível.
Temperatura do tanque:
 esta informação é necessária caso se deseje obter o fator de conversão com base na temperatura do tanque, em caso contrário pode ser repetida aqui a própria temperatura da amostra.
Após isso, clique no botão 
Edita
 
Para 
Derivados de Petróleo
 o botão Edita somente estará habilitado caso 
o perfil de sistema 
Fator de Conversão para Derivados de Petróleo, 
Ferramentas / Sistema / aba Estoque, esteja configurado como 
Utiliza dados da tabela de conversão.
 
A tela de alteração de tabela de conversão de combustíveis será aberta, já com as informações preenchidas nas telas anteriores.
