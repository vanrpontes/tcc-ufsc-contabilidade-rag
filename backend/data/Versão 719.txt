Importante




O perfil de sistema "Faz Apuração Fiscal pela:" não será mais verificado pelo sistema. Agora, toda apuração será realizada sempre pela Data de Referência. Para perfis configurados para Data Emissão ou Movimento, os campos de Data Referência na Entrada e Saída serão atualizados para a data correspondente.


O perfil de usuário "Não inclui pedido de venda sem liberação financeira" não será mais verificado pelo sistema. Essa alteração foi necessária para centralização e melhora do processo de aprovação de pedidos. Os pedidos que entravam nessa condição ficarão no sistema como não aprovados aguardando aprovação do financeiro.


O perfil de sistema "Utiliza créditos de outra empresa" foi substítuido pelo perfil "Pode utilizar Adiantamentos de outras empresas na baixa" que será utilizado na Baixa Manual, Automática e Débito/Crédito.
