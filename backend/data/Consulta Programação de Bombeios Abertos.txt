TITULO: Consulta Programação de Bombeios Abertos
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926194347
--------------------------------------------------
Para acessar a Consulta de programação de Bombeios Abertos via SispetroWeb clique no Menu localizado no lado esquerdo/superior da tela
Selecione a opção
 Base / Consulta Programação Bombeios Abertos
O SispetroWeb apresentará todas as programações de bombeio abertas, informando:
Data da programação
Produto a ser bombeado
Número do bombeio
Para incluir um pedido de bombeio para uma das programações, acesse 
Incluir Pedido de Bombeio
.
Para mais informações, ver 
Manutenção de Pedidos de Bombeio
.
