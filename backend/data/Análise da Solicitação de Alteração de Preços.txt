TITULO: Análise da Solicitação de Alteração de Preços
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878440123
--------------------------------------------------
Para aprovar ou rejeitar um pedido de alteração de preço, pode ser acessado de duas maneiras:
pelo menu Movimentação/ Vendas
/ 
Pedido Alteração Preços
 - botão Aprova / Rejeita;
pelo menu Consulta
/ 
Pedidos Pendentes de Alteração de Preço de Clientes
 - botão Analisa.
A tela seguinte é apresentada:
Deve ser informado alguma observação referente à análise no campo 
Observações
.
 Para aprovar ou rejeitar o pedido, selecionar o campo 
Ação
:
Ao 
rejeitar
 o pedido, a alteração de preço não é concluída, assim rejeitando o pedido.
 Ao 
aprovar
 o pedido, o sistema irá alterar os preços automaticamente conforme solicitado no pedido. A alteração dos preços é registrada no log, podendo ser consultada através do menu 
Ferramentas / Log Eventos
. A alteração é feita na tabela ItensTabelaPrecoUF.
Somente poderá aprovar ou rejeitar o pedido de alteração de preços se o usuário tiver permissão. Para habilitar a permissão, deve acessar o menu 
Ferramentas / Usuário / painel Perfil / painel Vendas
:
