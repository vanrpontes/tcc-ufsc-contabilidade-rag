TITULO: Mudança de depósito de carregamento
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878898217
--------------------------------------------------
Caso, após o pedido de venda haver sido emitido e aprovado (portanto, não permite mais nenhuma alteração), o setor de faturamento ou estoque resolver mudar o depósito de carregamento indicado pelo pedido para outra base armazenadora, pode-se utilizar este procedimento para realizar tal mudança no Sistema (sem necessidade de cancelamento de pedido e posterior redigitação). 


Abra a tela de Mudança de Depósito para Carregamento (Movimentação / Estoque / Muda depósito pedido). 


 




Digite o número do pedido que deseja modificar a base armazenadora de carregamento.


O Sistema pesquisará o pedido informando a base atual.


Indique a nova base armazenadora. Caso não saiba, aperte <enter> para pesquisar os depósitos cadastrados.




Aperte o botão 
OK
 para confirmar a mudança. 


Note que somente os pedidos para os quais não tenha sido emitida nenhuma Ordem de Carregamento ou Nota Fiscal é que poderão sofrer esta modificação.


Caso algum destes procedimentos já tenha sido executado, cancele a respectiva OC ou NF para poder modificar a base armazenadora do pedido.
