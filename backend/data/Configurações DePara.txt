TITULO: Configurações De/Para
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2957116861
--------------------------------------------------
Para o SisPetro interpretar as informações contidas no arquivo a ser importado, o usuário deve definir parâmetros de transformação através da manutenção do De/Para, ou seja, informar para o sistema que determinado valor no arquivo equivale a outro valor válido dentro do SisPetro.
Para acessar a tela de o usuário deve entrar no menu 
Ferramentas > Importação de Dados > 
Manutenção De/Para
Para configurar os processos:
De/Para Importação de Adiantamentos
De/Para Importação de Lotes
De/Para Importação de Saldo Final de Contas
De/Para Importação de Títulos
