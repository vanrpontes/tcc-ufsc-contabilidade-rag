TITULO: SCANC - Nova Categoria para Diesel A S10 no SCANC CTB 3.0.63 - Ago/2015
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2924285127
--------------------------------------------------
Conforme comunicado da Secretaria da Fazenda - MG responsável pelo SCANC, a partir de 08/2015 o Diesel A S10 terá uma nova categoria diferenciada dos demais Diesel no saldo de estoque no SCANC.
O SCANC liberou uma orientação do que precisa ser feito diretamente no programa SCANC CTB 3.0.63, quando liberado.
PUBLICAÇÃO DO SCANC CTB 3.0.63 (25 de Agosto de 2015)
https://scanc.fazenda.mg.gov.br/scanc/apresentarNoticia.do?method=apresentar&operacao=detalhar&id=226
Configuração Sispetro
Assim que o SCANC liberar o Código do Anexo que representará o Diese A S10 vá ate o Sispetro em:
Arquivo / Produto / Aba Dados Adicionais 
No quadrante Anexo, campo 
Código Anexo 
altere para o código informado pela Secretaria - MG. Esta alteração garantirá que todas as Notas Fiscais no registro tipo 40 fique com o código correto.
Caso seja necessário a inclusão de uma linha no Registro 62, durante a validação  será dada orientação como alterar o arquivo txt.
