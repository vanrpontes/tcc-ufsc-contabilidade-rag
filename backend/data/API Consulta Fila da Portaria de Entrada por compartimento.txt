TITULO: API Consulta Fila da Portaria de Entrada (por compartimento)
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/5119475713
--------------------------------------------------
Objetivo
Consultar quais frotas estão na portaria de entrada com dados por compartimento. Esta API retornará OCs repetidas pois estará informando dados relativos à compartimentação da OC portanto poderá duplicar OC 
1. Obter token autenticação
Introdução
Qualquer webservice do projeto 
SispetroWeb 
exigirá um token de autenticação. Este token é gerado a partir deste webservice e será válido por 24 horas. Assim, recomendamos que o sistema consumidor dos webservices aqui descritos salvem o token armazenando também a data e hora em que ele foi gerado para que, quando chegar próximo de sua validade, seja obtido novo token.
Para mais informações, ver 
API Obtenção de Token
2. Informações sobre Portaria de Entrada (por compartimento)
Introdução
Este webservice tem por finalidade retornar os dados da portaria de entrada por compartimento.

Será retornado um objeto JSON com uma matriz contendo os dados de cada registro na portaria de entrada e o status 200. 
O status da resposta sempre será 200.
