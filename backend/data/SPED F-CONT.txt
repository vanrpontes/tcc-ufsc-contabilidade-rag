Para a geração do SPED FCONT é preciso seguir alguns passos. Vejamos o que precisa ser feito!
Verificar e configurar os campos necessários para o bom funcionamento do SPED FCONT no Sispetro. 
Acessar o 
Conjunto de Dados
 (
menu Ferramentas/ Tabelas dos Conjuntos de Dados
) referente ao 
FCONT
.
Fazer o preenchimento dos dados
 (
aba Funções/ botão Preenche
), informando os dados padrões de Empresa, Data Inicial e Data Final:
Corrigir possíveis informações
, editando diretamente os dados.
Corrigir possíveis fórmulas
, o que exigirá a repetição do passo 2 (
aba Geral/ botão Fórmula
).
Configurar Campos da tabela
, se for necessário (
aba Geral/ botão Configuração
).
Gerar o arquivo texto
 (
aba Funções/ botão Gera
), informando a DLL 
sped_scont.dll
 na 
, além das informações padrões citadas no item 
Fazer o preenchimento dos dados.
 
A partir de 2015, o FCont foi substituído pela 
ECF - Escrituração Contábil e Fiscal
.
