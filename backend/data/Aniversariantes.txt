TITULO: Aniversariantes

--------------------------------------------------
O usuário pode ter acesso a uma relação de aniversariantes através da manutenção de 
 ou pelo Menu 
Ferramentas / Contatos / Aniversariantes
.




Relação de Aniversariantes pela manutenção de 
:

Para imprimir uma relação de aniversariantes, o usuário deve clicar no botão:


 




O sistema abrirá a tela de Relatório de Aniversariantes, onde o usuário poderá selecionar opções para impressão do relatório.




Para imprimir o relatório para uma entidade especifica, o usuário deve selecioná-la marcando o grupo 
Seleciona Entidade
.




Caso o usuário saiba o código da entidade, pode-se digitá-lo diretamente, caso contrário, pode-se selecioná-lo através da tela de pesquisa. Para ter acesso a tela de pesquisa, o usuário deve posicionar o cursor no campo 
Código Entidade
 e teclar <ENTER>.


Através do grupo 
Período
, é possível selecionar todas as datas de aniversário, selecionando a opção 
Todas as datas de aniversário
 ou selecionar o dia e mês marcando o grupo 
Informar um período
. Caso deseje, o usuário ainda pode omitir o dia e simplesmente fazer com que o sistema considere apenas o mês do período informado. Para isso, deve-se marcar a opção 
Considerar somente o mês para emissão do relatório
.


Por meio da opção 
Vínculo dos Contatos
, o usuário poderá selecionar o tipo de vinculo com os contatos:




Vinculados a clientes


Vinculados a Fornecedores


Sem vinculo com entidade


Todos




O botão 
Edita Layout
, permite ao usuário ter acesso a estrutura do relatório.


Para emitir o relatório deve-se utilizar o botão 
OK
 e para sair e retornar a tela de contatos o botão 
Relação de Aniversariantes
 pelo Menu 
Ferramentas / Aniversariantes
:


Ao acessar o menu o sistema abrirá a tela de aniversariantes.




A tela de aniversariantes possue duas opções que podem ser marcadas ou desmarcadas:




Não exibe mais essa tela hoje ao entrar no sistema:
 essa opção faz com que o sistema não exiba a tela de aniversariantes quando o usuário entrar no sistema várias vezes no mesmo dia, já no próximo dia, o sistema exibirá a tela.


Não exibe mais essa tela ao entrar no sistema:
 não exibirá mais a tela, a menos que a opção seja desmarcada.




O usuário ainda poderá emitir um relatório dos aniversariantes, através do botão 
Imprimir
.
