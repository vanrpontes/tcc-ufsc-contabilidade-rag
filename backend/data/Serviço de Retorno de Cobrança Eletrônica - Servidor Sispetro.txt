Configurações Servidor Sispetro
No servidor Sispetro foi criada a aba 
Serviço de cobrança Eletrônica 
onde iremos realizar a ativação do Serviço podendo também deixar o mesmo configurado para iniciar automaticamente, iremos também configurar para quais pastas onde serão salvos os arquivos e para onde o serviço deve mover após a leitura.
 
Geral
Iniciar Serviço:
 Botão para iniciar ou parar o Serviço.
Iniciar o Serviço automaticamente:
 Caso marcado inicia o Serviço automaticamente quando o Servidor Sispetro é iniciado.
 
Pasta para Leitura dos Arquivos de Retorno
Arquivos a serem Lidos: 
Caminho onde será salvo os arquivos de retorno vindo do banco para que seja realizado a leitura. 
Arquivos Processados: 
Caminho onde sera salvo os arquivos que foram processados, após o serviço realizar a leitura do arquivo caso o mesmo esteja OK e seja processado no Sispetro o serviço tira o arquivo de retorno do caminho de arquivos a serem Lidos e move para a pasta de Arquivos Processados.
Arquivos com erro: 
Casso o Sistema encontre algum erro ao realizar o serviço, o arquivo sera movido da pasta de arquivos a serem lidos para a pasta informada em Arquivos com erro.
 
Arquivos DLL indicados no Tipo de Cobrança:
 Nesse campo deve-se informar o caminho da pasta onde está salva as DLL dos bancos cadastradas no Tipo de Cobrança.
