TITULO: Como é preenchido automaticamente o Livro Fiscal
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2880176233
--------------------------------------------------
Os registros que irão compor o Livro Fiscal são gerados na gravação de uma NF ou são reprocessados utilizando o recurso de 'Reprocessamento de Nfs para Livro Fiscais' do Controladoria. Nem sempre os dados destas notas serão os dados que irão para o  Livro Fiscal. Para resolver isto criou-se o recurso de Fórmulas onde através de uma 
Tabela de Fórmulas
 podemos definir com mais exatidão como serão gravadas as informações no Livro Fiscal. Mesmo assim, algumas destas informações são geradas automaticamente. Para distinguir quais são geradas Automaticamente e quais são geradas por Fórmulas veja a tabela abaixo:
