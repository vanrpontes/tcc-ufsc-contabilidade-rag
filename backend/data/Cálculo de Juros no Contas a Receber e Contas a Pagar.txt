TITULO: Cálculo de Juros no Contas a Receber e Contas a Pagar
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878701758
--------------------------------------------------
Através do botão "Calc.Juros", o usuário poderá calcular o juros de um título. Pode-se utilizar o juros pré-cadastrado na entidade, ou lançá-lo no momento do cálculo. Esse recurso está disponivel tanto na manutenção do contas a receber como na manutenção do contas a pagar.
Pode-se calcular os juros para mais de um título, para isso, o usuário deve selecioná-los e depois clicar no botão "Calc.Juros". Para selecionar mais de um título, deve-se manter a tecla <SHIFT> pressionada e clicar com o mouse sobre os títulos desejados.
O sistema abrirá a tela de "Cálculo de Juros", exibindo os títulos selecionados e suas taxas de juros ao dia. A taxa de juros ao dia, será exibida se a taxa foi devidamente vinculada no cadastrado da entidade.
Para fazer a simulação, o usuário deve alterar o campo 
"Data Fluxo Caixa"
, para a data que se deseja pesquisar e clicar no botão 
"Atualiza"
. O sistema fará o 
cálculo do juros
, preenchendo as coluna "Valor Juros" e "Saldo + Juros". veja maiores detalhes de como é realizado o calculo do juros em 
Calculo de Juros em Títulos
Mudança de Taxa:
Mesmo tendo uma taxa de juros cadastrada para o cliente, ou não tendo nenhuma taxa de juros cadastrada, o usuário tem a possibilidade de cadastrar ou alterar uma taxa de juros para um ou mais títulos.
O usuário deve selecionar os títulos que receberão a nova taxa através do botão 
"Seleciona"
.
Os títulos selecionados estarão com a coluna "Selecionado" marcada. Após marcar os títulos, deve-se clicar no botão 
"Nova Taxa"
. Este botão só estará habilitado caso o usuário tenha permissão para alterar a taxa dos títulos. Esse perfil pode ser configurado através do menu 
Ferramentas / Usuário / Financeiro
:
Após clicar no botão "Nova Taxa", o sistema abrirá a tela 
"Nova taxa de Juros"
:
O usuário deverá preencher a nova taxa que será vinculada aos títulos selecionados e definir se o tipo de taxa será 
"Taxa de Juros Diária"
 ou 
"Taxa de juros Mensal"
. Para confirmar a alteração o usuário deverá clicar no botão "OK", dessa forma o sistema voltará para a tela "Cálculo de juros", mostrando o valor do juros para cada título selecionado.
Caso o usuário deseje que a nova taxa de juros seja gravada para cada título, deve-se clicar no botão 
"Salva Taxa"
. Dessa forma, a nova taxa será para os títulos selecionados.
Para finalizar e sair da tela o usuário deve clicar no botão 
"Sair"
.
