TITULO: Como emitir Nota Fiscal de Entrada de emissão própria para apropriação de ICMS do CIAP para o Estado de São Paulo
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2910814586
--------------------------------------------------
Para o Estado de São Paulo, é feita mensalmente a apropriação do ICMS do CIAP, conforme Portaria CAT 43/01.
Para emitir esse tipo de nota deve configurar:
Tabelas Auxiliares / Faturamento / Natureza Operação
:
Emitindo a nota fiscal:
Movimentação / Faturamento / Outras Entradas para emitir a Nota Fiscal desejada
:
Preencha os dados solicitados ou selecione da lista que pode ser exibida, em cada campo, pelo Sistema. A Natureza de Operação deve ser adequada à esta Nota Fiscal.
Acione o botão Avança para prosseguir com o preenchimento dos dados. 
Obrigatório preencher o campo 
QTDE VOLUMES:
Na tela de itens Informar o campo
 SIT. TRIB. com a informação correta - consultar sua área fiscal
:
Nos valores somente o campo
 VALOR ICMS
 deve ser preenchido:
Caso existam dúvidas quanto ao preenchimento vide maiores detalhes no tópico
