Aba Temperatura por Produto
Ao acessar a aba Temperatura por Depósito, a tela abaixo será exibida:
Esta tela permite informar dados como temperatura, densidade e número do laudo de todos os produtos para um determinado depósito que serão gravados para uso na 
Impressão da Nota Fiscal
.
Para informar os dados, clique no botão 
Edita
.
Através do botão 
Incluir Produtos
, o usuário adicionará os produtos ativos. Esse procedimento deve ser realizado quando houverem produtos criados posteriormente.
Este painel somente estará visível se no perfil de Empresas (Menu 
Ferramentas / Sistema / painel Nota Fiscal / caixa Temperatura e Densidade
), o campo 
No cadastro de produtos, cadastrar 
estiver configurado por depósito.
Para dar permissão ao usuário para editar esta tela, no perfil de usuário (Menu 
Ferramentas / Usuário / painel Produtos
), marcar a opção 
Pode alterar temperatura e densidade
.
