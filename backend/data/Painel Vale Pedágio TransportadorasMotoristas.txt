TITULO: Painel Vale Pedágio (Transportadoras/Motoristas)
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926197977
--------------------------------------------------
A partir da versão 9.17 foi adicionado à tabela transportadoras o painel Vale pedágio, onde são cadastrado os dados de cartão, utilizados para emissão de vale pedágio da gerenciadora de Pedágio Target.
Botão Inclui
Para realizar a inclusão de um cartão basta acionar o botão 
Incluir
.
Substituindo um cartão
Para os casos em que se deve substituir o cartão por outro, basta o usuário incluir um novo. Ao acionar o incluir o Sispetro já ira sugerir o ultimo cartão utilizado, onde o usuário pode selecionar o motivo da substituição. 
Só é possível incluir um novo cartão caso o anterior esteja com status igual á 
Processado ok
Botão Altera
Caso o cartão cadastrado 
NÃO 
esteja com o status igual a Processado Ok, é possível alterar o mesmo. Caso o cartão anterior esteja com Status igual a Processado OK, o usuário deve incluir um novo cartão, e informar o motivo da substituição.
Botão Inativa
Ao acionar o botão de inativa, o cartão será inativado para o Motorista em questão, podendo assim ser utilizado em outro motorista. 
O cartão só vai poder ser inativado se: 
O cartão não esteja vinculado a um pedido de pedágio ainda não processado.
O cartão esteja ativo.
O cartão já tenha sido processado na gerenciadora.
