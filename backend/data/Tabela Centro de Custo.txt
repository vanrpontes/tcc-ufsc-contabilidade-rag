TITULO: Tabela Centro de Custo
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210541
--------------------------------------------------
Nesta tabela são cadastrados os Centros de Custos a serem utilizados na 
Inclusão do Contas a Pagar
, 
, entre outros processos. 


A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - Tabelas Auxiliares / Controladoria / Centro Custo 




Para acrescentar um novo Centro de Custo acione o botão de inclusão de registro: 


 


O Sistema exibe a tela para preenchimento dos novos dados. 




O sistema irá sugerir um 
código
 para este Centro de Custo. Caso no menu Ferramentas / Sistema / painel Financeiro esteja marcada a opção 
"Pode modificar Centro de Custo"
 será possível alterar o código sugerido. 


Informe a 
Descrição
 do centro de custo.


Em seguida, informe a 
Máscara de Ordenação
 para este Centro de Custo.


Marque o perfil 
Ativo
 quando necessário para cada centro de custo. 


E se o Centro de Custo for do tipo sintético, então marque o perfil 
Sintético
.


É possível que o código do Centro de Custo seja apresentado com uma máscara, para facilitar a sua organização/visualização num esquema hierárquico. Para isso, no menu Ferramentas / Sistema / Financeiro Gerais preencha a máscara desejada no campo 
Máscara p/ Centro de Custo
: 


 


Com o exemplo de máscara acima, o centro de custo com numeração 1000, seria apresentado 1.000. 

Então poderia ser montada uma numeração, próxima a: 

1,000 - Administrativo (código 1000) 

1,001 - Secretaria 

1,002 - RH 

2,000 - Controladoria 

2,001 - Contabilidade 

2,002 - Fiscal 


Dessa forma, abaixo do Centro de Custo Administrativo, podemos visualizar os Centros de Custos Secretaria e RH. Abaixo do Centro de Custo Controladoria, temos os Centros de Custos Contabilidade e Fiscal. 


Acione o botão OK para atualizar os dados no Sistema. 


O Sistema retorna a tela de navegação. 


Para retornar ao Menu principal acione o botão Sair:
