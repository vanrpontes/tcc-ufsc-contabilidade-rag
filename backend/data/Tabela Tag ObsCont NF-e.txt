TITULO: Tabela Tag ObsCont NF-e
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/4317446145
--------------------------------------------------
Na tela de manutenção de tag ObsCont NF- e, são cadastrados os códigos e descrições das tags, que serão encaminhados no arquivo xml da NF-e, como placa de carreta, CFOP e afins, desta forma o cliente que receber o xml terá informações complementares que no momento não foram encaminhadas nos dados adicionais.
A tela que permite manipular esta tabela é acionada a partir do menu principal em 
Tabelas Auxiliares / Faturamento / Tag ObsCont NF-e
Botões:
Mais:
 realizar inclusão de uma nova tag ObsCont
Menos:
 realizar a exclusão de uma tag ObsCont
Ao clicar no botão 
mais, 
será apresentada a tela do cadastro da fórmula.
Botões:
 Edita: 
possibilita a edição de uma fórmula existente ou criação de uma, a partir das tabelas disponíveis.
 Para mais informações, ver 
Exemplos de Fórmulas
Testa
: permite testar uma fórmula cadastrada ou editar uma. 
Para mais informações, ver
 
Testar Fórmulas
