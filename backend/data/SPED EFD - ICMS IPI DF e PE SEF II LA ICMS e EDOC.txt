TITULO: SPED EFD - ICMS IPI, DF e PE (SEF II LA ICMS e EDOC)
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2907308036
--------------------------------------------------
Objetivo
Direcionar o usuário para  a geração da EFD. Para isso é preciso seguir alguns passos: 
Verificar e configurar os campos necessários para o bom funcionamento da EFD no Sispetro. Para mais informações, ver 
Configurações - EFD ICMS/IPI, EFD DF e EFD PE (SEF II LA ICMS e EDOC)
.
Acessar o 
Conjunto de Dados
 (
menu Ferramentas/ Tabelas dos Conjuntos de Dados
) referente ao 
Livro Fiscal Eletrônico - EFD
.
Fazer o preenchimento dos dados
 (aba Funções/ botão Preenche), informando além dos dados padrões de Empresa, Data Inicial e Data Final:
Data Inventário: 
Data do inventário para o período desejado;
Finalidade do Arquivo: 
Definir a finalidade do arquivo dentre as opções disponíveis:
Remessa do Arquivo Original;
Remessa do Arquivo Substituto;
Corrigir possíveis informações
, editando diretamente os dados.
Corrigir possíveis fórmulas
, o que exigirá a repetição do passo 2 (
aba Geral/ botão Fórmula
).
Configurar Campos da tabela
, se for necessário (
aba Geral/ botão Configuração
).
Gerar o arquivo texto
 (
aba Funções/ botão Gera
), informando as DLL 
sped_efd.dll para a EFD ICMS IPI, sped_df.dll para a EFD DF, sped_pe para a EFD PE SEF II LA ICMS e sped_edoc para a EFD PE SEF II EDOC 
 na 
Tabela Conjunto de Dados
, além das informações padrões citadas no item 
Fazer o preenchimento dos dados.
