TITULO: Tipos de Ocorrência

--------------------------------------------------
Através da tela de 
Tipos de Ocorrências
, o usuário poderá cadastrar as ocorrências 
que serão usadas no momento de se cadastrar uma ocorrência e uma pendência
.


Para acessar a tela de 
Tipos de Ocorrências
, o usuário deverá utilizar o Menu 
Ferramentas / Contatos / Tipos de Ocorrências
:




Para incluir um novo tipo de ocorrência, o usuário deve utilizar o botão 
.


O sistema se posicionará na no painel 
Detalhes
, aguardando a digitação do campo 
Descrição
:




O campo 
código
 é preenchido automáticamente pelo sistema, portanto, após preencher o campo 
descrição
, para salvar as modificações, basta o usuário clicar no botão:




O sistema salvará as alterações se posicionando no painel 
Navegação
:




Para cancelar qualquer alteração ou o processo de inclusão, o usuário deve utilizar o botão
