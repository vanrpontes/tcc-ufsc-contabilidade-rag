No painel 
Movimento Pedidos
, são apresentados os pedidos de venda cadastrados para o cliente selecionado. Cada registro possui uma coluna indicando a filial onde foi realizado o lançamento, facilitando assim a identificação por parte do usuário.
Para ter acesso é necessário habilitar o perfil de usuário,
 
Mostra painel movto. Pedidos, 
Painel Segurança (Usuários)
.
Os dados são filtrados a partir de uma determinada data de movimento. Ela pode ser alterada pelo botão 
Redefine data mínima movimento
.
Evite colocar uma data muito antiga pois o carregamento de muitos dados deixará a consulta lenta devido ao tráfego de dados.
O último período informado (em dias) será salvo no arquivo do usuário, sendo sugerido na próxima consulta.
