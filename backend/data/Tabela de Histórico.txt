TITULO: Tabela de Histórico
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210313
--------------------------------------------------
Nesta tabela são cadastrados todos os históricos de movimentação de mercadoria / financeiro contábil.


Todas as movimentações financeiras e de estoque utilizam, para facilitar a digitação e padronizar o histórico dos movimentos, esta tabela.


Esta tabela também recebe manutenção pelo Sispetro. A tela que permite a manipulação dos históricos padrão no módulo Contábil é acionada a partir do Menu principal com as opções -
 
Arquivos / Históricos.




Acione o botão de inserção de registro.




O Sistema exibe a tela para o cadastramento do novo histórico.




Preencha o campo de Descrição do novo histórico.


Quando o histórico exigir complemento, isto é, que além de selecionar o histórico, o usuário deva modificá-lo no momento de incluir/alterar uma movimentação, deve-se habilitar o campo
 
Exige Complemento.


Quando o histórico tiver de gerar um registro para a entidade associada ao movimento (por exemplo, histórico: Cheque Devolvido - sempre que este histórico for utilizado para determinada entidade em determinado movimento, um contador será incrementado e este movimento passará a ser visível em determinadas telas de consulta de entidades), habilite o campo
 
Gera registro para entidade.
 
Não é utilizado nos lançamentos contábeis.


Para facilitar os lançamentos contábeis, é possível especificar ou não uma conta que será utilizada para crédito e débito quando este histórico for selecionado. Assim, na tela de inclusão de
 
Lotes
 
já serão sugeridas as contas aqui informadas.


Acione o botão OK para atualizar os dados no Sistema.


O Sistema retorna à tela de navegação.


Para deixar a tela de Históricos acione o botão Ok ou Cancela.




Os históricos apresentados na cor vermelha estão inativos, ou seja, embora o Histórico apareça na tela de Manutenção de Históricos, ele não aparecerá nas telas de pesquisa. Para se ativar ou desativar um Histórico Padrão é necessário fazer a manutenção no módulo do Sispetro Normal.
