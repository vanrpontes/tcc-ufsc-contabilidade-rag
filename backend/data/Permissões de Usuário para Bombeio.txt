TITULO: Permissões de Usuário para Bombeio
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878701868
--------------------------------------------------
No painel Estoque, da configuração do usuário, há os seguintes perfis:






Pode Incluir/Cancelar Prog. Bombeio

Dá permissão para a tela de 
Programação de Bombeio






Pode incluir confirmação bombeio

Dá permissão para incluir 
Confirmação de Pedido de Bombeio






Pode cancelar conf. Bombeio

Dá permissão para cancelar 
Confirmação de Pedido de Bombeio






Pode incluir bombeio

Dá permissão para incluir 
Bombeio Físico






Pode cancelar bombeio

Dá permissão para cancelar 
Bombeio Físico






Pode incluir ajuste na distribuição do bombeio físico

Dá permissão para fazer 
Distribuição de Bombeio






Pode associar NF ao Bombeio

Dá permissão para 
vincular a NF ao Bombeio
