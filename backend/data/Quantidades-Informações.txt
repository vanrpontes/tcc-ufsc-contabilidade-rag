--------------------------------------------------
Pela tela de 
Cadastro de Produto
 é possível consultar quantidades e informações de determinado produto. Para isso deve-se selecionar um produto no painel 
Navegação
 (tela principal) e escolher o painel 
Quantidades/Informações
:
Caso deseje, clicar as opções:
Data Referência:
 data início do período das informações a serem visualizadas.
Mostra Vendas:
 caso queira que apareçam os dados de Vendas (mostra quantidade vendida, cliente, preço máximo, preço mínimo e preço médio).
Mostra Compras:
 caso queira que apareçam os dados de Compras (mostra quantidade comprada, fornecedor, preço máximo, preço mínimo e preço médio.
A cor 
Vermelha
 indica 
Vendas
 e a cor 
Verde
 indica 
Compras
.
Escolhida as opções desejadas, clique no botão 
Atualiza.
Este grid só é apresentado quando a opção "Mostra gráfico movimento produtos automaticamente" está selecionada no perfil de Sistema / Venda (
).
