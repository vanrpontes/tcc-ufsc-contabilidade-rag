Objetivo
Nessa aba, são feitas as configurações de 
Apuração de Valores de Armazenagem por Congênere
. O depósito deve estar com a opção 
Base Própria marcada. 
O acesso é feito através do Menu 
Arquivo / Depósito / Aba Apuração de Armazenagem.
Itens inativos agora ficarão destacados em 
vermelho
Selecione o Depósito que deseja configurar a apuração, em seguida clique na Aba 
Apuração de Armazenagem.
Abas
Aba Dados Gerais 
Nessa aba é feita a definição dos critérios de apuração, valor mínimo a ser cobrado e natureza para faturamento.
Para incluir ou editar os dados da aba em questão é necessário ter permissão de usuário 
Pode incluir, alterar e excluir Dados
 no 
Painel Armazenagem (Usuários)
, por padrão desmarcado. 
Quantidades para Apuração:
 determina qual estoque, volume e movimentação que será usado como critério na apuração de valores de armazenagem.
Dados por Congênere:
 neste quadro, é possível determinar se a congênere irá trabalhar com valor mínimo de movimentação, assim como definir a Natureza de Operação para emissão da Nota fiscal.
Para editar os dados, selecione uma congênere no grid, e clique no botão 
Altera.
Valor Mínimo?: 
ao marcar esse perfil, o Sispetro passa a trabalhar com um valor mínimo na apuração, ou seja, é possível parametrizar um valor de cobrança mínimo para apuração de armazenagem, caso a congênere em questão não tenha uma movimentação que atinja o valor estipulado, ou não tenha nenhuma movimentação na apuração, o valor mínimo estipulado será cobrado.
Nat. Operação Mínimo?: 
neste campo, selecione a Natureza de Operação que será utilizada para emissão da NF no valor mínimo configurado na apuração.
Nat. Operação Geral/Excedente: 
neste campo, selecione a Natureza de Operação que será utilizada para emissão da NF Geral, caso tenha valor mínimo definido, o valor excedente será gerada uma outra NF.
Naturezas Iguais
Caso parametrize a mesma natureza para os campos mínimo e geral/excedente, será gerada apenas uma única NF caso tenha valor excedente.
Produtos da Congênere: 
neste Quadro é possível determinar um valor mínimo a ser cobrado de uma congênere por produto, esse valor será um valor fixo total, caso não tenha movimentação do produto no período de apuração o valor mínimo não será cobrado.
Botões:
Incluí:
 inclui um 
produto de cada vez.
Inclui Produtos:
 inclui de uma única vez todos os produtos do tipo combustível.
Copia p/ Congêneres: 
copia todos os produtos e valores da congênere selecionada para uma nova congênere, uma nova tela será exibida para seleção da congênere destino.
Após marcar a congênere de destino, clique no botão
 Copia
 para executar a cópia dos produtos.
O botão 
Cancela 
descarta as alterações ou inclusões ainda não salvas.

Aba Desconto / Acréscimo
Possibilita incluir um desconto / acréscimo que será concedido no valor final da apuração.

Pode incluir, alterar e excluir Desconto 
no 
Painel Armazenagem (Usuários)
 que, por padrão, está desmarcado.   
Botões:
Inclui: 
inclui um novo desconto / acréscimo.
Congênere:
 empresa que deseja conceder o desconto.
Histórico:
 informe um histórico para identificar a operação.
Data:
 o valor será concedido se a data informada estiver dentro do período de apuração.
Acréscimo / Desconto: 
selecione se o cadastro do valor será acréscimo ou desconto
Valor:
 informe o valor total do desconto que será concedido.
Observações: 
campo livre para observações.
