O relatório 
Receitas / Despesas Frota 
 mostra as informações referentes as despesas e/ou receitas ocorridas por Frota, o motorista e outros. Utilizado para comparar os gastos e as receitas que foram geradas através do frete.
O acesso a esse relatório é através do menu: 
Relatório/ Transporte/ Receitas/ Despesas Frota
O layout do relatório será gerado conforme o arquivo 
ReceitaDespesaFrota.rtm
 que deve estar no mesmo caminho do executável. Caso não esteja, o sistema utilizará o layout padrão para impressão do relatório.
O usuário deve utilizar selecionar as opções de filtros existentes na tela, para obter resultados mais objetivos:
Filial
Esta opção permite verificar as viagens apenas de uma filial especifica, um grupo de filiais ou todas as filiais.
Frota Específico
Esse filtro limita o relatório apenas para as receitas/despesas que foram feitas do frota específico indicada no período.
É possível definir somente um dos itens (Frota, Carreta e/ou Treminhão) ou mais de um, representando um conjunto. Assim, é possível ver as receitas e despesas do frota específico ou do conjunto definido nos filtros para um período de viagem.
Período
Intervalo de data que se deseja emitir o relatório
Somente frotas próprios
Se marcada essa opção serão emitidas as receitas/despesas de frotas cadastrados como próprio, vide maiores detalhes na 
Manutenção Frota
.
Mostra Km/litro calculado p/ viagem
Se marcada essa opção será mostrado dados da quilometragem, caso haja, para o frota.
Exibe frotas inativos caso tenha receitas/despesas
Se desejar pode emitir receitas/despesas de frotas que foram inativados.
Botão Edita Layout
Permite ao usuário ajustar o layout, conforme a necessidade da empresa.
Para imprimir o relatório, o usuário deve clicar no botão 
 e para cancelar a impressão no botão 
.

Os dados selecionados na imagem acima serão calculados de acordo com os dados inseridos no 
Controle de Viagens,
 considerando o KM de Saída e o KM Retorno.
São descriminadas nas Despesas os eventos que ocorreram durante a viagem. Os mesmos são pré-cadastrados na 
Tabela de Eventos Frota
.
As receitas serão carregadas a partir da tela de 
Manutenção de Fretes
, podendo ser vinculado as Notas Fiscais.
Ao final de cada do período, considerando os filtros selecionados, o relatório totaliza as Receitas, Despesas e o Total Geral das Receitas x Despesas.
