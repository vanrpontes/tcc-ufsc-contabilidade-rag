--------------------------------------------------
Este serviço permite que sejam visualizados os usuários que estão acessando o SisPetro, bem como bloquear o acesso do usuário. 
Acessar o 
 e selecionar a 
aba
 Chaves
Chave:
 código interno gerado pelo SisPetro.
tipo:
 informação para controle interno do SisPetro.
Informação Adicional:
 nome do usuário.
Validade:
 uso interno do SisPetro.
Última Execução:
 data e hora que o usuário se conectou no SisPetro.
Origem:
 código interno de identificação gerado pelo SisPetro.
IP:
 número IP da máquina do usuário conectado.
Bloqueado:
 informa se a chave está ou não bloqueado para uso.
Data bloqueio
 - informa a data de bloqueio do usuário
O botão
 
 
bloqueia o usuário para utilização do SisPetro. Caso esteja utilizando o SisPetro, o usuário recebe uma mensagem na tela informando que a Chave foi bloqueada . Clicando em OK, o usuário é desconectado do SisPetro e desbloqueado automaticamente. É útil para remover usuários que por algum motivo não conseguiram se desconectar do SisPetro, ocupando licenças sem necessidade. 
O botão 
 apresenta os usuários que estão conectados no SisPetro.
