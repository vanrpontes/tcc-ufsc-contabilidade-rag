TITULO: Tabela de Grupos Contábeis
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210321
--------------------------------------------------
A tabela de Grupos Contábeis é utilizada para visualizar os grupos contábeis existentes para contabilização. A manutenção desta tabela é acionada a partir do Menu principal com as opções -
 
Arquivos / Grupos Contábeis. 




Nesta tela só será permitida a alteração do Grupo Contábil. 




Informe se para este grupo contábil deve ser forçada a Contabilização. Quando marcada, nas telas do Sispetro local que trabalhem com este grupo o usuário será sempre obrigado a contabilizar. No entanto, o perfil do usuário (Pode não contabilizar lançamentos) sobrepões esta configuração. Atente-se para só marcar esta opção quando você já tiver um código de contabilização criado para o grupo.


Acione o botão OK para atualizar os dados no Sistema.


O Sistema retorna a tela de navegação.


Para deixar a tela de Grupos Contábeis acione o botão Ok ou Cancela.
