Para trabalhar com a Gerenciadora de Análise de Crédito, se faz necessário realizar alguns cadastros, conforme KB 
Gerenciadora de Análise de Crédito - GAC
.
Nesta tabela são informados os dados de conexão com a Gerenciadora de Análise de Crédito. É somente permitido alterar os dados de conexão.
Para acessar:
Não é permitido incluir ou excluir uma Gerenciadora. A Gerenciadora Serasa foi incluída pela Futura.
