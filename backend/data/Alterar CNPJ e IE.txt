TITULO: Alterar CNPJ e IE
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2880798772
--------------------------------------------------
Quando são emitidas notas fiscais com CNPJ ou Inscrição Estadual incorretos, o Sispetro permite que se faça correções dos mesmos.


Essas correções são feitas somente para efeito de geração de arquivos magnéticos, tais como Sintegra, Scanc, etc., não afetando qualquer outra parte do Sispetro.


Para alterar o CNPJ/IE, em Menu Ferramentas->Altera CNPJ/IE:






Código da Entidade:
 informe o código da Entidade


Data Inicial
 e 
Data Final
: período das notas fiscais que deverão ser corrigidas o CNJP e/ou a IE.


CNPJ:
 informe o CNJP correto, se for o caso. Caso esteja correto, deixe como está.


IE:
 informe a IE correta, se for o caso. Caso esteja correta, deixe como está.




Caso o perfil do usuário permita, 
Menu Ferramentos->Usuário->aba Segurança: campos Pode Alterar Entidades e Pode Alterar CNPJ/CPF de Entidades
 , o CNPJ e a IE podem ser alteradas na Entidade também.




Muda na Identidade:
 marcando esta opção, o CNPJ e/ou a IE também serão alterados na tabela de Entidades.




Clique no botão 
OK
 e confirme a alteração:
