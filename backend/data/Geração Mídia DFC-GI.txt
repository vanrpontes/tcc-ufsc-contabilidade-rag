TITULO: Geração Mídia DFC-GI
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210472
--------------------------------------------------
A Declaração Fisco-Contábil (DFC) é um demonstrativo anual de informações das operações de entradas e saídas de mercadorias e serviços abrangidos pelo ICMS e pode ser gerado facilmente pelo SisPetro.


Através desta tela é possível gerar a mídia com a movimentação de um determinado período (anual) para ser transmitida ao programa específico de envio do DFC (Declaração Fisco Contábil), somente para o Estado do Paraná. A geração da GI também está disponível através dessa tela.


Acesse esta tela partir do menu Principal, nas opções
 
Ferramentas / Geração Mídia DFC.
 




Informe o período para a geração dos dados. O Sispetro não exige que esse período corresponda a um ano, porém o DFC exige. O que o Sispetro exige é apenas a existência de apurações encerradas para o período inicial e o final respectivamente, pois os dados serão gerados de acordo com essas apurações.


Informe os demais valores, caso for necessário para completar os dados de acordo com as especificações do programa DFC.


Cabe ressaltar que atualmente a geração do DFC/GI está somente disponível para o
 
Estado do Paraná.


É necessário configurar na
 
Tabela Natureza Operação (Sispetro Normal)
 
a categoria que a mesma pertence para que os dados sejam gerados corretamente. A categoria
 
Entrada por Transferências
 
deve ser utilizada para notas de entrada e somente para dentro do estado.


E para a geração da GI o sistema verifica no cadastro do
 
Cliente
 
se o mesmo é
 
Consumidor Final
 
ou não para gerar dados das saídas separadamente, conforme o layout.


No campo
 
Modelo
 
é definido o tipo de arquivo que será gerado, abaixo estão as opções:


* DFC - Simples/PR Faixa A


* DFC - Normal


* GI


Para a geração dos dados deve-se atentar ao Layout de Exportação, pois para o DFC é um layout e para a GI é outro, ou seja, são layouts diferentes, que deverão ser selecionados para a exportação dos dados.


Selecione o
 
Layout de Exportação
 
correto para a geração dos dados cadastrado para o DFC ou para a GI. Este layout de exportação deve ser carregado pela tela de
 
Manutenção de Exportação de Dados
 
utilizando um arquivo .rtm disponibilizado para tal. Informe também o caminho e o nome do arquivo onde serão gravados os dados.


Os layouts de exportação são disponibilizados para utilização, portanto, caso tenha dúvidas ou não possua o arquivo entrar em contato com o Suporte. Lembrando também que os mesmos podem sofrer alterações e devem ser atualizados para garantir a geração dos dados corretamente.


Os dados que serão considerados no DFC ou GI dependerão da
 
configuração de data a ser utilizada na Apuração
.

OK
 
para iniciar o processo. A mídia será gerada e ao final será apresentada a mensagem:
