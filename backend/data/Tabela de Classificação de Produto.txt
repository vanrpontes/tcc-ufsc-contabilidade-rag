TITULO: Tabela de Classificação de Produto
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210325
--------------------------------------------------
A tabela de Classificação de Produtos é utilizada para agrupar 
Tabela de Produtos
. A manutenção desta tabela é acionada a partir do Menu principal com as opções 
Arquivos / Classificação de Produtos.
Esta tabela é utilizada para sugerir dados na inclusão de um produto, que por sua vez sugere dados na inclusão de um bem.
Acione o botão de inserção de registro.
O Sistema exibe a tela para o cadastramento de uma nova classificação de produto.
Informe os dados necessários:
Código:
 um código ainda não cadastrado.
Descrição:
 que identifique a classificação.
Os dados a seguir serão sugeridos no 
Cadastro de Produtos do SisPetro Controladoria
 que utilizarem esta classificação, porém para cada produto poderão ser utilizados dados específicos:
Habilita depreciação: quando marcada indica que os bens deste produto participarão do processo de depreciação.
CC Custo Alienação:
 Conta contábil a ser utilizada no processo de alienação do bem, no momento da sua baixa.
CC Receita Alienação:
 Conta contábil a ser utilizada no processo de alienação do bem, no momento da sua venda.
CC Despesa Depreciação:
 Conta contábil a ser utilizada no processo de depreciação do bem.
CC Depreciação Acumulada:
 Conta contábil a ser utilizada no processo de depreciação do bem.
CC Imobilizado:
 Conta contábil que indica a conta de ativo onde é registrada a aquisição do bem.
Taxa Depreciação (% ao ano):
 Taxa anual para depreciação do bem, conforme legislação.
Vida útil (em meses):
 Vida útil (em meses) do bem em decorrência da taxa anual de depreciação. Ex.: Se um bem tem uma taxa de depreciação de 20% ao ano, sua vida útil é de 5 anos, portanto 60 meses. O sistema já sugerirá os meses de acordo com a taxa informada.
CC ICMS a Recuperar:
 Conta contábil a ser utilizada no caso de ter crédito.
CC ICMS a Recolher:
 Conta contábil a ser utilizada no caso de ter débito.
Nº Meses CIAP:
 Caso participe do CIAP, informar o número de meses. Normalmente são 48 meses a partir da data de aquisição.
Sai no SPED PIS/COFINS:
 Caso participe do SPED PIS/COFINS, deve marcar esta opção. Na inclusão de uma nova classificação, por padrão, já vem marcada esta opção.
Acione o botão 
OK
 para atualizar os dados no Sistema.
No SISPETRO local também tem uma tela de Classificação de Produtos, porém os dados utilizados para o módulo de Patrimônio (Bens) só poderão ser editados no módulo Contábil.
O Sistema retorna a tela de navegação.
Para deixar a tela de Classificação do Produto acione o botão 
OK
 ou 
Cancela
.
