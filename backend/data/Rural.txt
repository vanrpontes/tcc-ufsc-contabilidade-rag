Dados Gerais

O Banco Rural pode trabalhar individualmente como também em conjunto com o Banco Bradesco (237). Ou seja, se individualmente, emite o próprio boleto do Banco Rural. Se optar por trabalhar em conjunto com o Banco Bradesco, assim, a empresa que tem conta no Banco Rural, onde adquire o produto Cobrança Eletrônica, porém, ao imprimir o boleto, o imprime com as características do Banco Bradesco. Dessa forma, o cliente que recebe o boleto não precisa localizar uma agência do Banco Rural para efetuar o pagamento, podendo fazer isso perfeitamente no Banco Bradesco, que tem muito mais agências disponíveis em todo o país. Para atender essa necessidade usaremos a configuração de Banco Correspondente do SisPetro. Cabe lembrar que o arquivo de remessa é conforme o layout do Banco Rural.

Dados Técnicos




Nosso Número:
 deve ser impresso com a carteira (CC), código reduzido (SSSS), o nosso número (NNNNNNN) e o dígito verificador(D), no seguinte formato: CC/SSSSNNNNNNN-D.

Só será gravado no SisPetro, para visualização do Contas a Receber, o nosso número (NNNNNNN).

O dígito verificador é calculado no módulo 11, utilizando até a base 7.
