TITULO: Cancelar pedido Web
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926192077
--------------------------------------------------
Para cancelar um pedido Web:
Clique no Menu que fica no lado esquerdo/superior
Selecione a opção 
Pedidos / Visualizar
Informe o código ou o nome do cliente e a data para realizar o filtro.
Todos os pedidos incluídos via SispetroWEB serão apresentados - conforme filtro realizado. 
Clique no número do pedido
O pedido será aberto.
Selecione a opção 
Cancelar Pedido
, no fim da página.
Só poderão ser cancelados os pedidos que não estiverem com status 
Aprovados
 ou 
Faturados.
