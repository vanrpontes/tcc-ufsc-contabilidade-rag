TITULO: Layout para importação de lotes
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2900853622
--------------------------------------------------
Para que a importação seja feita, o arquivo .TXT precisa estar de acordo com o modelo desenvolvido pela Futura.
Os campos devem ser separados por 
ponto e virgula e a primeira linha deve conter o cabeçalho ;
Campo
Descrição
Número do lote
Número que servirá para agrupar os itens que devem fazer parte de um mesmo lote.
Data
Data do lote, formato dd/mm/yyyy
Código histórico
Código histórico (usado no lote gerencial)
Histórico
Texto sobre histórico, quando não informado será a descrição do Cód. histórico (usado no lote gerencial)
Número do documento
número que identifica este lote, informação opcional
Informações adicionais
Informações adicionais do lote
, informação opcional
Número NF
Número da nota fiscal, informação opcional
Valor do lote
Valor do lote: decimais informar com vírgula - não usar separador de milhar
Conta
Código da conta contábil
Valor
Valor do lançamento
: decimais informar com vírgula - não usar separador de milhar
Código histórico
Código histórico do item (usado no lote real)
Histórico
Texto sobre histórico, quando não informado será a descrição do Cód. histórico do item (usado no lote real)
Informações adicionais
Informações adicionais do item
, informação opcional
Débito ou crédito
D ou C
Centro de custos
Código do centro de custos
, informação opcional
