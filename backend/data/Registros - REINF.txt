Detalhamento dos Registros
R-2010
: é o registro que aponta os valores de Contribuição Previdenciária sobre Serviços tomados (como obra)
São gerados automaticamente ao clicar no botão 
Preenche
 da Manutenção do REINF. Para mais informações, ver 
Preenchimento Automático REINF - R2010
R-2020
: é o registro de Retenção de Contribuição Previdenciária - Serviços Prestados
Os dados devem ser informados manualmente seguindo o Guia Prático do Reinf. Este evento não foi testado por falta de informações.
R-2040
: é o registro dos valores decorrentes de recursos passados para Associações Desportivas (doações)
Os dados devem ser informados manualmente seguindo o Guia Prático do Reinf. Este evento não foi testado por falta de informações.
R-2070
: é o registro que envia, por beneficiário, quais as retenções feitas das contribuições citadas (Retenções na Fonte - IR, CSLL, Cofins, PIS/Pasep) 
Removido pela Receita Federal para remodelação na versão 1.4
R-2099: 
Fechamento dos Eventos
Gerado automaticamente, porém, em alguns casos deve ser alterado manualmente, dependendo dos eventos que estão sendo enviados. Por padrão, todos os eventos (registros R-20xx) estão marcados como N.
R-5011
: Retorno da Consulta de Dados
Gerado automaticamente
R-2098:
 Reabertura e a Consulta de Dados de Fechamento
Gerado automaticamente
R-4010:
 é o registro que aponta pagamentos, créditos a beneficiário pessoa física
São gerados automaticamente ao clicar no botão 
Preenche
 da Manutenção do REINF. Para mais informações, ver 
Preenchimento Automático REINF - R-4010
R-4020:
 é o registro que aponta pagamentos, créditos a beneficiário pessoa jurídica
São gerados automaticamente ao clicar no botão 
Preenche
 da Manutenção do REINF. Para mais informações, ver
 
Preenchimento Automático REINF - R4020
R-4040:
  é o registro que aponta pagamentos, créditos a beneficiários não identificados
Os dados devem ser informados manualmente seguindo o Guia Prático do Reinf.
R-4080:
 é o registro de Retenção no recebimento
Os dados devem ser informados manualmente seguindo o Guia Prático do Reinf.
R-4099:
 fechamento/reabertura dos eventos da série R-4000
Os dados serão sugeridos conforme preenchimento dos registros da série R-4000
R-9000: 
Exclusão de Eventos
Este evento ocorre quando já foram enviados eventos incorretos e devem ser corrigidos.
R-1000: 
Informações do Contribuinte (a ser enviado somente se houver alteração)
São gerados automaticamente ao clicar no botão 
Preenche
 da Manutenção do REINF. Para mais informações, ver 
Preenchimento Automático REINF - R-1000
