TITULO: Pré-Adiantamento a Fornecedores
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878701884
--------------------------------------------------
O pré-adiantamento a fornecedores, como o próprio nome diz, é uma etapa que antecede ao adiantamento, ou seja, o usuário inclui um pré-adiantamento no SisPetro, o qual não movimenta o financeiro, e permite que outro ou o mesmo usuário aprove ou não este pré-adiantamento. Após a aprovação, o usuário da área financeira efetiva este pré-adiantamento, informando os dados da conta de onde irá sair este adiantamento e outras informações relevantes ao procedimento de adiantamento.
Para incluir um pré-adiantamento a fornecedores, acessar o menu 
Movimentação / Financeiro / Contas a Pagar / Manutenção Pré-Adiantamento a Fornecedores.
Funcionalidade dos Botões
Inclui:
 inclui um pré-adiantamento. 
Para mais informações, ver
 
Inclusão de Pré-Adiantamento
;
Altera:
 altera um pré-adiantamento com status pendente selecionado na tela. 
Para mais informações, ver
 
Alteração de Pré-Adiantamento
;
Exclui:
 exclui um pré-adiantamento com status pendente selecionado na tela.
Cancela:
 cancela um pré-adiantamento com status pendente ou aprovado. 
Para mais informações, ver
 
Cancelamento de Pré-Adiantamento
.
Aprova:
 aprova o pré-adiantamento selecionado na tela. 
Para mais informações, ver
 
Aprovação de Pré-Adiantamento
.
Efetiva:
 efetiva o pré-adiantamento com status aprovado. 
Para mais informações, ver
 
Efetivação de Pré-Adiantamento
.
Layout Default:
 retorna a tela do painel Principal no formato padrão.
Legenda:
 exibe a legenda com as cores apresentadas na tela.
Sair:
 sai da tela de Manutenção de Pré-Adiantamento a Fornecedor.
As funções incluir, alterar, excluir, cancelar e aprovar dependem de permissão de perfil de usuário. Para habilitar a permissão, deve acessar o menu Ferramentas / Usuário / painel Perfil / painel Financeiro

Filtro dos Dados
Período de Inclusão:

Por padrão, o período apresentado será de 30 dias contados da data atual.
O usuário poderá alterar as datas, ficando gravadas como padrão para esse usuário.
