TITULO: Inclusão de Transportadoras Web
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926198477
--------------------------------------------------
Para incluir uma transportadora no SispetroWeb, é necessário habilitar o direito de usuário, localizado em 
~Menu Ferramentas / Usuários - Perfil - Fretes
:
O menu de transportadora só é habilitado caso o perfil de cliente definido no SispetroMon esteja como Congênere ou Ambos (
Menu SispetroWeb / Definição Perfil Cliente
).
Clique no Menu que fica no lado esquerdo/superior
Selecione a opção 
Tabelas / Transportadoras
Selecione a opção 
Incluir
 na parte inferior direita da tela:
Preencha com as informações da transportadora:
Clique em 
Salva
 ou 
Cancela
 para finalizar a inclusão.
Toda transportadora incluída pelo SispetroWeb fica bloqueada e depende do desbloqueio para a utilização. Para desbloquear, acesse 
.
