Descrição do produto para a ANP
Implementada tabela interna com informações da descrição dos produtos, onde o código referência é o código DPMP do cadastro de 
.
Esta tabela não terá manutenção feita pelos usuários. Caso identifique necessidade de manutenção, acione o Suporte da Futura.
O Sispetro não bloqueará a emissão em casos do código não existir na tabela. Para esses casos, o sistema da Sefaz retornará a informação de Falha de Schema, apresentando a informação de que a Tag da Descrição está vazia.

Importação de NF:
Para a importação das Notas Fiscais na vesão 4.0 o frota não vem mais no XML.
Implementado um parâmetro na Natureza de Operação (
Importação NF (Painel Geral)
 / 
Vínculo de OC e PMM em Notas Fiscais Importadas
) que permite definir se a empresa deseja ou não 
salvar na Nota Fiscal o frota indicada na OC na Importação de NF. 
Para isso, além de marcar o Perfil na Natureza de Operação, pode-se desmarcar o perfil de usuário "Exibe OC's com a mesma Placa da NF" e ter um De/Para de "" (branco) para um frota Padrão.
Mudanças no arquivo .XML 
FCP (Fundo de Combate a Pobreza)
Ajustes nas Tags de impostos, que variam de acordo com o CST utilizado nos itens
Quando o CST for 060
Se o produto for Combustível, enviar o grupo ICMSST
Nos dados do Remetente será utilizado o PMPF da Origem
Nos dados do Destinatário será utilizado o ICMS Retido
Se o produto 
NÃO
 for Combustível, enviar o grupo ICMS60
Para envio do ICMS Próprio as tags utilizadas serão FCP:
vBCFCP - Base de Cálculo
pFCP - Alíquota
vFCP - Valor do FCP
Para envio do ICMS Substituição, as tags utilizadas serão FCPST:
vBCFCPST - Base de Cálculo
pFCPST - Alíquota
vFCPST - Valor do FCP
Para envio do ICMS Retido (ICMS60), as tags utilizadas serão FCPRet:
vBCFCPRet - Base de Cálculo
pFCPRet - Alíquota
vFCPRet - Valor do FCP
Para envio do ICMS Retido (ICMSST) as tags utilizadas serão FCPSTRet:
vBCFCPSTRet - Base de Cálculo
pFCPSTRet - Alíquota
vFCPSTRet - Valor do FCP
Inclusão de totais, como Tags, com o Valor do FCP
vFCP - Valor do FCP Próprio
vFCPST - Valor do FCP ST
vFCPSTRet - Valor do FCP Retido
