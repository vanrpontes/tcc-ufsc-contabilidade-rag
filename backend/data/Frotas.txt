Para acessar a tela de Manutenção de Frotas, acessar o 
Menu
 
Início > Tabelas Aux > Frotas.
Na tela de Manutenção e em algumas outras telas do SispetroApp, podemos realizar alguns filtros através dos botões abaixo:
Legenda:
Digite alguma informação para efetuar a pesquisa
Incluir um Frota
Editar Frota selecionado
Buscar dados mais recentes
Filtrar dados
Imprimir
Definir rodapé da tabela
Selecionar colunas a serem exibidas na tela
Resetar layout da tabela para layout padrão
No Rodapé da Tela, temos alguns botões que permitem a navegação entre as páginas listadas e também podemos definir o número de linhas por página:
Para editar um Frota, selecione o registro desejado e clique no botão 
Para salvar as alterações, clique no botão
 
Confirmar
 
ou
 
Cancelar
 
para descartar.
