TITULO: Provisão de PMM no Fluxo de Caixa
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/4764270593
--------------------------------------------------
Pode ser realizado o provisionamento de uma compra e este controle pode ser realizado na tela de Fluxo de Caixa, para realizar o controle do provisionamento é necessário realizar um PMM que deve estar vinculado a uma OC, após a realização do vinculo é necessário realizar um recebimento de material para vinculando a OC para realizar o abatimento desta forma será possível gerenciar pela tela de Fluxo de Caixa as compras de mercadorias, e abatimentos de lançamentos futuros.
Para lançar um PMM é necessário ir no menu: 
Movimentação de Material / Pedido de Movimentação de Material / Pedido de Movimentação de Material
utilizando o tipo de recebimento  
via transferência entre depósitos
Realizar a OC de Transferência entre Depósitos  é necessário habilitar o campo 
Importa Pedido Mov. Mat
Informar o Número do PMM
Realizar o recebimento através de Ordem de Carregamento:
Na tela de Fluxo de Caixa será possível habilitar o campo Provisão de PMM para exibir os dados de Efetivo e Provisão
Ao clicar duas vezes no item é possível visualizar os PMM que são vinculados a provisão.
