TITULO: Movendo relatórios - Versão 7.2
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2887680051
--------------------------------------------------
Apenas na versão 7.2 do Sispetro Normal não se deve mover / arrastar um relatório de uma determinada pasta para outra do Gerenciador de Relatório, essa ação ocasiona uma mensagem de erro de Formato de Matriz inválido e inutilizará o relatório
.




Caso exista esta necessidade 
Não Execute
 o processo de mover / arrastar o relatório, 
deve ser feito o seguinte procedimento
:




No Gerenciador de relatórios (
Relatórios/ Gerenciador de Relatório
) localize o relatório


Edite o relatório, e no menu Arquivo acesse a opção 
Salvar Como


Informe o novo local onde o relatório deverá ficar e clique no botão Salvar.


Volte a pasta anterior e selecione o relatório, e em seguida clique no botão Excluir




Na próxima versão estaremos disponibilizando uma correção para este processo.
