Nesta tabela estão cadastradas as informações sobre os produtos utilizados pelo sistema.


A tela para manipulação desta tabela é acionada no Sispetro normal, a partir do Menu principal com as opções -
 
Arquivos / Produtos.
 
No caso da geração da DIC precisamos informar o campo
 
Código do Produto Relevante,
 
que encontra-se no painel
 
Detalhes / Dados Adicionais,
 
devemos fazer uso dos códigos conforme PORTARIA N.º 1.505/2002-SEFAZ, de 26 DE NOVEMBRO de 2002.




Abaixo estão listados os códigos para cada produto Relevante referente a combustíveis.
