TITULO: Como alterar o número de Nota Fiscal sugerido
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878440424
--------------------------------------------------
Para alterar o número da Nota Fiscal sugerida pelo Sistema, deve-se selecionar pela tela da 
Natureza de Operação
, que normalmente utilizamos para nossas Notas Fiscais de Saída, uma natureza de saída. 


Acione o painel Detalhes desta tela para verificar a série controladora do número de Notas Fiscais para esta Natureza de Operação. 


O campo Entrada/Saída normalmente é o 
SAI
 (mas poderia ser qualquer outro código - utilize o correto). 


Agora, deve-se ir até a manutenção de 
Série de Notas Fiscais
 (Tabelas Auxiliares / Série de Notas Fiscais) e encontrar a série que desejar (posicione o cursor sobre a série com o mesmo código encontrado anteriormente). 


Acione o painel Detalhes desta tela e mude o número para o 
próximo número
 que deseja imprimir. 


Aperte OK para confirmar.
