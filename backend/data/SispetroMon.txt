Objetivos
Este documento é direcionado para detalhar o que é, como funciona e como deve ser configurado o Sispetro Monitor (SispetroMon).
O que é o SispetroMon
O SispetroMon é uma plataforma onde é feita a parametrização e o monitoramento do SispetroWeb/SispetroApp.
Principais Funções do SispetroMon
Monitorar o SispetroWeb/SispertroApp a partir de seu log.
Permitir editar os arquivos de configuração do SispetroWeb/SispertroApp numa interface Web.
Disponibilizar interface Web para controlar/editar todos os parâmetros necessários para realizar os procedimentos acima.
Sincronizar arquivos com a nuvem (Ordens de Carregamento).
Rastrear e apagar arquivos (Ordens de Carregamento) que foram incluídos a mais de 24 horas nas pastas 
Processados
 e 
Erros
.
Monitorar pasta para inclusão automática OC's no SispetroWeb/SispetroApp
Monitorar Servidores Sispetro
