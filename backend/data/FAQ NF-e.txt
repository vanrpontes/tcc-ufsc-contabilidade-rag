Dúvidas Frequentes
Qual layout da NF-e devo usar?
A partir do dia 01/04/2011 o layout exigido pelo Governo é o 2.0, lembrando que este está disponível a partir da versão 7.7 do SisPetro.
Aonde é configurado este layout?
É um perfil de sistema, acessado por: Ferramenta/ Sistema/ Aba nota fiscal / Versão dos dados SEFAZ.
Qual é o modelo de NF que preciso usar?
Para notas fiscais eletrônicas é preciso usar o modelo 55, acessado por Tabelas Auxiliares / Faturamento / Natureza de Operação / aba detalhes, aba Dados adicionais => Modelo NF.
Quando utilizava a versão 1.0, nenhuma nota eletrônica ocorria erros, depois que mudei o layout ocorrem erros frequentes.
Sim, isso pode acontecer pois na versão 1.0 não eram validados muitos registros que a versão 2.0 valida.
Como posso identificar os erros da NF-e, falha de schema , base de cálculo difere do calculado...?
Os erros são identificados pela própria Sefaz, o SisPetro não valida se todos os campos estão de acordo com a legislação e produto. Aconselhamos a instalarem o programa da Sefaz do Rio Grande do Sul, chamado AssinadoRS, baixado pelo 
AssinadorRS
 , nós temos um "passo a passo" de como configurar e validar o arquivo .XML, vide help 
AssinadorRS
Os erros mais corriqueiros são: CFOP, NCM, CST configurados errados, assim gerando erros de ICMS Subst, base ICMS errada, PIS e CONFIS...
