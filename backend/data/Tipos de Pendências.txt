TITULO: Tipos de Pendências

--------------------------------------------------
Através da tela de 
Tipos de Pendências
, o usuário poderá cadastrar as 
pendências que serão usadas ao se cadastrar uma pendência
.


Para acessar a tela de 
Tipos de Pendências
, o usuário deverá utilizar o Menu 
Ferramentas / Contatos / Tipos de Pendências
:




Para incluir um novo tipo de pendência, o usuário deve utilizar o botão 
.


O sistema se posicionará no painel 
Detalhes
, aguardando a digitação do campo 
Descrição
:




O campo 
código
 é preenchido automáticamente pelo sistema, portanto, após preencher o campo 
descrição
, para salvar as modificações, basta o usuário clicar no botão:




O sistema salvará as alterações se posicionando no painel 
Navegação
:




Para cancelar qualquer alteração ou o processo de inclusão, o usuário deve utilizar o botão
