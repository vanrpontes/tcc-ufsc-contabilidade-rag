--------------------------------------------------
Para efetivar o pedido de pré-adiantamento a fornecedor, acessar o menu 
Movimentação / Financeiro / Contas a Pagar / 
Manutenção de Pré-Adiantamento a Fornecedores
 - botão 
Efetiva
.
Alguns campos são preenchidos conforme cadastrado no pré-adiantamento, como Código Entidade, #Documento, Data Movimento, Valor Original, Código Histórico, Histórico e Informações Adicionais. Para o preenchimento dos demais campos, ver tópico 
Inclusão de Adiantamento a Fornecedores
.
Ao efetivar o pedido, será incluído o adiantamento a fornecedor.
A efetivação do pedido de pré-adiantamento será permitido se:
o pedido deve estar 
aprovado
.
o usuário deve ter permissão para efetivar. Para habilitar a permissão, deve acessar o menu 
Ferramentas / Usuário / painel Perfil / painel Financeiro
:
na efetivação será possível realizar a alteração da data de movimentação de pré-adiantamento, a nova data será exibida na tela de efetivação, necessário habilitar 
Pode alterar data de movimento na efetivação de pré-adiantamento
.
 
Temos a opção de trabalhar com grupo econômico para fornecedores, onde ao trabalhar com grupo econômico será possível alterar o Fornecedor para o Fornecedor representante do grupo econômico. Veja também 
Grupo Econômico - Fornecedores
