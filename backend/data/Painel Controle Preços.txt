Conhecendo os perfis existentes:
Perfil Controla preços por cliente?
Este perfil permite configurar o Sispetro para trabalhar com preços por cliente. Veja maiores informações para configurar preços por cliente em 
Preços (Clientes)
.
Perfil Controla Data de Vigência da Tabela de Preços em Clientes
Este perfil permite configurar data de vigência na tabela de preço no cadastro de cliente. Veja maiores informações para configurar preços pro cliente em 
Preços (Clientes)
.
Perfil Apresentação dos Produtos
Somente do tipo de combustível:
 
Lista os produtos com o Tipo de Produto Combustível conforme definido no 
Cadastro de produtos
. Esta opção vai apresentar os produtos independente se no cadastro de produtos está marcada a opção "Sai na lista de preços (permite venda)".
Todos os produtos configurados para sair na lista de preços: 
Lista todos os tipos de produtos. Esta opção só vai apresentar os produtos que no 
Cadastro de produtos
 a opção "Sai na lista de preços (permite venda)" esteja marcada.
Note que isso não afetará em nada como o Sistema irá buscar os preços para cada cliente pois o Sispetro continuará buscando os preços a partir das tabelas de preço. Porém, como será verificado na Aba Preços (de cada cliente), ele disponibiliza uma série de ferramentas para facilitar a geração de novas tabelas de preços (que poderão ser únicas para cada cliente) além da facilidade de se editar os preços a partir da tela de clientes e da possibilidade de definir uma data de vigência para cada produto.
