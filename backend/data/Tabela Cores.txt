TITULO: Tabela Cores
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878439621
--------------------------------------------------
Nesta tabela são cadastrados os corantes, que normalmente são adicionados para dar cor a um produto, e que são vinculados aos produtos no 
Cadastro de Clientes
.
A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Vendas / Cores
:
Para acrescentar uma nova Cor acione o botão de inclusão de registro 
.
O Sistema exibe a tela para preenchimento dos novos dados.
Informe uma 
Descrição
 e um 
Código de Impressão
 e acione o botão 
OK
 para atualizar os dados no Sistema.
O Sistema retorna a tela de navegação.
Para retornar ao Menu principal acione o botão 
Sair
