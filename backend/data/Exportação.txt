TITULO: Exportação
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210335
--------------------------------------------------
Neste painel é possível cadastrar códigos de exportação para as contas, diferenciado por filial. Assim, os lançamentos desta conta espcíficos de uma filial podem ter também um código de exportação específico. No caso de não ser necessário esta distinção, basta colocar o mesmo código de exportação para todas as filiais (isso o sistema já sugere assim que é preenchido o primeiro código de exportação).




Informe o código de exportação para a filial na coluna específica. Ao terminar de digitar todos, clique no botão
 
Salva
, para cancelar clique no botão
 
Cancela.


Cabe lembrar que, caso alguma filial não tenha o código de exportação preenchido, isto acarretará na não exportação dos dados desta filial.


Na parte inferior da tela é exibida a descrição da conta posicionada.
