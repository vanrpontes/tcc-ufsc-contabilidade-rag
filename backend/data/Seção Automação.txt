Antes de utilizar a configuração abaixo acione nosso Suporte Técnico.
Nesta seção são informados os parâmetros necessários para conexão com o banco de dados SQL Server para 
Automação View
.
DataSource:
 Nome da fonte de dados ODBC criada para conexão com o banco de dados SQL Server.
Usuario:
 Nome do usuário no banco de dados SQL Server para conexão.
Senha:
 Senha para conexão no banco de dados SQL Server
No arquivo scg.ini a seção de automação deve ficar similar a figura a seguir:
[Automacao]
DataSource=AutomacaoView
Usuario=sa
Senha=010101
Veja maiores detalhes em como providenciar a criação da fonte de dados ODBC em 
Informações Técnicas para Automação da View
.
