TITULO: Tabela de Comissão de Vendas

--------------------------------------------------
Esta tabela de comissão permitirá ao usuário cadastrar todos os tipos de comissão existentes.
Este painel permite a inclusão desta tabela, é aciona a partir do Menu 
Tabelas Auxiliares / Vendas / Comissão:
Para acrescentar uma nova Comissão à tabela, acione o botão de inclusão de registro 
.
O SisPetro exibe a tela para preenchimento dos dados.
Digite o conteúdo para os campos:
Descrição: 
Informe uma descrição para a comissão.
Comissão Ativo: 
Marcando a comissão estará ativo, quando desmarcado ficará inativo.
Acione o botão OK para atualizar os dados no SisPetro.
Nesta tela exibe os itens onde estão cadastrados os códigos, descrição e percentual de comissão 
No rodapé temos os seguintes botões:
Incluir: 
Permite cadastrar o percentual de comissão dos produtos, selecione o produto e insira o percentual, click em OK para salvar
Produto: 
Informe o código do produto
Sequencial: 
Automático
Código Tabela Comissão de Vendas: 
Automático. 
Excluir: 
Exclui o produto da tabela de comissão de vendas
Edita: 
Permite alterar o percentual do produto cadastrado
Incluir todos: 
Inclui todos os produtos na tabela de comissão de vendas
Percentual Único: 
Permite cadastrar o percentual de comissão único para todos os produtos
