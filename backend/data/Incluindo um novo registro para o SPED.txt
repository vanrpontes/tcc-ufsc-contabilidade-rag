TITULO: Incluindo um novo registro para o SPED
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2879128169
--------------------------------------------------
A inclusão de um novo registro no Conjunto de Dados não afetará as tabelas reais (por exemplo Nota Fiscal), ele afetará somente as tabelas preenchidas conforme o Preenchimento dos Dados para o SPED. Sendo assim, os registros originais não serão afetados.
Para a inclusão de um novo registro para o SPED, o usuário deverá acessar a tela de Conteúdo de Tabela (Ver tópico para 
Edição de Dados
).
E na tela de 
Edição/ Inclusão
, deverá informar os dados solicitados, conforme a tabela selecionada.
Importante: Os campos destacados na cor verde NÃO poderão sofrer alterações, pois são campos gerados automaticamente ou através de cálculos. No quadro de Tabelas Dependentes, o Sispetro traz todas as tabelas vinculadas à tabela selecionada anteriormente.
Após informar os dados solicitados, o usuário deve clicar no botão 
Salvar
.
 
O usuário poderá cancelar a inclusão a qualquer momento, para isso ele deve clicar no botão 
Cancelar
 
E confirmar (ou não) o cancelamento através da mensagem:
 
Após salvar os dados para inclusão de um novo registro, o usuário poderá visualizar o registro através da tela de Conteúdo de Dados.
Após a inclusão dos dados, o usuário poderá gerar o arquivo txt do SPED.
Para sair da tela, o usuário deverá clicar no botão "Fechar".
