TITULO: Desbloqueio Manual de Crédito (Adiantamento)
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878701870
--------------------------------------------------
Para realizar o desbloqueio de créditos (adiantamento) bloqueados, o usuário deve acessar o menu 
Movimentação/Financeiro/Contas a Receber/ Desbloqueia Créditos
.




Deve ser informado um período, nos campos 
Data Desbloqueio Inicial
 e 
Data Desbloqueio Final
, utilizando a tecla <TAB> para saltar de um campo para outro. Após informar a data de Desbloqueio Final, o usuário deve utilizar a tecla <TAB> para que o sistema processe as informações e mostre na tela os créditos (adiantamentos) bloqueados no período selecionado.




Para desbloquear os créditos, basta o usuário clicar no botão 
OK
. Assim, todos os registros mostrados na tela serão desbloqueados. Para evitar que determinados registros sejam desbloqueados por engano, o usuário deve restringir os registros exibidos, através da data de desbloqueio.




Exemplo:


O crédito que deve ser desbloqueado, pertence a uma data especifica. Ao entrar na tela o usuário deve informar essa data nos campos 
Data Desbloqueio Inicial
 e 
Data Desbloqueio Final
.




Ao informar uma data especifica, apenas os registros dessa determinada data serão exibidos na tela, evitando assim, que seja informado dados de outros períodos e por engano seja feito o desbloqueio.


Apesar de ter na tela o número de registros restringido, o usuário pode precisar desbloquear apenas um registro, caso se tenha mais de um registro exibido para o período informado.


Neste caso, o usuário pode utilizar o botão 
Muda Data Desbloqueio
 para trocar a data original do desbloqueio dos outros registros que não serão desbloqueados. Esse recurso pode ser usado toda a vez que for necessário alterar a data de desbloqueio de algum registro.




O usuário deve estar posicionado sobre o registro que se deseja mudar a data e clicar no botão 
Muda data desbloqueio
. O sistema abrirá uma nova tela, onde deverá ser informada uma nova data de desbloqueio para aquele registro.




Para confirmar a mudança de data, deve-se utilizar o botão 
OK
 e o botão 
Cancela
 para abortar o processo.


Ao fechar a tela de mudança de data de desbloqueio de créditos, o sistema retornará para a tela de Desbloqueio. Se o objetivo era apenas trocar a data do desbloqueio, o usuário deve fechar a tela de desbloqueio, clicando no botão 
Fechar
. Caso o usuário clique no botão 
OK
, todas os créditos mostrados na tela serão desbloqueados.
