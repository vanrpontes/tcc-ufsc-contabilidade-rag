Na versão 9.15 foi implementado a tabela de cadastro de motivos de bloqueio para transportadora, através dessa tabela o usuário cadastra os motivos à serem utilizados no bloqueio das transportadoras, veja mais em 
.
A tela que permite a manipulação desta tabela é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Transportes / Distancias
:
A tabela só ficara habilitada para uso caso o usuário esteja com o perfil 
'Pode bloquear/desbloquear Transportadoras'
 habilitado. Veja mais em 
Painel Fretes (Usuários)
Após acionar o botão incluir sera apresentado a tela abaixo, onde usuário inclui uma descrição e seleciona se o motivo ira ficar ativo ou não.
 
Só sera permitido realizar a exclusão de motivos que não foram utilizados, o motivo em questão foi utilizado em algum bloquei de trasportadora será apresentado a mensagem baixo.
