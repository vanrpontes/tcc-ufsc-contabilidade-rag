TITULO: Movimento entre Depósitos
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2885845059
--------------------------------------------------
Esta tela possui a finalidade de efetuar movimentações entre depósitos internos da empresa.
Por exemplo:
 Depósito Almoxarifado, onde é possível movimentar produtos existente para outro depósito de outra filial.
Para utilizar esta tela o acesso deve ser feito através do menu 
Movimentações/ Estoque/ Manutenção de Movimento entre Depósitos
.
Para incluir uma movimentação é necessário clicar no botão 
Inclui
.
Na tela de 
Inclusão de Movimento entre Depósito
 o usuário deverá informar:
 
Data Movimento
: a data de movimentação do produto entre os depósitos.
Quadro Origem: 
dados do depósito de origem
Empresa:
 Código da empresa de origem, onde se encontra o produto. Pode ser a mesma empresa.
Depósito:
 Código do depósito de origem.
Cód. Histórico:
 Código do Histórico a que se refere a movimentação.
Descrição:
 Descrição do Histórico utilizado.
Movimenta Estoque Físico:
 Marque essa opção para que o estoque físico seja movimentado junto com o escritural.
Quadro Destino: 
dados do depósito de destino
Empresa:
 Código da empresa de destino, onde se encontra o produto. Pode ser a mesma empresa.
Depósito:
 Código do depósito de destino.

 Código do Histórico a que se refere a movimentação.
Descrição:
 Descrição do Histórico utilizado.
Movimenta Estoque Físico:
 Marque essa opção para que o estoque físico seja movimentado junto com o escritural.
Quadro Itens:
Itens que serão movimentados.

Pontos importantes a considerar:
A opção 
Movimenta Estoque Físico
 somente será habilitada quando o Depósito for 
Base Própria
.
As opções de seleção de 
Tanque, Temperatura Tanque, Temperatura Amostra e Densidade
, encontradas na tela 
Itens da Movimentação entre Depósitos
, somente estarão habilitadas caso a opção 
Movimenta Estoque Físico
 esteja marcada.
Ao informar os códigos dos tanques, os dados de temperatura e densidade são carregados automaticamente, caso os tanques tenham tais dados cadastrados.
Clicando no botão 
Régua
, abre a tela Dados da Medição, para informar os dados da leitura dos tanques origem e destino:
No campo 
Observações
 informar dados de trena, termômetro, densímetro, etc.
Será de inteira responsabilidade do usuário caso volte para a tela de inclusão de itens e altere algum dado obtido da tela de Medição, como quantidade 20c, temperatura, etc. Não será feita qualquer validação.
Após incluir os dados das medições dos tanques, as informações são atualizadas conforme as medições:
 
Tela 
Itens da Movimentação entre Depósitos:
 após incluir as informações
 
Em uma única inclusão de movimentação entre depósitos, podem ser incluídos quantos itens desejar. Para cada item movimentado fisicamente, serão impressos 2 páginas de medição: origem e destino.
Para imprimir e alterar layout dos relatórios, clique no botão 
Imprime.
 
Relatório dos dados da transferência.
 
CACL do tanque origem
 
CACL do tanque destino
 
Ao clicar em 
IMPRIME
, primeiro é gerado o relatório dos dados da transferência, clicando em Fechar, gera o CACL do tanque origem (caso tenha mais de 1 1tem, será gerado 1 CACL para cada item), clicando em Fechar novamente é gerado o CACL do tanque destino (caso tenha mais de 1 item, será gerado 1 CACL para cada item).
Para cancelar a movimentação, deve-se clicar no botão 
Cancelar
.
Para atualização da tela, deve-se utilizar o botão 
Atualizar
.
Para localização de algo específico na tela, deve-se utilizar o botão 
Localiza
.
Para criar um filtro específico, deve-se utilizar o botão 
Filtra
.
Para sair da tela, clique no botão 
Sai
.
