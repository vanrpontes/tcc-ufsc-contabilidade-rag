TITULO: Tabela de Subgrupo de Despesas
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878211036
--------------------------------------------------
Nesta tabela são cadastrados os Subgrupos de Despesas a serem utilizados no 
Controle de Despesas com Frota
.
O 
Subgrupo de Despesas
 é uma subdivisão do Grupo de Despesas para melhor organizar os eventos relacionados ao frota.
Exemplo:
 Grupo: Pneus
 Subgrupos: Compra, Consertos, Recauchutagem
A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Transportes / Frotas / SubGrupo de Despesas
:
Para acrescentar um novo SubGrupo de Despesas acione o botão de inclusão de registro 
.
O Sistema exibe a tela para preenchimento dos novos dados.
Informe a 
Descrição
 para o SubGrupo de despesas e qual o 
Grupo
 que este subgrupo pertencerá. O grupo será cadastrado em 
Grupos de Despesas
.
Caso haja a necessidade de ativar/ inativar um subgrupo, marque/ desmarque o campo 
Ativo
.
Acione o botão 
OK
 para atualizar os dados no Sistema.
O Sistema retorna a tela de navegação.
Para retornar ao Menu principal acione o botão 
Sair
 
.
