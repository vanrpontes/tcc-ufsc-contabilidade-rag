Aba Tabela de Preços/Tipo Cliente
Ao acessar a aba Tabela de Preços / Tipo Cliente, a tela abaixo será exibida:
Nesta tela será permitido vincular uma tabela de preço a um determinado Tipo de Cliente (
), para uma empresa especifica ou para todas as empresas, dessa forma permite o agrupamento de clientes com características similares. Com este vínculo Tipo Cliente x Tabela de Preços será possível ter um  preço específico para cada Depósito.
Este recurso foi desenvolvido devido a grande diferença de custos entre as bases utilizadas pelas distribuidoras para armazenar os seus produtos. 
Para habilitar esta tela é preciso que o usuário tenha o perfil em Ferramentas > Usuário > aba Perfil > aba inferior Vendas > 
Pode Alterar Tabela de Preços por Tipo de Cliente no Depósito
Ação ao não encontrar Tabela de Preço para um Tipo de Cliente
Usa somente a Tabela de Preço do Cliente: 
utilizando essa opção será utilizada a tabela de preço configurada no tipo de cliente que está cadastrado no cliente.
Impede a Venda: 
ao incluir um pedido de venda caso não tenha uma tabela de preço cadastrada para o tipo de cliente que foi informado no cliente para a empresa logada, a venda não será permitida.
Usa a Tabela de Preço de um Tipo Cliente Específico: 
utilizando essa opção será utilizada a tabela de preço configurada no tipo do cliente que está cadastrado no cliente, mas caso não seja encontrado o tipo de cliente que foi informado no cliente do depósito, será utilizada a tabela de preço cadastrada no Tipo de Cliente Padrão.
Tipo de Cliente Padrão
Tipo de Cliente Padrão: 
Caso seja utilizado a opção de
 Usa a Tabela de Preço de um Tipo Cliente Específico
, ao incluir um pedido será utilizado o tipo de Cliente padrão informado nesse campo, caso o cliente informado no pedido de venda não tenha um tipo de cliente cadastrado no depósito.
