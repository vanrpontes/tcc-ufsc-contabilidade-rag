TITULO: Referências Bancárias (Clientes)
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210642
--------------------------------------------------
A partir do painel 
Detalhes
 é possível acionar o painel 
Referências Bancárias
 localizado no menu inferior da tela.
Esta janela salva as informações vindas dos bancos aos quais o cliente tem relacionamento. Na parte superior, inclui-se os bancos e, na parte inferior, para cada banco, as informações vindas de cada um deles. Pode-se comparar as informações vindas de cada um deles para apontar discrepâncias.
Para ter acesso é necessário habilitar o perfil de usuário, 
Vizualiza Referências Bancárias, 
Painel Segurança (Usuários)
.
Bancos Restritos para Cobrança Eletrônica:
Permite cadastrar os bancos nos quais o cliente não terá cobrança eletrônica (Tipo de Cobrança). É utilizado no 
filtro de tipo de cobrança
.
Quando o cliente configurado com determinados bancos, ao emitir uma nota de venda, fará a verificação no 
filtro de tipo de cobrança
. Caso encontre um tipo de cobrança com um dos bancos relacionados nessa tela, passará para o próximo tipo de cobrança e assim sucessivamente, até encontrar um tipo de cobrança cujo banco não esteja relacionado.
