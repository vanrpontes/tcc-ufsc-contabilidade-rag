TITULO: Tabela Período DCP
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878898634
--------------------------------------------------
A tabela de Período para DCP, que é acessada pelo menu 
Tabelas Auxiliares / Estoque / Período DCP
, deve ser atualizada para cada produto sempre que mudar o período. O período 
 é necessário para que se emita o DCP e o cadastro de composição dos produtos também é útil para o 
.
Esta atualização deve ser feita sempre incluindo num novo registro para o período e nunca alterando simplesmente um período existente, para que fiquem registrados todos os períodos existentes e possibilite gerar novamente um DCP de um período anterior.
É importante salientar que deve ser incluído períodos para todos os produtos utilizados no DCP.
No caso de produtos que tenham composição, quando houver mudança na composição deve-se informar a nova composição assim que for criado o novo período DCP. Desta forma, o DCP pode ser gerado em um período com uma determinada composição e outro período com outra composição diferente. A composição também será utilizada no DPMP para operações de Remessa/Retorno para a unidade interna do Informante.
