TITULO: Adiantamentos de Clientes
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878701792
--------------------------------------------------
Pela tela de manutenção de adiantamentos de clientes (menu
 
Movimentação / Financeiro / Contas a Receber / Manutenção de Adiantamento de Clientes
), pode-se realizar uma série de procedimentos com os créditos de clientes.
Botões:
Visualiza:
 Permite visualizar os dados do adiantamento selecionado pelo usuário.
Inclui:
  Permite realizar a inclusão de adiantamentos, veja o help 
Roteiro de Inclusão de Adiantamento de Clientes
.
Transf. p/ Entidade:
  Permite realizar a transferência de adiantamentos para outras entidades, veja maiores detalhes em  
.
Transf. p/ Conta: 
Permite realizar a transferência de um adiantamento para uma conta contábil, veja maiores detalhes em  
.
Transf. Empresas:
 Permite realizar transferência de adiantamento para outras empresas, veja maiores detalhes em  
Transferência de Créditos (Adiantamentos) entre Empresas
.
Transf. p/ Ñ. Identific:
 Permite realizar a transferência de adiantamentos não identificados para entidades.
Caso aja a necessidade de transferir um adiantamento não identificado para uma conta contábil, a solução é primeiro realizar uma transferência para uma entidade, e depois realizar uma transferência da entidade para a conta contábil.
Paga ao Cliente:
 Permite realizar pagamentos ao adiantamento, veja maiores detalhes em 
Pagamento de Crédito (Adiantamentos) a Cliente
.
Cancela:
 Permite realizar o cancelamento de adiantamentos, o botão só ficara habilitado para uso caso o usuário esteja com o perfil de usuário "cancela" habilitado. Veja mais em 
Painel Financeiro (Usuário)
.
Recibo:
 realiza a impressão do recibo de adiantamento.
Atualiza:
 Atualiza os dados na tela.
Sai:
 Sai da tela
Algumas versões atrás implementamos a Inclusão de Adiantamentos Multi-Empresa (o usuário consegue incluir adiantamento para uma empresa diferente da empresa conectada), porém, a Manutenção de Adiantamentos ainda se limitava a apresentação somente dos adiantamentos da empresa conectada. Visando uma maior versatilidade criamos na versão 10.6 a opção de trabalhar com a tela de Manutenção de Adiantamento Multi-Empresa, para isso o perfil de Sistema
 
'Adiantamento Multi-empresa'
 
deve estar estar selecionado a opção de
 
'Conforme permissão do usuário'
, e o Perfil de usuário
 
'Empresas X Adiantamentos'
 
deve estar selecionado a opção
 
'Pode escolher a empresa'
.
Dessa forma serão apresentados adiantamentos correspondentes às empresas que o usuário tem acesso.
Dica:
 
Ao trabalhar com a opção de Adiantamento Multi-Empresa sera criado a coluna de "Empresa' no final do grid da aba de lançamentos, o usuário pode ir ate o final do grid clicar e arrastar essa coluna para onde for melhor para o usuário.
