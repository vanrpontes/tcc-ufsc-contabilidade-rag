TITULO: Alterar pedido Web
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926192137
--------------------------------------------------
Para alterar um pedido Web:
Clique no Menu que fica no lado esquerdo/superior
Selecione a opção 
Pedidos / Visualizar
Informe o código ou o nome do cliente e a data para realizar o filtro.
Todos os pedidos incluídos via SispetroWEB serão apresentados - conforme filtro realizado. 
Clique no número do pedido
O pedido será aberto.
O usuário poderá incluir ou excluir novos itens de pedido ou alterar a observação do pedido (para incluir ou excluir itens do pedido, acesse 
).
Para gravar a alteração, selecione a opção 
Finalizar
 ou clique em 
Retornar
 para voltar para a tela de manutenção de pedidos.
Só poderão ser alterados os pedidos que não estiverem com status
 Aprovados
 ou 
Faturados
.
Para cancelar um pedido, acesse 
.
