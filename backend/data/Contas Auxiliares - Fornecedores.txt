--------------------------------------------------
Neste painel Contas Auxiliares é possível cadastrar combinações de contas para o fornecedor selecionado, empresa e produto. 

Vide tela: 




Utilize o botão 
Inclui Produtos e Empresas
 para que sejam incluídas as combinações de empresas e produtos (tipo combustível) cadastrados. Informe nas colunas NF Entrada e NF Saida 
contas gerenciais
 a serem utilizadas a cada combinação. 


Normalmente, estas contas informadas são utilizadas para a contabilização real de transferências de produtos entre filiais. Ou seja, se o fornecedor selecionado representar uma filial e tiver movimentando um produto numa determinada empresa, que conta de estoque deverá ser movimentada na entrada ou na saída. 


Porém os dados deste painel poderão ser utilizados por cada empresa para outra finalidade que exijam definições de contas específicas para as combinações de entidade X produto X empresa. 


Utilize o botão 
Salva
 para salvar os dados informados.
