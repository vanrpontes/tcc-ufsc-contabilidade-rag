TITULO: Tabela Moedas
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210973
--------------------------------------------------
Esta tabela armazena as moedas utilizadas no Sistema. Contém, no mínimo, a moeda corrente no país.
 Caso a Empresa utilize em qualquer operação (compra, venda, contas a pagar, contas a receber, controles fiscais, etc) outras moedas, é necessário seu cadastramento prévio.
A tela que permite a manutenção do cadastro de moedas no Sistema é acionada a partir do Menu 
Arquivos / Moedas
:
Para adiconar uma nova moeda ao Sistema, acione o botão de inserção de registro. 
Será exibida a tela que permite o cadastramento.
Deve ser atribuído um 
código
 e uma 
descrição
 para a nova Moeda.
O sistema permite que seja inserida uma taxa diária para a Moeda estrangeira utilizada, para que isso seja possível marque a caixa de seleção 
Informa Taxa
, será disponibilizada uma nova aba chamada 
Conversão
.
Para que a aba Conversão fique disponível, é necessário que o usuário tenhas as devidas permissões para essa operação. Acesse o 
Perfil de usuário
 na Aba 
Financeiro
, marque o perfil abaixo:
 
Na aba Conversão será possível incluir, excluir e editar a taxa da moeda.
Será possível inserir apenas uma Taxa por dia, caso a mesma seja alterada várias vezes durante o mesmo dia, utilize o botão de edição para alterar.
Acione o botão 
OK
 para efetivar a inclusão e/ou alteração.
