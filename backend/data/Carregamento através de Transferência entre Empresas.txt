O carregamento através de uma transferência entre empresas ocorre quando o usuário emite uma Ordem de Carregamento de Transferência entre empresas. Este tipo de ordem de carregamento normalmente é emitida quando a empresa necessita transferir algum produto de sua base para uma outra empresa.


Para saber como emitir a ordem de Carregamento de Transferências, veja o tópico

Portaria de Entrada
