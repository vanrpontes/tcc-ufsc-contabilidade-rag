Objetivo
Através dessa manutenção é possível cadastrar a mensagem padrão que será utilizada para enviar mensagem ao usuário. 
O acesso é feito através do menu 
Tabelas Auxiliares / Comunicações / Mensagem Padrão
Aba Navegação
Aba Detalhes
Nessa aba o usuário irá informar a descrição e qual será a mensagem enviada 
Código:
 incluído automaticamente pelo SisPetro
Descrição: 
informar qual será a descrição da mensagem 
Processo: 
qual o processo do SisPetro que será enviada a mensagem. Atualmente temos apenas no processo de 
Chamada de Motorista na Portaria
  na 
Portaria de Entrada
 
Portaria
ID Modelo: 
modelo de mensagem que esta sendo utilizado
Modelo de Mensagem
: informar qual a mensagem que será enviada no processo escolhido. 
Vincula Modelo Selecionado:
 ao clicar irá vincular a mensagem selecionada no grid, levando informação para o campo superior ID Modelo
Atualiza Modelos: 
traz a relação de mensagem pré-cadastradas na Twilio ou demais integradoras de mensagem.
