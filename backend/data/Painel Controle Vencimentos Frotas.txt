No painel 
Controle Vencimentos
 é possível cadastrar os documentos pertinentes o Frota / Veículos, que necessitem de verificação periódica e possuem data de vencimentos.
Para que o usuário possa alterar e incluir controle de vencimentos do Frota, veja o help 
Painel Fretes (Usuários)
Para efetuar o Controle Vencimentos com o Frota é preciso ter o documentos necessários cadastrados no Sispetro. Para efetuar o cadastro, veja maiores informaçõs em 
.
Como efetuar o vínculo com o Frota?
O vínculo deverá ser feito caso a caso. Portanto, para cada Frota / Veículo é preciso vincular os respectivos documentos necessários. Isto porque cada documento possui sua data de validade, o que não pode ser generalizado ao vincular com o Frota
Vejamos o que é preciso ser feito:
Acessar o painel 
Navegação
 e selecionar o Frota desejado.
Acessar o painel 
Controle Vencimentos
.
Clicar na primeira linha em branco da coluna 
Sequencial Controle
. 
Observe que o campo ficará disponível da seguinte forma:
Clique no botão 
...
 
Na tela de Pesquisa de Controle de Vencimentos, selecione um já cadastrado
Clique no Botão 
Ok
Complete as demais informações referente ao documento para o Frota / Veículo selecionado:
Tipo de Controle: 
será informado conforme o cadastro efetuado em 
.
Número do Documento:
 informar conforme o Frota / Veículo
Data Vencimento:
 Data de vencimento do documento.
É permitido informar uma data com, no máximo, 30 anos contados a partir da data de inclusão/alteração deste campo
KM Vencimento: 
 Quilometragem de vencimento do veículo.
Data última: 
data da última atualização.
KM última:
 Quilometragem da última atualização.
Repita o processo para todos os documentos necessários.
Clique em 
Ok
 para finalizar o vínculo
Havendo algum campo preenchido de forma incorreta, o Sispetro emitirá mensagens de alerta, impossibilitando a finalização do processo.
