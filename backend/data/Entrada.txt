Para ajustar o Estoque escritural com uma entrada, isto é, somar a quantidade introduzida aos saldos existentes, acione a partir do menu principal as opções 
Movimentação / Estoque / Entradas / Outras Entradas
Para que o usuário tenha acesso a essa tela o mesmo deve estar com o
 
perfil de usuário
 
Pode fazer ajuste manual de Entrada Escritural 
marcado.
 
Insira os seguintes campos
: 
 
Referência
: Um número de documento (número do contrato, por exemplo). Opcional. 
 
Depósito
: Código do Depósito de entrada do ajuste. Obrigatório. 
 
Histórico Ajuste
: Código do histórico do ajuste (o motivo).Obrigatório. 
 
Observações
: Qualquer observação que se faça necessário.Opcional.
Aperte o botão Avança.
Para cada produto que deseja ajustar, digite:
Código do Produto
: Caso não saiba, aperte <enter> para abrir a tela de pesquisa de produtos e escolha o produto desejado.
Quantidade
: Quantidade que será somada ao saldo já existente (disponível).
Custo
: Custo da mercadoria.
Aperte o botão OK para confirmar o ajuste.
