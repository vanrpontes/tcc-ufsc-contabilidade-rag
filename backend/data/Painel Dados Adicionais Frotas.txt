O painel 
Dados Adicionais
 permite ao usuário informar os dados cadastrais do Frota.
Informe os dados necessários:
Quadro Proprietário: 
Informar os dados referentes ao Proprietário do Veículo
CNPJ/CPF
Nome
IE 
(se houver)
Tipo: 
Três opções: 
TAC Agregado
, 
TAC Independente
 ou 
Outros
. Caso tenha dúvidas sobre como preencher essa informação sobre a TAC (Transportador Autônomo de Cargas), procure a legislação correspondente.
UF
Marca:
 Marca do veículo.
Modelo:
 Modelo do veículo.
Ano Fabricação:
 Ano que foi fabricado o veículo.
Ano Modelo:
 Ano do modelo do veículo.
Chassi:
 número do chassi do veículo.
P
oderá ser validado conforme perfil de sistema 
Painel Fretes
, quadro Cadastro de Frotas, opção Impede frotas com Nº de Chassi repetido. Caso esteja marcada, não permitirá incluir ou alterar qualquer frota com mesmo número de chassi, validando também caso esteja em branco. Não valida conteúdo: valida somente se está ou não preenchido. Estando preenchido não deve ser repetido.
Valor:
 Valor do veículo.
Vendido:
 Indica que o veículo foi vendido.
Comprador:
 Nome do comprador do veículo.
KM Atual:
 Quilometragem atual do veículo.
Alienado:
 Indica se o carro alienado.
#Frota Cliente:
 Número do frota no cliente.
Combustível:
 
tipo do combustível, no caso um dos produtos que esteja marcado na 
 com o tipo Combustível.
Observações:
 Descrição complementar do veículo.
Tanque sofreu alterações?:
 Permite informar se o tanque do veículo sofreu alterações a fim de auxiliar a equipe de segurança a reforçar a vistoria do veículo
Dados da alteração do tanque: 
permite informa quais alterações foram feitas no tanque do veículo.
Após informar os dados, clique no botão 
OK
 para confirmar a inclusão no sistema.
Para abortar o processo, clique no botão 
Cancela
