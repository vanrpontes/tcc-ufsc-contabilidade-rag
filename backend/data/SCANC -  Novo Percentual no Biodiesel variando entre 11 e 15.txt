TITULO: SCANC -  Novo Percentual no Biodiesel variando entre 11 e 15%
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2936701098
--------------------------------------------------
A partir de Setembro de 2019 o percentual mínimo de B100 no Biodiesel é de 11%, porém a ANP permitiu que se trabalhasse também utilizando os percentuais de 12 a 15%.
Para as
 
Distribuidoras que comercializaram nesse mês de Setembro Biodiesel utilizando mais de um Percentual de mistura e estão na versão 10.2
, é necessário, para geração do arquivo do
 
SCANC
, utilizar um
 
novo Layout de Exportação (SCANC_v102.rtm).
Para utilizar o novo layout o primeiro passo é carregar o novo RTM na manutenção de Exportação de Dados, para isso deve-se seguir os passos abaixo:
Acessar o menu Ferramentas / Exportação de Dados / Manutenção Exportação Dados.
Clicar no botão Incluir. 
Indica uma descrição, e seleciona a opção TextFile, no campo Tipo de Arquivo.
Clica em 'Carrega de Arquivo'
Localiza o layout e clica em Abrir.
Com o novo layout as NF serão enviadas para o SCANC conforme o percentual utilizado em cada produto, corretamente.
Na tela do Anexo, como ainda não foi atualizada, os saldos continuarão utilizando um único percentual (indicado na inclusão do Anexo), porém, como o SCANC recalcula os saldos, não causará problemas para a empresa. Essa tela calculará os saldos corretos a partir da versão 10.3.
