TITULO: Alteração da Data Base
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2879127771
--------------------------------------------------
A data base é automaticamente gravada pelo Sispetro no momento de sua passagem pela portaria de saída, sendo ela de Entrada ou de Saída. 
 Porém, as vezes, é necessário alterarmos uma das data posteriormente, seja pelo fato de que a data foi gravada incorretamente como também porque o motorista demorou efetivamente para iniciar a viagem.
Assim, para alterarmos uma das data, deve- se proceder da seguinte forma:
Abrir tela de 
manutenção de carregamentos/descarga (Movimentação / Base / Manutenção Carregamento e Descarga
).
Localizar o carregamento ou descarga que se deseja alterar disponível no painel 
Carregto/Descarga
.
Clicar no botão 
Altera Data
Na tela de 
Alteração Data Base
 altere a informação desejada, podendo ser 
Data de Entrada
 ou 
Data de Saída
Ao finalizar, clique em 
Ok
Caso não queira efetuar as alterações desejadas, clique em 
Cancela
 
Necessário ter o perfil de usuário 
Pode mudar data Entrada/Saída Base na Man. Carregto
 em Ferramentas > Usuários > aba Perfil > aba inferior Carregamento 
marcado
. Veja maiores informações sobre o perfil em 
.
