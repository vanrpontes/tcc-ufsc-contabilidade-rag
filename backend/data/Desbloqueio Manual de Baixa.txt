TITULO: Desbloqueio Manual de Baixa
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878701653
--------------------------------------------------
Para realizar o desbloqueio de baixas bloqueadas, o usuário deve acessar o menu 
Movimentação / Financeiro / 
Contas a Pagar ou Contas a Receber
 / Desbloqueia Baixas
:
Deve ser informado um período, nos campos 
"Data Desbloqueio Inicial"
 e 
"Data Desbloqueio Final"
, utilizando a tecla <TAB> para saltar de um campo para outro. Após informar a data de Desbloqueio Final, o usuário deve clicar no botão 
"Atualiza"
 para que o sistema processe as informações e mostre na tela as baixas bloqueadas no período selecionado.
Para desbloquear as baixas, basta o usuário clicar no botão "OK". Assim, todos os registros mostrados na tela serão desbloqueados. Para evitar que determinados registros sejam desbloqueados por engano, o usuário deve restringir os registros exibidos, através da data de desbloqueio.
Exemplo:
A baixa que deve ser desbloqueada, pertence a uma data especifica. Ao entrar na tela o usuário deve informar essa data nos campos "Data Desbloqueio Inicial" e "Data Desbloqueio Final".
Ao informar uma data especifica, apenas os registros dessa determinada data serão exibidos na tela, evitando assim, que seja informado dados de outros períodos e por engano seja feito o desbloqueio.
Apesar de ter na tela o número de registros restringido, o usuário pode precisar desbloquear apenas um registro, caso se tenha mais de um registro exibido para o período informado.
Neste caso, o usuário pode utilizar o botão "Muda data desbloqueio" para trocar a data original do desbloqueio dos outros registros que não serão desbloqueados. Esse recurso pode ser usado toda a vez que for necessário alterar a data de desbloqueio de algum registro.
O sistema abrirá uma nova tela, onde o usuário deverá selecionar as baixas que desejam ser alteradas a data do desbloqueio e informar uma nova data. Mais informações em 
Alteração de Data de Desbloqueio de Baixa
.
