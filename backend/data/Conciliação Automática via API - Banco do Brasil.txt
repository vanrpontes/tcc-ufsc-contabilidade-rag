Aba Tipo Lançamento: no arquivo tipo JSON que o banco envia ao realizarmos a consulta via API, cada lançamento tem um código de identificação. A Futura já disponibiliza vários tipos de lançamentos com esses códigos, porém o usuário deve verificar se:
O Tipo de lançamento que a API enviou após a consulta está incluído. 
Caso não esteja, incluir e preencher todas as informações requisitadas.
Se tem o tipo de lançamento, verifique as informações como operação, Tipo, Antecipação do Lançamento (no caso de adiantamento) estão condizentes com o lançamento em questão.
