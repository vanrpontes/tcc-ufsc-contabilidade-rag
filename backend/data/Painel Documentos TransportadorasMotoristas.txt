TITULO: Painel Documentos (Transportadoras/Motoristas)
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/4617437185
--------------------------------------------------
O painel Documentos
 permite ao usuário controlar os arquivos de Transportadoras e Motoristas dentro do Sispetro. O acesso a tela se da através do Menu 
Tabelas Auxiliares / Transportes / Transportadoras/Motoristas.
Para acesso a esta função se faz necessário a habilitação do perfil de usuario:
Menu Ferramentas / Usuário / Aba superior Perfil / Aba inferior 
Frete
 / Quadro Manutenção de Transportadora / Opção Pode incluir/excluir Documentos (Arquivos) de Transportadoras/Motoristas?
Transportadora: 
código da transportadora
Nome Pasta:
 nome gerado automaticamente pelo sistema conforme cadastro
Visualizar: 
abre arquivo selecionado no painel
Atualizar: 
atualiza lista de documentos
Gerenciamento de Arquivos: 
botão para incluir e gerenciar arquivos
 
Arquivos Vinculados a esse Processo
Carregar Arquivo
Pesquisar no Computador: 
abre a janela para pesquisa de arquivo a ser carregado
Opções de Vínculo
Desvincular: 
remove vínculo do arquivo com a transportadora/motorista
Geral
Visualizar:
 abre arquivo selecionado no painel
Gravar:
 salva alterações realizadas
Legenda:
 status dos arquivos de acordo com as cores
Cancelar:
 desfaz alterações realizadas
