TITULO: Créditos Não Identificados - Conciliação Automática
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878898289
--------------------------------------------------
Créditos não identificados são os recebimentos na conta de banco cuja origem não foi encontrada. 
Exemplo: Quando um cliente deposita algum valor e esquece de enviar o comprovante. Neste caso, teremos uma movimentação da conta de banco cuja origem o responsável pela conciliação não conseguirá identificar.


Assim, nestes casos, ele simplesmente lança este valor para um crédito não identificado, concilia este movimento (pois o mesmo aparece no extrato bancário) e, quando o cliente enviar o comprovante, transfere o montante deste movimento para o cliente correto.


Para lançar um crédito não identificado, clique no botão 
Créd. Não Identificado.




Informe:




Uma conta para lançamento contábil
 (normalmente já virá sugerida uma conta) da contrapartida do banco (conta de créditos não identificados no seu plano de contas).


Número do Documento:
 O que aparece no extrato.


Data Movimento:
 A Data de movimentação no extrato.


Valor:
 O montante a ser créditado.


Código/Histórico:
 Código e descrição do histórico do movimento.




Aperte 
OK
 para confirmar o lançamento.
