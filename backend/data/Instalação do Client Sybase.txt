TITULO: Instalação do Client Sybase
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926196055
--------------------------------------------------
Para instalar em uma maquina somente os driver do Sybase para conexão com o banco de dados.
Baixa no site da SAP o arquivo de instalação 
SQLA16_Windows_Client.exe . 
Este procedimento server para o client do Sybase Anywhere 12 ou Sybase Anywhere 16, sendo necessário baixar o client de cada versão.
Para baixar Clique 
Aqui
Clique duas vezes no executável
Clique em 
Next
Escolha a linguagem 
English
Clique em
 
Next
Neste ponto da Escolha a opção de Personalizar a instalação.
Caso esteja instalando somente os driver do Sybase para utilização do Bando de dados, escolha  somente a opção 
SQL Anywhere Cliet
Caso queira fazer a instalação das ferramentas de gerenciamento do Banco de dados como 
Interactive SQL 
e
 Sybase Central 
escolha também a opção 
Administration Tools.
