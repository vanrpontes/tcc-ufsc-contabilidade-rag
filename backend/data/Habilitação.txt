TITULO: Habilitação

--------------------------------------------------
Para habilitar o gerenciamento de segurança do gerador de relatórios, devemos habilitar no painel Segurança do Sistema (Menu 
Ferramentas / Sistema
) o controle 
Habilita segurança Gerador de Relatórios
 (conforme abaixo). 


 


Assim, após habilitar este controle, será permitido ao usuário definir o perfil de cada usuário com relação a relatórios e pastas. 


Nota: Este controle se aplica a todas as filiais, isto é, habilitando-o você estará habilitando a segurança para todas as filiais de uma vez só. 


Note que, somente habilitando o controle, não irá mudar em nada o comportamento dos usuários com relação às pastas e relatórios. Somente depois de configurar o perfil de cada usuário e atribuir segurança às pastas é que efetivamente você estará utilizando a segurança em seu benefício.
