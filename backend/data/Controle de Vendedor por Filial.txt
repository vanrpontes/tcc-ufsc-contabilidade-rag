TITULO: Controle de Vendedor por Filial
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210624
--------------------------------------------------
Para ter o controle de mais de um vendedor por cliente em filiais diferentes, o usuário deve configurar através do menu 
Ferramentas / Sistema / Clientes
, a opção:
Caso essa opção não seja configurada, o sistema utilizará somente um único vendedor por cliente para todas as empresas. Com esta opção marcada (Controla vendedor por filial para os clientes), o sistema passará a filtrar os detalhes do cadastro na tela de clientes, daqueles que não pertençam ao vendedor em determinada filial, podendo porém, ser visualizado apenas os dados njo painel de navegação.
Através do botão 
"Define vendedor por filial"
, localizado ao lado do campo 
"Vendedor"
, o usuário pode vincular um vendedor em cada filial para o cliente.
Ao clicar no botão 
"Define vendedor por filial"
, o sistema abrirá uma tela onde o usuário poderá vincular ou desvincular o vendedor a uma determinada filial, através dos botões 
"Vincula Vendedor"
 e 
"Desvincula Vendedor"
.
Caso não tenha um vendedor vinculado em cada filial, será considerado o vendedor informado na tela de clientes, para as filiais que não tiverem o vendedor vinculado.
