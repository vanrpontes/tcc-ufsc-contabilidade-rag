TITULO: Tabela Unidades
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878900331
--------------------------------------------------
Objetivo
A tabela armazena as unidades de medida da empresa, com manutenção disponível em 
Tabelas Auxiliares / Estoque / Unidades
.
Nela é possível cadastrar novas unidades informando código, descrição, código DCP, indicar se é peso e salvar as alterações.
Esta tabela armazena todas as unidades de medida utilizadas pela Empresa.
Existe uma tela no Sistema que permite a manutenção do cadastro de Unidades. Para ativá-la

acione a partir do menu principal as opções - Tabelas Auxiliares / Estoque / Unidades.
Acione o botão de inserção de novo registro.  
O Sistema exibe a tela que permite o cadastramento da nova Unidade de medida.
Preencha os campos:
Código
 da Unidade - KG, LT, PC, ...
Descrição
Código DCP
 - Código equivalente para uso no DCP.
Se a Unidade é considerada uma unidade que reflete o peso do produto, clique com o mouse sobre o campo 
Unidade é Peso
.
Acione o botão OK para atualizar os dados no Sistema.
 Para retornar ao Menu principal acione o botão Sair
