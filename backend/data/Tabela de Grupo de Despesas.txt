TITULO: Tabela de Grupo de Despesas
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878211033
--------------------------------------------------
Nesta tabela são cadastrados os Grupos de Despesas a serem utilizados no 
Controle de Despesas com Frota
.
O 
Grupo de Despesas
 pode ser utilizado para dividir o setor de Transportes em vários grupos, e assim ter uma melhor descrição na inclusão de novos Eventos relacionados ao Frota. Além disso,pode-se ter um 
plano de contas interno
 para o setor em questão, sem interferência do/ com setor contábil, pois esta tabela (junto com a tabela de subgrupo de despesas) é utilizada somente no módulo de Transportes.
Exemplo:
 Grupo: Pneus
A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Transportes / Frotas / Grupo de Despesas
:
Para acrescentar um novo Grupo de Despesas acione o botão de inclusão de registro 
.
O Sistema exibe a tela para preenchimento dos novos dados.
Informe a 
Descrição
 para o grupo de despesas. O grupo poderá ter 
subgrupos de despesas
.
Caso haja a necessidade de ativar/ inativar um grupo, marque/ desmarque a opção de 
Ativo
.
Acione o botão 
OK
 para atualizar os dados no Sistema.
O Sistema retorna a tela de navegação.
Para retornar ao Menu principal acione o botão 
Sair
 .
