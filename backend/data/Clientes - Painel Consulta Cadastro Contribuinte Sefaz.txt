TITULO: Clientes - Painel Consulta Cadastro Contribuinte Sefaz
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926183394
--------------------------------------------------
Este painel apresenta as consultas realizadas ao Cadastro de Contribuinte da Sefaz e deve ser acessado a partir de 
Arquivos / Clientes.
Para verificar todas as consultas, automáticas ou manuais, realizadas para determinado cliente é preciso ir na aba 
"
Consulta Cadastro Contribuinte Sefaz
". 
Será apresentada a lista de todas as consultas, com todas as informações necessárias, veja abaixo.
Usuário
: Informa o usuário responsável pela consulta
Data/Hora
: Informa a data e hora da consulta
Status Resposta
: Apresenta o código do status da resposta realizada
Descrição Resposta
: Apresenta a descrição do status da resposta da consulta realizada
Data Resposta
: Informa a data da resposta recebida
Situação
: Informa a situação do cliente, recebida nessa consulta
Status Processamento
: Informa o status de processamento da consulta
Imprime Lista
: Imprime a lista de consultas realizadas para este cliente
XML Enviado
: Apresenta o xml enviado para realizar a consulta
XML Retorno
: Apresenta o xml de retorno desta consulta
Clientes com duas Inscrições Estaduais 
O Sispetro verifica as IEs, tanto do Cadastro principal/
Dados Gerais
 quanto da aba 
Outros Endereços
. Caso uma das duas esteja com alguma restrição ou inativa, a consulta terá resultado 
Não habilitado
.  Por esse motivo é importante se atentar aos dados, tanto da aba Dados Gerais quanto Outros Endereços.
