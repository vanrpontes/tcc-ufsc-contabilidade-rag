Bem vindo ao 
Módulo de Controladoria
 do Sispetro.
Este módulo, embora faça parte do  Sispetro, é completamente estanque, isto é, pode ser usado sem que o Sispetro Normal esteja ativo. Projetamos ele desta forma de maneira a dar ao contabilista toda  flexibilidade para a contabilização dos diversos eventos que devem ser  contabilizados.
Além disso, pelo fato de estar totalmente integrado ao  Sispetro, todos os procedimentos realizados nele (faturamento, inclusão baixa,  etc) podem ser configurados individualmente no Sispetro de maneira com que cada  um deles gere os dados contábeis que, de outra forma, teriam de ser incluidos  manualmente.
Esta flexibilidade permite que:
Cada procedimento seja tratado de maneira diferenciada com relação à geração dos  dados para a contabilidade.
Mesmo se você não quiser utilizar o módulo contábil, poderá utilizar somente a parte  geradora do Sispetro para atualizar o seu próprio sistema de contabilidade via  exportação contábil.
Todos os eventos sejam contabilizados independente da atuação do usuário (inclusive  cancelamentos).
Além disso,  o módulo de Controladoria foi projetado para ser acessado via rede local ou  mesmo via Internet (já foi sintonizado para manipular e trafegar a menor  quantidade possível de dados entre o servidor e sua estação).
