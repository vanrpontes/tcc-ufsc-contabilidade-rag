TITULO: Bloqueio de Venda por Atividade Econômica
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210652
--------------------------------------------------
Abaixo seguem as informações de como bloquear a venda de determinados produtos para o cliente.
Há um perfil de usuário 
Pode Vender Produto Bloqueado,
 que se sobrepõem as configurações de Bloqueio, ou seja, se estiver marcado 
nunca
 será bloqueado. Seu no
Bloqueio por Empresa
No perfil de Empresa 
Ferramentas \ Sistema
, aba 
Clientes
, localize o perfil 
Padrão para inclusão de clientes
.
Nesse perfil, visualizaremos todas as Atividades Econômicas disponíveis no Sispetro.
O Campo 
Vende todos Produtos
 define se a Atividade Econômica selecionada terá alguma restrição ou não de venda, caso a opção Vende todos Produtos esteja HABILITADA, não haverá bloqueio para nenhum produto dessa empresa nessa atividade econômica, salvo exceções de determinados clientes que tenham sua própria configuração de bloqueio (veremos isso no Bloqueio por Cliente).
Caso a opção Vende todos Produtos esteja DESABILITADA, deveremos configurar apenas os produtos que serão permitidos para venda.
Para habilitar ou desabilitar o campo Vende todos os produtos, selecione a atividade econômica desejada e clique no botão 
Editar
, marque ou desmarque a opção e 
Gravar
.
Para definir os produtos que serão permitidos ou não, utilize o botão 
Produtos Permitidos
.
Somente será permitido o acesso a esse botão quando a Atividade Econômica selecionada não estiver com o campo Vende todos Produtos marcado.
Será aberta a tela abaixo:
Utilize os botões de 
Inclusão e Exclusão
 para finalizar a operação.
Somente será permitido o acesso a esse botão quando se o usuário possuir privilégios: Perfil de usuário 
"
Pode Alterar Permissão Dados de Vendas Produtos"
Bloqueio por Cliente
Acesse o cadastro de Clientes, aba 
Venda Produto
:
Da mesma forma como funciona no perfil de Empresa, utilizando os botões de 
Inclusão e Exclusão
 defina os produtos que deseja que sejam bloqueados.
