TITULO: Bancos - Saída por Origem-Destino e Centro de Custo
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878898398
--------------------------------------------------
Este relatório apresenta a análise de despesas, também por Centro de Custo.


O acesso a esse relatório é através do Menu 
Relatórios / Financeiro / 
Saída por Destino/Origem por C.Custo
:




Pode ser realizado filtro por:




Unidade de Negócios:
 onde pode ser selecionado uma Filial específica, um grupo de Filiais, um conglomerado ou mesmo todas as Filiais cadastradas.



 






Filial:
 caso tenha sido selecionado 
Filial (escolher)
 em 
Unidade de Negócios
, será habilitado este campo onde pode ser informada ou selecionada uma filial. O relatório só trará as despesas relativas a esta filial.


Grupo de Filiais:
 caso tenha sido selecionado 
Grupo de Filiais (escolher)
 em 
Unidade de Negócios
, será habilitado este campo onde pode ser informado ou selecionado um grupo de filiais. O relatório só trará as despesas relativas a este grupo de filiais.


Conglomerado de Filiais:
 caso tenha sido selecionado 
Conglomerado de Filiais (escolher)
 em 
Unidade de Negócios
, será habilitado este campo onde pode ser informado ou selecionado um conglomerado de filiais. O relatório só trará as despesas relativas a este conglomerado de filiais.


Filtra Conta Origem:
 caso tenha sido marcada esta opção, vai habilitar o campo 
Conta de Banco
 para que seja selecionado uma determinada conta de banco.


Conta de Banco:
 deve ser informada ou selecionada a Conta de Banco que deve ser filtrada. O relatório só trará as despesas relativas a esta Conta de Banco.



 






Filtra Conta Destino:
 caso tenha sido marcada esta opção, vai habilitar o campo 
Conta de Destino
 para que seja selecionado uma determinada conta destino.


Conta de Destino:
 deve ser informada ou selecionada a Conta de Destino que deve ser filtrada. O relatório só trará as despesas relativas a esta Conta de Destino.



 






Filtra Centro de Custo:
 caso tenha sido marcada esta opção, vai habilitar o campo 
Código do Centro de Custo
 para que seja selecionado um determinado centro de custo.


Centro de Custo:
 deve ser informado ou selecionado o Centro de Custo que deve ser filtrada. O relatório só trará as despesas relativas a este Centro de Custo.


Somente despesas sem Centro de Custo informado:
 se esta opção for marcada, o campo 
Centro de Custo
 será desabilitado e só serão apresentadas as despesas sem Centro de Custo.



 




O relatório pode ter até 3 níveis de 
Agrupamento
, sendo por 
Origem(Banco)
, 
Destino(Despesa)
 ou 
Centro de Custo
:




Se for marcado como 
resumido
 o 
Nível 3
, só aparecerão os totais, para o agrupamento realizado.

Se for marcado como 
resumido
 o 
Nível 2
, será desabilitado o Nivel 3 e só aparecerão os totais, para o agrupamento realizado.

Se for marcado como 
resumido
 o 
Nível 1
, serão desabilitados os Niveis 3 e 2, e só aparecerão os totais, para o agrupamento realizado.


Deverá ser informado o 
Período
 ao qual se deseja imprimir o relatório.




O layout do relatório será gerado conforme o arquivo 
MovimentoBancoSaidaCCusto.rtm
 que estiver no caminho do executável. É possível editar o Layout do relatório através do botão 
Edita Layout
