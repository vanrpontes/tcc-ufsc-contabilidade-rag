TITULO: Relatórios de Viagens
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878900261
--------------------------------------------------
O relatório de viagens, mostra as informações referentes as viagens que foram controladas pelo sistema.


O acesso a esse relatório é através do menu 
Relatório / Transporte / Viagens
.




O layout do relatório será gerado conforme o arquivo 
Viagens.rtm
 que estiver no mesmo caminho do executável. Caso não exista esse arquivo, o sistema utiliza um layout padrão para imprimir o relatório.


O usuário deve utilizar as opções de filtros existentes na tela, para obter resultados mais objetivos:


Filial
:

Esta opção permite verificar as viagens apenas de uma filial especifica, um grupo de filiais ou todas as filiais.


Frota Específica e Transportadora Específica:

Esse filtro limita o relatório apenas para as viagens que foram feitas do frota ou transportadora selecionados.


Status:

Pode-se selecionar o relatório pela situação da viagem: Pendente, Fechadas (s/lancto), canceladas,Lançadas Pagto, Todas (Menos Canceladas).


Ordenado Por:

Esse campo determina o tipo de classificação que o relatório deve ter. Primeiramente o relatório será ordenado por frota, depois pelo filtro escolhido pelo usuário (Viagem,transportador ou Data de saída).


Período:

Intervalo de data que se deseja emitir o relatório.


Imprime adiantamentos, Imprime Eventos e Imprime Nota Fiscais:

Estas opções faz com que o relatório mostre os adiantamentos, eventos e notas fiscais que estão vinculadas a uma viagem.


Botão Edita Layout:

Permite ao usuário ajustar o layout, conforme a necessidade da empresa.


Para imprimir o relatório, o usuário deve clicar no botão "OK" e para cancelar a impressão no botão "Cancelar".
