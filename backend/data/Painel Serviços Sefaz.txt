 Serviços SEFAZ
Fornecedor não cadastrado
Ao receber um resumo de um Fornecedor não existente no SisPetro, irá seguir a opção de manifestação cadastrada no perfil de sistema. Para o XML completo não irá incluir manifestação caso o Fornecedor não esteja devidamente cadastrado no sistema.

Tipo Manifestação: 
define qual o evento de Manifestação padrão. 
Este evento será sugerido sempre que for incluída uma manifestação e/ou quando o usuário não tiver permissão de alterar o Tipo de Manifestação (perfil de usuário).

Consulta Destinatário
Permite realizar a consulta de destinatário de NF-e e de CT-e
NF-e:
Último NSU Consultado nas NF-es emitidas para a empresa
Último sequencial para pesquisa dos eventos de consulta de destinatário. O NSU (Número Sequencial Único) é fornecido pela Sefaz a cada consulta realizada. Caso não tenha mais notas disponibilizadas pela Sefaz, o último NSU não altera. O SisPetro utiliza este sequencial na 
Consulta de Destinatário de NF-e/CT-e
.
Serviços:
Faz Consulta através de Serviço no Servidor Hermes: 
habilita o 
Serviço automático para Consulta de Destinatário de NF-e/CT-e
.
 
CT-e:
Último NSU Consultado nos CT-es emitidos para a empresa
Último sequencial para pesquisa dos eventos de consulta de destinatário. O NSU (Número Sequencial Único) é fornecido pela Sefaz a cada consulta realizada. Caso não tenha mais conhecimentos disponibilizados pela Sefaz, o último NSU não altera. O SisPetro utiliza este sequencial na 
Consulta de Destinatário de NF-e/CT-e
.
Serviços:
Faz Consulta através de Serviço no Servidor Hermes: 
habilita o 
Serviço automático para Consulta NF-e/CT-e Destinatário
.
