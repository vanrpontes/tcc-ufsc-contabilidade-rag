A partir da versão 9.15 é preciso indicar no Servidor SisPetro a pasta onde as DLLs de Cobrança Eletrônica estão, para que a cobrança seja enviada via e-mail. Caso contrário, será exibido ao usuário a mensagem de erro.
A geração de arquivo de Remessa é algo repetitivo que é realizado todos os dias onde na maioria das vezes a Empresa realiza a geração dos arquivos uma vez por dia em um determinado horário. Pensando nisso a partir da versão 9.9 foi criado o Serviço de geração de arquivo de remessa de cobrança eletrônica.
Depois de configurado, o serviço funcionará automaticamente gerando os arquivos de remessa para os Tipos de cobrança que estão configurados, e no horário definido pelo usuário no perfil de sistema.
Veja as configurações necessárias abaixo.

Painel Detalhes - Tipo de Cobrança
O valor de corte é indicado para as empresas que enviam seu arquivo de remessa para o banco ou representante ate um determinado valor, criando um segundo arquivo para os demais arquivos. Caso a empresa não trabalhe com essa opção basta deixar o valor de corte igual a zero.
Veja abaixo um exemplo de como funciona a geração de arquivo de remessa utilizando o valor de corte.
Exemplo:

Conforme podemos ver nas imagens acima, o serviço pega os lançamentos ate atingir o valor de corte, caso o próximo lançamento for ultrapassar o valor de corte o Sistema gera o Primeiro arquivo com os lançamentos ate aquele momento e os demais são gerados no segundo arquivo. Pegando o exemplo acima o nosso valor de corte era 1000,00 e tínhamos 5 lançamentos, o serviço gerou um arquivo de 800,00 composto por um lançamento de 500,00 e outro de 300,00, e como o terceiro lançamento era de 400,00 onde ultrapassaria o valor de corte, o Serviço pegou esse terceiro lançamento e junto com os demais gerou o segundo arquivo.
Manutenção de Cobrança Eletrônica
Através da tela de 
Manutenção de Cobrança Eletrônica
 podemos ver os arquivos de remessa que foram gerados tanto via Serviço quanto manualmente, de qual grupo de corte o mesmo participa
Caso aja necessidade também  podemos acionar o serviço manualmente para que sejam gerados os arquivos via serviço, para isso basta acionar o botão 
Aciona Serviço, 
após acionar essa opção o Serviço inicia gerando os arquivos de remessa que estão configurados para gerar via Serviço. Ao acionar o botão será apresentado uma tela informando a hora que está configurado para gerar o serviço e se o usuário deseja dar continuidade ao processo.
