TITULO: Consultas - Financeiro
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2887221315
--------------------------------------------------
As consultas tridimensionais de Contas a Pagar e Contas a Receber do Financeiro permitem ao usuário verificar quais os títulos pendentes atualmente ou pelo último fechamento financeiro. Esta tela pode ser acessada pelo Menu 
Consulta / Financeiro / Contas a Pagar Pendente
 ou
 
Consulta / Financeiro / Contas a Receber Pendente
.
Sua consulta abrange dados como:
Data Vencto Original
Usuário
Data Vencto
Razão Social
Portador
Comprador
Vendedor
Tipo Documento
Semestre Vencto
Ano Vencto
Trimestre Vencto
Data Emissão
Trimestre Emissão
Semestre Emissão
Ano Emissão
Mês Emissão
Mês Vencto
Histórico
Código Histórico
Valor Original
Valor Bruto
Valor Pago
Saldo Pendente
Ano
Mês
Cód Empresa
Tipo Cobrança
Observe que nesta consulta está sendo filtrado por empresa e os tipos de cobrança utilizados nos títulos de cada empresa. Em contra-partida, temos o total de Valores Originais, Valores Brutos, Valores Pagos e Saldos Pendentes.
Para maiores informações de como utilizar esta consulta, vide o tópico 
Consultas Tridimensionais
.
Os filtros disponíveis para utilizar esta consulta são:
Filial:
 Podendo ser somente uma Filial, Grupo de Filiais e Conglomerado de Filiais.
Layout Usuário:
 Permite ao usuário salvar vários tipos de layout's referentes a esta consulta. Muitas vezes o mesmo usuário precisa ter mais de um tipo de layout para a geração desta consulta, com base nisso, este campo possibilita ao mesmo salvar várias telas, desde que o nome seja alterado. Utilize os botões 
Salvar
 e 
Cancelar
.
Tipo Posição:
Quais títulos irão aparecer nesta tela
Posição Atual:
 O que realmente está pendente
Posição Fechamento Financeiro:
 a posição de pendências com base no fechamento financeiro informado no campo 
Data Fechamento
.
Este procedimento será o mesmo para o Contas a Receber Pendente.
