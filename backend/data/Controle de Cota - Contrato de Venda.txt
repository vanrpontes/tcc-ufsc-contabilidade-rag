Para o Contrato de Venda é necessário indicar a cota de venda e o fluxo de caixa.
O fluxo de caixa previsto é gerado normalmente pelas condições de pagamento estipulada, porém poderá ser alterado para que acompanhe o fluxo de entrega das cotas.
É imprescindível que se informe como será o controle de cota diária, cuja soma deverá ser igual a quantidade informada no contrato.
Ao finalizar um Contrato de Venda, será apresentada a tela a seguir:
Através do botão 
Inclui Cota Rápida
 é possível estipular como o SisPetro pode preencher a cota automaticamente. Para isso será necessário informar os dados  na seguinte tela:
É possível que o SisPetro divida a quantidade total entre o período informado, ou então que se fixe a cota diária, onde o SisPetro, a partir da data inicial irá lançar a cota diária até atingir a quantidade total.
Clique em OK após definir a opção mais adequada.
O SisPetro apresentará a distribuição que ele fez automaticamente, porém esta distribuição poderá ser alterada à vontade, desde que a quantidade da soma das cotas seja sempre igual à quantidade total informada no contrato.
Pelo botão 
Cria fluxo p/ cota
 é possível ajustar o fluxo de caixa previsto que foi gerado inicialmente pelas condições de pagamento para que seja de acordo com as cotas diárias. Sendo assim, será gerado para cada dia de retirada informado um fluxo equivalente à quantidade vezes o valor unitário do produto. Este processo será útil quando foi informada uma determinada condição de pagamento, mas o pagamento será feito de acordo com a retirada do produto (não condizendo com a condição informada).
Pelo botão 
Atualiza Valores
 é possível atualizar os valores de cota retirada a partir de Notas Fiscais, Bombeios, e OC de transferência.
Controle de cota no contrato de Venda
A partir da versão 8.42 é possível realizar a validação das cotas diárias informadas no contrato de venda.
Para realizar o controle das cotas no contrato de venda primeiramente é necessário marcar o perfil de sistema 
Bloqueia Cota do Contrato de Venda.

Alteração de Cotas
É possível alterar o valor e a data das cotas, para que seja possível realizar essa alteração é necessário habilitar o perfil de usuário
 
Altera Cota dia e Fluxo Financeiro no Contrato de Venda.
 Para mais informações, ver 
Painel Vendas (Usuários)
.
Com o perfil de usuário devidamente habilitado na tela de Manutenção Contrato de Venda na aba de Cota Dia será habilitado o botão 
, permitindo assim realizar as alterações das cotas.
