Acesse esta tela através do menu Ferramentas/ Usuários.
Os usuários do sistema poderão ser incluídos, excluídos, inativados e o seu perfil referente as permissões dentro do sistema atualizados. O acesso a tela de "Manutenção de Usuário" deve ser feita a partir do menu 
Ferramentas / Usuário:
   
Para incluir um novo usuário, basta clicar no botão de inclusão:
O sistema irá se posicionar no painel "Detalhes", onde será possível incluir um novo usuário dentro do sistema:
Deverão ser preenchidos os seguintes campos:
   
Nome Usuário:
 Trata-se do nome que o usuário do sistema irá informar no momento de sua entrada no sistema.
Tipo de Criptografia:
 neste campo deverá ser informado o padrão criptográfico empregado para armazenar a senha deste usuário no banco de dados, lembrando que quanto mais criptografada a informação melhor será a segurança, contudo há uma degradação de performance, para realização da descriptografia por parte do sistema em todo instante que o usuário necessite entrar.
Nome Completo do Usuário:
 Campo apenas informativo para futuras consultas por parte do Administrador do sistema.
Setor:
 Setor ao qual o usuário está vinculado. (Maiores informações no tópico 
Setores.
)
Ativo:
 No caso da necessidade de desabilitar o acesso a um usuário do sistema, basta apenas desativá-lo desmarcando essa opção. Estando inativo, o usuário não conseguirá ter acesso ao sistema.
Foto do Usuário:
 Para vincular uma foto ao usuário, basta dar 
"duplo clique"
 com o botão esquerdo do mouse em cima do quadrado branco que exibirá a foto selecionada, ao dar o duplo clique, será aberta uma tela, solicitando o arquivo de imagem a ser carregado.
Mostra novidades da versão do Sispetro ao iniciar:
 Estando marcada esta opção, ao entrar no sistema, será aberta a tela "O que há de novo?", que descreve todas as alterações e correções implementadas na versão corrente.
Botão "Nova Senha":
 No momento da inclusão do usuário, o botão "Nova Senha", não estará habilitado. Apenas quando o usuário estiver cadastrado que este botão terá os seus recursos disponíveis. Para maiores informações sobre este botão, acesse o tópico 
Nova Senha.
