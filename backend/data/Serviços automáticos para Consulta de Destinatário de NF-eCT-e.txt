TITULO: Serviços automáticos para Consulta de Destinatário de NF-e/CT-e
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2925002800
--------------------------------------------------
É possível configurar o SisPetro para que faça a Consulta de Destinatário de forma automática. Para mais informações, ver 
Consulta de Destinatário de NF-e/CT-e
Abaixo segue os passos necessários para habilitar o processo automático.
Configuração de Sistema
Na Configuração de Sistema, aba 
Serviços Sefaz
 
Faz Consulta através de Serviço no Servidor SisPetro
Servidor Hermes
Serviço de Consulta Destinatário NF-e
: configurar os tempos de execução, intervalo de execução e usuário
Serviço de Consulta Destinatário CT-e
: configurar os tempos de execução, intervalo de execução e usuário
Na tela 
Consulta de Destinatário de NF-e/CT-e
,
 
foi criada uma nova tela onde o usuário poderá verificar de forma mais fácil as pesquisas feitas. Para mais informações, ver 
Consultas de Destinatário Efetuadas
Os processos de baixar e salvar os arquivos .XML das Consultas não foram alterados e funcionam da mesma forma na Consulta Automática e Manual.
