TITULO: Saída

--------------------------------------------------
Para ajustar o Estoque escritural com uma saída, isto é, subtrair a quantidade introduzida aos saldos existentes, acione a partir do menu principal as opções 
Movimentação / Estoque / Saídas / Outras Saídas


 


Insira os seguintes campos: 




Referência
: Um número de documento (número do contrato, por exemplo). Opcional.


Depósito
: Código do Depósito de saída do ajuste. Obrigatório.


Histórico Ajuste
: Código do histórico do ajuste (o motivo).Obrigatório.


Observações
: Qualquer observação que se faça necessário.Opcional.


Setor
: A quem destina-se a saída do estoque.




Para ser possível informar o setor, deve configurar a empresa. Acesse a partir do Menu 
Ferramentas / Sistema / painel Estoque
, e marque a seguinte opção: 




Aperte o botão Avança.




Para cada produto que deseja ajustar, digite: 




Código do Produto
: Caso não saiba, aperte <enter> para abrir a tela de pesquisa de produtos e escolha o produto desejado.


Quantidade
: Quantidade que será subtraida ao saldo já existente (disponível).




Aperte o botão OK para confirmar o ajuste.
