TITULO: Tabela de Termos
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210327
--------------------------------------------------
A tabela de Termos é utilizada para armazenar os termos de abertura e fechamento a serem utilizados nos Livros ou relatórios contábeis. A manutenção desta tabela é acionada a partir do Menu principal com as opções -
 
Arquivos / Termos. 
Acione o botão de inserção de registro. 
 
 
O Sistema exibe a tela para o cadastramento de um novo Termo. 
Informe uma
 
descrição
 
e o caminho do arquivo de
 
layout
 
a ser utilizado (clique no ... para abrir a tela de estrutura de diretórios para escolha de um arquivo). Este arquivo de layout pode ser editado nesta mesma tela, clicando no botão
 
Edita Layout
.
Para imprimir automaticamente o período e o número do livro que está sendo impresso (
Apuração
, 
Entradas
 e 
Saídas
, na Edição de Layout, coloque três Objetos do tipo "Label" e renomeie-os respectivamente por lblDataInicioApuracao, lblDataFimApuracao e lblNumeroDoLivro. Posicione-os no texto conforme desejado.
O nome dos objetos deve respeitar rigorosamente as letras minúsculas e maiúsculas, por ser case sensitive
Acione o botão OK para atualizar os dados no Sistema.
O Sistema retorna a tela de navegação.
Para deixar a tela de Termos acione o botão Ok ou Cancela. 
A Futura disponibiliza layout's padrões dos termos de abertura e encerramento da Apuração Fiscal, Livro de Entrada e Livro de Saídas. Nesses layout's já estão configurados para imprimir automaticamente o período e o número do livro, não tendo necessidade de editar os layout's.
Caso deseje utilizar os layout's padrões, enviar e-mail ao Suporte da Futura solicitando os mesmos.
