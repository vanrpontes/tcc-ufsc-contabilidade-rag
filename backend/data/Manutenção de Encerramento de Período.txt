--------------------------------------------------
Periodicamente deve ser feito o encerramento das contas de Receita e Despesa para apuração do resultado da empresa (lucro ou prejuízo). Para visualização dos encerramentos, bem como manutenção dos mesmos é necessário acionar a partir do Menu 
Movimentação / Controladoria / Manutenção de Encerramento Período
:
Na tela de manutenção de encerramentos é possível visualizar os dados de cada encerramento, inclusive a conta de resultado utilizado e os valores lançados em cada conta de receita ou despesa que precisou ser fechada.
Para fazer o encerramento de um determinado mês, clique no botão 
O encerramento das contas deve ser feito para cada empresa, portanto o sistema já sugere a empresa na qual você está conectado. Faça o processo e repita-o, conectando antes na empresa desejada.
O sistema também verificará qual foi o último encerramento feito e apresentará a data na tela e sugerirá o mês atual como data do próximo encerramento. No caso de ser o primeiro fechamento, o mês anterior será preenchido com um mês a menos do mês do encerramento atual. 
O período a ser informado no encerramento deve ser, preferencialmente, conforme o período de apuração de lucro que a empresa trabalha (anual ou trimestral), porém o usuário poderá informar o período que julgar necessário
.
Informe até que data de movimentação deve ser feito o levantamento de dados para o fechamento das contas. Lembre-se que após confirmado o processo, não poderá ser feita nenhuma movimentação com data anterior a esta informada. Se necessário for, deve-se cancelar o encerramento, fazer as movimentações e depois providenciá-lo novamente.
Informe a conta de resultado que receberá os lançamentos de crédito e débito que fecharão cada conta de receita e despesa.
Informe um histórico que represente este processo.
