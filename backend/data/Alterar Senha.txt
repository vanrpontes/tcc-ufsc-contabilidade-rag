TITULO: Alterar Senha
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2965439122
--------------------------------------------------
Para acessar o menu Troca Senha Usuário, 
acessar o 
Menu
 
Início > Outros > Alterar Senha. 
Nessa tela, temos os campos:
ID:
 p
reenchimento automático do sistema
Senha Atual:
 informe a senha atual
Nova Senha:
 informe a nova senha
Confirmar nova Senha:
 d
igite a nova senha novamente
Para salvar as alterações, clique no botão
 
Confirmar
 
ou
 
Cancelar
 
para descartar.
