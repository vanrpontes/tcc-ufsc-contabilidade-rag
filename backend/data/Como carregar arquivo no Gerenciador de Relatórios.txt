TITULO: Como carregar arquivo no Gerenciador de Relatórios
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2910486801
--------------------------------------------------
O Gerenciador de Relatórios do SisPetro permite não só a criação de novos relatórios como também a alteração de arquivos RTM. 
Através deste Help iremos descrever como carregar arquivo RTM no Gerenciador de Relatórios.
 
Acesse o menu 
Relatórios/Gerenciador de Relatórios/Novo Relatório
Menu
Arquivo/Carregar do Arquivo/Selecione o arquivo desejado/Abrir
OBS: É importante ressaltar que somente arquivos RTM criados no SisPetro Normal serão carregados nesta tela. Caso o relatório tenha sido criado no SisPetro Remoto será apresentada uma mensagem informando erro.
O link abaixo consta orientações sobre como converter o relatório para que funcione no SisPetro Normal.
 
Após carregar o arquivo com sucesso e fazer suas devidas alterações, o SisPetro dispõe de duas formas para salvar o arquivo: 
Salvar como... 
ou 
Salvar para Arquivo...
Salvar como:
 Através dessa opção o relatório será salvo dentro do Gerenciador
Salvar para Arquivo: 
Com esta opção o relatório será exportado 
para que possa inclusive ser encaminhado.
