TITULO: Tabela Demonstrativo de Resultado
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210536
--------------------------------------------------
A tabela de Demonstrativo de Resultado é utilizada para criar uma estrutura para o Demonstrativo de Resultado e será utilizada na 
 para as contas do tipo Receita e Despesa.
A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - Tabelas Auxiliares / Controladoria / Demonstrativo de Resultado 
 
A máscara do campo de Ordenação é definida na configuração do Sistema, painel 
Contab. Real
.
Para acrescentar um novo demonstrativo de Resultado, acione o botão de inclusão de registro:
O SisPetro exibe a tela para preenchimento dos novos dados. 
  
Informe um 
código para ordenação
, sem pontos, pois a máscara preencherá os pontos necessários. Indique se esta estrutura é Sintética (só descritiva) ou 
Analítica
 (agrupa valores das contas que utilizam este tipo de demonstrativo). Informe o 
sinal
 que este item representará no Demonstrativo de Resultado quando a estrutura for analítica. Marque a opção 
Ativa
 para que este item possa ser utilizado.
O 
Tipo de Agrupamento DRE Vendas
 será utilizado para impressão do 
, indicando em que item do Demonstrativo será agrupada a Receita de Vendas (Notas Fiscais), Custos (Custos sobre NF) e Devoluções de Vendas (Notas Fiscais de Devolução). 
ATENÇÃO: só deverá ter um único item de cada tipo
.
Acione o botão OK para atualizar os dados no SisPetro.
O SisPetro retorna a tela de navegação.
Para vincular contas ao item do Demonstrativo, utilize o painel Contas ou a tela de Manutenção de 
Plano de Contas
.
No painel Contas é possível visualizar as contas que já estão vinculadas ao item do Demonstrativo, vide tela: 
 
Utilize o botão 
Seleciona Contas
 para vincular uma ou mais contas a este item de Demonstrativo. Ao clicar no botão será apresentada a tela para seleção: 
 
Utilize as teclas <SHIFT> ou <CTRL> para selecionar mais que uma conta e clique em 
OK
. Este recurso é bem mais prático que o vínculo a partir da tela de Plano de Contas. 
ATENÇÃO: Caso a conta selecionada esteja vinculada a um outro item do demonstrativo, ela perderá o vínculo anterior e passará a ser vinculada a este novo item
.
Para retornar ao Menu principal acione o botão Sair: 
 
Para emitir o demonstrativo utilize a opção 
Impressão do Demonstrativo de Resultado
.
