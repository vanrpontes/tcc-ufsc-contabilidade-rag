TITULO: 5. Relatórios
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210253
--------------------------------------------------
Neste menu será possível emitir relatórios contábeis e relatórios fiscais específicos. Tais relatórios, em sua maioria, poderão ter o seu layout alterado de forma que possa ficar personalizado. Consulte em cada item específico a recomendação para atingir esta ação.


Para alguns relatórios sua apresentação será sempre em tela cheia facilitando a visualização. E a tela de seleção de parâmetros não é fechada após a emissão do mesmo, assim o usuário pode emitir outro relatório alterando os parâmetros desejados, caso seja necessário.


Utilize a opção "Gerenciador de relatórios" para produzir e emitir relatórios personalizados para sua empresa.




Emissão de relatórios em impressoras matriciais




Como as impressoras matriciais imprimem mais rápido com as suas fontes residentes (fontes próprias), existe possibilidade de se definir no Sispetro Local, menu Ferramentas/Sistema, painel "Contab.Real" se os relatórios contábeis vão utilizar fonte de impressora. Quando marcada esta opção, deve ser selecionada a impressora matricial a ser utilizada e em seguida se escolher a fonte desejada (escolha sempre uma fonte que antes do nome tenha o desenho de uma impressora). Assim, todos os relatórios contábeis constantes no menu Relatórios serão impressos com esta fonte, independente da fonte definida no arquivo de layout (arquivo .rtm).




Salvando o relatório para o formato PDF:




O Sispetro Controladoria possibilita salvar os relatórios oficiais, exceto os relatórios localizados no menu Relatórios/Auxiliares, para o formato PDF. Para isso, basta seguir os seguintes passos:




No momento da impressão, marcar a opção 
"Imprimir para Arquivo",






Escolher o 
"Tipo"
 do arquivo como sendo PDF File;






Na linha 
"Onde"
, digitar o caminho no qual o arquivo será criado;






Clicar no botão OK.






Lembrando que o usuário precisa ter um programa especifico para ler os arquivos criados no formato PDF.




Escolha de filiais




Na escolha de filiais para impressão dos relatórios só estarão disponíveis as filiais que façam parte do mesmo grupo empresa da filial que você está conectado. 




Relatórios consolidados




Na escolha de um relatório consolidado, desmarcando a opção filial específica, só farão parte os dados referentes às filiais que façam parte do mesmo grupo empresa da filial que você está conectado. 




Edição de layout




No caso de necessitar alguma alteração no layout de algum relatório, utilize o botão 




Será exibida a tela do Gerenciador de relatórios para a edição do layout. Para isso é necessário verificar a cada relatório qual o arquivo .rtm utilizado. Altere posição e largura. As alterações de fonte só terão validade se não estiver definida a opção para impressora matricial citada acima. Quanto às alterações de agrupamento de dados, é importante consultar a Futura se a ordenação interna dos dados deste relatório permite a alteração. Independente da alteração a ser feita, tire sempre um backup do arquivo .rtm antes de iniciar o trabalho.
