TITULO: Tabela Preços
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878439632
--------------------------------------------------
Esta tabela armazena as as tabelas de preços utilizadas pela Empresa, armazenando preço de tabela, preço mínimo permitido e preço máximo permitido por produto.
Uma Empresa pode trabalhar com várias tabelas de preços. A condição para se aplicar uma ou outra, por exemplo, pode ser o perfil do cliente comprador.
O vínculo da Tabela de Preço com o Cliente deverá ser feita via 
Cadastro de Clientes > Padrão para Vendas
.
Seu cadastro poderá ser feito através do menu 
Tabelas Auxiliares > Vendas > Tabela Preços:
.
Para incluir um novo registro, clique no botão de 
Inclui 
Na tela de inclusão a tela abaixo será exibida:
Informe os campos abaixo na inclusão na aba Detalhes:
Código: 
Código para a tabela de preço, não podendo ter repetição.
Descrição: 
Descrição da tabela de preço a ser utilizada.
Percentual?: 
Este perfil deverá ser marcado quando a tabela for de preço por alçada, vide maiores detalhes no tópico 
.
Bloqueia pedido de venda caso preço não exista ou seja zerado: 
se marcado, 
não
 permitirá um 
pedido de venda
 cujo preço esteja com zerado para o estado ou não exista. Como padrão, este campo já vem marcado quando incluir uma nova tabela de preço.
Ao término, clicar no botão 
Ok.
Para abortar o processo, clicar no botão 
Cancela.
Para atribuir preços a nova tabela, informe para o produto desejado na aba Preços da Tabela
Ao sair do campo de preço, o valor já será salvo automaticamente.
