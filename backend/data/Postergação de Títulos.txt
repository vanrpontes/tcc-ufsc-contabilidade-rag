Postergação de Títulos em Lote
 
A partir da versão 9.9 é possível realizar a postergação de títulos em lote, dessa forma facilitando o processo como um todo.
 
O primeiro passo é selecionar os títulos que irão ser realizados a postergação, para isso basta selecionar mais de um título pressionando a tecla CTRL ou SHIFT do teclado e clicando nos títulos.
 
Após selecionar os títulos basta clicar no botão 
, que a tela de 
Acréscimo de Juros em Lote
  será aberta.
Na tela de 
Acréscimo de Juros em Lote
 iremos informar os dados para acréscimo, veja maiores detalhes abaixo:
 
Grid Títulos Selecionados:
 No grid de títulos Selecionados serão apresentados os lançamentos, a coluna de juros é atualizada conforme a Porcentagem informada no campo '% Juros ao Dia' versos a quantidade de dias entre a data de vencimento e a data para cálculo.
Data para Cálculo:
 É o campo de referencia para o cálculo do juros, e também para alteração da data de vencimento, caso selecionado a opção de 'Altera data de vencimento Conforme a Data para Cálculo'.
Data da Contabilização:
 Nesse campo é informado a data para data de contabilização.
Altera Data de Vencimento Conforme a Data para Cálculo:
 Caso marcado altera a data de vencimento para a data informada no campo Data para Cálculo. 
Essa opção foi disponibilizada na versão 9.10.
Código Histórico:
 Nesse campo é informado o código do histórico.
Histórico:
 Descrição do Histórico.
Informações Adicionais:
 Aqui é informado as informações adicionais.
Código de Acréscimo:
 Nesse campo informamos um código de acréscimo.
Conta de Contabilização Real:
 Nesse campo informamos uma conta de Contabilização Real.
Botão Calcula Juros:
 Ao acionar esse botão é realizado o calculo do juros.
 
Após preencher todas as informações da postergação, basta clicar no botão 
Calcula juros
 para que os juros sejam calculados
 
e clicar em Confirma.
