TITULO: Irregularidades nas Entidades para Livro Fiscal
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210430
--------------------------------------------------
No Livro Fiscal, os dados das Entidades (fornecedores e clientes) tem que estar preenchidos, devido exigência legal. Neste relatório, acionado pelo menu
 
Relatórios / Auxiliares / Fiscal / Irregularidades nas Entidades p/ Livro Fiscal
, é possível se relacionar todas as entidades que tiveram nota fiscal emitida num determinado período e que possuem algum dado imprescindível incompleto.


Informe o período desejado para verificação das irregularidades.


O layout do relatório será gerado conforme o arquivo
 
IrregularidadeLF.RTM
 
que deve estar localizado no mesmo diretório que o executável. Sua existência é
 
imprescindível.
 
Para alterar o layout utilize o botão 




Clique no botão OK para confirmar e o relatório será apresentado em tela. Para imprimí-lo clique no botão
