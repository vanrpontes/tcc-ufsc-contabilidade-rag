TITULO: Tabela de Locais Eventos Frota
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878211031
--------------------------------------------------
Nesta tabela são cadastrados os Locais de Eventos a serem utilizados no 
Controle de Despesas com Frota
.
A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Transportes / Frotas / Locais Eventos
:
Para acrescentar um novo Local de Evento acione o botão de inclusão de registro 
.
O Sistema exibe a tela para preenchimento dos novos dados.
Informe a 
Descrição
 para o Local do Evento. Caso este Local represente um depósito da empresa, de onde o valor abastecido para um determinado produto precisa ser movimentado, marque 
Tem Depósito?
 e preencha o 
Depósito
.
Caso haja necessidade de ativar/ inativar um local de evento, marque/ desmarque a opção 
Ativo
.
Acione o botão 
OK
 para atualizar os dados no Sistema.
O Sistema retorna a tela de navegação.
Para retornar ao Menu principal acione o botão 
Sair
 .
