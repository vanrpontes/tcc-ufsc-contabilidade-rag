Utilize este relatório, acionado pelo menu
 
Relatórios / Auxiliares / Contábil / Contas Contábeis de Produtos,
 
para conferência das contas contábeis gerenciais utilizadas no cadastro de Produtos. Tais contas são de Receita de Vendas (utilizada geralmente nos códigos de Contabilização das Notas Fiscais de Venda), de Estoque (utilizada geralmente nos códigos de Contabilização das Notas Fiscais de Compra) e de Custo.


Os produtos a serem conferidos são aqueles cuja contabilização da Nota Fiscal utilize uma das contas citadas acima, na dúvida confira todos. É possível, assim informar uma
 
 
específica ou deixar esta opção desmarcada para que sejam emitidos todos os produtos.


Além de conferir se foram associadas as contas corretas aos produtos, utilize o
 
 
para conferir se estas contas gerenciais estão devidamente vinculadas.


O layout do relatório será gerado conforme o arquivo
 
ContaProduto.RTM
 
que deve estar localizado no mesmo diretório que o executável. Sua existência é
 
imprescindível.
 
Para alterar o layout utilize o botão 




Clique no botão OK para confirmar e o relatório será apresentado em tela. Para imprimí-lo clique no botão
