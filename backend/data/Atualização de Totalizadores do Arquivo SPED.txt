TITULO: Atualização de Totalizadores do Arquivo SPED
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2879128171
--------------------------------------------------
Após efetuar a 
Edição dos Dados
, é importante efetuar a atualização dos totalizadores dos registros, para que ao efetuar a 
Geração do Arquivo
 as informações saim corretamente.
Para atualizar os totalizadores, o usuário deve acessar o menu 
Ferramentas / Totalização do Livro Eletrônico
.
Informar o Conjunto de Dados que deverá ter seus totalizadores atualizados após alterações de conteúdo.
Após informar o Conjunto de Dados, o usuário deverá informar os parâmetros para seleção, conforme imagem:
Na tela de 
Edição/Inclusão
, o usuário deverá informar os parâmetro ou confirmar os parâmetros trazidos na tela:
Informações Padronizadas:
Sequencial:
 Número sequencial do preenchimento;
Empresa:
 Selecionar a empresa que irá efetuar o preenchimento;
Data Inicial:
 Data inicial do período desejado;
Data Final:
 Data final do período desejado;
SQL:
 Para uso interno e nos testes do Suporte
Clique no botão 
Salvar
, para atualização dos totalizadores.
Após isso, o sistema irá atualizar o totalizadores do Conjunto de Dados definido, e deverá enviar uma mensagem de conclusão da atualização:
Após isso, clique no botão 
Ok
, e em seguida no botão 
Sair
, da tela de Totalização do Livro Eletrônico.
Com isso, o usuário já poderá gerar o arquivo do SPED considerando as edições efetuadas no arquivo. Para maiores informações, vide tópico 
Geração do Arquivo para o SPED.
