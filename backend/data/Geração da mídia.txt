Através do painel 
Geração Mídia
 é possível gerar o arquivo texto a ser importado no SIMP (Sistema de Informações de Movimentação de Produtos).




Informe o 
layout de Exportação
 onde foi configurado o layout do DPMP. Para facilitar, no menu 
Ferramentas / Sistema / painel Estoque
 é possível configurar o layout de exportação padrão para que ele seja sempre sugerido nesta tela:




Informe o 
caminho para geração da mídia magnética
, com o path completo (drive, diretório, nome do arquivo e extensão. Utilize o ... para localizar o diretório na árvore de diretórios.




Ex.: c:\arquivos\dpmp112004.txt


Quando a empresa tem várias filiais, geralmente para todas as filiais é um único Código ARI e cada filial tem o seu código de instalação. Como a ANP exige que para um mesmo código ARI seja enviado um único arquivo, é possível para isso, utilizar a opção: 


Quando esta opção estiver marcada, serão verificadas todas as empresas que tenham o código ARI igual ao da empresa que foi informada na tela e o arquivo será gerado com dados de todas estas empresas.


Assim, só para reforçar, é necessário que, para as outras empresas, já tenha sido feito o processo de geração do DPMP de forma individual. Só o momento da geração da mídia que será agrupado.


O arquivo texto deverá ser criado no caminho especificado. Utilize esse arquivo para, no SIMP (Sistema de Informações de Movimentação de Produtos), fazer a opção de Carga.


É importante que só seja gerado o arquivo após ter resolvido as 
críticas
 apontadas. No caso de ser para mais de uma empresa, as críticas devem ser resolvidas em todas as empresas deste Código ARI, de forma individual.
