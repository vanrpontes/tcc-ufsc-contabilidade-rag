TITULO: Ordem de Carregamento Web
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926194349
--------------------------------------------------
O SispetroWeb permite que as congêneres incluam as ordens de carregamento e descarga de terceiros diretamente na armazenadora.
Para mais informações, ver:
Incluir Ordem de Carregamento Web
Manutenção de Ordem de Carregamento Web
O menu de Ordem de Carregamento só é habilitado caso o perfil de cliente definido no SispetroMon esteja como 
Congênere
 ou 
Ambos
 (
Menu SispetroWeb / Definição Perfil Cliente
).
