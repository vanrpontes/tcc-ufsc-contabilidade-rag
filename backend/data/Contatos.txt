TITULO: Contatos

--------------------------------------------------
Através da manutenção de contatos que o usuário irá cadastrar um contato, vinculando ou não esse contato a uma entidade.
Para acessar a tela de manutenção de contatos o usuário deve utilizar o Menu 
Ferramentas / Contatos / Contatos
:
Para incluir um contato, o usuário deve clicar no botão 
. O SisPetro irá se posicionar no painel 
Detalhes
, onde o usuário poderá preencher os dados referente ao contato:
O campo 
#Código
 é preenchido automaticamente pelo SisPetro , portanto, o usuário deverá preencher os demais campos manualmente.
O contato cadastrado pode ser vinculado ou não a uma entidade (cliente ou fornecedor), isso se dá preenchendo o campo 
Código Entidade
 dentro do grupo 
Vincula Contato a uma entidade
, porém, isso depende do perfil de usuário, localizado através do Menu 
Ferramentas / Usuário / Contatos
:
Visualiza Contatos sem cliente vinculado: 
quando marcado vai exibir contatos sem clientes vinculados.
Pode incluir/alterar/excluir Manutenção de Ocorrências: 
quando habilitado o perfil de usuário tem acesso a incluir/alterar ou excluir ocorrências.
Pode incluir/alterar/excluir Contatos:
 quando habilitado o perfil de usuário tem acesso a incluir/alterar ou excluir contatos.
Quando este perfil estiver desmarcado, o SisPetro força o usuário a vincular o contato a uma entidade. Mesmo não preenchendo o campo 
Código Entidade
, o SisPetro solicitará ao usuário o seu preenchimento quando o registro for salvo.
Além disso, no painel Navegação, só serão apresentados os contatos que tiverem vínculo com alguma entidade.
Estando marcado o perfil nas configurações do usuário, o SisPetro permitirá ao usuário optar por desmarcar ou não o grupo 
Vincula Contato a uma entidade. Desmarcando o grupo, não será necessário vincular o contato a uma entidade. E no painel Navegação, serão apresentados tanto os contatos com vínculo como aqueles sem vínculo com alguma entidade.
Para gravar o registro, o usuário deve utilizar o botão 
Salvar
. Após salvar o registro, o sistema irá se posicionar no painel navegação, onde será apresentado os registros cadastrados.
Os registros cadastrados podem ser alterados ou excluídos. Para isso, basta o usuário se posicionar sobre o registro desejado no painel Navegação e clicar no painel Detalhes. Através do botão Editar, o usuário poderá alterar os dados do registro e por meio do botão Excluir, apagar esse contato.
O contato não será apagado, caso se tenha ocorrências que estejam cadastradas vinculadas a ele.
*É possível através da tela de contatos, acessar a tela de 
Ocorrências
, 
Pendências
 e imprimir uma relação de 
aniversariantes
, além de atualizar as informações em tela. Para isso, basta o usuário clicar no botão correspondente no rodapé da tela.
Para fechar a tela e retornar ao menu principal do SisPetro , o usuário deve utilizar o botão 
Fechar
.
Agrupando as Informações em tela:
Caso queira, o usuário poderá agrupar as informações em tela, arrastando a coluna desejada para o cabeçalho da tela.
Por exemplo: Pode-se arrastar a coluna Nome para o cabeçalho da tela e agrupar as informações por empresa. Assim, o usuário terá todos os contatos agrupados por empresa.
Basta clicar no sinal 
+
, para que as informações sejam apresentadas:
O usuário pode criar grupos com vários níveis, basta ir arrastando as colunas para o cabeçalho.
Pode-se incluir novas colunas na tela, ou até mesmo tirá-las, para isso o usuário deve clicar no controle localizado no lado esquerdo da tela.
O controle mostrará os campos disponíveis para serem apresentados na tela. Caso deseje retirar uma coluna, basta clicar no nome correspondente.
Após utilizar o recurso de agrupamento na tela de contato, o usuário pode restaurar o formato de tela original, bastando clicar no botão 
.
Aspectos Importantes:
O SisPetro permite que o usuário tenha acesso a determinadas entidades, conforme configuração de usuário para filtrar o vendedor, ou seja, se no perfil do usuário estiver configurado para filtrar as vendas por um determinado vendedor, o usuário só terá acesso aos clientes vinculados a esse vendedor.
Essa configuração fará com que no momento de incluir um contato, na tela de pesquisa de entidades a serem vinculadas, embora o SisPetro mostre todas as entidades, após o cadastrado, apenas será apresentado no painel navegação, os contatos que o usuário possa ver. Portanto, o usuário poderá cadastrar um contato e vinculá-lo a uma entidade que ele não tenha acesso, mais esse contato não ficará a disposição no painel Navegação.
