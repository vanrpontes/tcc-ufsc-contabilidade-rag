TITULO: Tabela Feriados
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210601
--------------------------------------------------
Nesta tabela são cadastrados os feriados a serem considerados pelo Sistema.


A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Financeiro / Feriados
:




Para acrescentar um novo Feriado à tabela, acione o botão de inclusão de registro 
.


O Sistema exibe a tela para preenchimento dos novos dados.




Preencha os seguintes campos:




Descrição:
 informe a descrição do feriado.






Data:
 data do feriado.






Tipo:
 informe se o feriado é federal, estadual ou municipal. Se o feriado for estadual você deverá informar a 
UF
, se for municipal informe o 
código DNC
 da cidade (caso não saiba, tecle <ENTER> que será exibida a tela de pesquisa das cidades).




Os feriados cadastrados serão levados em consideração no vencimento da Nota Fiscal (caso na tela 
Sistema - Painel Financeiro
 esteja marcada a opção 
Muda data de vencimento para próximo dia útil
), na data de vencimento dos lançamentos de contas a pagar e receber (será questionado caso o vencimento caia num feriado se deseja trocar a data para o próximo dia útil), no pedido de venda quanto ao cálculo se a data de entrega está dentro do limite de entrega (de acordo com a opção 
Dias de Entrega Posterior permitido
, na tela 
Sistema - Painel Vendas
).


Acione o botão 
OK
 para atualizar os dados no Sistema.


O Sistema retorna a tela de navegação.


Para retornar ao Menu principal acione o botão 
Sair
 
.
