TITULO: Tabela Status NF-e
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210599
--------------------------------------------------
Nesta tabela são cadastrados os códigos e descrições de mensagens como resultado do processamento da solicitação, motivos de não atendimento da solicitação e motivos de denegação de uso, utilizado na nota fiscal eletrônica, que após solicitar um determinado serviço para o web service da Secretaria da Fazenda Estadual, será retornado o código deste status de resposta.


A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Faturamento / Status NF-e
:




Para acrescentar um novo Documento acione o botão de inclusão de registro 
.


O Sistema exibe a tela para preenchimento dos novos dados.




Acione o botão 
OK
 para atualizar os dados no Sistema.


O Sistema retorna a tela de navegação.


Para retornar ao Menu principal acione o botão 
Sair
 
.
