TITULO: Tabela Tipo Vendedores
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878439630
--------------------------------------------------
Nesta tabela poderão ser cadastrados tipos para classificar os vendedores, utilizados para diferenciação dos vendedores no cadastro de cliente.


A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Vendas / Tipo Vendedores
:




O Sistema cadastra automaticamente um tipo de vendedor denominado NORMAL. Para acrescentar um novo Tipo de Vendedor à tabela, acione o botão de inclusão de registro 
.


O Sistema exibe a tela para preenchimento dos novos dados.




O sequencial é gerado automaticamente pelo sistema. Informe apenas a descrição do novo tipo de vendedor.


Acione o botão 
OK
 para atualizar os dados no Sistema.


O Sistema retorna a tela de navegação.


Para retornar ao Menu principal acione o botão 
Sair
 
.
