Neste painel é possível habitar algumas opções para utilização do Auto Atendimento no SispetroApp:
Quadro Auto Atendimento Base
Auto Atendimento Habilitado:
 se marcado, habilita opção no SispetroApp ao Modo Auto Atendimento. Para mais informações, ver 
.
Ação para Login Motorista: 
definirá, por padrão, se irá permitir ou não o login do motorista no SispetroApp. Possui duas opções:
Permitir:
 permite ao motorista, se habilitado, acessar o Auto Atendimento
Não Permitir: 
não permite ao motorista acessar o Auto Atendimento. 
Para mais informações, ver 
Permite Escolher Depósito de Venda:
 nessa opção, habilitamos ou não a possibilidade de alterar o depósito de venda no SispetroApp
Qtde de Dias para Exibição do Pedido:
 nesse campo podemos configurar o nº de dias (anteriores) para exibição de pedidos
Quadro Venda Spot
Auto Inclusão Usuário
Habilita AutoCadastramento usuários:
 habilita o AutoCadastramento de usuários, com base no perfil de usuário selecionado no campo abaixo:
Perfil para cópia:
 permite definir um usuário/perfil padrão para copiar as permissões ao novo usuário.
 Quadro Inclusão Clientes
Permite Inclusão Clientes:
 permite que seja realizada a inclusão de Clientes.
Código Bloqueio:
 permite selecionar um código padrão de bloqueio.
Quadro Geral
Habilita Controle de Alterações:
 com essa opção marcada, o SisPetro irá data e hora que o pedido de venda foi alterado. Será gravado também a data e hora que algum dado do cadastro do cliente inclusive posição financeira foi alterado. 
Não 
será possível verificar qual dado foi alterado. Essas alterações serão utilizadas na API de vendas
