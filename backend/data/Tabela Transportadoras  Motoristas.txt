Objetivo
Esta tabela arm
azena o cadastro das empresas de transporte ou motoristas que executam o serviço de retirar o produto do depósito onde está armazenado, e transportá-lo até o destino (endereço do cliente cada
strado para a entrega, distribuidora ou usina).
Esta tabela armazena também todo o histórico de fretes executados pela transportadora / motorista.
Para acessar a tabela de Transportadoras / Motoristas do SisPetro, acione a partir do Menu principal

Esta opção é aplicada somente a empresas transportadoras. Não se aplica a distribuidoras.

Ação Motorista sem vínculo c/ Transportadora: se trata de bloquear a movimentação dos registros que não tenham algum vínculo com transportadoras. Podendo ser configurado de duas formas, veja abaixo.
Assume que é autônomo: 
caso seja configurado desta forma o SisPetro não bloqueará nenhum registro. Assim podendo ser utilizado os registros que não tenham quaisquer vínculos com transportadoras ou motoristas.
Bloqueia ação: 
caso seja configurado desta forma, o SisPetro não deixará ser utilizado registros que não tenham vínculo. Ou seja, caso haja algum motorista sem vínculo com transportadora, este não poderá ser utilizado.
No caso de uma nova transportadora ou motorista, este perfil será sugerido da forma que foi indicado no perfil de sistema 
Painel Fretes
.
Caso haja algum motorista que é uma exceção a regra da distribuidora, por exemplo, todos os demais motoristas precisam ter vínculos com transportadoras. Porém há um que, por qualquer motivo, não precise, neste caso basta marcar que "Assume que é autônomo". Desde modo este motorista, mesmo não tendo vínculo, poderá ser utilizado.

Tem Frota?:
 este perfil permitirá vincular uma frota ao cadastro principal, habilitando o campo Frota.
Frota: 
quando informamos que a transportadora/motorista tem frota próprio, o SisPetro habilita este campo para que seja feito o vínculo. Após o vínculo, serão apresentadas, automaticamente, as informações de quantidade de lacres, capacidade KG e as placas.
Altera dados Frota:
 se a transportadora/motorista tem frota próprio e o mesmo está selecionado, ao clicar nesta opção, é possível fazer alteração nos dados do frota.
Compartimentação
: a transportadora/motorista tem frota próprio, é possível alterar a compartimentação dos frotas vinculados.
Pertence a uma empresa Transportadora:
 caso o motorista pertence a uma empresa transportadora, poderá ser indicado, porém, para o SisPetro serão utilizadas as empresas transportadoras que estiverem vinculadas no painel 
Empresa Transportadora
.
Recebe diária?:
 com esta opção selecionada, o SisPetro apresenta em relatórios a quantidade de diárias dos motoristas em viagens, para que seja feito o cálculo e pago o valor ao mesmo.
Permite Averbação:
 indica se permite que a carga seja assegurada. Caso esteja marcado, este campo permite a geração do valor da fatura no arquivo RTM de Averbação para notas não canceladas, e que não sejam do tipo retirada/venda.
Salário: 
informação do salário do motorista para fins de relatórios
% Desconto de frete:
 valor padrão para a Transportadora. O SisPetro possibilita que o desconto possa ser informado por transportadora e filial, não apenas por transportadora. Para mais informações, ver 
Painel Empresa Transportadora (vínculo Transportadora x Motorista)

Descarta Tabela de Distâncias da Transportadora
: quando o cenário é de um motorista vinculado a uma transportadora, a tabela de Distâncias normalmente utilizada seria a da Transportadora (que estabelece os preços que serão utilizados por seus motoristas). Nesse cenário esse campo deve ficar desmarcado. Porém, pode se ter um caso onde o Motorista terá uma tabela específica para ele, então se marca esse campo para que o SisPetro descarte a tabela da Transportadora vinculada ao Motorista e use a tabela dele. Para mais informações, ver
