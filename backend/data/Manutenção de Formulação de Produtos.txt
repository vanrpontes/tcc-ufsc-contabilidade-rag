TITULO: Manutenção de Formulação de Produtos
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2898133072
--------------------------------------------------
Essa tela visa a criação de uma formulação de um produto que será produzido através da tela de 
Manutenção de Ordem de Fabricação
Antes de acessar a tela, certifique-se de que seu usuário do Sispetro contenha as devidas permissões operacionais. Para configuraras permissões do usuário, vá no perfil de 
usuário
 na aba, 
Produção de Produtos.
O acesso será feito através do menu 
Movimentação \ Base \ Manutenção de Formulação de Produtos
.
Para um determinado produto, poderemos elaborar uma ou mais fórmulas. Cada fórmula conterá um ou mais produtos tanque e seu percentual para obtenção do produto destino. Vinculado a uma fórmula, teremos também o código do histórico de movimentação de estoque.
Ao acessar o menu, será exibida uma tela conforme a imagem abaixo:
Para incluir uma nova formulação, utilize o botão 
Incluir
, será aberta a tela de inclusão, conforme o exemplo abaixo:
Informe os dados para os campos abaixo:
Produto:
 Código do produto que será produzido.
Depósito:
 Código do deposito onde o produto já produzido será armazenado.
Histórico:
 Código do histórico de movimentação de estoque.
Após salvar, será necessário informar os produtos que farão a composição da fórmula, para isso utilize o botão 
Incluir
 que está posicionado dentro do quadro 
Composição da Fórmula
:
Informe o produto da composição (Produto Primário)
Informe o Percentual % de que será usado desse produto para a produção.
Aperte o 
OK
Repita o processo para os demais produtos primários.
Após inserir todos os produtos primários, certifique-se que o total percentual das composições some 100%.
Finalize e está concluído.
Produto correlato não faz parte da composição das fórmulas.
