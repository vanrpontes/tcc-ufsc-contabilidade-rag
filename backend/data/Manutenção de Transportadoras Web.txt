TITULO: Manutenção de Transportadoras Web
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926198445
--------------------------------------------------
Para alterar uma transportadora no SispetroWeb:
Clique no Menu que fica no lado esquerdo/superior
Selecione a opção 
Tabelas / Transportadoras
Selecione a transportadora pelo nome ou pelo código e clique no código:
O SispetroWeb apresentará a tela para alteração:
Clique em 
Salva
 ou 
Cancela
 para finalizar a alteração.
O menu de transportadora só é habilitado caso o perfil de cliente definido no SispetroMon esteja como Congênere ou Ambos (
Menu SispetroWeb / Definição Perfil Cliente
).
