Nesta tela é possível realizar a configuração das posições que serão utilizadas no status de posição Frota.
O acesso a esta tela se dá pelo Menu Tabelas Auxiliares / Transportes / Frotas / Posição Frota.
 
Para acrescentar uma nova posição de Frota acione o botão 
Incluir
Na tela para preenchimento dos novos dados, informe:
Descrição:
 nome para este controle de posição.
Considera Em Viagem:
 Como será o controle, podendo ser por:
Indiferente: 
status a ser exibido referente a viagem
Não: 
status a ser exibido referente a viagem
Sim: 
status a ser exibido referente a viagem
