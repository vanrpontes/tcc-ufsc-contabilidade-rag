A partir do painel 
Detalhes
 é possível acionar o painel 
Patrimônio / Referências
 localizado no menu inferior da tela, para mostrar a tela abaixo que será preenchida com todas as propriedades, referências bancárias e comerciais do cliente, além dos acionistas.
Para alterá-la o usuário deve ter o privilégio do perfil,
 
Alteração de Cliente
 
e ao perfil de usuário 
Pode Alterar Dados do Patrimônio do Cliente.
 
Para mais informações, acesse 
Painel Segurança (Usuários)
.
A partir da versão 10.27, para alterar os dados de Patrimônio / Referência, basta que no perfil do usuário tenha a opção 
Pode alterar dados do patrimônio do cliente
 marcado. Para mais informações ver 
Painel Segurança (Usuários)
.
