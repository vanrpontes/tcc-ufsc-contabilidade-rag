TITULO: Manutenção de Processos - Adiantamentos
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878899517
--------------------------------------------------
No processo de importação de Adiantamentos, o sistema precisa saber qual o tipo de layout (Petrobrás ou Futura) e o De/Para que será usado.


Essas informações são vinculadas na manutenção de processos, através do Menu 
Ferramentas / Importação Dados / 
Manutenção de Processos
:




Para iniciar o cadastro o usuário deve clicar no botão 
. O sistema se posicionará no painel 
Detalhes
, aguardando o cadastro das informações.




O usuário deve informar preencher os campos:


Importação:
 Deve ser informado o layout que o sistema irá utilizar para esse processo (Layout Futura ou Petrobrás).


De/Para:
 Deve ser informado o De/Para deste processo. É necessário que antes de iniciar o cadastro dos processos, o usuário já tenha cadastrado todos os De/Para, ou pelo menos o De/Para que será utilizado nesse processo.




Confirmando a operação através do botão 
, o sistema irá se posicionar no painel 
Navegação
, mostrando o registro gravado.




Caso deseje cancelar a gravação, o usuário deve clicar no botão 
.
