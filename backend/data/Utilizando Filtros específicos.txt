Os filtros específicos por coluna poderão ser utilizados para:




Cód Produto


Produto


Volume


Volume 20c


Custo Médio


Valor


Cód Depósito


Depósito




Para utilizar os filtros, o usuário deverá:




Clicar na coluna desejada e selecionar a seta indicativa de filtro








Com as opções em tela, selecionar as que são desejadas








Conforme o usuário for definindo as opções desejadas, a tela será atualizada.








O mesmo pode ser feito para as demais colunas citadas.






Para todos os filtros existe a opção Customizar:




Com essa opção, o usuário poderá acrescentar outros critérios de filtros ao campo solicitado através da tela a seguir:




Observe que na tela acima o campo utilizado para definir filtro específico foi o 
Tanque
. O usuário poderá colocar filtros onde as condições são: 




Igual


Diferente


É menor que


É menor ou igual a


É maior que


É maior ou igual a


Parecido com


Não é parecido com


É vazio


Não é vazio




Além disso, o usuário poderá definir mais de uma condição para o campo selecionado. Para isso ele deverá indicar as opções 
E (AND)
 ou 
OU (OR)
.
