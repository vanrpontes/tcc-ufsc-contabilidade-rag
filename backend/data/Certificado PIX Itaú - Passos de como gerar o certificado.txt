Obtenção de Certificado
informar os passos necessários para configurar o Postman para acessar as API's do Banco Itaú.
Geração dos Arquivos *.csr e *.key com programa Git Bash
Siga os passos abaixo
Abra o 
Git Bash
Vá até a pasta onde deseja gerar os arquivos
Execute o comando abaixo:
openssl req -new
 
-subj "//CN=CLIENT ID RECEBIDO DO ITAU\OU=SisPetro\L=Sao Paulo\ST=SP\C=BR"
 
-out Futura.csr -nodes -sha512 -newkey rsa:2048
 
-keyout Futura.key
Exemplo do comando com os dados da Futura: 
openssl req -new -subj "//CN=1exxaf03-05aa-4x2c-96xx-c24c0073ex6x\L=Sao Paulo\ST=SP\C=BR" -out Futura.csr -nodes -sha512 -newkey rsa:2048 -keyout Futura.key
Informações importantes
O arquivo csr gerado precisa ter 17 linhas, menos ou mais que isso o arquivo não foi gerado corretamente.
Gerar o arquivo no prompt de comando do DOS pode gerar problemas no arquivo (má formação).
Caso o arquivo não seja gerado com a quantidade de linhas que deveria, tente trocar o nome dos arquivos gerados.
Obter Arquivo de Certificado via Postman
No 
Postman
, é necessário fazer um post na url 
https://sts.itau.com.br/seguranca/v1/certificado/solicitacao
Na aba 
Authorization
, informe o parâmetro 
Bearer Token
, informando o TOKEN recebido através de e-mail do Itaú.
Na aba 
Headers
, informe o parâmetro 
Content-Type
 com o valor 
text/plain
Na aba 
Body
, selecione a opção 
raw
 e informe o conteúdo do arquivo csr gerado no inicio desse manual.
Na aba 
Settings
, desabilite a opção 
Enable SSL certificate verification
Clique em 
Send
. Voltando a aba 
Body
 o Postman deverá ter recebido o arquivo do certificado digital gerado pelo Itaú.
Gerando o arquivo com extensão .crt
Copie todo o conteúdo do certificado recebido, desde 
-----BEGIN CERTIFICATE-----
 até 
-----END CERTIFICATE-----
 e salve na pasta dos seus arquivos com a extensão 
.crt
Senha (secret) retornada com o certificado 
Outra informação importante retornada nesse método é a senha ( 
Secret
 ) vinda acima, essa senha será necessária para recebimento do token temporário com validade de 5 minutos.
Atenção salve a informação do 
secret, 
pois será utilizado posteriormente para configurações no Sispetro em 
Client_secret.

Para realizar o teste utilize o programa Git Bash.
