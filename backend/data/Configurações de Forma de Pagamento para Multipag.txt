TITULO: Configurações de Forma de Pagamento para Multipag
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926205381
--------------------------------------------------
Para configurar a forma de pagamento para utilizar o Multipag (CNAB 240), é necessário atentar-se as informações do Código do Banco e do Código do Banco Adicional.
É fundamental conferir junto ao Banco os códigos de 
forma de pagamento
, que podem ser alteradas pela instituição bancária sem prévio aviso.
Além disso, é importante conferir quais serviços estão disponíveis para o Banco que será utilizado para este pagamento.

código do banco
