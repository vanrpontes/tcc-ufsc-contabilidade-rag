TITULO: Códigos de Contabilização Padrão na Transferência de Adiantamentos
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878701837
--------------------------------------------------
Ao fazer o processo de transferência, dependendo da configuração da empresa, o usuário é obrigado a informar o código de contabilização para terminar o processo de transferência, porém, muitas vezes há mais de um código de contabilização disponível para o usuário selecionar.




Neste caso a solução seria configurar o sistema para sugerir de forma automática o código de contabilização para aquele processo de transferência.




Para isso o usuário deve acessar as configurações da empresa, Menu 
Ferramentas / Sistema / Painel Contab.Real
 e colocar os código de contabilização no grupo Transferências.


 


Apenas estarão disponíveis para seleção os códigos que estão configurados como Grupo Contábil de Transferência de Crédito. 


O grupo transferência, possue os seguintes campos:




Adiantamento entre clientes:
 O código de contabilização aqui informado será sugerido nas telas de "Transferência de Créditos (clientes)", "Transferência de Adiantamentos em Lotes" (cliente) e "Pagamento ao cliente";


Adiantamento entre fornecedores:
 O código de contabilização será sugerido nas telas de "Transferência de Créditos (fornecedores)", "Transferência de Adiantamentos em Lote" (fornecedores);


Adiantamento entre Empresas - Clientes:
 O código de contabilização será sugerido na tela de "Transferência Crédito entre Empresas" Clientes e na tela de "Débitos / Créditos de Clientes" (Painel Dados p/ Transf.Créditos Entre Empresas);


Adiantamento entre Empresas - Fornecedores:
 O código de contabilização será sugerido na tela de "Transferência Crédito entre Empresas" Fornecedores e tela de "Débitos / Créditos de Fornecedor" (Painel Dados p/ Transf.Créditos Entre Empresas);




Estando configurado de forma correta, ao entrar em uma dessas telas, o sistema irá sugerir o código de contabilização, porém, isso não impede o usuário de alterar o código de contabilização caso seja necessário.


Para deixar o código de contabilização nas configurações da empresa (grupo Transferências) em branco, o usuário pode utilizar o botão 
Limpa Código
.




Estando dessa forma, o sistema não irá sugerir nenhum código para o usuário, porém, dependendo da configuração da empresa, o usuário será obrigado a selecionar um código para finalizar o processo.
