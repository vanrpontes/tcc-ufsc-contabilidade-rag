TITULO: Tabela Gerenciador de Pedágio X Empresa

--------------------------------------------------
Essa tela foi removida a partir da versão 9.27, agora a configuração do Gerenciador x Empresa é realizado na tela de 
Tabela de Gerenciador de Pedágio
.
 
A tabela de gerenciador de pedágio x empresa é destinada a realização do vinculo das empresas com o gerenciador de vale pedágio, é através dessa tabela que informamos o gerenciador de vale pedágio e qual a identificação utilizada.
Para vincular uma empresa a um gerenciador de pedágio basta acionar o botão 
Altera Empresa.
Gerenciador de Pedágio:
 Nesse campo é selecionado o gerenciador de pedágio.
Identificação do Gerenciador de Pedágio:
 Nesse campo é informado a identificação do gerenciador de pedágio que representa a filial, normalmente cada filial tem o seu próprio ID de identificação.
Identificação do gerenciador de Pedágio Responsável:
 è o ID de "login" nos serviços  que pode ser o da própria filial quando trabalhar de forma descentralizada, ou o login da "filial responsável" quando trabalhar de forma centralizada e só uma empresa é responsável por todas.
Ambiente:
 Nesse campo definimos o ambiente de envio dos vale pedágios, podendo ser em 
Produção 
ou
 Homologação
.
Ativo:
 Esse campo define se o vinculo do vale pedágio com a empresa esta ativo ou inativo.
Obtém Vale Pedágio?:
 Esse campo define se para essa empresa vai obter o Vale Pedágio ou não.
 Só é possível alterar o gerenciador de pedágios de uma determinada Empresa, casó não tenha nenhum Pedido de Pedágio pendente, ou aguardando alguma ação do usuário.
