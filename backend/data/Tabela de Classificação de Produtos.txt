TITULO: Tabela de Classificação de Produtos
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878211196
--------------------------------------------------
O intuito desta tabela é organizar os produtos, fazendo com que a Empresa determine da maneira mais simples a classificação de seus produtos.
Está diretamente associada à classificação, as Contas Contábeis de Receita de Vendas, Custo de Vendas, Estoque e Conta de Despesa. No cadastramento de cada produto deve ser associada sua classificação. Maiores detalhes no tópico 
.
Para acessar a tabela de Classificação de Produtos, o usuário deve utilizar o Menu 
Tabelas Auxiliares / Vendas / Classificação Produtos
:
Acione o botão de inserção de novo registro.
O Sistema exibe a tela abaixo.
O Sistema já traz as Contas Contábeis de Receita de vendas, Custo de venda, Estoque e de Despesa definidas como padrão. O padrão é definido no perfil da empresa, através do Menu 
Ferramentas / Sistema / painel Contas Contábeis
. O usuário deve se certificar que as contas estão corretas, se não estiverem, deve-se limpar os campos e teclar <Enter> para selecionar a conta do cadastrado de contas contábeis do Sistema.
Devem ser preenchidos os seguintes campos:
Código
 da nova Classificação
Descrição
% Comissão
% Lucro Bruto
 (só estará disponível para os usuários em que no perfil - painel Produtos - esteja marcada a opção 
Pode visualizar % lucro bruto
)
Tipo do Item no Livro Eletrônico:
 Definir o tipo de item para a classificação de produto específica. Esta informação é de importância ao SPED.
Para gravar as informações o usuário deve utilizar o botão 
OK
.
Será apresentado o seguinte questionamento:
Ao responder 
SIM
, o sistema atualizará no cadastro de produtos com esta classificação o % de lucro bruto do produto de acordo com o % de lucro bruto desta classificação.
