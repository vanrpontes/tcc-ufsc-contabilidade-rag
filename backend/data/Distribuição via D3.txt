TITULO: Distribuição (via D3)

--------------------------------------------------
Este procedimento visa ajustar as quantidades que irão ser faturadas pelo fornecedor (Petrobrás) para as companhias participantes deste bombeio.
Assim, o usuário receberá como informação, a quantidade efetiva do bombeio, medida pelo fornecedor. Esta quantidade pode ser diferente do bombeio medido no tanque (pois o processo de medição é diferente). Assim, ele receberá esta informação, informando ao fornecedor o rateio do bombeio entre as empresas participantes.

Manutenção Bombeios

Além dessa movimentação para cada filial, será feita uma movimentação de Saída (Escritural e/ou Física) conforme o Volume Físico, para a filial que recebeu o Bombeio Físico (a filial conectada). No caso de ser uma Redistribuição serão apresentadas as quantidades preenchidas anteriormente e possibilitando alterá-las. Ao confirmar, não serão geradas novas movimentações, mas sim será feita a alteração das quantidades das movimentações anteriormente realizadas.
Cancela
Possibilita o cancelamento de toda a distribuição feita.
Quando confirmamos uma distribuição do bombeio, o sistema zera a distribuição provisória incluída automaticamente pelo sistema na confirmação do bombeio físico. E, quando cancelamos uma distribuição de bombeio, o sistema também zera esta distribuição provisória. Desta forma, é importante lembrar que, quando incluimos uma nova distribuição, após o cancelamento de outra, é necessário desabilitarmos a opção "Zera Provisório" para que o sistema não zere a distribuição provisória novamente.
