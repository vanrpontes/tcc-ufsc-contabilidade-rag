Para aprovar o pedido de pré-adiantamento a fornecedor, acessar o menu 
Movimentação / Financeiro / Contas a Pagar / 
Manutenção de Pré-Adiantamento a Fornecedores
 - botão 
Aprova
.




Clicar no botão 
OK
 para aprovar o pré-adiantamento.


Somente poderá aprovar o pedido de pré-adiantamento a fornecedor se o usuário tiver permissão. Para habilitar a permissão, deve acessar o menu 
Ferramentas / Usuário / painel Perfil / painel Financeiro
