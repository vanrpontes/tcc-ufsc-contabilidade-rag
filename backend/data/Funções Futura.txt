TITULO: Funções Futura
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878898208
--------------------------------------------------
No Gerador de Relatórios, conforme explicado em manual próprio, é possível codificarmos ações conforme os eventos disponíveis. Em linguagem mais simples, é como criar um programa dentro do relatório indicando o que o relatório deve fazer em determinada situação. Esta programação (codificação) permite o uso de funções (também esclarecidas no manual do Gerador de Relatórios), sendo que algumas delas são específicas da Futura.
 Nos tópicos a seguir, teremos o esclarecimento de como usá-las:
Comissão de Vendas
Conversão Moeda Real
Conversão Real para Moeda
Módulo 11
Extenso Não Monetário
Pega Valor Outros Impostos Base Cálculo
Pega Valor Outros Impostos Alíquota
