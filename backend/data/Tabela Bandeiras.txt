Esta tabela armazena as bandeiras dos postos revendedores.
A tela que permite a manipulação desta tabela é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Vendas / Bandeiras
:
A partir da versão 8.21 foi criado um novo campo "Bandeira ANP", veja abaixo como preenche-lo.
Para acrescentar uma nova Bandeira à tabela, acione o botão de inclusão de registro 
O Sistema exibe a tela para preenchimento dos dados.
Digite o conteúdo para os campos: 
Código
 e 
Descrição da Bandeira e Bandeira ANP.
Descrição: 
Para uso interno da empresa 
Bandeira ANP: 
Deve ser preenchido exatamente como está na 
. Trabalhará no bloqueio de vendas para bandeiras diferentes ou atualização das bandeiras, no cadastro de cliente. Veja mais informações em 
.
Bloqueada para Vendas: 
Caso a bandeira esteja bloqueada para vendas, marcar este perfil, assim não será possível incluir pedidos para tal. Esse perfil só funcionara se no Perfil de Sistemas, aba Vendas, perfil 
Bloqueio de Pedido por Bandeira.
Acione o botão 
OK
 para atualizar os dados no Sistema.
O Sistema retorna a tela de navegação.
Para retornar ao Menu principal acione o botão 
Sair.
