TITULO: Aba Restriçao Usuário
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2925003315
--------------------------------------------------
É possível restringir um ou alguns vendedores a determinados usuários. Para que a restrição funcione é preciso indicar no vendedor a qual usuário este será restrito. Veja abaixo
Clicar no botão 
Incluir. 
Será aberta a seguinte tela:
 
Deverá ser indicado o usuário a qual este vendedor será restrito.
Após esse passo é preciso indicar no Perfil de Usuário a restrição. Veja mais informações em 
Restrição Vendedor
Para casos de vínculo com usuários de perfil (maiores detalhes em 
Perfil de Usuário
), o acesso será para todos os usuários vinculados a este usuário.
