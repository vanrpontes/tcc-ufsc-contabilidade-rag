TITULO: Lê Arquivo
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878899805
--------------------------------------------------
Na tela de 
Manutenção de Importação de Ordens de Carregamento
 acione o botão 
 para dar início ao processo de importação de dados.


Ao acionar o botão aparecerá a seguinte tela:




Informe no campo 
Arquivo Origem
 o caminho onde se encontra o arquivo que será importado.


No painel 
Etapas
 são apresentadas as etapas que o arquivo irá passar até realmente fazer a importação.


Acione o botão 
Importa
 para iniciar o procedimento.




Depois de importar os dados para um arquivo temporário, o sistema irá mostrar no painel 
Dados
 os dados desse arquivo, porém para isso acontecer deve-se marcar a opção 
Visualiza os dados antes de transferí-los para o arquivo destino
 que irá mostrar os dados do arquivo antes de importá-lo.


Acione o botão 
Continua
 para dar continuidade no processo de importação. Se a leitura do arquivo for concluída com sucesso, o sistema apresentará uma mensagem e dará início ao processo de importação de OC, caso contrário, apresentará mensagem informando que houve erro e essa informação poderá ser consultada no painel 
Erros
, conforme tela abaixo:




Os erros acontecem quando os dados do arquivo foram incluídos diferentes do layout padrão definido ou se deixou de informar algum dado ou por qualquer outro motivo. Portanto o sistema irá rejeitar o arquivo mostrando no painel 
Erros
 quais foram as rejeições encontradas.


Acione o botão 
Sai
 para voltar para a tela de 
Manutenção de Importação de Ordens de Carregamento
.
