Nesta aba é possível incluir as informações das entregas da OC. Para mais informações, ver 
O acesso é feito através do Menu Movimentação / Estoque / Manutenção Ordem de Carregamento / Aba Entregas
Botões
Inclui / Altera 
Resumo: 
o motorista / usuário irá informar as datas e hora prevista e real das entregas 
as data e hora da entregas dessa tela podem ser preenchidas tanto pelo usuário na base quanto pelo motorista no App. Para mais informações sobre, ver 
Quadro Início Entrega (Saída Base)
Previsto:
 data e hora Prevista que o motorista sairá para entrega. 
Real: 
data e hora Real que o motorista sairá para entrega. 
Quadro Finalização Entrega
Previsto:
 data e hora Prevista que o motorista irá finalizar a entrega. Será preenchido com a última data prevista de entrega, caso na OC tenha mais que uma entrega
Real:
 data e hora Real que o motorista finalizou a entrega. 
Quadro Volta à Base
Previsto:
 data e hora Prevista que o motorista irá retornar a Base. 
Real:
 data e hora Real que o motorista voltou à Base
Quadro Entregas: 
Através do botão 
Altera Datas
 é possível preencher as datas Prevista de Real de cada uma das OC. 
Para salvar as informações, basta clicar no botão 
OK
. Caso queira sair, clique no botão 
Cancela
Quadro Quilometragem:
 
Início (Saída Base): 
o motorista deve informar o KM marcado no odômetro do veículo antes de iniciar as entregas.
Final (Chegada à Base):
 o motorista deve informar o KM marcado no odômetro do veículo após 
finalizar 
as entregas e retornar a base
Quadro Situação Entrega: 
o usuário / motorista irá informar qual a situação atual da entrega dele. Ela pode ser: 
Viagem não iniciada
Em Andamento
Realizada Última Entrega
Chegada à Base
Excluí: 
excluí as informações de entrega da OC
