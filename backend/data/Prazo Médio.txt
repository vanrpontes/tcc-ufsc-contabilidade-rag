TITULO: Prazo Médio
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878440255
--------------------------------------------------
Este relatório mostra o prazo médio de vendas (em dias) por cliente, vendedor ou produto. É possível fazer vários filtros e obter resultados agrupados da forma desejada.
O acesso a esse relatório é através do menu 
Relatórios / Vendas / Prazo Médio
:
Informe os filtros para obter os resultados desejados:
Filial:
 Pode-se obter os resultados por filial, todas as filiais, por grupo de filiais ou por conglomerado de filiais.
Período:
 Informe o período que se deseja obter os resultados.
Agrupamento:
 Pode-se selecionar a forma de agrupamento para melhor visualização dos dados no relatório. Os níveis são divididos em Nível 1 e Nível 2, e nessa ordem são mostrados os dados.
 
Analítico / Sintético:
 A opção Analítico mostra de forma detalhada e a opção Sintética apresenta de forma resumida.
O layout do relatório será gerado conforme o arquivo 
PrazoMedio.rtm
 que estiver no caminho do executável. É possível editar o Layout do relatório através do botão 
"Edita Layout"
:
Clique 
OK
 para obter o relatório.
 
O cálculo feito pelo relatório para chegar no prazo médio é:
(Quantidade NF * Prazo NF) / Quantidade Total.
