TITULO: Processos - Fiscal
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2879390057
--------------------------------------------------
O roteiro de implantação do módulo Fiscal é mais simples que o do módulo contábil, podendo ser feito num tempo mais curto. É composto dos seguintes itens:


1.
 
Identificação dos tipos de fórmulas
 para os livros fiscais necessários (valor contábil/isentas/outras) - Setor Contábil


2.
 
Verificação de dados incompletos de Entidade
 das Notas que irão para o Livro Fiscal - Setor Contábil


3.
 
Inclusão/Ajuste das fórmulas do Livro Fiscal
 - Suporte Futura


4.
 
Processamento de Notas Fiscais de meses anteriores
 - Setor Contábil


5.
 
Inclusão de Apuração Fiscal para conferência dos dados
 - Setor Contábil


6.
 
Ajuste das Fórmulas do Livro Fiscal e Reprocessamento de Notas Fiscais
 - Setor Contábil/Suporte Futura
