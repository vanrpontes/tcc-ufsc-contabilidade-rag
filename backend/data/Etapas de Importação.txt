TITULO: Etapas de Importação

--------------------------------------------------
Nesta tabela são armazenadas as 
Etapas de Importação
 utilizados para organizar os processos de um PMM que seja de importação. Aqui o usuário poderá incluir, alterar e excluir estes Processos.

Para acessar a tela de Etapas de Importação, o usuário deve utilizar o menu: 
Tabelas Auxiliares / Movimentação de Material / Etapas de Importação
.




O sistema abrirá a tela , mostrando dados já cadastrados. Para inserir um novo código, o usuário deve clicar no botão 
.




O sistema entrará em modo de edição, permitindo que o usuário preencha o campo:


Processo:
 
Processo
 ao qual essa Etapa será vinculada.


Descrição:
 O nome da Etapa de Importação a ser cadastrado. Neste caso o sistema aceitará letra e números, se for o caso.


O campo código é sugerido pelo próprio sistema e não poderá ser alterado.


Para finalizar o cadastro, o usuário deve clicar no botão 
. O sistema se posicionará na tela de navegação, onde o usuário poderá consultar uma Etapa de Importação que foi cadastrada ou as já existentes.


Para consultar algum registro, basta o usuário clicar na coluna que se deseja pesquisar ("Código"), desta forma o sistema organizará as informações e posicionará o cursor no campo "Pesquisa", aguardando a digitação por parte do usuário




A coluna que o usuário clicar, ficará em amarelo, demonstrando assim, que o sistema fará a pesquisa por essa coluna.
