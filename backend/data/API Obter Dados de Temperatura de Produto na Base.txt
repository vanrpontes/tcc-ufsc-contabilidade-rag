TITULO: API Obter Dados de Temperatura de Produto na Base
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/4722393089
--------------------------------------------------
Objetivo
Obter os dados de temperatura e densidade para um determinado produto na base conforme o tanque ativo no momento para este produto.
1. Obter token autenticação
Introdução
Qualquer webservice do projeto 
SispetroWeb 
exigirá um token de autenticação. Este token é gerado a partir deste webservice e será válido por 24 horas. Assim, recomendamos que o sistema consumidor dos webservices aqui descritos salvem o token armazenando também a data e hora em que ele foi gerado para que, quando chegar próximo de sua validade, seja obtido novo token.
Para mais informações, ver 
Documentação - API Obtenção de Token

Retorno Sem Sucesso
Em formato JSON, será obtido um token caso os dados informados sejam validados e autenticados, a empresa esteja vinculada ao usuário, usuário esteja ativo e válido e empresa esteja ativa e válida. 
O status da resposta sempre será 200.

    "densidade": "0.75"

Retorno Sem Sucesso
Se, por algum motivo, não for possível gerar o token, será gerado uma resposta com formato JSON a seguinte resposta (
O status da resposta sempre será diferente de 200)
