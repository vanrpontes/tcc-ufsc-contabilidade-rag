TITULO: Inclusão de Frota Web
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2927269621
--------------------------------------------------
Para incluir um frota no SispetroWeb, é necessário habilitar o direito de usuário, localizado em 
Ferramentas / Usuários - Perfil - Fretes
:
O menu de Frotas só é habilitado caso o perfil de cliente definido no SispetroMon esteja como 
Congênere
 ou 
Ambos
 (
Menu SispetroWeb / Definição Perfil Cliente
).
Clique no Menu que fica no lado esquerdo/superior
Selecione a opção 
Tabelas / Frotas
Selecione a opção 
Incluir
 na parte inferior direita da tela:
Preencha com as informações do Frota:
Clique em 
Salva
 ou 
Cancela
 para finalizar a inclusão.
