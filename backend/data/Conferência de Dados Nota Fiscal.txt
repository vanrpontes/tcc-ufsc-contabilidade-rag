TITULO: Conferência de Dados Nota Fiscal
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2924291411
--------------------------------------------------
É possível, durante a inclusão manual de uma Nota Fiscal, conferir se os dados digitados estão corretos. 
Será aberta a seguinte tela
Os dados informados aqui, nesta tela, serão confrontados com os digitados. Após clicar no botão Finaliza o SisPetro analisará e caso haja alguma divergência, será exibida
Não será permitido finalizar a Nota Fiscal caso os dados não sejam iguais.
