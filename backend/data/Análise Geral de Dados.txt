TITULO: Análise Geral de Dados
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878440182
--------------------------------------------------
Com a finalidade de permitir a realização de consultas personalizadas, encontra-se disponível no Sispetro a tela de Analise Geral de Dados, a qual pode se acessada pelo menu 
Consulta / Analise Vendas / Análise Geral
. 




Sendo disponibilizadas pelo campo "Categoria" as consultas criadas, conforme a permissão do usuário. 

Nesta tela encontram-se disponíveis os menus: 




Manutenção Categoria
: permite acesso ao cadastro de Categorias (consultas) permitidas de serem realizadas


Exportação
: por meio deste menu é permitido ao usuário a realização da exportação das informações presentes no grid para um dos três formatos "XML" (Linguagem de Marcação muito empregada na comunicação de informações na internet), "XLS" (Excell) e "HTML" (Formato de página de internet)


Impressão
: por meio deste menu é permitido ao usuário realizar a impressão, visualização das informações presentes no grid, além de configurar a impressão (o que deverá ser apresentado ou não, cores, fontes, etc..) e a página.


Layout
: por meio deste menu é disponibilizado ao usuário a opção de salvar o layout do grid, isto é, manter a ordem das colunas, agrupamentos realizados, ordenação dos dados. Esse recurso é especialmente importante para casos onde desejamos visualizar a informação sempre no mesmo formato como por exemplo. Em uma consulta de todas as vendas por período poderiamos agrupar o campo data da venda, e ordená-los por valor e salvar o layout, sendo assim todas as vezes que realizassemos a consulta e selecionácimos o Layout salvo, e os dados no grid seriam dispostos assim encontra-se no layout selecionado.


Grupos
: Este recurso, facilita ao usuário abrir todos os agrupamentos sem a necessidade de clicar sobre cada grupo por vez, isso quer dizer que ao invés de abrirmos todos os agrupamentos para exportarmos para o excell (por exemplo) pressionariamos o menu "Abre Todos" presente neste menu e todos os agrupamentos apresentariam então as informações que os compõem.


Filtro
: Neste menu serão disponibilizados alguns filtros que poderam ser realizados nos dados selecionados, sendo divididos em dois grupos:
	


Específico
: Podemos realizar a filtragem dos dados fazendo uso de uma tela de consulta específica para a informação (o que nos permitirá consultar a informação a ser empregada na consulta, como por exemplo a se selecionar a opção "Entidade", seria aberta uma tela com onde deverá ser informado o código da entidade, mas caso não se saiba o código especificamente é permitido consultar a Pesquisa de Entidade). Estando disponíveis as seguintes opções: Entidade, Vendedor, Transportadora, Condição Pagamento, Tipo Cobrança, Empresa e Produto.


Genérico
: Neste caso é apresentada uma tela de pesquisa onde estarão disponíveis todos os campos presentes no grid, contudo deverá ser informado o valor diretamente se nenhuma pré-consulta, como é permitido no caso do filtro "Específico".






Atualiza: Após informada as datas "De" e "Até", bem como a "Categoria" será por meio deste menu que os dados serão atualizados no grid.
