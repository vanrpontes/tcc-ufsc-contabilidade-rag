TITULO: Questionamentos para travamento do Servidor Hermes
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2927273724
--------------------------------------------------
Para eventuais travamentos do Servidor Hermes, utilizaremos os questionamentos abaixo para que possamos identificar possíveis problemas:
Está rodando somente o Hermes (sem o servidor SisPetro) para o normal?
Quantos Hermes estão sendo executados? 
Serviços juntos ou em outro Hermes?
É possível conectar o Controlador Hermes neste caso?
Se tiver mais de um Hermes, qual ficou parado?
Há quanto tempo eles estavam sendo executados?
Como estava a memória e processamento do servidor no momento?
