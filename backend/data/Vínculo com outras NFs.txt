TITULO: Vínculo com outras NFs
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2921074139
--------------------------------------------------
Caso a natureza de operação esteja configurada para 
Permite vincular NF a outra NF emitida
, o botão Vinculo Docto Fiscal ficará habilitado para uso, também pode ser acessado pelo atalho "ALT+D". Para maiores informações, vide tópico de 
Tabela de Natureza de Operação
.
Isto é, se este perfil estiver configurado como 
Sugere referenciar outras NFs
 ficará a critério do usuário vincular manualmente outras notas a esta que está sendo incluída.
Caso este perfil esteja configurado como 
Força referenciar outras NFs
 o usuário será obrigado a vincular uma ou mais notas a esta nova nota fiscal. Caso ele não vincule, o sistema enviará uma mensagem e não permitirá finalizer a nota.
Para efetuar o vínculo com outras notas fiscais, vide o tópico de 
Seleção de Notas para Vínculos
.
Icon
Importante: Para as notas fiscais de devolução e de complemento, este vinculo será feito automaticamente. Basta o usuário informar a origem, e o sistema efetuará o vínculo através dos dados fornecidos. Para maiores informações, vide tópico para Notas Fiscais de
 
Devolução de Cliente
,
 
Complemento de Preços
 
e
 
Complemento de Impostos
.
