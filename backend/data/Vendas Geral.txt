TITULO: Vendas Geral
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878440233
--------------------------------------------------
Este relatório mostra uma situação geral de Vendas. É possível fazer vários filtros e obter resultados agrupados da forma desejada. 


O acesso a esse relatório é através do menu 
Relatórios / Vendas / Vendas Geral
: 




Informe os filtros para obter os resultados desejados: 




Filial:
 Pode-se obter os resultados por filial, todas as filiais, por grupo de filiais ou por conglomerado de filiais.


Vendedor:
 Marque essa opção se deseja filtrar por vendedor, e informe o código do mesmo logo em seguida.


Produto:
 Marque essa opção se deseja filtrar por produto, e informe o código do mesmo logo em seguida.


Comprador:
 Marque essa opção se deseja filtrar por comprador do pedido, e informe o código do mesmo logo em seguida.


Cliente:
 Marque essa opção se deseja filtrar por cliente, e informe o código do mesmo logo em seguida.


Níveis de Agrupamento:
 Pode-se selecionar a forma de agrupamento para melhor visualização dos dados no relatório. Os níveis são divididos em Nível 1, Nível 2 e Nível 3, e nessa ordem são mostrados os dados. É possível alterar as opções de agrupamento conforme for necessário, as opções são: Cliente, Vendedor e Produto.


Período:
 Informe o período que se deseja obter os resultados.




Caso deseje obter os resultados de forma resumida, marque a opções 
Relatório Resumido
. 

Se a empresa desejar pode-se alterar o layout utilizando o botão 
"Edita Layout
". 


Clique 
OK
 para obter o relatório
