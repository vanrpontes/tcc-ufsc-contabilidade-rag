TITULO: Notas Fiscais Emitidas
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878899422
--------------------------------------------------
Objetivo
Este relatório mostra todas as Notas Fiscais (Entrada, Saída e conhecimentos) que foram movimentadas no sistema. É possível fazer vários filtros, selecionar o tipo de operação, fazer ordenação desejada e etc.
O acesso a esse relatório é através do menu 
Relatórios > Faturamento > Notas Fiscais Emitidas:
O layout do relatório será gerado conforme o arquivo 
NFEmitidas.rtm
 que estiver no mesmo caminho do executável, sua existência é imprescindível.
Esse relatório possui os seguintes filtros:
Filial:
 pode-se obter os resultados por filial, todas as filiais e por grupo de filiais. Se for por filial, informe o código da filial, se é por grupo de filiais, informe o grupo.
Cliente/Fornecedor específico:
 marque essa opção se deseja filtrar por um determinado cliente/fornecedor e informe o código logo em seguida.
Vendedor específico:
 marque essa opção se deseja filtrar por vendedor e informe o código logo em seguida.
Comprador específico:
 marque essa opção se deseja filtrar por comprador da nota e informe o código logo em seguida.
Tipo Nota Fiscal: 
selecione se o relatório em questão irá listar:
Nota fiscal:
 quando esse tipo é selecionado o relatório irá mostrar os tipos de operação abaixo
Conhecimento:
 quando esse tipo é selecionado o relatório irá mostrar os tipos de operação abaixo
Ordem:
 selecione se deseja ordenar o relatório por:
Vendedor
Entidade
Período
#Nota Fiscal.
Razão Social Entidade Faturada
Razão Social Comprador
Tipo de Operação:
 marque os tipos de Notas Fiscais que deseja imprimir, por padrão todos os tipos são apresentados marcados.
Intervalo # Nota Fiscal:
 informe o intervalo de número de notas fiscais para obter resultados mais precisos, caso for necessário.
Período:
 informe o período que se deseja obter os resultados.
Filtra CFOP?:
 marque essa opção caso deseje filtrar por um CFOP específico.
Incluir NFs Canceladas?: 
marque essa opção caso deseje incluir as notas canceladas
É possível editar o relatório em questão clicando no botão 
Edita Layout  
Clique 
OK
 para gerar o relatório
