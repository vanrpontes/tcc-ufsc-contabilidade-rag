TITULO: Tabela de Aplicações Frota
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878211038
--------------------------------------------------
Nesta tabela são cadastradas as Aplicações de Despesas com Frotas a serem utilizadas no 
Controle de Despesas com Frota
.
A 
Aplicação Frota
 é um indicador de uma utilização ao qual o evento está relacionado. E assim, poderemos dividir o Frota para agrupagamento e organização das despesas.
Exemplo:
 Sistema de propulsão (motor, combustível, óleos e afins)
 Transmissão (câmbio, eixos, diferencial, etc)
 Chassis
A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - 
Tabelas Auxiliares / Transportes / Frotas / Aplicação
:
Para acrescentar uma nova Aplicação acione o botão de inclusão de registro 
.
O Sistema exibe a tela para preenchimento dos novos dados.
Informe a 
Descrição
 para a Aplicação, ou seja, uma denominação para onde vai ser aplicada a despesa quando cadastrada.
Caso seja necessário ativar/ desativar uma Aplicação, marque/ desmarque o campo 
Ativo
.
Acione o botão 
OK
 para atualizar os dados no Sistema.
O Sistema retorna a tela de navegação.
Para retornar ao Menu principal acione o botão 
Sair
