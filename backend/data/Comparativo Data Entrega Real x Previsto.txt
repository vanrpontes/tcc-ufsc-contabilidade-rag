TITULO: Comparativo Data Entrega Real x Previsto
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2880798863
--------------------------------------------------
Este relatório tem por objetivo permitir ao usuário fazer um comparativo de informações. Considerando a data de entrega real e o que foi previsto.


Para isto deve-se acessar esta tela através dos menus: 
Relatórios/ Transportes/ Comparativo Data Entrega Real x Previsto
.




Informe a 
Data Inicial
 e a 
Data FInal
 para pesquisa. Em seguida, informe se o *Tipo* é *Retira* ou *Entrega*.


Após finalizar, clique no botão *Ok*. 


Para cancelar a pesquisa, clique no botão *Cancelar*.
