Nesta tabela são cadastradas as tabelas progressivas a serem utilizadas na 
.
A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - Tabelas Auxiliares / Controladoria / Tabela Progressiva. 
Para acrescentar uma nova tabela progressiva, acione o botão de inclusão de registro: 
 
O Sistema exibe a tela para preenchimento dos novos dados. 
Informe a 
descrição
 e o 
valor limite por dependente
 (este valor está sendo cadastrado, mas ainda não está sendo utilizado devido não termos a informação do número de dependentes das entidades).
No painel "Faixas", conforme tela apresentada: 
Cadastre os valores de 
ínicio
 e 
fim
 desejados, sempre iniciando do 0,00 e terminando com um valor nas casas de bilhões. Atente-se para não deixar intervalos sem serem abrangidos. Portanto, se uma faixa termina num valor, a próxima faixa deverá iniciar neste valor acrescido de 0,01. Informe 
alíquota
 e a 
dedução
.
Como é utilizada a tabela progressiva
:
O sistema verificará em que faixa o valor da base de cálculo se encaixa.
Identificando a faixa, o sistema aplica sobre a base de cálculo a alíquota da faixa
Sobre o valor obtido é então diminuído o valor informado no campo dedução da faixa
 Veja maiores detalhes em 
Cálculo de Descontos-Acréscimos
.
Acione o botão OK para atualizar os dados no Sistema.
O Sistema retorna a tela de navegação. 
 Para retornar ao Menu principal acione o botão Sair:
