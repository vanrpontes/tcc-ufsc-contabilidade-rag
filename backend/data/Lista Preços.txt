Através do Menu Relatórios 
/Vendas / Lista Preços
, o usuário poderá emitir um relatório referente as tabelas de preço cadastradas no sistema. 




Através do Grupo "Filial", pode-se selecionar uma entidade especifica, um grupo de filiais ou todas as filiais para a composição dos dados do relatório. 




Pelo grupo "Classificação de Produto", o usuário pode selecionar todas as classificações que estejam vinculadas aos produtos ou apenas uma determinada classificação ou ainda um determinado intervalo. 




Mais informações sobre classificação de produtos no tópico 
Tabela Classificação Produtos
. 


A opção "Seleciona Tabela de Preço", vem marcada como padrão, forçando o usuário a selecionar uma determinada tabela de preços. 




Não sendo o caso de se emitir o relatório para uma determinada tabela, o usuário deve desmarcar essa opção, assim, o sistema apresentará todas as tabelas que se encaixam nos filtros selecionados. 


O usuário deve selecionar dentro do grupo "Opções de Tabela", o tipo de tabela de preço que será pesquisado: "Geral" ou por "UF". Sendo por "UF", o usuário deve selecionar no campo "UF", se são todas as UFs, ou uma especifica.O filtro também pode ser aplicado a vigência. O sistema exibe os valores praticados na data escolhida, independentemente de existir vigência com a mesma data, caso a data não seja coincidente o valor retornado é o da vigência imediatamente anterior a data em questão. 


 


O sistema permite que seja apresentado no relatório os produtos com preço zerado, ou seja, que não possuem preço cadastrados. Para isso, deve-se marcar a opção: 




Para facilitar a visualização e conferência do relatório, o usuário pode imprimir o relatório zebrado e ordenar o relatório. Para imprimir o relatório zebrado, o usuário deve marcar a opção: 




Desta forma, ao ser impresso, o relatório terá uma linha com o fundo branco e outra linha com o fundo cinza. 




A ordenação, consiste em organizar os dados apresentados em tela, classificando-os conforme selecionado no grupo "Ordem". 




Cada opção do Grupo "Ordem", possue a opção de ordenar os dados por "Código de Produto", "Alfabética", "Classificação" ou "UF", este último caso se esteja utilizando as opções de tabela por UF. 


O usuário pode também fazer alterações no relatório, utilizando o botão 
"Edita LayOut"
. Lembrando que se poderá alterar a estrutura do relatório, alterando fontes, configurando tamanho de pagina e apenas incluindo campos que estejam disponíveis. 


Para a emissão do relatório, o usuário deve utilizar o botão "OK", caso contrário, deve ser utilizado o botão "Cancela".
