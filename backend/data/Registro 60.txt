É o registro que informa o Estoque para cada tipo de produto.


Os dados a serem preenchidos nesse registro são:


01 - Tipo de registro - 60

02 - Código do Produto SEF - código fixo conforme manual do SCANC

03 - Data do Estoque

04 - Quantidade do produto

05 - Valor BC ST Estoque

06 - Quantidade em estoque proveniente de transferência
