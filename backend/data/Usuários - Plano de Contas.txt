A partir da versão 8.10 foi criado o 
Perfil para grupos de usuários
 , note que ao atualizar uma conta, valerá para todos os usuários do grupo. Veja maiores informações no help de 
 
Perfil para grupos de usuários
 
Para visualizar os usuários que podem acessar uma conta é necessário marcar uma opção localizada no painel 
Detalhes
, pois somente os usuários que tiverem permissão poderão fazer a movimentação utilizando determinada conta. É dada esta permissão a partir do Menu 
Ferramentas / Usuário / Perfil / painel Plano Contas
:
Do lado esquerdo aparecerão as Contas Disponíveis, que só irão aparecer ali se estiver com aquela opção marcada no painel 
Detalhes
 citada logo acima na tela do 
Plano de Contas
.
Selecione a conta que será permitida o acesso para o usuário e utilize os botões de seleção que se encontram no centro da tela para mover a conta para o lado direito, onde ficam as Contas Selecionadas que o usuário poderá fazer movimentação.
Portanto, para consultar os usuários que tem permissão para movimentar a conta, deve-se primeiro selecionar a conta no painel 
Navegação
 e em seguida acionar o painel 
Usuários
.
