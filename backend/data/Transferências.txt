TITULO: Transferências
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878898271
--------------------------------------------------
Para realizar uma transferência de uma conta de banco para outra (por exemplo, para realizar um DOC ou depósito de um fundo fixo para o banco e vice-versa), procedemos com o procedimento Transferência.


Assim, clique no botão 
Transferência
 para iniciar o procedimento.




Preencha os seguintes campos:




Data:
 Data da Transferência.


Conta Destino:
 A conta para onde o montante irá ser transferido (lembrar que a conta origem é a conta anteriormente selecionada - a que estamos trabalhando).


Valor:
 Valor do movimento.


#Documento:
 Número do documento relativo a transferência (DOC, etc).


Histórico:
 Histórico da transferência entre as contas (irá aparecer no extrato das duas contas).




Caso sejam transferidos cheques, utilize o botão 
Lista Cheques Recebidos
 para selecionar/incluir os cheques desejado. Veja maiores detalhes sobre esta tela em 
.


De acordo com o tipo de documento o valor da transferência pode ou não ser diferente do valor total dos cheques incluídos/selecionados. Vide na 
Tabela Tipo Documento
 o item 
Valida valor do movimento associado
 para ajustar esta opção.


Se estiver habilitado o módulo Contábil, será exibida a opção 
Pode Contabilizar
. Para esta opção poder ser desmarcada ou ser trocado o código de contabilização é necessário que no menu 
Ferramentas / Sistema, painel Contab. Real
 esteja marcada a seguinte opção:




No caso de transações que hajam cobrança de juros, ou seja, para a conta destino só vai um percentual do valor transferido, já que a outra parte vai servir para pagamento de juros, marque a opção 
Desconta valores (juros,etc)
 e informe a 
conta de juros
, caso não saiba o código da conta, acione a tela de pesquisa para selecionar, serão mostradas apenas contas do tipo Despesa. Não existe perfil no sistema que habilite mostrar todos os tipos de contas. Informe também o 
valor dos juros
. A diferença apresentada será o valor que movimentado na conta destino.


Aperte o botão 
OK
 para confirmar a transferência.
