TITULO: Atualizar Sistema
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2965439124
--------------------------------------------------
Para acessar a tela de Atualização do Sistema, acessar o 
Menu
 
Início > Outros > Atualizar Sistema.
Através dessa tela, podemos visualizar os campos abaixo:
SispetroWeb Versão Atual:
 exibe a versão atual do SispetroWeb
URL SispetroWeb:
 exibe o IP da máquina do SispetroWeb
URL S3:
 exibe o IP e porta do webservice do Hermes S3
URL Hermes:
 exibe o IP e porta do webservice do Hermes Serviços
Versão Atual App:
 exibe a versão atual do SispetroApp
TESTE COMUNICAÇÃO HERMES:
 testa a comunicação das portas do webservice configuradas no SispetroMon e HermesControlador
VERIFICAR ATUALIZAÇÃO:
 verifica se existem novas atualizações disponíveis e permite a realização da atualização
Cancelar:
 volta para a tela acessada anteriormente
Se não houverem novas atualizações, será apresentada a mensagem conforme abaixo:
Caso não queira efetuar a atualização, clique no botão 
Cancelar.
