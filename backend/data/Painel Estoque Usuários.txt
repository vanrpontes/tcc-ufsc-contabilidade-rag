Conhecendo os perfis existentes:
Quadro Fechamento de Estoque: 
através do quadro de Fechamento de Estoque que iremos definir as permissões de liberação para fechamento de estoque, podendo permitir ao usuário realizar o Fechamento e abertura de um determinado estoque. Essa permissão é muito útil para empresas onde o usuário que realiza o fechamento do Estoque Escritural não é o mesmo que fecha o Estoque físico ou Master.
Quadro Master: 
o Fechamento de estoque master é o Fechamento do dia, após realizar esse fechamento dependendo de como a empresa trabalha não é mais possível realizar movimentações para esse dia, ou ate mesmo realizar outros fechamentos com data anterior ao ultimo fechamento Master.
Permite abrir fechamento de estoque: 
permite que o usuário possa abrir o Fechamento de estoque Master.
Só abre fechamento de estoque com nº dias permitidos: 
permite apenas que seja aberto o fechamento de estoque Master caso não tenha ultrapassado o numero de dias permitidos para abertura de fechamento de estoque, limite de dias informado no perfil de sistema 
Dias anteriores permitidos para abertura do estoque/custo
.
Pode fazer fechamento de estoque: 
permite que o usuário possa realizar o Fechamento de estoque Master.
Quadro Escritural: 
através desse quadro iremos configurar e dar as devidas permissões para os usuários que poderão realizar o fechamento de estoque escritural.
Permite abrir fechamento de estoque: 
permite que o usuário possa abrir o Fechamento de estoque Escritural.
Só abre fechamento de estoque com nº dias permitidos: 
permite apenas que seja aberto o fechamento de estoque Escritural caso não tenha ultrapassado o numero de dias permitidos para abertura de fechamento de estoque, limite de dias informado no perfil de sistema 
Dias anteriores permitidos para abertura do estoque/custo
.
Pode Confirmar Entrada/saída:
 habilita disponibiliza que o usuário possa realizar a confirmação de Entrada/Saída.
Permite Altera Data Movimento no fechamento de Estoque: 
permite que seja alterado a data de movimento no fechamento de estoque.
Pode fazer fechamento de estoque: 
permite que o usuário possa realizar o Fechamento de estoque Escritural.
Pode alterar Tolerância das Descargas ?:
 permite o usuário alterar a tolerância de descarga da nota fiscal na tela de fechamento de estoque escritural, que é utilizada na 
Apuração de Fretes a Pagar
.

Quadro Movimento entre Depósitos
Pode incluir movimento: 
permite ao usuário incluir Movimentos entre Depósitos.
Pode cancelar movimento: 
permite ao usuário cancelar Movimentos entre Depósitos.
Quadro Depósito
Pode incluir/Excluir depósito: 
permite que o usuário Incluir/Excluir Depósito.
Pode alterar no depósito verificação de estoque: 
permite que o usuário alterar no depósito a verificação de estoque.
Pode Criar/Alterar Depósito Virtual: 
permite que o usuário Crie/Altere o Depósito Virtual.
Pode Vincular/Desvincular Depósito: 
permite que o usuário Vincule/Desvincule o Depósito.
Pode habilitar restrição de Usuário:
 b
loqueia as movimentações de determinado depósito por usuário. Para que seja possível informar quais usuários poderão ou não fazer movimentações no depósito é preciso habilitar este perfil. Para mais informações, ver 
Restrição por Usuário
Pode dar manutenção em Dados de Lacres e Envelopes: 
permite que o usuário altere as informações referentes a Lacres e Envelopes na Manutenção de Depósitos.
Pode dar Manutenção em Horários Alternativos: 
permite ao usuário dar manutenção a horários extraordinários para Congêneres no 
Depósitos - Aba Agendamento

Permite abrir Fechamento de Custo: 
permite que o usuário abra o fechamento de custo.
Só abre fechamento de custo com nº dias permitidos: 
permite apenas que seja aberto o fechamento de custo caso não tenha ultrapassado o numero de dias permitidos para abertura, limite de dias informado no perfil de sistema 
Dias anteriores permitidos para abertura do estoque/custo
.
Pode alterar custo médio na tela de Atualização: 
permite que o usuário altere o custo médio na tela de atualização de custos.
Quadro Bombeio 
Pode incluir/Cancelar Prog. Bomb.:
 permite que o usuário inclua e cancele Programação de Bombeio.
Pode incluir confirmação de Bombeio:
 permite que o usuário inclua confirmação de Bombeio.
Pode cancelar conf. Bombeio:
 permite que o usuário cancele confirmação de Bombeio.
Pode cancelar bombeio:
 permite que o usuário cancele o Bombeio.
Pode alterar Movimentação de Estoque Bombeio Físico: 
permite que o usuário altere movimentações de Estoque em Bombeio Físico. 
Trabalha em conjunto com o perfil de
 
Depósitos - Aba Dados Adicionais
.
Pode associar NF ao Bombeio: 
permite que o usuário associe NF ao Bombeio.
Pode alterar Movimentação de Estoque Bombeio Físico: 
permite que o usuário altere as Movimentações de Estoque referentes aos Bombeio Físico.
Quadro Ajuste de Estoque
Pode fazer ajuste de Estoque Inicial (Entrada/Saída):
 permite que o usuário realize ajustes de estoque Físico, tanto de entrada quanto de saída.
Ação sobre verificação de Ajuste Manual: 
define a verificação que o Sispetro irá realizar no Ajuste Manual conforme opções abaixo:
Não verifica nunca: 
não realiza nenhuma verificação, e permite que o usuário realize ajustes com qualquer quantidade, e quantas vezes quiser.
Respeita o Depósito: 
realiza uma verificação conforme perfil informado no depósito.
Pode fazer ajuste manual de Estoque Físico: 
permite que o usuário realize ajustes manuais de estoque Físico.
Pode fazer ajuste manual de Entrada Escritural:
 permite que o usuário realize ajustes Entrada de estoque Escritural.
Pode fazer ajuste manual de Saída Escritural:
 permite que o usuário realize ajustes de Saída de estoque Escritural.
Pode realizar movimento entre filiais/congêneres: 
permite que o usuário realize movimentações entre Filiais e Congêneres.
Pode ver saldo em estoque de todas as empresas: 
permite que o usuário visualize o saldo de estoque de todas as empresas.
Pode cancelar Pedido de Outras Saídas/Entradas: 
permite que o usuário cancele pedidos de Outras Saídas/Entradas.
Quadro ANP - Estoque Diário
Pode dar manutenção?:

Pode imprimir/gerar planilha?: 
habilita os botões 
Imprimir 
e 
Gerar Planilha
.
