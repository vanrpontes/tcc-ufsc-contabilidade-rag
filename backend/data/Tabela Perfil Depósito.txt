TITULO: Tabela Perfil Depósito

--------------------------------------------------
Para utilizar um perfil genérico em diversos depósitos o primeiro passo e configurar um 
Perfil de Depósito
 que fica em Tabelas Auxiliares / Estoque / Perfil de Depósito.
Para incluir, clique no 
Descrição:
 descrição do Perfil de deposito. 
% Sobre Movimentação de Entrada:
  nesse campo é informado a porcentagem sobre a movimentação de Entrada.
% Sobre Movimentação de Saída
: nesse campo é informado a porcentagem sobre a movimentação de Saída.
Nº de Dias para Verificação:
 nesse campo é informado o numero de dias para verificação.
Nº de Ajustes (Diários) por Depósito/Produto/Empresa:
  nesse campo é informado a quantidade máxima de ajustes que será permitido por depósito.
