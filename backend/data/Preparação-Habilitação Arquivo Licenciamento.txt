TITULO: Preparação-Habilitação Arquivo Licenciamento
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2879128262
--------------------------------------------------
A preparação do arquivo de licenciamento é parte fundamental para o correto funcionamento do Sispetro. Um arquivo corretamente configurado e validado faz com que o processo de validação do usuário fique transparente não ocasionando nenhum problema no seu dia a dia. 


Assim, é importante entender o processo de preparação e validação do mesmo, que são: 




Captura das credenciais:
 O Sispetro monitora todas as estações de trabalho que eventualmente utilizem o mesmo para seu trabalho diário. Com as credenciais de cada uma delas, vai formando uma tabela com as suas credenciais. Assim, para que uma determinada estação de trabalho possa ser incluída na lista de estações a ser ativadas, deve ter tido pelo menos um acesso para que o Sispetro a registre.






Edição do Arquivo Licenciamento:
 Processo que um usuário habilitado escolhe quais estações de trabalho deverá fazer parte da lista de estações para ativação. O usuário só pode escolher dentre as estações que eventualmente tenham enviado suas credenciais (isto é, que o Sispetro as tenha capturado) marcando-as para ativação. Neste momento também, deve-se preencher os dados da empresa que comprou os direitos de uso do Sispetro, tipo de licenciamento e número de estações ativadas.






Geração Arquivo:
 Após conferência, deve-se gerar o arquivo de licenciamento quando o Sispetro irá juntar todas as informações com relação ao licenciamento num arquivo (com extensão LIC e com nome que o usuário desejar) salvando-o para posterior envio via correio eletrônico para a Futura Tecnologia.






Recebimento/Registro Arquivo Habilitado:
 Após uma validação, a Futura Tecnologia ativará o arquivo de licenciamento ativando todas as estações de trabalho no arquivo de licenciamento tornando os dados reais (e não propostos como antes da ativação). Após ativar o arquivo de licenciamento tornando-o habilitado, Futura Tecnologia enviará o arquivo para o cliente que o registrará no Sispetro.
