Dados Gerais
 O Banco J Safra trabalha em conjunto com o Banco Bradesco (237). Assim, a empresa tem conta no Banco J Safra, onde adquire o produto Cobrança Eletrônica, porém, ao imprimir o boleto, o imprime com as características do Banco Bradesco. Dessa forma, o cliente que recebe o boleto não precisa localizar uma agência do Banco J Safra para efetuar o pagamento, podendo fazer isso perfeitamente no Banco Bradesco, que tem muito mais agências disponíveis em todo o país. Para atender essa necessidade usaremos a configuração de Banco Correspondente do SisPetro. Cabe lembrar que o arquivo de remessa é conforme o layout do Banco J Safra, muito parecido com o layout do Bradesco, mas não idêntico.

 deve ser impresso com a carteira (CC), código do cliente no Safra (SSSSS), o nosso número (NNNNNN) e o dígito verificador(D), no seguinte formato: CC/SSSSSNNNNNN-D. 
 
 Só será gravado no SisPetro, para visualização do Contas a Receber, o nosso número (NNNNNN).
O dígito verificador é calculado no módulo 11, utilizando até a base 7.
