A partir da versão 8.4 é possível emitir o relatório de ficha de bens, este apresenta todos os dados dos bens como dados de depreciação, CIAP, contas gerenciais, valor original, valor atual, dados do SPED e observações. É acessado pelo menu 
Relatórios/Ficha de Bens.
O layout do balanço será gerado conforme o arquivo FichaBem
.rtm
 que estiver no caminho do executável, sua existência é 
imprescindível
. Para alterar o layout utilize o botão 
Edita layout.
É possível filtrar:
Filial
Grupo de filiais
Conglomerado de Filiais 
Todas as filiais 
Por
 produto
 ou
 bem,
 lembrando que se for filtrado por produto o relatório tratá todos os bens deste.
