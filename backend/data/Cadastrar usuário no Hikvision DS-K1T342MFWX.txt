TITULO: Cadastrar usuário no Hikvision DS-K1T342MFWX
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/4651778049
--------------------------------------------------
Objetivo
Cadastrar usuário no Hikvision para acessar
 o dispositivo DS-K1T342MFWX ou realizar o vínculo com o cadastro do Motorista do SisPetro.
 
As informações contidas aqui é um apanhado obtido do manual do fabricante. Para mais informações, veja o manual conforme modelo  
Passos Necessários
Acesso ao dispositivo Hikvision DS-K1T342MFWX:
Abra um navegador web em seu computador.
Na barra de endereço do navegador, digite o endereço IP do dispositivo Hikvision DS-K1T342MFWX e pressione Enter.
Isso deve abrir a página de login do dispositivo.
Fazer login com a conta admin:
Insira as credenciais de login, geralmente o nome de usuário é 
admin
 e a senha é definida pelo usuário durante a 
configuração inicial
 do dispositivo.
Clique em 
Login
 ou pressione Enter para acessar o painel de controle do dispositivo.
Ir até a guia User (Usuário):
No painel de controle do dispositivo, localize e clique na guia ou menu 
User
 (Usuário). Isso pode estar localizado na barra de navegação superior ou em um menu lateral, dependendo da interface do dispositivo.
Acionar um usuário e cadastrar as informações necessárias:
Na guia 
User
 (Usuário), você deve encontrar opções para adicionar.
Clique em 
Add
 (Adicionar usuário) ou em uma opção semelhante para começar a adicionar um novo usuário.
Preencha as informações necessárias para cadastrar o novo usuário. Isso geralmente inclui employee ID (CPF do Motorista), name, gender, level, entre outras informações relevantes.
Para carregar a imagem do usuário clique no campo conforma imagem abaixo:
O formato da imagem deve ser JPG ou JPEG ou PNG e o tamanho deve ser inferior a 200 K. 
Após preencher todas as informações necessárias, clique em 
OK
 (Para Salvar).
