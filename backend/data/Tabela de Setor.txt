TITULO: Tabela de Setor
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210323
--------------------------------------------------
tabela de Setor é utilizada para indicar a localização dos bens. A manutenção desta tabela é acionada a partir do Menu principal com as opções - 
Arquivos / Setor. 




Acione o botão de inserção de registro. 




O Sistema exibe a tela para o cadastramento de um novo setor. 




Informe uma
 
descrição
 
para o novo setor.


Acione o botão OK para atualizar os dados no Sistema.


O Sistema retorna a tela de navegação.


Para deixar a tela de Setores acione o botão Ok ou Cancela.
