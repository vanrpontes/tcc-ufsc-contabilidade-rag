TITULO: Inclusão-Alteração de Pré-Adiantamento
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878701893
--------------------------------------------------
Para incluir/alterar o pedido de pré-adiantamento a fornecedor, acessar o menu 
Movimentação / Financeiro / Contas a Pagar / 
Manutenção de Pré-Adiantamento a Fornecedores
 - botões 
Inclui
 e 
Altera
.
Ao incluir um pré-adiantamento o sistema fará a checagem se existe outro pré adiantamento pendente para o fornecedor, caso a opção de Sistema - menu 
Ferramentas / Sistema / painel Financeiro - "Na inclusão avisa se tiver adiantamentos pendentes", 
esteja marcada, avisando assim sobre o item existente.
Na alteração do pré-adiantamento, somente é permitido alterar se o pré-adiantamento estiver com status pendente.
Somente poderá incluir, alterar ou excluir o pedido de pré-adiantamento a fornecedor se o usuário tiver permissão. Para habilitar a permissão, deve acessar o menu 
Ferramentas / Usuário / painel Perfil / painel Financeiro
:
A partir da versão 9.22 temos a opção de trabalhar com grupo econômico para fornecedores, onde ao trabalhar com grupo econômico será possível selecionar um PMM do Fornecedor informado no PMM ou um PMM do Fornecedor responsável pelo grupo econômico. Veja também 
Grupo Econômico - Fornecedores
.
