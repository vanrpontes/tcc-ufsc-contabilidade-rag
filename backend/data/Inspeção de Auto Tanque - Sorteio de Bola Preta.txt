Histórico
Bola Preta é um procedimento de sorteio aleatório (ou forçado, se o usuário assim desejar), durante a operação de carregamento e descarga, para uma inspeção mais minuciosa. Ele é chamado assim pois antigamente este sorteio era realizado colocando bolas brancas e pretas num recipiente e sorteando uma bola que, se fosse preta, indicaria a necessidade de realização de uma inspeção adicional no frota.
Esta inspeção é sempre realizada
 
após
 
o procedimento de portaria de entrada, possibilitando a configuração para
 
carregamento, descargas ou ambas operações.
Configurações necessárias no SisPetro

Cancelar uma inspeção
Liberar ou não uma OC (frota) selecionada para inspeção
Inspecionar uma OC (frota) não selecionada para inspeção
Incluir o resultado de uma inspeção
Pode visualizar dados de sorteio

Frota:
Permite Inspeção:
 se marcada, o frota (OC) será selecionado para sorteio e/ou permitir inspeção;
Inspeção Imediata:
 se marcada, o frota (OC) obrigatoriamente será inspecionado.

Portaria de Entrada:
 não há qualquer alteração para o Bola Preta. Se o depósito estiver habilitado para fazer Bola Preta, no momento em que é realizada a leitura (seja manual, Basecom, Infitech, SmartCard ou VIew) da OC para fazer o carregamento ou descarga, a OC entra para sorteio (sem interferência do usuário). O resultado é visualizado na Portaria de Saída.
