Através do menu
 
Relatórios / Auxiliares / Contábil / Códigos de Contabilização
 
é possível emitir uma relação dos códigos de contabilização, detalhando ao invés do código do campo (quando informado entre <>) a devida identificação do mesmo (ex.: NotaFiscal.NumeroNF).


Este relatório é muito útil para conferência.


O layout do relatório será gerado conforme o arquivo
 
CodigoContabilizacao.RTM
 
que deve estar localizado no mesmo diretório que o executável. Sua existência é
 
imprescindível.
 
Para alterar o layout utilize o botão 




Clique no botão OK para confirmar e o relatório será apresentado em tela. Para imprimí-lo clique no botão
