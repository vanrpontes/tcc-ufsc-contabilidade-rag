Esta aba trabalha em conjunto com o 
Restrição de Clientes por usuário
. Serve para incluir usuário de acesso, ou caso seja informado um usuário existente, o cliente é adicionado a lista de clientes que o usuário tem acesso. Pode ser uma alternativa, para incluir um novo usuário do SisPetro Web também, porém é necessário saber qual usuário que ele se baseará.
Caso o usuário necessite ter acesso apenas a determinados clientes, o Administrador do Sistema pode optar em ir cliente a cliente liberando o acesso. Veja abaixo como fazer.
Clicar no botão "Incluir", abrirá a seguinte tela
Deverá indicar o E-mail do usuário, ou abrir a tela de pesquisa. Caso o usuário não tenha cadastro o sistema não localizará, mas mesmo assim é necessário informar.
Os campos "Nome do usuário" e "Usuário Modelo" serão automaticamente preenchidos. Se o e-mail já estiver previamente cadastrado. Caso não seja localizado, deve informar o nome e em qual usuário, este novo, se baseará.
Feito isso, o novo usuário foi incluído. Para visualizar se está de acordo, vá em 
Restrição de Clientes por usuário
