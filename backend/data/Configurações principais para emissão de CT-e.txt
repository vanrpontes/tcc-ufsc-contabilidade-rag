Objetivo
A NF-e (modelo 57) que e conhecida como Conhecimento de Transporte é um documento digital que serve para formalizar o transporte de produtos entre locais.
Para realizar emissão de CT-e, e necessário:

Informações Adicionais
Em 
Painel Filial >
 
Impressão NF, caso
 precise, informar algumas observações da nota 
Para que as observações sejam separadas para fisco e para o próprio contribuinte, deverá preencher as observações dentro de tags (<tag>....</tag>), conforme o exemplo abaixo. As tags utilizadas para Informações Adicionais são:
Informações Adicionais de interesse do Fisco:
 
<infAdFisco> ..... </infAdFisco>
Informações Complementares de interesse do contribuinte:
 
<infCpl> ..... </infCpl>
Estas tags serão utilizadas para separar o conteúdo do campo 
Observações 
em campos distintos, no layout do DANFE de forma que você possa utilizar estes campos no seu layout (por exemplo, colocando o campo que informa somente as informações adicionais para o Fisco, num determinado local no seu layout de DANFE).
O preenchimento deve ser feito conforme mostrado no exemplo acima, para que, ao montar o XML, considere as observações informadas pelo usuário.
