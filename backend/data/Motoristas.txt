Para acessar a tela de Manutenção de Transportadoras/Motoristas, acessar o 
Menu
 
Início > Tabelas Aux > Motoristas.
Na tela de Manutenção e em algumas outras telas do SispetroApp, podemos realizar alguns filtros através dos botões abaixo:
Legenda:
Digite alguma informação para efetuar a pesquisa
Incluir Transportadora/Motorista
Editar Transportadora/Motorista de venda selecionado
Buscar dados mais recentes
Filtrar dados
Imprimir
Definir rodapé da tabela
Selecionar colunas a serem exibidas na tela
Resetar layout da tabela para layout padrão
No Rodapé da Tela, temos alguns botões que permitem a navegação entre as páginas listadas e também podemos definir o número de linhas por página:
Para editar uma Transportadora/Motorista, selecione o registro desejado e clique no botão 
Para salvar as alterações, clique no botão 
Confirmar
 ou 
Cancelar 
para descartá-las.

Toda transportadora incluída pelo SispetroApp fica bloqueada e depende do desbloqueio para a utilização. Para desbloquear, acesse no SisPetro Normal o menu: 
Tabelas Auxiliares > Transportes > 
Transportadoras/Motoristas
.
Para definir o padrão de bloqueio, acesse no SisPetro Normal o menu: 
Ferramentas > Sistema > Painel 
Ordem Carregto
