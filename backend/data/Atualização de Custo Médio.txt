Objetivo
Permitir a atualização de custo médio em Notas Fiscais (entradas ou saídas) e Ajustes (entradas ou saídas) quando identificado que o custo médio esta incorreto, sem a necessidade do processo de cancelamento, exclusão e inclusão da nota fiscal ou do ajuste.

Painel Estoque (Usuários)
A partir da versão 10.23 do SisPetro, é possível ajustar o custo médio não só das notas fiscais de entrada como também dos pedidos de ajustes de entrada/saída. Foram implementados também nesta versão, os filtros de busca por 
Depósito
 e pelo 
Tipo Pesquisa.  
O campo 
Tipo Pesquisa
 permite selecionar Notas Fiscais ou pedidos de Ajustes.
Importante
As movimentações de saídas de estoques que não participam do CMV interferem no cálculo do custo médio. Por isso é importante, tanto para as notas fiscais como para os ajustes, ter o valor unitário correto:
Ajustes:
 o valor unitário de um ajuste de saída utiliza o último custo médio fechado.

Informe a Empresa, o Período, o Depósito, selecione o Tipo de Pesquisa e a Natureza/Histórico e clique no botão Atualizar para carregar os dados:
Notas Fiscais: 
No campo Custo Médio Atual informa o valor unitário da nota fiscal
No campo Novo Custo Médio sugere o valor unitário da nota fiscal, podendo ser alterado para o valor desejado
Selecione as notas que deseja atualizar o custo médio. Pode selecionar individualmente ou clicar no botão Marca/Desmarca. Por padrão, todas as notas estão marcadas - campo Calcula.
Para alterar o valor do custo médio:
Individualmente: quando as notas tem diferentes custos médios altere o campo Novo Custo Médio informando o novo custo para cada nota 
Em lote: quando as notas tem os mesmos custos médio
Informe o valor no campo Custo Médio e clique no botão Alterar
Processando as alterações - utilizando alteração em lote
Informe o novo custo médio no campo Custo Médio
Verifique se todas as notas estão marcadas - campo Calcula
Estando tudo correto clique no botão Alterar. Será apresentada a mensagem:
Caso esteja tudo de acordo, na mensagem clique em Sim. Voltará para a tela e o campo Novo Custo Médio apresentará o custo médio informado.
Para que as alterações sejam gravadas, clique no botão Salvar
Na mensagem clique em OK para finalizar as alterações. As notas serão apresentadas com o novo custo no campo Custo Médio Atual e no campo Novo Custo Médio apresentará o valor unitário original das notas. 
Para sair da tela, clique em Fechar
Ajustes: 
No campo Custo Médio Atual informa o valor unitário do ajuste
No campo Novo Custo Médio sugere o valor unitário gravado no pedido, podendo ser alterado para o valor desejado
Selecione os ajustes que deseja atualizar o custo médio. Pode selecionar individualmente ou clicar no botão Marca/Desmarca. Por padrão, todos os ajustes estão marcados - campo Calcula.
Para alterar o valor do custo médio:
Individualmente: quando os ajustes tem diferentes custos médios altere o campo Novo Custo Médio informando o novo custo para cada ajuste
Em lote: quando os ajustes tem os mesmos custos médio
Informe o valor no campo Custo Médio e clique no botão Alterar
Processando as alterações - utilizando alteração em lote
Informe o novo custo médio no campo Custo Médio
Verifique se todos os ajustes estão marcados - campo Calcula
Estando tudo correto clique no botão Alterar. Será apresentada a mensagem:
Caso esteja tudo de acordo, na mensagem clique em Sim. Voltará para a tela e o campo Novo Custo Médio apresentará o custo médio informado.
Para que as alterações sejam gravadas, clique no botão Salvar
Na mensagem clique em OK para finalizar as alterações. Os ajustes serão apresentados com o novo custo no campo Custo Médio Atual e no campo Novo Custo Médio. 
Para sair da tela, clique em Fechar
Produtos compostos
A partir da versão 9.30 serão apresentados além do produto composto, os produtos primários em azul.
Dicas de filtros
Para facilitar a consulta foi adicionada uma tela mais dinâmica, onde é possível filtrar pela coluna desejada. Para isso basta arrastar a coluna desejada para a parte cinza da tela.
Veja um exemplo abaixo onde a coluna 
#Número NF
 foi arrastada e com isso o SisPetro faz o agrupamento pelo número da nota fiscal.
