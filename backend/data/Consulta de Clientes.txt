TITULO: Consulta de Clientes
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2880798720
--------------------------------------------------
O SIspetro disponibiliza algumas formas de pesquisa ao cliente. Uma delas é a tela de 
Consulta de Clientes
, que pode ser acessada através do menu 
Consultas/ Clientes




Esta tela permite ao usuário informar o número do CPF ou CNPJ do cliente e, automaticamente, o SIspetro trará as demais informações do cliente. Isto facilita para o usuário que é vendedor, pois enquanto mantém contato com o cliente, ele pode efetuar uma pesquisa ao cliente de forma rápida, podendo verificar, por exemplo, qual o status do cliente ou quaL vendedor está vinculado ao seu cadastro.


Ao informar se é 
Pessoa Jurídica
, o Sispetro habilitará o campo 
CGC/CPF
 formatado para a digitação de dados do CNPJ do cliente. Caso contrário, o campo 
CGC/CPF
 será habilitado e formatado para a digitação de dados do CPF do cliente.


Ao sair do campo 
CGC/CPF
 (com as teclas <ENTER> ou <TAB>), o Sispetro irá verificar, automaticamente, a existência dos dados informados no banco de dados. Existindo os dados, os demais campos (
Razão Social, Endereço, Número, Complemento, Munícipio, UF, Status e Vendedor
) serão preenchidos automaticamente.

Caso o sistema não encontre os dados informados, emitirá a mensagem abaixo:




Para efetuar uma nova pesquisa, deve-se clicar no botão 
Nova Pesquisa
.




Para sair da tela, clique no botão 
Fechar
.
