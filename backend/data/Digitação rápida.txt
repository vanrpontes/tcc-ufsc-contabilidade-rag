TITULO: Digitação rápida

--------------------------------------------------
Para incluir vários lotes de uma forma rápida, utilize no Menu Principal as opções
 
Movimentação / Digitação / Rápida.




A filial sugerida para inclusão dos lotes gerados nesta tela será a filial que o usuário estiver conectado. Porém é possível que seja escolhida outra filial, desde que ela faça parte do mesmo grupo de empresas.


Informe 
Data
, 
Número de documento
 e 
Código de Histórico
. Se no cadastro deste histórico tiver sido informado as contas de débito e crédito, estas já serão sugeridas. O sistema permite o lançamento de registros com data retroativa, desde que não se tenha travamento contábil para o período. Mesmo lançando o lote com data retroativa, o sistema grava a data atual de inclusão do registro, esse campo pode ser utilizado relatórios. Maiores informações sobre travamento contábil no tópico 
.


Informe as 
contas para débito e crédito
, ou confirme-as. As contas podem ser informadas pelo código reduzido, padrão do sistema, ou então pelo código extendido. Para informar o código extendido, deve ser marcada no perfil da empresa, Sispetro Normal / Ferramentas / Sistema / painel Contab.Real, a opção 
"
Permite pesquisar conta pelo código extendido"
 e além disso, também deve ser configurado por usuário a opção em Ferramentas / Usuários / painel Contabilidade 
"Pesquisa conta real pelo código extendido"*
.*


Informe o
 
Valor
, o
 
Histórico
 
e as
 
Informações Adicionais.
 
Acione o botão
 
Inclui.


Este lote será incluído no sistema, mas não consolidado ainda. Marque a opção
 
Repete os valores para o novo registro
 
para que os valores sejam repetidos (podendo ser alterados) e facilite a digitação.


Na parte inferior da tela será exibido o valor total e o número de lotes incluídos. Ao terminar as inclusões, utilize o botão
 
Consolidar
 
para providenciar a
 
consolidação
 
destes lotes. Se por algum motivo não for feita a consolidação dos lotes por esta tela, utilize a tela
 
Manutenção de Lançamentos
 
para providenciá-la, mas neste caso, deverá ser feito lote a lote.
