TITULO: Inclusão de código de Benefício Fiscal
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/4013719703
--------------------------------------------------
Para realizar a inclusão de um novo código de Benefício Fiscal basta ir no menu Ferramentas > Tabelas do Conjunto De Dados > Escolher o Conjunto de Dados 2, encontrar a tabela LFEAjusteInfo e clicar 2 vezes.
Será aberto a tela de Dados tabela: LFEAjusteInfo.
Após isso basta acionar o botão Incluir, e cadastrar conforme desejado.
