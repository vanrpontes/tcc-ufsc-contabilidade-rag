TITULO: Impostos NF
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878899418
--------------------------------------------------
Este relatório apresenta os valores dos impostos informados nas Notas Fiscais (Entrada / Saída) que foram movimentadas no sistema. É possível fazer vários filtros, por unidade de negócio, tipo de nota (Entrada/Saída), produto e data de emissão da NF.


O acesso a esse relatório é através do menu Relatórios / Faturamento / 
Impostos NF:




O layout do relatório será gerado conforme o arquivo 
ImpostosNF.rtm
 que estiver no mesmo caminho do executável, sua existência é 
imprescindível
.


Informe os filtros para obter os resultados desejados:




Filial:
 Pode-se obter os resultados por filial, todas as filiais e por grupo de filiais. Se for por filial, informe o código da filial, se é por grupo de filiais, informe o grupo.


Produto Específico:
 marcando essa opção, serão apresentadas somente as notas fiscais que possuam o produto selecionado em sua composição.


Tipo de Nota:
 Existem apenas duas opções 
"Entrada"
 e 
"Saída"
, mas esse filtro poderá ser utilizado para filtragem somente das notas emitidas para fora do estado ou somente as emitidas para o próprio estado da empresa se este assim o desejar, caso não queira realizar a filtragem das notas basta selecionar a opção "Todas".


Período:
 Através deste filtro o usuário deverá informar o período de emissão das notas que serão analisadas.


Relatório Resumido:
 Marcando essa opção o relatório irá apresentar somente os valores sumarizados das notas, sem entrar em detalhes como número da nota, data de emissão, valor unitário e quantidade. Essa opção por padrão vem desmarcada, para que o usuário possa realizar uma verificação nota a nota dos impostos.




Neste relatório as notas fiscais canceladas não serão apresentadas.


Clique 
OK
 para obter o relatório
