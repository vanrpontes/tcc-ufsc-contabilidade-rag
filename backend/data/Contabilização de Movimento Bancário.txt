TITULO: Contabilização de Movimento Bancário

--------------------------------------------------
Através do menu
 
Relatórios / Auxiliares / Contábil / Contabilização de Movimento Bancário
 
é possível emitir a movimentação de uma conta gerencial do tipo Banco num período com a indicação se tal movimentação foi contabilizada e no caso positivo quais as contas movimentadas. 




Informe uma conta gerencial específica, ou desmarque a opção para todas as contas gerenciais do tipo Banco. Informe também o período.


O layout do relatório será gerado conforme o arquivo
 
ContabilizacaoBanco.RTM
 
que deve estar localizado no mesmo diretório que o executável. Sua existência é
 
imprescindível.
 
Para alterar o layout utilize o botão 




Clique no botão OK para confirmar e o relatório será apresentado em tela. Para imprimí-lo clique no botão
