TITULO: Serviço de Monitoramento para Servidores Sispetro
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2926194017
--------------------------------------------------
A partir da versão 9.12 foi implementado um serviço de monitoramento dos Servidores Sispetro.
Este serviço irá monitorar o status de cada servidor cadastrado, sendo o status do banco e da internet.
Para configurar o Servidor Sispetro, entre no Painel "Servidor Monitoramento" e defina uma porta que esteja livre, para que o sistema configure um WebService:
Para ter certeza de que a porta está livre, cliquem em "Testar Porta" e o sistema apresentará a mensagem:
Caso não esteja livre, apresentará a seguinte mensagem:
Para que o Servidor Sispetro inicie o serviço do WebService automaticamente, selecione a opção "Iniciar WebService Automaticamente".
Finalizadas as configurações acima, selecione a opção "Iniciar WebService". Neste momento foram finalizadas as configurações deste Servidor Sispetro.
Este procedimento deve ser feito para cada Servidor Sispetro iniciado.
 
Para parar o monitoramento, selecione a opção "Parar WebService".
 
Finalizadas as configurações do Servidor Sispetro, deve-se configurar o SispetroMon. Para maiores informações, acesse SispetroMon.
