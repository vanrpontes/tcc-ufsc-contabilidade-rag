TITULO: Encerramentos
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878210372
--------------------------------------------------
Trimestralmente ou anualmente (depende de cada empresa) deve ser feito o encerramento das contas de Receita e Despesa para apuração do resultado da empresa (lucro ou prejuízo). 
Para visualização dos encerramentos, bem como manutenção dos mesmos é necessário acessar a tela de encerramento, a partir do Menu
 
Movimentação / Manutenção Encerramentos:
Na tela de manutenção de encerramentos, serão visualizadas apenas enceramentos da empresa conectada, além de ser possível visualizar os dados, inclusive a conta de resultado utilizada e os valores lançados em cada conta de receita ou despesa que precisou ser fechada, através dos painéis 
"Itens Receitas"
 e 
"Itens Despesa".
Para fazer o encerramento de um determinado mês, o usuário deve clicar no botão:
O sistema abrirá a seguinte tela:
O encerramento das contas deve ser feito para cada filial, portanto, o sistema já sugere a filial na qual o usuário está conectado. Deve-se fazer o encerramento conectando antes na filial desejada.
O sistema também verificará qual foi o último encerramento feito e apresentará a data na tela, sugerindo o mês atual como data do próximo encerramento. No caso de ser o primeiro fechamento, o mês anterior será preenchido com um mês a menos do mês do encerramento atual.
 
Lembre-se que o encerramento pode ser feito por período.
Deve-se informar a conta de resultado que receberá os lançamentos de crédito e débito que fecharão cada conta de receita e despesa, além de informar o histórico que represente este processo.
O usuário deve informar até que data de movimentação deve ser feito o levantamento de dados para o fechamento das contas. Após confirmado o processo, não poderá ser feita nenhuma movimentação com data anterior a esta informada. Se necessário for, deve-se cancelar o encerramento, fazer as movimentações e depois providenciá-lo novamente.
Pressione o botão Ok para gravar o encerramento.
