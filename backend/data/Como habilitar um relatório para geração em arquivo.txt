TITULO: Como habilitar um relatório para geração em arquivo
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2927272361
--------------------------------------------------
Para habilitar um relatório para que o mesmo seja salvo em arquivo, selecione a opção "Report" na Estrutura do Relatório e marque as opções "AllowPrintToArchive" e "AllowPrintToFile", conforme mostra a figura abaixo:
Para que os dados seja gerados em arquivo txt, selecione o menu 
Arquivo / Configurar impressão para Arquivo
, marque as
 Faixas
 e selecione os Campos (
Controles Disponíveis
) que deseja que sejam gerados no arquivo, com a devida separação, definida no campo
