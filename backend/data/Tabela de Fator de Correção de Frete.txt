Nesta tabela são cadastrados os fatores de correção para frete.
O acesso é feito através do menu 
Tabelas Auxiliares / Transportes / Tabela de Fator de Correção de Frete. 
Para cada tipo de frota temos o cálculo de frete de forma diferenciada, ou seja, um valor para cada tipo de frota. 
Surgiram novas situações que se fez necessário um fator para ajustar o frete cobrado, a exemplo do 
Bitruck. 
Para acrescentar um novo Frota acione o botão
 
Incluir
O SisPetro exibe a tela para preenchimento dos novos dados, painel
 
Detalhes.
Campos
ID: 
definido pelo SisPetro automaticamente
Descrição:

Fator de Correção

informar um valor, positivo ou negativo, que será
