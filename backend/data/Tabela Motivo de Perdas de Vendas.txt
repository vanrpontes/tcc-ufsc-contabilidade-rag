TITULO: Tabela Motivo de Perdas de Vendas
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/2878439623
--------------------------------------------------
A partir do Menu 
Tabelas Auxiliares / Vendas
 pode-se cadastrar os principais motivos de perdas de vendas.
Esses motivos serão utilizados na inclusão de Perdas de Vendas da tabela Cliente, vide maiores detalhes em 
Perdas de Venda
:
Para acrescentar um novo Motivo acione o botão de inclusão de registro 
.
O Sistema exibe a tela para preenchimento dos novos dados.
O código é preenchido automaticamente, portanto deve-se apenas cadastrar uma descrição para o novo registro.
Para inativar um Motivo basta desmarcar a opção 
Ativo
. No painel navegação os motivos inativos são mostrados com a cor de fundo 
vermelha
.
Clique 
OK
 para salvar o novo registro.
