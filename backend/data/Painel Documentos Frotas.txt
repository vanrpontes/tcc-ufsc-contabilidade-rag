TITULO: Painel Documentos (Frotas)
URL: https://futura.atlassian.net/wiki/spaces/SPT/pages/4616552451
--------------------------------------------------
O painel Documentos permite ao usuário controlar os arquivos de Transportadoras e Motoristas dentro do SisPetro. O acesso à tela se da através do Menu 
Tabelas Auxiliares / Transportes / Frotas / Manutenção Frota
Para acesso a esta função se faz necessário a habilitação do perfil de usuário:
Menu Ferramentas / Usuário / Aba superior Perfil / Aba inferior 
Frete
 / Quadro Manutenção de Frotas / Opção Pode incluir/excluir Documentos (Arquivos) de Frotas?
Frota: 
código da frota
Nome Pasta:
 nome gerado automaticamente pelo sistema conforme cadastro
Visualizar: 
abre arquivo selecionado no painel
Atualizar: 
atualiza lista de documentos
Gerenciamento de Arquivos: 
botão para incluir e gerenciar arquivos
 
Arquivos Vinculados a esse Processo
Carregar Arquivo
Pesquisar no Computador: 
abre a janela para pesquisa de arquivo a ser carregado
Opções de Vínculo
Desvincular: 
remove vínculo do arquivo com a frota
Geral
Visualizar:
 abre arquivo selecionado no painel
Gravar:
 salva alterações realizadas
Legenda:
 status dos arquivos de acordo com as cores
Cancelar:
 desfaz alterações realizadas
