Avisos importantes
Para que as consultas da ANP e Sintegra funcionem corretamente, as DLL SisPetroANP.dll e SisPetroSintegra.dll devem ser salvas, a partir desta versão, na pasta onde está executável do Servidor SisPetro.
No Relatório de Fretes, foram excluídos os condutores de dados Frota, FrotaCarreta, FrotaTreminhão e Transportadora, para as empresas que utilizam esses condutores, será necessário fazer uma adaptação no layout RlFrete.rtm.
Geral
Sybase 16
A partir dessa versão estaremos atualizando o SisPetro para rodar e atualizar as rotinas do Sybase 16. UPA (CCF 26764)
Manutenção Fornecedor – Aba principal
A tela foi alterada para não apresentar as colunas desnecessárias. CPC (CCF 25729)
Criação de Usuário
A rotina de criação de usuário a partir dos dados de um usuário já existente estava mantendo o nome do usuário de origem dos dados como usuário de perfil. Usuários criados nessa situação não respeitam o seu perfil utilizando sempre os perfis do usuário de origem. Esse problema foi corrigido, porém, é necessário alterar no cadastro de usuários os usuários nessa condição. 
UPA (CCF 26925)
Perfil de Usuário "Inclui adiantamento quantificado de produto"
O Perfil "Inclui adiantamento quantificado de produto" estava oculto. Foi corrigido. 
PRFC (CCF 27036)
Acesso Sispetro Web
Disponibilizada a aba "Acesso Web" no Cadastro de Clientes e Cadastro de Usuários onde é possível vincular os usuários do Sispetro aos Clientes para acesso ao Sispetro Web. 
O Sispetro Web já está em fase de testes, com previsão de disponibilização para comercialização em Março/2015, onde, nessa fase, será permitida a inclusão de Pedido de Venda pela Web, seja por Vendedores Externos, Clientes, etc.

Tela de Manutenção de Rateio de Estoque
Foi incluída nessa tela a opção de filtrar os registros por período.

CPC (CCF 21848)
Tela de Alteração de Itens de Carregamento
A tela de Alteração de Itens de Carregamento apresentava no campo Volume NF a somatória da Quantidade do Item da NF, foi realizada uma alteração para nesse campo seja apresentada a somatória da Quantidade Ambiente do Item da NF.
 CPC (CCF 26038)
Atualização do Fator de Conversão e Quantidade 20c
Na versão 8.18, foram feitos ajustes na atualização do Fator de Conversão e Quantidade a 20c. Nessa versão, essa alteração foi replicada para o Faturamento Automático, 
o que também corrige o fato do campo Qtde20 ir sempre zerado nos Pedidos de OS dos produtos básicos
. 
CPC (CCF 26073, CCF 26856)
Faturamento
Validação de Chave de Acesso em Emissão de Terceiros
A validação de chave de acesso em Notas Fiscais de Terceiros será realizada apenas ao finalizar a nota. PRFC (CCF 26412)
Indicação do Número da NF para Emissão Própria
Foi criada uma nova configuração na tabela de Série da NF para possibilitar que o usuário informe o Número da Nota quando a série for do Tipo Emissão Própria. 
A alteração descrita não se aplica para o Faturamento Automático, o mesmo continuará com a regra anterior e será alterado em uma próxima versão. As Distribuidoras não devem alterar a numeração de Notas Próprias.
 
A partir dessa versão, não será permitido incluir/alterar uma Série com o Tipo Emissão "Indefinido".

Foi realizada uma alteração no SisPetro para não permitir que no cadastro de Cliente, seja salvo um cliente com o Endereço de Consumo indefinido. O campo também foi disponibilizado no Remoto. CPC (CCF 25967)

Prevenção de dados serem apagados indevidamente
Na tela de Geração de Arquivo de Distribuição da NF-e a opção "Apaga conteúdo do diretório onde os arquivos serão criados" será sugerida desmarcada para evitar que o conteúdo do diretório seja apagado devido usuário não ter reparado na opção. GSN (CCF 26843)
 
Vínculo de NF em empresa transportadora 
Apresentar no campo "Tipo NF" todas as opções de NF, mesmo quando a empresa for transportadora, ao Vincular um Documento Fiscal. 
CPC (CCF 26887)
Erro de CPF na Geração do XML do CT-e
O arquivo XML do CT-e estava considerando o campo errado para geração de dados referente ao CPF do Remetente, Destinatário e Retirada. Após correção, será considerado corretamento o valor do campo "CNPJ/CPF".
 PRFC (CCF 26978)
Acréscimo/Desconto no Tipo de Cobrança para Faturamento Automático
No faturamento automático, em algumas situações em que o tipo de cobrança possuía acréscimo ou desconto, a parcela a ser gerada não estava sendo calculada corretamente. Esse problema foi corrigido.

Transporte
Manutenção de Frete - botão Pagto
Na tela de Manutenção de Frete é possível selecionar vários Fretes e gerar o Contas a Pagar pelo botão "Pagto", ao fazer isso, o SisPetro realiza uma validação em cada um dos fretes selecionados. Nessa versão, foi realizada uma alteração para otimizar a performance dessa validação.

. CPC (CCF 26855)

Ao lançar uma OC de Venda, caso o usuário altere a Transportadora, o SisPetro altera automaticamente os dados do Pedido Pendente de Faturamento. Nessa versão, foi realizada uma alteração para não permitir isso, já que o dado alterado foi o da OC e não do Pedido.
