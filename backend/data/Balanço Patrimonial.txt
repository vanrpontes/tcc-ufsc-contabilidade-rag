O Balanço Patrimonial pode ser emitido a partir do menu
 
Relatórios / Balanço.
 
O layout do balanço será gerado conforme o arquivo
 
balanco.rtm
 
que estiver no caminho do executável, sua existência é
 
imprescindível
. Para alterar o layout utilize o botão 
Para incluir informações sobre o Responsável Legal pela empresa, acesse 
Dados do Responsável Legal pela empresa nos Relatórios Contábeis
.
 
Atentar-se as atualizações do arquivo .rtm a cada atualização do Sistema. Para que as alterações feitas sejam consideradas, é necessário sobrepor o .rtm existente pelo atual.
 
Opções do relatório:
 
 
Filial Específica:
 
Caso marque, informe a
 
filial
 
para qual deseja emitir o balanço
Caso desmarque a opção
 
"Filial específica", 
 será gerado 
um balanço com dados consolidados de todas as filiais pertencentes ao mesmo Grupo Empresa.
Detalhar por filial:
 caso marque, serão gerados dados detalhados por filial
Até Mês/Ano Base:
 informe até que
 
mês/ano
 
serão coletados os dados
Comparativo:
 caso marque, informe 
Até Mês/Ano para comparação, 
que deve ser 
menor
 que Mês Ano/Base.
 
É importante lembrar que, se a empresa trabalha com códigos estendidos e selecionar a opção de comparativo, a partir do processamento dos novos códigos estendidos, o período da emissão dos relatórios deve ser em função da data inicial deste período. Não é recomendado emitir um relatório com um período que contemple dois ou mais códigos estendidos diferentes, pois os resultados podem ser afetados.
Para maiores informações, acesse 
Considera contas inativas:
 recomendamos sempre marcar, pois pode acontecer de contas contábeis que foram movimentadas serem inativadas ou delas terem saldos.
Imprime Termo Final:
 caso marque, será gerado um texto padrão na última página do Balanço e com os dados dos responsáveis pela empresa.
Nível:
 informe o nível desejado para impressão 
do balanço - 
1, 2, 3, 4 ou 5.
Clique no botão
 
OK
 
para confirmar e o relatório
 será apresentado em tela. Para imprimir clique no botão
