--------------------------------------------------
Nesta tabela são cadastrados os grupos de contas para hierarquização do 
Plano de Contas
 gerencial. 


A tela que permite manipular esta tabela é acionada a partir do Menu principal com as opções - Tabelas Auxiliares / Controladoria / Grupo Contas Contábeis. 


 


Para acrescentar uma novo grupo contábil acione o botão de inclusão de registro: 


 


O Sistema exibe a tela para preenchimento dos novos dados. 


 


Preencha os dados: 




Descrição
: Nome do grupo.


Ordem no Relatório
: Qualquer texto ou conjunto de números que sirva como ordenação para a hierarquização do plano.




Acione o botão OK para atualizar os dados no Sistema. 


O Sistema retorna a tela de navegação. 


Para visualizar quais as contas que fazem parte deste grupo, acione o painel Contas Contábeis. 


 


Ordene os dados conforme necessário clicando no cabeçalho do campo. 


Para retornar ao Menu principal acione o botão Sair:

: Para as contas que são geradas automaticamente no Sistema deve-se ter um grupo de contas padrão já especificado. Para isso na tela "Sistema", painel "Contas Contábeis/Históricos", deverão ser preenchidos os grupos para as contas exibidas abaixo:
