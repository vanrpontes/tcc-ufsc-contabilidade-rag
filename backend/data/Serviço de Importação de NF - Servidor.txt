Serviço de Leitura de XML NF-e/CT-e
O Serviço de leitura é responsável por realizar a leitura dos arquivos XML, ao iniciar o serviço o Sistema ira realizar a leitura dos arquivos que se encontram dentro do diretório apontado na Pasta de Arquivos a processar, e caso seja um arquivo valido do tipo XML, sera realizado a importação dos dados para o Sispetro, e o arquivo XML sera movido da pasta Pasta de Arquivos a processar para a Pasta de Arquivos Processados, caso o arquivo contenha algum erro, o mesmo será movido para a Pasta Arquivos com erro.
Iniciar Serviço:
 Ao acionar esse botão o serviço é iniciado ou parado. 
Iniciar o serviço automaticamente
: Caso deseje que o serviço seja iniciado junto com o servidor, marcar esta opção.
Serviço de aplicação de De/Para
O serviço de De/Para é responsável por executar os De/Para nas Notas Importadas. Após o Sispetro realizar a leitura do arquivo, caso o Serviço de De/Para esteja Ativo, o Sispetro ira realizar automaticamente o De/Para para as notas de acordo com o De/Para configurado no processo informado no perfil de Sistema, e com os perfis configurados devidamente. Pasta para a importação de XML
Pasta de Arquivos a Processar: 
Deve ser indicado um diretório para a leitura dos arquivos. O sispetro ira realizar a leitura dos arquivos dentro do diretório indicado e ira mover o mesmo para a pasta de Processados ou com erro caso tenha algum. Arquivos que não são do tipo XML serão movidos para a pasta de com erro. Após realizar a leitura do arquivo, o serviço tenta mover o arquivo para o diretório referente aos aquivovos processados ou para a Pasta de arquivos com erro, e caso não consiga arquivo é perdido.
Pasta de Arquivos Processados:
 Nesse campo informamos o diretório de onde irão ficar os arquivos Processados. Depois de realizar a leitura, caso não seja encontrado nenhum erro no arquivo xml o SisPetro ira realizar a inclusão da nota e moverá o arquivo para o diretório informado nesse campo.

Arquivos com erro: 
Nesse campo informamos o diretório para os arquivos com erro, nesse diretório também será gravado o arquivo que contem o motivo do erro. Caso seja encontrado algum erro na leitura, os arquivos serão movidos para esta pasta.
