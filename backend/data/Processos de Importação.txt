Nesta tabela são armazenados os 
Processos de Importação
 utilizados para organizar os processos de um PMM que seja de importação. Aqui o usuário poderá incluir, alterar e excluir estes Processos.

Para acessar a tela de Processos de Importação, o usuário deve utilizar o menu: 
Tabelas Auxiliares / Movimentação de Material / Processos de Importação
.




O sistema abrirá a tela , mostrando dados já cadastrados. Para inserir um novo código, o usuário deve clicar no botão 
.


 

O sistema entrará em modo de edição, permitindo que o usuário preencha o campo:


Descrição:
 O nome do Processo de Importação a ser cadastrado. Neste caso o sistema aceitará letra e números, se for o caso.


O campo código é sugerido pelo próprio sistema e não poderá ser alterado.


Para finalizar o cadastro, o usuário deve clicar no botão 
. O sistema se posicionará na tela de navegação, onde o usuário poderá consultar um Processo de Importação que foi cadastrado ou os já existentes.


Para consultar algum registro, basta o usuário clicar na coluna que se deseja pesquisar ("Código" ou "Descrição"), desta forma o sistema organizará as informações e posicionará o cursor no campo "Pesquisa", aguardando a digitação por parte do usuário




A coluna que o usuário clicar, ficará em amarelo, demonstrando assim, que o sistema fará a pesquisa por essa coluna.
